        &lt;h3&gt;第三卷第七十章  人不学会改变，只会被淘汰&lt;/h3&gt;
<br/>
<br/>        聪明的人无需多说，哪怕是一句简单的话语，就可以明白其中的意思，对于软件上的事情他们确实不了解，按照寒伯安和他的设想，后期对软件上的要求肯定是非常高的，他们和陈子昂现在是合作关系，他们对软件的设计和要求并不是很懂，只能依靠软件开发商来做。
<br/>
<br/>        现在的app也都是在不停的进行修改和升级，时代发展太快，谁不是边走便摸索着，如果等到所有的东西都成熟的话，黄花菜都凉了。
<br/>
<br/>        在进行几大汽车厂家的考察上，他们将投放的汽车定位在普通轿车、商务7座车和豪华轿车上，目前仅仅以租赁为形式，会在多个旅游城市投放汽车，在人们出游的同时享受到愉快的出行体验。
<br/>
<br/>        郝景文在考察汽车市场的时候不仅仅要考察汽车的实用性还要看汽车的安全西能及后期的维修成本。
<br/>
<br/>        在众多的汽车品牌中，郝景文暂定了几款车，雪佛兰，别克、大众、起亚、上汽几个品牌。至于后期到底用什么样的品牌，回去还是要和寒伯安和江俞轩商量，像他们的物流系统一样，先期会投入一部分车辆，最后在根据市场的需求投入更多的车辆。
<br/>
<br/>        他们的定位区别与一般的出租车，不但要在车型上选型好，还要在服务上体现人性化。
<br/>
<br/>        公司人员配备还没有到位，他的考察也只停在考察阶段，一切还得等寒伯安回来了在商定具体的运作方式。
<br/>
<br/>        郝景文顺道回了一趟家，第一，看看老父亲，当然父亲也不老，五十多岁的年纪，也正是干事业的黄金时期。
<br/>
<br/>        第二，视察一下物流情况，自从兄弟们加入到他们的团队以来，这个就是他值得骄傲的一个物流部了，清一色的同一品牌同一型号的车，清一色的服装，各个斗志昂扬，将送货干出了高大上的感觉，整个区域的私人发货基本都会找他们。
<br/>
<br/>        当郝景文回到家里时，家里热热闹闹的好一番景象，他们原本就是做生意，茶叶生意会销往全国。
<br/>
<br/>        原来是找别人发货，现在是别人来找他们发货，门口的货物堆积如山，人们忙忙碌碌的装货卸货，送货车排成队，有条不紊的按照单子领着待送的货物。
<br/>
<br/>        “文仔，回来啦，快过来帮忙。”父亲发现了郝景文。
<br/>
<br/>        “文仔才回来，你不让他歇一会，来、抽根烟。”大兄弟阿莫递过来一支烟。
<br/>
<br/>        阿莫大学毕业一年，学的管理，刚好可以用上，算是对口的工作，虽然说是经验有些欠缺，但是阿莫很认真，人生难得认真二字。
<br/>
<br/>        “怎么样？还顺手吗？”郝景文接过烟，拿在手中轻轻的转着，他没有抽烟的习惯，可抽可不抽。
<br/>
<br/>        “好着呢，你看看大家，都充满了斗志。隔壁冷三叔说我们是拿着鸡毛当令箭，把一个送货的活干出了当爷的感觉。”阿莫笑咪咪的说。
<br/>
<br/>        “哈哈哈，我们要的就是这个感觉。在现代社会中，行业没有贵贱之分，只有贡献大小。别看不起我们这些服务行业，这些服务行业解决了很多就业问题。也是为社会做了贡献的。”
<br/>
<br/>        “嘿嘿，人不学会改变，只会被淘汰。”阿莫看着冷三叔那边说。
<br/>
<br/>        这一段时间物流红红火火，很多发货的都来找他们发货，他们的速度很快，近的地方一两天就到了，稍微远一点的一个礼拜，对于他们茶叶行业来说，茶叶是很精贵的东西，怕潮，怕压，运输途中是要很注意的，而且送到客户手中时间越短他们的回本也越快。
<br/>
<br/>        别的人都会来找他们，而冷三叔就是对他们不屑一顾，还是自己跑某政去发货，说是正规部门让人放心。
<br/>
<br/>        “别人淘汰不淘汰我们不管，我们只要不被社会所淘汰就好。”郝爸爸帮着把几盒包装好的茶叶放到车上，走过来接着阿莫的话说。
<br/>
<br/>        “文仔，你搞的这个事情确实不错，我们业内人都竖大拇指，说是比平时的速度快了一倍多。”郝爸爸很满意儿子做的这个事情。
<br/>
<br/>        这份事情如果不做片面考虑的话，确实不失为一件好事情，给他们的输送解决了时间过长的问题，将时间缩短了很多。
<br/>
<br/>        “爸，辛苦你了。”郝景文知道，最近这一段时间也是靠着父亲的帮衬，这个事情才红红火火的开展下去。
<br/>
<br/>        父亲原本是一个业界大佬，说话做事情都是有分量的人，现在竟然帮着他做这些琐粹的事情。
<br/>
<br/>        “哈哈，我天天和他们在一起，我都变的和他们一样充满了活力。”郝爸爸说的是真话，原来和一帮子朋友在一起就是喝茶，聊天、讲茶道，时间久了，虽然悟出了一些道理，可是随着时间的推移，总感觉缺少了一些什么。
<br/>
<br/>        现在和一帮年轻人在一起，他知道平时他缺少什么了，是缺少激情，看着年轻的孩子们热情似火的奔跑着，他的心也跟着一起燃烧起来了。
<br/>
<br/>        “文哥，这次回来又有什么好的项目？兄弟们可是还想和你一起干呢。”阿莫问。
<br/>
<br/>        “先把手头这个事情干好，这个是长远的事业，不是一天两天的激情就说了算，我们这个算是服务行业，服务大众，服务人民，只有做好了服务，才能长长久久。”
<br/>
<br/>        “文文说的很对，现在的人们越来越注重感官享受，服务虽然是看不到摸不着的一个词汇，但是却是在无形中影响着人们对生活幸福指数的体验。”郝爸爸点点头。
<br/>
<br/>        “是啊，就拿我们去给客户送货，及时送到，然后还满脸笑容开开心心的给客户说一声谢谢，客户不但会开开心心的给我们说谢谢，还会说，辛苦你们了啊，注意安全啊等等关切的话，感觉就是朋友之间的相处，和谐而温暖。”
<br/>
<br/>        阿莫毕竟还是才出社会，他不知道，当人们在承受着工作或者是生活的压力时，当沉浸在忙忙碌碌的琐粹事情里，有一张真诚的笑脸，有一句温暖的“谢谢”，一句关切的问候，虽然平凡，虽然简单，却拉近了人与人之间的距离，让人觉得这个世界很美好。
<br/>
<br/>        郝爸爸和郝景文都是在社会上摸爬滚打了很多年的人，他们都知道生活中残酷的一面和温情的一面。
<br/>
<br/>        在繁杂的社会中拥有一颗温暖光明向上的心是多么的重要和美好，他会让整个生活温馨而美好。
