　　“行，那就有劳毛哥了。”
<br/>　　“不用跟我客气，我也是为了斩妖除魔，不是单纯为了帮你。我这就去了。”毛小方拱了拱手，直接就出门去了。
<br/>　　叶少阳突然想到一件事，赶忙把他叫住，让他把云儿一起带走，正好可以送到翠云住的地方去，让翠云陪着她，两个人正好也有个伴，也省的云儿在这里整天提心吊胆的。
<br/>　　毛小方也觉得这样不错，于是叶少阳去隔壁把云儿叫过来，跟她说了，云儿有点不舍得离开叶少阳，不过也确实想离开这个镇子，越远越好。
<br/>　　“少阳哥，你会去找我吗？”云儿问道。
<br/>　　叶少阳笑道：“当然了，我姐还在那边呢，我这边处理好了就过去找你们。”
<br/>　　“你可一定要来，我等你。”云儿咬了咬嘴唇说道。
<br/>　　叶少阳笑着点点头。
<br/>　　云儿跟着毛小方一起离开。
<br/>　　叶少阳跟美华继续讨论有关三足金蟾的事。
<br/>　　“对了，我听说，这三足金蟾在这里，是为了镇压蛤蟆谷里的古墓，你知道那古墓么，里面到底有什么？”叶少阳问道。
<br/>　　美华沉吟道：“我只知道那下面有一个阴巢尸穴，本来我也动过据为己有的想法，不过找不到入口所在，直到最近，那里被洪水冲了，墓**露出来，每晚有鬼魂游行，我去探过一次，但是刚下去不久，就遭遇一股极其强大的力量，我知道不是对手，就没有再下去，我却也不清楚古墓下面的事。”
<br/>　　叶少阳点点头，关于这座古墓的事，也只有先放放再说了，先集中精力对付这个三足金蟾再说。
<br/>　　叶少阳突然想起了镇长选了三十个姑娘给三足金蟾挑选的事，跟美华说了，美华表示回去立刻让自己手下过来调查，肯定能找到关押这些姑娘的地方。
<br/>　　“你要不要跟我一起回洞府？”美华问道。
<br/>　　叶少阳想了一下，觉得还是算了，自己过去也没什么必要，还是留在这里等着，也算是陪着云儿，免得她一个人在这里担惊受怕。
<br/>　　美华走之前，盯着叶少阳看了一会，然后笑了笑，转身出门，搞得叶少阳莫名其妙。
<br/>　　他们两个都走了之后，叶少阳找客栈老板要了饭菜，自己吃了，然后去镇上逛了一会。
<br/>　　天黑之后，镇上也没什么地方可以逛的，叶少阳溜达了一圈也就回客栈了，坐在床上吐纳。
<br/>　　经过这些天的吐纳调息，叶少阳感到自己的丹田越来越稳固，能够容纳的罡气也越来越多，也就是说，丹田修复的速度，远远超过了他最初的预想，不过一天最多也只能吐纳四个周天，不然丹田就会过于疲劳，反而不利于恢复。
<br/>　　想到五通神那个畜生，叶少阳恨得不行，有一种有力气使不出来的感觉，如果是以前的自己，对付这五通神，自己一个人完全可以搞得定，哪里还用这么费事。
<br/>　　吐纳结束之后，叶少阳躺下睡觉，但是刚睡了没有一会，外面有人敲门，开门一看是美华来了。
<br/>　　“我找到那些姑娘被关押的地方了。”美华一进门就说道。
<br/>　　叶少阳立刻让她带自己过去看。
<br/>　　穿过镇子，两人来到山脚下一座大宅子的前面，这座宅子已经在镇子的最边上了，宅子很大，有一种深宅大院的感觉。
<br/>　　美华告诉他，这里就是镇长的家，不过镇长现在不在这住，那些姑娘都关在这座宅子里，有人在附近守护，想要进去得小心一点。
<br/>　　两人一起绕到宅子的后面，正在想办法进去，美华突然指着一个让叶少阳看，叶少阳转头看去，是一团黑气，正在从远处飞来。
<br/>　　“三足金蟾！”美华沉声说道，拉着叶少阳往后退到安全地带，口中说道：“还真是巧了，我们刚到，这三足金蟾就到了。”
<br/>　　叶少阳想了一下道：“也不算巧。云儿被我带走了，这蛤蟆精是个老色狼，肯定又想找个替代品。我去看一看。”
<br/>　　美华一把拉住他，说道：“你这么去，会被发现的，很危险。”
<br/>　　“知道，我元神出窍，你保护一下我的肉身。”
<br/>　　两人向后来到山林里藏好，叶少阳故技重施，元神出窍之后，朝那座宅子飞了过去，这时候五通神三足金蟾也进入了宅子。
<br/>　　元神不会被识破。叶少阳放心地进入宅子，宅子的一楼，是一个开阔的大间，靠墙站着一排姑娘，都是稚气未脱的少女，在一些村民打扮的人监督之下，很紧张地站成一排。
<br/>　　叶少阳大致看了一下，差不多也有二三十个。
<br/>　　三足金蟾在一个穿的体面的男子的陪同下，从那三十个姑娘面前一个个看过去。
<br/>　　这些姑娘也是第一次见到三足金蟾，十六岁的少女，一般都不想嫁人，但也不排除有些内心对五通神有些想象的，毕竟他是神，能服侍他，自己也会感到光光荣。不过这种想法也都是少女一厢情愿的美好想象，把五通神想象成了神话故事里听到的那种翩翩仙人，结果见到真人之后……所有人都失望透顶。
<br/>　　这些姑娘都是从镇上选出来的最漂亮的姑娘，没有人能看得上五通神蛤蟆一般的模样，谁也不想牺牲最美好的三年青春，去服侍这样一个丑八怪……
<br/>　　因此当三足金蟾去看哪一个的时候，那个姑娘都低下头去，露出非常紧张的样子。
<br/>　　“抬起头来。”三足金蟾驻足在一个姑娘面前，说道。
<br/>　　那姑娘迟疑了一下，不得不抬头，却用一种求救的目光望着陪同三足金蟾的那个男子。
<br/>　　三足金蟾一张大口张开，露出满意的笑容，说道：“就这个了。”
<br/>　　“啊，她啊……”那男子露出为难的样子。
<br/>　　“怎么了？”三足金蟾皱眉说道。
<br/>　　男子立刻跪在地上，说道：“大神啊，这是我家独生女……”
<br/>　　三足金蟾也有些意外，说道：“邱文书，你家女儿，怎么会在这里。”
<br/>　　邱文书道：“是……镇长依照大神的吩咐，当年凡是年纪符合的女子都选上了，那时候我还没在镇上做事……如今也不得不让我女儿一起前来候选，我家就这一个独生女，求大神开恩。”
