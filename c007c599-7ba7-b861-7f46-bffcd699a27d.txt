　　苏莫和聂乾、韩盈三人，离开了那座小镇之后，飞行了几百万里，便再次找到了一个合适的巫族小镇。
<br/>　　然后，苏莫如法炮制，再次吞噬了那座小镇，又得到了大量的储物戒、兵器、和巫族武者的尸体。
<br/>　　这些尸体，都是他日后的资源，基本上武帝境的武者，体内神丹能保持数十年，其内的玄力都不会流逝太多。
<br/>　　接下来的半个月的时间里，苏莫三人打一枪换一个地方，疯狂的吞噬洗劫。
<br/>　　有韩盈的洞听耳战魂之助，无论是寻找目标，还是外围警戒，都是丝毫没有难度。
<br/>　　半个月的时间，三人洗劫了六个小镇，单是储物戒就不下五十万个。
<br/>　　虽然这些储物戒的主人，大多修为不是很高，其内的资源财富不多，但胜在储物戒的而数量多啊！
<br/>　　苏莫直接分给了韩盈和聂乾一人两万个储物戒，让两人几乎乐的合不拢嘴了。
<br/>　　无尽大地深处，苏莫三人相对而坐。
<br/>　　三人在各自清点着自己的战利品。
<br/>　　“哈哈！苏兄，我这两万个储物戒中，共有下品灵晶十四万，极品灵石八千三百多亿，各自兵器六万多件！”聂乾满脸潮红，大笑着说道，他从没想到赚钱这么容易。
<br/>　　如此多的财富，足够他修炼到武尊之境，也不用愁了。
<br/>　　“苏兄，我的比聂兄的还要多点！”韩盈亦是面带笑容。
<br/>　　“很好，不过这还不够，我们还要继续洗劫！”苏莫点了点头，他自己得到财富，远不是韩盈和聂乾能比的。
<br/>　　他身上有几个储物戒内的空间，都几乎被塞满了，全部都是储物戒，他也懒得清点其内的财富。
<br/>　　等回到古周星之后，他会留下一部分，其它的全部交给苍穹神宫。
<br/>　　“苏兄，我们现在已经灭了六个巫族小镇，接下来再想洗劫，怕是危险性大增了！“聂乾的脸色凝重了下来。
<br/>　　“是啊！巫族之内，管理颇有规则，每一个小镇，都有一座巫族城池领导，然后所有的城池，分别归于九大卫城领导，最后再由天冥圣城统一管理，六个小镇的突然消失，估计消息现在已经传递到了天冥圣城。”韩盈叹道，这些都是他这几个月来了解到的消息。
<br/>　　在苏莫进入巫族之后，他和聂乾轮流出去历练，了解了很多巫族的情况。
<br/>　　“这无妨，我还有分身在巫族天冥圣城！”苏莫微微一笑说道，他和分身心意相通，这段时间以来，分身经常活跃在天冥圣城和几大卫城之中，与一些长老人物相识。
<br/>　　以分身在巫族的地位，什么风吹草动，都能探知得到。
<br/>　　目前，消息并未传到天冥圣城，毕竟时间太短了，就算有些城池发现了消失的小镇，也会先自行查探一番的。
<br/>　　“走，我们继续行动！”随即，苏莫带着聂乾和韩盈，离开了地底，继续出去洗劫。
<br/>　　……
<br/>　　天冥圣城。
<br/>　　宗府内，议事大殿。
<br/>　　今日，大殿中座无虚席，很多巫族的核心长老、少司均在大殿之中。
<br/>　　巫尘作为新晋的少司，自然也在殿中。
<br/>　　大祭司高坐于主座之上，正在召开会议。
<br/>　　族长常年在主脉之中，天冥星一脉的大小事务，基本上都是大祭司在管理。
<br/>　　“诸位，最近天冥星上不太平，据各大城池来报，最近有很多我族小镇无端消失！所有族人也尽皆消失无踪！”大祭司目光环顾众人，朗声说道。
<br/>　　为此，他之前用庞大的灵识意念，在天冥星上查探了一番，不过并没有查探到什么线索。
<br/>　　他的灵识虽然庞大，可以笼罩方圆千万里，但天冥星实在太大了，他的灵识根本不可能全部笼罩，连百分之一的范围都覆盖不了。
<br/>　　“什么？居然还有这种事？”众巫族核心长老闻言，均是诧异不已。
<br/>　　小镇连同族人一起消失了？
<br/>　　以前可是从未发生过这种情况。
<br/>　　巫尘面色淡淡，静坐在座位上，看着众巫族高层谈论。
<br/>　　“大祭司，各地城池有查探出什么线索吗？”有核心长老问道。
<br/>　　“没有什么线索！”
<br/>　　大祭司摇了摇头，道：“此事比较诡异，小镇几乎是被连根拔起，连一根草都没留下！”
<br/>　　众长老闻言心中讶然，连一根草都没留下，这未免太诡异了吧！
<br/>　　不过众人也没有太过在意，毕竟只是几个小镇而已，不算什么大事。
<br/>　　“此次，老夫召集诸位，是问谁愿意负责此事，外出查探一番！”大祭司沉声问道，这等小事他自然不会亲自追查，安排给下面的族人便可。
<br/>　　巫尘闻听此言，顿时眸光微闪，随即立刻起身，向大祭司抱拳道：“大祭司，我愿意前去查探！”
<br/>　　“你？”大祭司目光望向巫尘，随即沉吟一番，道：“你的修为怕是有些低了，可能会有危险！”
<br/>　　大祭司不想让巫尘离开，因为他觉得巫尘隐藏的很深，估计有些秘密。
<br/>　　他给了巫尘少司令牌，监察巫尘的动向，但上次令牌上的意念，突然被阻挡了，这让他心中略有惊疑。
<br/>　　这更让他确定了巫尘不简单，身上或许隐藏了什么大秘密。
<br/>　　他也没有撤了巫尘的少司之位，倒是要看看巫尘能玩出什么花样。
<br/>　　“大祭司，我会小心的，而且以我的实力，等闲的武帝境武者，根本不是我的对方！”巫尘脸色郑重的说道。
<br/>　　大祭司闻言，面现沉吟之色。
<br/>　　巫尘见此，继续说道：“大祭司，承蒙您看重，让我成为族中少司，我自然要为您分忧，此事就交于我吧，我定然查探清楚此事。”
<br/>　　“嗯！既然你有如此心意，那此事便交由你去查探了！”
<br/>　　大祭司没有多想，便点了点头，面现欣慰之色，随即又道：“这样吧，以免你势单力薄，带些人手过去吧！”
<br/>　　“可以！”巫尘没有反对，立刻点了点头。
<br/>　　而后，大祭司命人前去传唤，招来了十几名族中天才高手。
<br/>　　这十几名天才高手，个个修为都不低于巫尘，而其中修为最高的两人，修为高达武帝境六重。
<br/>　　这两人不是别人，正是巫玛和巫幽兰。
<br/>　　这让巫尘心中一沉，大祭司居然让巫玛和巫幽兰，以及这么多高手跟他同行，这明显是不信任他啊！
