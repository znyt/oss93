<br/>　　好像在说，吸血蛇怎么会听你的，你算什么东西。

<br/>　　江行泽竭斯底里，异常狰狞的脸如同一只发了狂的猫，尖叫的露出尖利的牙。

<br/>　　能被白镇人民尊为蛇王，能在他师父蛇博士的手中逃生，如今，这条蛇好像变的更加的厉害。

<br/>　　所以，他决定上去加一把火。
<br/>　　如果换做平时，江行泽不敢如此对吸血蛇咆哮的拳打脚踢，更别说一口一个畜生，但现在此时的他几乎已经失去理智。
<br/>　　那一跪，将他所有的理智都给跪没了。

<br/>　　吸血蛇一股恐怖的气息席卷出来，周身火焰迸发，呈大火焚烧之势，一双铜铃般的蛇目死死的盯着疯狂的江行泽，散发强烈杀意与深渊般绝望的恐怖气场。

<br/>　　江行泽不禁的倒退，嘴唇和面颊惨白而拉长了，吸血蛇的杀意他太了解了。

<br/>　　闻言，江行泽大吃一惊，难道吸血蛇认他为主，就是想利用他，利用自己逃出山野集团，利用自己，来到蛇沟深处吗。

<br/>　　江行泽满脸恐惧起来，吸血蛇杀意如临深渊的笼罩在自己的身上，悄然间，江行泽响起了牢笼中师父陈默对他说的话。
<br/>　　泽儿，你是控制不了这条蛇的..．

<br/>　　简单，随意，三十米巨大的吸血蛇，现在想吃一个人，太轻松了，就跟人咬一口鸡腿似的。
