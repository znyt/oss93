        &lt;h3&gt;第三卷第八十四章  冯睿请客&lt;/h3&gt;
<br/>
<br/>        冯睿准备叫陈子昂和陈子霖还有潘青松及陈子乔到家里来吃顿饭，陈子乔调到西城也有一段时间了，但是他们一直都没有联系，陈家就数陈子乔有出息，现在已经是某部门的一把手了。
<br/>
<br/>        原本想着叫陈子乔，但是陈子乔要叫着其他人，冯睿也就将陈家在西城的人都叫上了。对于陈子昂和江大少的事情，她还没有来得及和弥清云和蔡少芬汇报，这一段时间比较忙，不过陈子昂和张函再也不可能有交往了。
<br/>
<br/>        这让她似乎失去了依仗。
<br/>
<br/>        前几天弟弟找到她，说他们公司也想得到那块地盘，让她给陈子建大声招呼，如果陈子建他们单位去给别人勘查的时候就说有问题，等最后给他们公司勘查的时候陈子建再去。
<br/>
<br/>        从陈子昂那次请客吃饭回来之后，感觉陈子建对她有些不满，这一段时间一直冷冷谈谈的，她也不知道是什么原因。
<br/>
<br/>        所以有些话她想开口但是看着陈子建那冰冷的眼神，便打住了，叫上陈家姊妹来家里，也算是缓和一下家里陈子建的状态吧。
<br/>
<br/>        果然，她对陈子建说叫上他们一起吃饭的时候，陈子建高兴的答应了。
<br/>
<br/>        “冯睿，这是这么多年以来，兄弟姐妹们第一次来家里聚吧？”陈子建开心的说。
<br/>
<br/>        “怎么可能呢，前几年子昂不是来家里住了几天吗？”冯睿倒是记得之前陈子昂在他们家住了几天。
<br/>
<br/>        “你说的都对。”陈子建憨憨的笑着，冯睿终于开窍，能让家里的人到他们家里来了。
<br/>
<br/>        陈子建一笑，冯睿心里就松了一口气。
<br/>
<br/>        夫妇俩个人去超市买了很多新鲜的素菜和海鲜，连同保姆三个人在家里做了满满一桌子菜出来了。
<br/>
<br/>        陈子霖对陈子昂说：“晚上吃饭叫上江俞轩吧。”
<br/>
<br/>        “为什么要叫上江俞轩？”
<br/>
<br/>        “江俞轩这几天要用到二哥，还有那次吃饭他说他和你连孩子都有了，而且还是双胞胎。说话也不想一想再说。”陈子霖有些抱怨，虽然怼得二嫂无话可说，可是后期又该怎么处理，还不是让二嫂笑掉大牙。
<br/>
<br/>        陈子昂当时有些震惊江俞轩的话，但是后来她也没有去深究，她就当江俞轩胡说，最近的事情太多，哪有时间容她想那些烂七八糟事情。
<br/>
<br/>        陈子霖这样一说，她还是觉得有点问题，江俞轩是无心之言还是知道了她有俩个孩子？
<br/>
<br/>        “哥，你信江俞轩的话吗？”
<br/>
<br/>        “信他个鬼。”陈子霖都知道江俞轩和张倩楠要结婚的事情了。
<br/>
<br/>        “二哥和二嫂就信了。”
<br/>
<br/>        “所以，我让你叫上江俞轩。”
<br/>
<br/>        “好吧，我打电话问问看看他有没有时间。”虽然江俞轩还是在他们公司，但是最近却在忙江家的事情。
<br/>
<br/>        “俞轩，下午有时间吗？一起吃下午饭？”陈子昂拨通江俞轩的电话，他们经常直接就是开场白。
<br/>
<br/>        “想我啦？”江俞轩正在看方案。
<br/>
<br/>        “就算是吧，我二哥打电话让我们去他家吃饭。”
<br/>
<br/>        “我去好吗？”
<br/>
<br/>        “你说呢？”陈子昂有些幽怨的声音。
<br/>
<br/>        “行，你下午来接我。”江俞轩想着明天也是要请陈子建他们，不如今天见上一面，况且他还可以请陈子建看看手镯的事情。
<br/>
<br/>        “这么近你还要我来接你？”他们都在一条线上，就差了两站路，江俞轩走都走得到。
<br/>
<br/>        “为了表达我们俩个的恩爱，你必须来接我，不过你可以换一双运动鞋。”
<br/>
<br/>        “好吧。”陈子昂每天有跑步的习惯，运动鞋是常备的。
<br/>
<br/>        .........
<br/>
<br/>        “江大少，你现在牛气的不行啊，吃顿饭还要我们俩个来接你。”陈子霖和陈子昂一起步行到了他们的售楼中心，下午江俞轩在售楼中心开会。
<br/>
<br/>        “反正我没有让你来接。”俩个人还是见面就斗嘴。
<br/>
<br/>        “你以为我想来啊。”
<br/>
<br/>        “哈哈哈，知道你不想来，但是你这不是来了吗？”
<br/>
<br/>        江俞轩嘚瑟的样子让陈子霖心里挠得难过。
<br/>
<br/>        “哼！”陈子霖又吃亏了。
<br/>
<br/>        “子昂，你二嫂怎么突然就想起来请我们吃饭了？”江俞轩不再和陈子霖说笑转身问陈子昂。
<br/>
<br/>        “我也不知道啊。”
<br/>
<br/>        “我刚好找你二哥有事情，还可以蹭一顿饭，哈哈哈。”
<br/>
<br/>        “是勘查的事情吗？”
<br/>
<br/>        “是的，还有一件事情，大哥让我帮他找个东西，可以问问你二哥。”
<br/>
<br/>        “哎，有求于人，还这么牛气，怕也只能是你江大少了。”陈子霖接过话。
<br/>
<br/>        “哈哈哈，谁让我是陈子昂的搭档呢。”
<br/>
<br/>        三个人说说笑笑的朝陈子建家里走去。
<br/>
<br/>        陈子建一直住在单位分的房子里，三居室，一百三十多平米，就在单位的后楼，冯睿本来想搬走，但是陈子建不愿意，上班几步路就到了，多方便的。
<br/>
<br/>        听到敲门声，陈子建快步打开房门，陈子昂三个人站在门外，陈子建见到江俞轩也来了，心里更是欢喜，看见陈子昂和江俞轩并排站在一起，他总有一种终于将女儿嫁出去了的喜悦，又有一种失落。
<br/>
<br/>        想着自己这些想法，心里竟然也莫名的感觉好笑，他最小的妹妹终于是找到了幸福，而且还是江俞轩这么优秀的人。
<br/>
<br/>        “快，进来。”陈子建搓着手，弟弟妹妹们都有着落了，他这个不称职的哥哥心里那一似愧疚也减少了许多。
<br/>
<br/>        餐桌上的菜散发着淡淡的香味，冯睿也是做得一手好饭菜，清蒸鲈鱼、剁椒蒜蓉烤生蚝、干锅番茄鸡、蒜烧鲍鱼、夏威夷木瓜味淋虾、姜仔河鱼、红烧排骨、葱爆羊肉片、醋溜土豆丝、炝拌佛手瓜、凉拌笋尖、水煮菠菜、枸杞木耳炒山药、羊肉萝卜汤、糖醋花生。满满的摆了一桌子，厨房里的保姆还在忙碌着。
<br/>
<br/>        “二嫂。”陈子昂将手中的礼物给了冯睿，毕竟离家之后这是第一次来二哥二嫂家，而且还是和江俞轩一起，虽然俩个人误会了，那先就误会着吧，她也懒得解释。
<br/>
<br/>        “来就来了，还拿什么东西啊。”二嫂将东西拿到房间，悄悄的打开看了一下，给她买的是条钻石项链，这条项链是她之前有去中心街看到的那条，十几万，她舍不得买。
<br/>
<br/>        陈子昂到底是嫁给了江家，财大气粗啊，一出手就是这么大方，可怜的冯睿到现在都不知道她的这个小姑子到底有多少资产，是干什么的，因为在她的意识中，陈家永远都是没有出息的农民。
<br/>
<br/>        陈子霖这几年虽然是火火火火，可是怎么能比得上他们这些人有前途呢？
<br/>
<br/>        冯睿将项链戴在脖子上试了有试，看了又看，竟然不舍得拿下来了。
<br/>
<br/>        “冯睿，子乔他们来了。”陈子建在外面喊着。
<br/>
<br/>        “也不知道陈子乔有没有给我拿礼物，戴着项链让他们看看。”冯睿舍不得取下项链。
<br/>
<br/>        “哎呀，子乔、青松、子悦你们来啦，你们看子昂给我买的礼物，我太喜欢了。”冯睿说。
<br/>
<br/>        陈子乔并没有给冯睿拿东西，她就拿了几瓶酒，一些茶叶。
<br/>
<br/>        陈子悦倒是给陈子建和冯睿一人拿了一套品牌服装。
<br/>
<br/>        只有潘青松清廉如水，也是给陈子建提了几瓶酒。
<br/>
<br/>        陈子昂给冯睿买的礼物太珍贵了，大家都是场面上的人一眼就看出来了，而且陈子悦就发现这条项链是出自他们卖场。
<br/>
<br/>        气氛微微的有些尴尬。
<br/>
<br/>        “都来了，快上座，你嫂子都准备了一下午。”陈子建对冯睿的做法越发的反感了，但是又不好多说什么，等以后一定要找个机会说一说。
<br/>
<br/>        现在的兄弟姐妹都已经成家立业了，而且个个都算是有身份地位的人，冯睿还当他们是小孩子对待，从来都不顾及他们的感受。
<br/>
<br/>        “快，快，你看我这一高兴就忘记招呼你们了。”冯睿笑着说。
<br/>
<br/>        江俞轩和潘青松坐到了上座，陈子建和陈子霖相陪，陈子昂挨着陈子霖坐，陈子乔挨着陈子建坐，冯睿挨着陈子乔坐，陈子悦在冯睿和陈子昂的中间。
<br/>
<br/>        虽然是一家人，餐桌上的礼仪还是要讲的，江俞轩和潘青松来到了陈家就是贵客了。
