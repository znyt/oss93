　　望着转身离去的巫人王，苏莫眼帘低垂，心念电转。
<br/>　　但是，思来想去，他都找不到丝毫能逃过此劫的办法。
<br/>　　现在大祭司坐镇与此，他根本没有逃走的机会。
<br/>　　良久，苏莫叹了口气，只能等到进入真圣界之后，再看看有没有什么脱身之法了。
<br/>　　这一次是真正的生死危机，搞不好就彻底完了。
<br/>　　以往他遇到危机，还能靠自己强大的实力度过去，但这次危机他的实力没有任何用处了。
<br/>　　“任兄，你继续说！”
<br/>　　少倾之后，苏莫对任云都说道，还是先了解一番真圣界内的情况吧，看看有没有逃走的机会。
<br/>　　“苏兄……！”任云都张口欲言，想要询问苏莫为何会得罪巫族，但巫族之人就在不远处，他也不好询问。
<br/>　　“任兄，你无需多问，继续说吧！”苏莫道。
<br/>　　任云都无奈的点了点头，沉默了片刻，便继续讲解道：“通圣三关，第一关为圣门关，武者需通过圣门，才能到达第二关，一旦失败，就会自行被抛出真圣界。”
<br/>　　“圣门中有什么？”苏莫问道，既然是第一关，这圣门肯定会有考验的方法。
<br/>　　“什么都没有？”任云都道。
<br/>　　“什么都没有怎么考验？”苏莫疑惑。
<br/>　　“这圣门就是一种考验，会自行检测武者的天赋，无论是战魂天赋，还是血脉体质天赋，亦或是精神力天赋等等一切天赋，只要是天赋达不到要求的人，根本别想通过圣门。”任云都说道。
<br/>　　“原来如此！”苏莫心中恍然，这个其它地方的考验也没有什么不同，只不过是换了一种形式而已。
<br/>　　“这第一关也是淘汰人最多的地方，会有九成以上的人被直接淘汰！”任云都说道。
<br/>　　苏莫点头。
<br/>　　“第二关为圣雨关，所有通过第一关的人，会被自动分组，然后承受圣雨的攻击，谁能撑到最后，谁就能过关！”任云都继续说道。
<br/>　　苏莫没有在出声询问，静静的聆听，这第二关就是综合实力的考验了，实力、防御力、闪避能力、应变能力等等的综合考验。
<br/>　　“第三关为挑战关，自由挑战，想通过此关，必须连胜三场，只要败了一场就会被淘汰。”任云都道。
<br/>　　“明白了！”苏莫点头，心中也有数了，巫族的人想要杀他，只能在第二关和第三关。
<br/>　　而其中第三关更为合适，这一关肯定会有一番生死战了。
<br/>　　在苏莫和任云都交谈之间，此地依旧不断的有大量的天才到来，汇聚的人数越来越多。
<br/>　　黑鳞族的人也来了，这是一种全身长满黑色鳞甲的人类，除了双眼和头发上之外，全身上下都不慢了细密的鳞甲，寒光森森，让人震惊。
<br/>　　此族的天才也不少，足足来了三四千人，个个气息强大无比。
<br/>　　时间不短的流逝，很快便过去了一日的时间，距离真圣界的开启。越来越近。
<br/>　　咻！
<br/>　　就在此刻，天际的尽头响起犀利的破空之声，一柄巨大的飞剑，洞穿长空，急速而来。
<br/>　　此剑长远数千丈，巨大无比，犀利的剑光即便相隔万里，都让人全身冰寒，气血凝固。
<br/>　　“这是……灵剑！”苏莫见此顿时心中一惊，这柄灵剑之强，堪称恐怖。
<br/>　　当然，这柄灵剑的本体并没有数千丈长，只是其上的散发的剑光凝而不散，让得此剑长达数千丈。
<br/>　　而在这柄巨大的灵晶之上，伫立着秘密麻麻的人影，黑压压的一大片。
<br/>　　“是三神宗的人！”
<br/>　　“三神宗的人来了！”
<br/>　　“不知道姜神夜来了没有？”
<br/>　　无数人惊呼出声，三神宗是古灵星河人族之中，最强大的势力，没有之一。
<br/>　　可以说，在古灵星河之中，能单独与巫族、黑鳞族、或者飞灵族这种强大异族抗衡的势力，唯有三神宗。
<br/>　　三神宗，原本不叫三神宗，是后来更改的名字，这是因为十万年内，三神宗连出三位虚神境大能。
<br/>　　甚至有人预测，若是姜神夜将来也能成就虚神之位，则会改名四神宗。
<br/>　　不仅如此，三神宗内，武尊境的天才之中，还有天赋不下于姜神夜的人，同样拥有虚神之资。
<br/>　　当然，想要达到虚神境，已经不单单是资质的问题了，需要天赋、机遇、气运等等因素。
<br/>　　可以这么说，就算天赋再强，再妖孽，都不一定能达到虚神境。
<br/>　　咻！
<br/>　　瞬息之间，巨大的灵剑来到了现场，剑光敛去，消失不见，只剩下万余名三神宗之人。
<br/>　　一众三神宗弟子之中，为首的是一名潇洒的中年男子，此人一声白衣，长发飘舞，相貌堂堂，气度非常不凡。
<br/>　　这名男子即便已是中年之态，给人的感觉依旧如同一位年轻的美男子，风华无双，其面带微笑，让人如沐春风。
<br/>　　“是逍遥剑圣！”
<br/>　　“三神宗的逍遥剑圣竟然亲自带队！”
<br/>　　“好潇洒的逍遥剑圣！”
<br/>　　不少人惊呼出声，这位中年男子，名为林逍遥，人称逍遥剑圣，一生潇洒，放荡不羁，声名远播，名动古灵星河。
<br/>　　逍遥剑圣，曾经是古灵星河无数女子的梦中情/人，即便现在已经不再年轻，依旧能让无数女子为之倾倒。
<br/>　　“见过逍遥剑圣！”
<br/>　　“见过逍遥剑圣！”
<br/>　　“见过逍遥前辈！”
<br/>　　很多大势力的强者，纷纷向逍遥剑圣行礼，就连吴晋和唐婉导师都不例外。
<br/>　　不少势力中年轻的女弟子，望着逍遥剑圣，想到对方的逍遥轶事，不禁眸中异彩连连。
<br/>　　苏莫眼眸望着逍遥剑圣，一种直觉告诉他，这是一位本命剑客，无比强大的本命剑客。
<br/>　　“诸位免礼！”
<br/>　　逍遥剑圣微微一笑，袖袍一挥，所有行礼之人，便感觉到一股无形的力量，将他们托了起来。
<br/>　　“多谢逍遥剑圣！”众人朗声说道。
<br/>　　逍遥剑圣微微点头，随即转头看向了不远处的巫仑，轻笑道：“今日这是什么风？居然把天冥星巫族一脉的大祭司吹来了！”
<br/>　　“林逍遥，你不是也来了！”大祭司面色淡淡的说道。
<br/>　　“呵呵，闲来无事，来观摩观摩门下弟子的水平！”逍遥剑圣笑道。
<br/>　　“彼此彼此！”大祭司淡声道，他可不会说自己来此，是为了杀一个人族的小辈。
