　　苍穹神宫，白玉广场。
<br/>　　“苏莫怎么还不现身？”
<br/>　　“时辰已经不早了！”
<br/>　　“难道有什么事耽搁了？”
<br/>　　嘈杂的议论声此起彼伏，现在已经到了中午时分，而苏莫居然迟迟不现身，不少人都等急了！
<br/>　　嗖！
<br/>　　就在此刻，破空声响起，一声大红色长袍的苏莫来了。
<br/>　　唰！
<br/>　　苏莫速度极快，瞬息之间便飞临白玉广场，而后降临在了礼台之上。
<br/>　　“感谢诸位来参加在下的婚礼！”伫立在礼台上，苏莫向四方抱拳，朗声说道。
<br/>　　“苏宫主客气了！”
<br/>　　“苏宫主大婚，我等自然不能错过！”
<br/>　　“恭喜苏宫主，贺喜苏宫主！”
<br/>　　不少人纷纷开口，言语客气至极。
<br/>　　苏莫点了点头，朗声道：“好了，时间也不早了，诸位也久等了，现在婚礼正式开始！”
<br/>　　随着苏莫言罢，现场立刻安静了下来，变得落针可闻。
<br/>　　随即，破空声再次响起，众人抬头看去，便见到天空之上，夕儿缓缓的飞了过来。
<br/>　　夕儿同样穿着一身大红色的衣裳，裙摆飘飘，身姿纤柔，其头上盖着大红色的盖头，盖头上还绣着栩栩如生的凤凰图案。
<br/>　　夕儿并不是一个人，而是在水一的搀扶之下，缓缓飞来。
<br/>　　少倾，水一带着夕儿落在了礼台上，站在了苏莫的身边。
<br/>　　“婚礼开始，新郎、新娘点天香！”水一朗声高喝，随即他的手中出现了两只三尺来长的大檀香，递给了苏莫和夕儿。
<br/>　　两人将檀香点燃，插在了面前的大鼎之中。
<br/>　　“现在进行叩拜大礼，一拜天地！”水一再次高喝。
<br/>　　苏莫和夕儿面朝大鼎，齐齐一拜。
<br/>　　“二拜高堂！”
<br/>　　苏莫和夕儿转身，向苏洪和帝释叩拜。
<br/>　　“夫妻对拜！”
<br/>　　苏莫和夕儿互相叩拜。
<br/>　　很快，两人拜完了天地，已经算是真正的夫妻了。
<br/>　　“送入洞房！”
<br/>　　随着一声高喝响起，苏莫牵着夕儿的手，身形一展，飘飞而去。
<br/>　　苏莫带着夕儿，回到了他居住的宫殿中。
<br/>　　宫殿中已经精心的布置了一番，布置了一间华丽的卧房。
<br/>　　雕花大床，罗纱幔帐，温馨而奢华。
<br/>　　苏莫牵着夕儿，走进了房间，坐在了床边，随后轻轻的掀起了对方的红盖头。
<br/>　　夕儿绝美的容颜，顿时展露在了苏莫的眼前。
<br/>　　今日的夕儿极美，从未有过的美，娇嫩的脸庞上略施粉黛，粉腻酥融娇欲滴，一双柳眉弯如月，两潭泓瞳似清泉。
<br/>　　夕儿的双眸中，蕴含惊人的柔情，仿佛能融化世间的一起。
<br/>　　四目相对，虚空中荡起一阵涟漪。
<br/>　　两人就这样彼此望着对方，静静的凝视。
<br/>　　直到良久之后，苏莫轻轻的吻了一下夕儿粉.嫩的额头。
<br/>　　“夕儿，我去招呼宾客，你等我回来！”苏莫轻声道。
<br/>　　“嗯！”夕儿轻抿红.唇，微微点头。
<br/>　　随即，苏莫离开了房间，再次回到了白玉广场上。
<br/>　　此刻，广场上礼台已经拆除，摆上了酒宴，宴请所有宾客。
<br/>　　只不过，酒宴上并没有菜肴，有的仅仅是灵果和灵酒。
<br/>　　对于众人来说，到了他们这个级别，也不需要任何菜肴了，灵果更为合适。
<br/>　　这些灵果等级虽然不是太高，但也不低，对武皇境的强者，都有不小的效用。
<br/>　　苏莫面带微笑，身形走过一席又一席，在每一席停留敬酒，与众人笑谈。
<br/>　　整个白玉广场上，一片欢腾，人声鼎沸。
<br/>　　最后，苏莫来到了苍穹神殿之中。
<br/>　　苍穹神殿中，安排了一席特殊的宾客，这些人个个身份惊天动地，俱是苍穹世界巅峰的存在。
<br/>　　这一席上，有帝释、海明江、太玄宗宗主究楚北、独孤世家家主、战神殿殿主、南海赤云岛岛主等，均是一众准帝强者。
<br/>　　此席之上，唯一不是准帝境界之人，那就是吞天龙皇。
<br/>　　东海蛟龙一族的族长并未来此，由吞天龙皇代为前来祝贺。
<br/>　　虽然吞天龙皇修为不如众人，但他相比于众人，却也并不弱势。
<br/>　　因为众所周知，吞天龙皇的实力，比一般的武皇境九重武者强的多，非常接近准帝强者。
<br/>　　“各位，久等了！”苏莫大步走进殿中，向众人抱拳。
<br/>　　“哈哈，苏宫主快来就座！”海明江哈哈一笑，率先招呼苏莫就座。
<br/>　　随即，苏莫在海明江的身边坐了下来。
<br/>　　“今日大婚，诸位不远万里来此，在下感激不尽，先痛饮三杯，以表谢意！”
<br/>　　苏莫环视众人，微笑着说道，随即他自斟三杯灵酒，均是一饮而尽。
<br/>　　“苏宫主爽快！”
<br/>　　“哈哈！”
<br/>　　众人大笑，场面显得颇为和睦。
<br/>　　这时，一名面上带着短须的中年人站了起来，向苏莫抱拳道：“苏宫主，老夫独孤世家家主独孤鸿海，我独孤家以前多有得罪，还望苏宫主莫要介怀。”
<br/>　　独孤世家家主亲自向苏莫赔罪了，希望能化解和苏莫的恩怨。
<br/>　　以前，独孤世家准备抢夺苏莫的十亿魔剑，闹出了一点不愉快，但现在苏莫已经彻底成长起来，而且前途无可估量。
<br/>　　所以，独孤鸿海希望能化解这点恩怨。
<br/>　　“独孤家主客气了，我并未放在心上！”苏莫微笑着说道，其他他对于独孤世家，倒是并没有什么记恨。
<br/>　　毕竟，独孤世家并未真正的向他出手，唯一的一次是独孤兖准备追他，但被吞天龙皇阻挡了。
<br/>　　“哈哈，苏宫主大度，那老夫就自罚三杯，酒尽怨消！”独孤鸿海大笑道。
<br/>　　“请！”苏莫微笑。
<br/>　　而后，独孤鸿海自罚了三杯，以逝双方恩怨。
<br/>　　苏莫见此，心中沉吟，或许独孤世家并没有和黄泉魔宗同流合污，不然的话也不会向他道歉。
<br/>　　对方此举，一是道歉化解以前的恩怨，二是向苏莫表明，独孤世家不会与苏莫为敌。
<br/>　　“苏宫主，你我第一次见面，本座先自报家门，本座乃战神殿殿主，战空！”
<br/>　　这时，又一名强者开口了，向苏莫抱拳，此人年约四十岁许，身形高大壮硕，面容较为粗犷。
<br/>　　“辛会！”苏莫向对方回礼，他微微打量了一眼此人，发现对方面容和战乱有五分想像，估计是战乱的长辈。
<br/>　　仿佛是知道苏莫心中所想，战空大笑道：“苏宫主，战乱正是小儿，你们也算不打不相识了！”
<br/>　　“原来如此！”苏莫点头，原来是此人是战乱的父亲。
