        “你不是要跟我大哥订婚吗？你怀我的孩子，若是让大哥知道了。我大哥他……”傅昼景欲言又止。
<br/>
<br/>        季思妍伸出手来，紧紧地握住傅昼景的手，苦苦哀求起来：“你大哥拒绝了这门婚事，更何况，联姻的事情，是我爸爸提出的，又跟我有何关系？我爱的人又不是你大哥！阿景，我求求你，好吗？你可以和筱颖一起走，但是我只想要一个属于你和我的孩子，陪我过接下来的日子！好不好？阿景，我求求你。”
<br/>
<br/>        “思妍，你有没有想过，你给我生的孩子，将来要背负着什么？非法私生？这对你有什么好处？”傅昼景一脸不悦地皱了皱眉头。
<br/>
<br/>        “阿景，算我求你，好吗？我只想要个孩子！”季思妍心一横，一边解开自己身上的浴巾，一边赤脚落地，带着哀求的哭腔，一步步朝傅昼景靠近。
<br/>
<br/>        她伸手环住他的腰，踮起脚尖吻上了他的脖子，在他耳畔呵气如兰。
<br/>
<br/>        季思妍的双手缓缓向上移去，慢条斯理地解开傅昼景衬衫上的纽扣，然后吻上了他漂亮的锁骨，娇柔欲滴，信誓旦旦地说道：“阿景，我只是想要一个孩子陪着我！因为我爱你，所以，我只想跟你生孩子。我的身心都接受不了其他男人了！阿景，我求求你，好吗？给我一个孩子，好不好？我向你保证！从此以后，我不会再打扰你和筱颖！”
<br/>
<br/>        傅昼景忽然间面无表情，稍微拉开些他和季思妍的距离后，又突然将她横抱了起来，往那边大床走去。
<br/>
<br/>        “阿景，你……答应了，对吗？”季思妍软软地贴在傅昼景的怀中，受宠若惊地询问。
<br/>
<br/>        傅昼景并未回答季思妍的话，而是将她放到床上后，没有任何前奏地直接扑上她。
<br/>
<br/>        “喀嚓”的金属质地的脱落声，皮带解开后，他利索地褪去了身上的衣物。
<br/>
<br/>        他不曾给她任何一个暧昧的吻，他是直来直去，像是在做任务一样，对她敷衍了事。
<br/>
<br/>        所以，他这是真的完完全全地已经不再爱她了，对吗？
<br/>
<br/>        季思妍咬着唇，眼泪哗啦啦地往下流。
<br/>
<br/>        另一边的酒店套房里，童筱颖洗完澡后，穿着自己的睡裙，披散着半湿不干的长发，将狗狗小爱抱在了怀中，坐在大床上，看电视剧。
<br/>
<br/>        因为看得入迷，她完全忘了时间，更没在意傅昼景是否去得太久了还未回来。
<br/>
<br/>        不知不觉中，童筱颖抱着狗狗小爱躺床上睡了过去。
<br/>
<br/>        当她再睁眼的时候，天已经微微亮。
<br/>
<br/>        身边，傅昼景不知何时归来的，只见他剑眉微蹙，身上穿着白色浴袍，侧躺在被褥上。
<br/>
<br/>        房间里开了空调，不盖被褥，还是有点儿冷意袭来。
<br/>
<br/>        童筱颖看着傅昼景的睡颜，甜甜地笑了笑，然后轻轻地起身，拉起被褥的一角。
<br/>
<br/>        当她想要给他盖上被褥的时候，他那双从浴袍里露出来的双腿，顿时吸引了她的注意力。
<br/>
<br/>        童筱颖笑容渐敛，微微歪着脑袋，一本正经地打量起傅昼景的两条大长腿来。
<br/>
<br/>        傅昼景的双腿修长是没错，但是小腿有些微微浮肿，与大腿不成正比。
<br/>
<br/>        童筱颖下意识地抬起手来，缓缓地将手伸进了傅昼景的浴袍内，在他的腰后轻轻地捶了捶。
<br/>
<br/>        “痛——筱颍，别闹了！”傅昼景剑眉微蹙，眼皮子也不抬一下地说道，“我有点儿累，让我再睡一会。”
<br/>
<br/>        “小卫，你最近是不是经常腰痛呀？”童筱颖匍匐在傅昼景的身边，关心地问道。
<br/>
<br/>        “经常加班熬夜，坐在办公室里坐久了，自然是会腰痛。没关系的，不打紧。”傅昼景这才缓缓地睁开了双眼，嘴角微扬，伸手将童筱颖搂入怀中，轻轻地抱住。
<br/>
<br/>        童筱颖下意识地往傅昼景的怀里钻了钻，担忧地说道：“我们今天去了澳洲后，我陪你去医院里做个身体检查好不好？”
<br/>
<br/>        “我真的没事！”傅昼景右手肘撑在枕头上，侧身看着童筱颖，微微低头在她的额头上轻啄了一下，眸底染着些许笑意，“你不用太担心我。”
<br/>
<br/>        “去嘛！去嘛！好不好？就做个身体检查！我陪你一起做！”童筱颖开始不依不饶地撒起娇来。
<br/>
<br/>        她虽不是什么正经八经的医生，但是跟随在外婆身边那么多年，也有些耳濡目染。
<br/>
<br/>        小腿轻微浮肿，经常腰痛的话，多半是肾出了问题。
