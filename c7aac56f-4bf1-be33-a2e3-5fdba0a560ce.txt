　　苏莫继续参悟，努力掌控第十重时光神殿的法则。
<br/>　　又过了近十个时辰之后，他终于如愿以偿，掌控了第十重时光神殿的时间法则。

<br/>　　苏莫睁开了眼眸，暗暗的松了空气，随即他立刻大量四方。
<br/>　　这第十重时光神殿之中，人也不少，约莫有四十余人，那名之前的中年人也赫然在列。

<br/>　　近二十个时辰，相当于一天半左右的时间，倒也还算充足。
<br/>　　随即，苏莫控制此殿的时间规则，仔细的查看了一下时间流速。
<br/>　　“不到六千倍，应该在四千五百倍左右！”苏莫心中暗道，这个流速的话，他在这里能修炼十九年左右。
<br/>　　他并没有失望，十九年的时间，已经非常的不错了。
<br/>　　他并不奢望，一次性给他百年时间，若是那样的话，他的资源也不够消耗的。
<br/>　　“刚才倒是有些浪费了！”苏莫暗叹了一口气，之前杀了许焱，连对方储物戒都毁灭了。

<br/>　　他并没有大规模的吞噬五行源石，只是不断的取出源石，手握源石吸纳其中的源力。

<br/>　　不过，这也无妨，他吸纳源石中源气的速度，也不是常人可比，比同阶武者快了十倍都不止。
<br/>　　一把又一把的源石，被苏莫吸纳尽了源力，化为了齑粉，苏莫的修为缓缓的增长。
<br/>　　三年过去了，苏莫的修为，达到了二品中级的巅峰。
<br/>　　五年过去了，苏莫的修为，成功的达到了二品高级。
<br/>　　他毫不停歇，不停的吸纳源石中的源气。
<br/>　　又过去了一年，大殿之中，慕容雷利和公孙嘉两人，也纷纷的掌控了第十重时光神殿的时间法则。
<br/>　　公孙嘉比慕容雷利，仅仅是早了一刻钟，显然两人的悟性相差无几。
<br/>　　“慕容兄，时间已经不多了，我们只能留在第十重时光石殿了。”公孙嘉望着慕容雷利，无奈的说道。
<br/>　　“嗯，这里正好可以节约不少的时间！”慕容雷利点了点头，若是时间允许，他自然还想前进，但是时间已经不多了。
<br/>　　“呵呵，慕容兄，以我们的天赋，将来走过三十三重时光神殿，也不是没有可能！”公孙嘉淡然一笑道。
<br/>　　这是他的梦想，穿过三十三重时光神殿，到底时空的彼岸，才能去争取超脱纪元的希望。

<br/>　　“希望数十万年之内，能完成这个目标！”慕容雷利点了点头，想要穿过三十三重时光神殿谈何容易。

<br/>　　“哦？”公孙嘉闻言，双眉一挑，随即仔细的打量了苏莫一番，不禁微微摇头。
