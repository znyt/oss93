<br/>    无数的白色利齿之间，陈明朝着前方行去，终于是踏足在血肉之上，不过这龙鲲的血肉，坚韧的如同玄铁！
<br/>    
<br/>        陈明看得到身后四人已经到了他的身旁，陈明道：“紧紧跟着我，不要走散了！”
<br/>    
<br/>        四人都是道：“我们明白！”
<br/>    
<br/>        陈明看向周围，夜明珠绽放光华，陈明顺着这巨大的通道前进，这里是龙鲲的喉咙，如同一个巨大的洞窟！
<br/>    
<br/>        足足前进了一里地，陈明终于是看见了一个身穿青衣的身影，此刻正在水流之中漂泊，陈明看得出来，卓青瑶已经有些脱力了。
<br/>    
<br/>        紫霞仙子也是急忙喊道：“快看，卓青瑶在这里！”
<br/>    
<br/>        陈明哪里还会等，直接摇身一变，化作黑色的玄龟，四爪飞动，将卓青瑶托在背后，卓青瑶感受到陈明的到来，有些心虚地喊了一句，“师父，你也想到进龙鲲腹中这个主意了？”
<br/>    
<br/>        陈明冷声道：“回去了再教训你！”
<br/>    
<br/>        苏青阳三人也是有些惊奇，没有想到，居然能够在这里遇到卓青瑶。
<br/>    
<br/>        卓青瑶道：“这龙鲲太大了，进来之后根本找不到路，完全不知道它的要害在哪儿，而且它的血肉，太坚韧了，很难劈出一条路来。”
<br/>    
<br/>        陈明道：“笨，只要朝着一个方向劈开一条口中，进入这龙鲲的经脉之中，顺着经脉前行，一定能够找到气血最为充盈的心脏！”
<br/>    
<br/>        卓青瑶恍然大悟道：“原来还可以这样！”
<br/>    
<br/>        陈明道：“你们四个，朝着一个方向劈出一条路来，我们只要找到了龙鲲的经脉，跟着它的经脉前行，一定能够找到这龙鲲的心脏！”
<br/>    
<br/>        四个人对于陈明的安排也没什么二话，顺着一堵墙壁就开始劈砍，开辟出一条通道来，这点伤害，对于龙鲲来说，根本算不得什么，甚至连感觉都不会有。
<br/>    
<br/>        几人顺着劈开的口子前行，足足前行了十多丈，一阵鲜血喷出，陈明直接带着卓青瑶冲了进去，龙鲲的经脉，就像是一个通道，里面有着源源不断流动的鲜血。
<br/>    
<br/>        陈明顺着血流前进，“跟我来！”
<br/>    
<br/>        几人在经脉之中顺着血流的方向前行，也不知道自己到了何处，足足过了三个时辰，几人终于是来到一个新的地方，这里如同是另外一个世界，周围都是血腥味浓厚的鲜血，就如同是一座巨大的宫殿。
<br/>    
<br/>        陈明看向周围，朝着周围游走了一阵，然后道：“这里很是宽阔，足足有着一座宫殿那么大，应该就是龙鲲的心脏了，只要击破了龙鲲的心脏，它必死无疑！”
<br/>    
<br/>        刘芒笑道：“既然已经来到了这个大家伙的心脏，就要它的命！”
<br/>    
<br/>        刘芒摸出自己的寒霜龙枪，挥枪就是朝着周围的红色墙壁一枪劈去，一枪落下，墙壁直接打开了一道口子，不过却是没有劈穿，只是劈开了一道豁口，六人忽然感受到周围一阵晃动！
<br/>    
<br/>        从外面传来如同雷鸣般的嘶吼，刘芒道：“那龙鲲发现我们了！”
<br/>    
<br/>        而在眼前的那一道豁口，竟然血肉重生，逐渐愈合！
<br/>    
<br/>        不过十息的时间，刚刚那一道豁口已经消失不见！
<br/>    
<br/>        雪自悟道：“这龙鲲有着如此巨大的身躯，恢复的速度绝对不能按照常理推断，我们即便是进入了它的心脏，也很难直接杀掉它！”
<br/>    
<br/>        忽然，周围忽然有着无边的法力涌来，想要将六人压成碎片！
<br/>    
<br/>        苏青阳道：“龙鲲已经发现我们了，它不会让我们在这里胡作非为的，它已经散开了它的法力，想要靠着法力直接压死我们！”
<br/>    
<br/>        紫霞仙子也是道：“这龙鲲的身躯巨大之极，如果只是以法力论，它的法力足足比得上数位大君，根本不是我们能够对抗的！龙鲲正在调集它所有的法力，全部汇聚到心脏之中。”
<br/>    
<br/>        陈明口中吐出无数柄灵剑，瞬间组成一个大阵，然后道：“我开启阵法阻挡这些法力，你们速速将它的心脏劈穿，让它流血而死！”
<br/>    
<br/>        苏青阳看向陈明，问道：“堪比数位大君的法力，你真的撑得住？”
<br/>    
<br/>        陈明道：“拼法力量，或许差了点，但是我还撑得住一会儿，现在，没有时间废话了，你们尽快！”
<br/>    
<br/>        玄关剑舞在陈明的身旁展开，将所有的鲜血隔绝开来，只露出身旁的一堵血肉墙壁，四个绝顶天才可以感受到压制着自己的法力瞬间消失，几人对视了一眼，瞬间拿出自己的兵器，施展仙术朝着身前的血肉墙壁劈去！
<br/>    
<br/>        轰！
<br/>    
<br/>        轰！
<br/>    
<br/>        轰！
<br/>    
<br/>        一声声巨响在身边响起，卓青瑶看了一眼这四人的实力，远远高于自己，自己还是不要上去添乱了。卓青瑶跪坐在陈明的背上，拿出一瓶丹药开始恢复自己的气血之力。
<br/>    
<br/>        苏青阳，紫霞仙子，刘芒都在全力轰着身前的血肉墙壁，一阵血肉翻飞，落在身前的，却是如同玄铁一般坚硬的碎块。
<br/>    
<br/>        雪自悟看到三人劈开的豁口，双手穆然间结印，“仙术，千里冰凝！”
<br/>    
<br/>        白雪落下，在他的脚下，一层层冰层凝结，瞬间蔓延到豁口之上，将豁口处的血肉冻住，让龙鲲的恢复速度骤降！
<br/>    
<br/>        雪自悟看向陈明，他很怕陈明现在撑不住，要知道，陈明现在，可是在承受着堪比数位大君的法力，“宗师，你还好吧？”
<br/>    
<br/>        陈明道：“只是法力，并不是运转法力的法术，还行！”
<br/>    
<br/>        陈明此刻也是发现了，法力对拼，他似乎确实不虚啊，不是因为别的，而是因为太给力了！
<br/>    
<br/>        拥有属性的法力，就像是一柄柄利剑，而龙鲲的法力，就像是一根根木头，虽然对方量多，但是目前还能撑得住，而且，论法力量，陈明可是有着近百个道宫的！
<br/>    
<br/>        雪自悟点点头，道：“那就好，我等继续！”
<br/>    
<br/>        四人无意中看到陈明只见陈明稳如泰山，对于陈明的实力，越发震惊，现在的局面，可谓是陈明一个人顶住龙鲲的压力，而他们，则是没有任何后顾之忧的在进攻。
<br/>    
<br/>        足足半刻钟之后，一道巨大的豁口终于是被他们劈穿，血液如同河流一般从豁口之中流出，周围一阵天旋地转，那是龙鲲在嘶吼，在震怒！
<br/>    
<br/>        有人在它的身躯之中搞破坏！
<br/>    
<br/>        而且，还是在它的要害。
<br/>    
<br/>        正在这时，六人分明看到，滚滚的鲜血汇聚在一处，周围的血液瞬间清空，在心脏之中汇聚成一条长达二十丈的血龙，血龙之上的威势，堪比大君！
