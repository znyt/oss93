　　随着灵柩打开，阎魔鬼圣的魔魂之中，立刻涌出了一股魔念，向芊寻月涌去。
<br/>　　苏莫神情一凝，不过他并未阻止，这老魔若是敢夺舍芊寻月，他立刻便将对方的魔魂彻底吞噬。
<br/>　　他现在武魂晋升到天级九阶，威力大增，想灭杀阎魔鬼圣，已经没有什么问题了。
<br/>　　不过，苏莫显然多虑了，阎魔鬼圣也知道自己的处境，并不敢夺舍芊寻月。
<br/>　　其魔念在芊寻月身上仔细查看了一番，便收了回来。
<br/>　　“怎么样？”苏莫紧张的问道，非常担心对方再说出无法救活的言语。
<br/>　　“神魂消散，只剩下一缕执念，而且弱到了极致！”
<br/>　　阎魔鬼圣叹了口气，继续道：“她这个状态应该维持了很长的时间吧？若是几年前还有办法救活，但现在已经救不活了！”
<br/>　　轰！
<br/>　　阎魔鬼圣的话语，如同一声猛雷，在苏莫的脑海中炸响，立刻让他身形剧震，脸色苍白。
<br/>　　“怎么会这样？”苏莫呢喃自语，心中感觉非常的悲戚，救不活了吗？
<br/>　　这个结果让他无法接受！
<br/>　　虽然他和芊寻月没有多少感情，但对方毕竟是他的女人，有过夫妻之实。
<br/>　　“鬼圣前辈，你是武圣境的超级强者，难道都没有办法吗？”苏莫神情怔怔的问道。
<br/>　　“没有办法！”阎魔鬼圣叹道，他已经查看清楚了芊寻月的状况，这完全可以说是一个死人，别说是他，就算是虚神境大能，也不一定能救活。
<br/>　　苏莫闻言，脸色更为苍白，他缓缓的伸出手掌，抓住了芊寻月的玉手，紧紧的攥在手中。
<br/>　　浓郁的伤感，在苏莫心中蔓延，让得他心绪低落无比。
<br/>　　“其实，也并不是绝对的无解！”这时，阎魔鬼圣的声音再次响起。
<br/>　　“什么办法？”苏莫闻听此言，眸中顿时精光爆闪，急忙出声问道。
<br/>　　而这时，阎魔鬼圣却沉默了起来，半晌都没有声音。
<br/>　　这让苏莫心中大急，不过，他也不傻，顿时明白了对方的意图。
<br/>　　这么重要的事情，对方肯定不想如此轻易的告诉他。
<br/>　　“鬼圣前辈，我苏莫指天发誓，只要突破到武尊之境，便立刻还你自由，并且为你找一具上好的肉身，若违此时，天诛地灭！”苏莫面色郑重，指天发誓道。
<br/>　　“呵呵！汝不必如此，吾只是在思考而已！”阎魔鬼圣笑了笑，即便苏莫不发誓，他也会告诉苏莫。
<br/>　　因为他的命运，还在苏莫的掌控之中。
<br/>　　他之所以没有马上回答，就是想看看苏莫的态度。
<br/>　　“前辈，到底是什么方法？”苏莫急声问道。
<br/>　　阎魔鬼圣沉默了少倾，幽幽的说道：“传闻在这无尽宇宙之中，有一座轮回之门，可以逆天改命，汝只要找到了轮回之门，便能救活此女！”
<br/>　　“轮回之门？”苏莫呢喃自语，心中惊讶，生死轮回乃是天道规则，居然还有这种奇物。
<br/>　　“轮回之门，可以让人在天道的规则之外转世重生，从而拥有全新的生命，玄妙无比！”阎魔鬼帝说道。
<br/>　　“那这轮回之门在何处？”苏莫沉声问道。
<br/>　　“不清楚！”
<br/>　　阎魔鬼圣摇头，继续道：“这轮回之门的事情，还是吾幼年之时听闻的传说，到底是否存在，没有人知道！”
<br/>　　听闻对方的话语，苏莫面无表情，心已经沉入了谷底。
<br/>　　对方幼年之时？
<br/>　　那已经是十几万年前的事情了，而且对方也只是听闻的传说，很可能根本就是不存在的事情。
<br/>　　不过，这终究是一丝希望！
<br/>　　哪怕希望再渺茫，总比没有希望要好。
<br/>　　他会竭尽所能，找到这所谓的轮回之门，然后救活芊寻月。
<br/>　　“那这轮回之门，是一件宝器？还是某种自然形成的玄奇之地？”苏莫再次问道，想要多了解一番轮回之门的消息。
<br/>　　“不清楚！这是亿万年前的传说，吾也只是偶尔听闻而已。”
<br/>　　阎魔鬼圣再次摇头，继续道：“而且，就算真有轮回之门的存在，是否真有传闻中的这种能力，也尚未可知！”
<br/>　　苏莫闻言心中暗叹，看来只能等以后他实力强大了，再找寻轮回之门的线索了！
<br/>　　他目前实力太低，别说在无尽的宇宙星空中，就算在这古周星，都只能算是一般的武者。
<br/>　　“多谢前辈了！之前态度不佳，还望恕罪！”苏莫叹了口气，向对方道谢，并表示歉意。
<br/>　　“无妨！”阎魔鬼圣道。
<br/>　　少倾之后，苏莫锊了捋芊寻月有些凌乱的发丝，轻轻的盖上了玄天灵柩，便将玄天灵柩收了起来。
<br/>　　而后，他盘膝而坐，准备修炼。
<br/>　　修炼才是根本，没有实力的话，一切都是虚妄。
<br/>　　刚刚闭上双眼，就在此时，阁楼外响起了脚步声，随即阁楼的大门被人敲响了。
<br/>　　嘭！嘭！嘭！
<br/>　　沉闷的声音响起，来人敲的极重，让得整座阁楼都微微震颤。
<br/>　　苏莫眉头一皱，随即灵识一扫，顿时有些意外。
<br/>　　因为来人，乃是太史轩。
<br/>　　苏莫心中疑惑，对方来找他做什么？他和对方不仅不熟，甚至还有些过节。
<br/>　　不过，苏莫还是起身打开了房门。
<br/>　　“何事？”
<br/>　　望着门外气度非凡，如翩翩公子一般的太史轩，苏莫淡淡的问道。
<br/>　　“苏莫，你的天赋很不错，加入我太史世家一脉如何？”太史轩背负着双手，面带微笑，开门见山的说道。
<br/>　　苏莫闻言一怔，随即立刻恍然，原来对方是来拉拢他的。
<br/>　　之前和冷江一战，他的天赋暴露了大半，已经引起了对方派系的看重。
<br/>　　“不好意思，没有兴趣！”苏莫摇了摇头，他不会加入任何的派系，更不会加入太史世家的派系。
<br/>　　对于太史轩，他没有任何好感，此人目中无人，嚣张无比，让他根本不屑与之为伍。
<br/>　　对方现在对他客气有佳，不过是想要拉拢他而已。
<br/>　　太史轩闻言脸色一沉，他没想到苏莫拒绝的如此干脆，根本没有一丝犹豫。
<br/>　　“苏莫，我们太史世家，可是古灵星河一等一的大势力，你不再考虑了一番？”
<br/>　　太史轩眯着眼睛问道，他主动发出邀请，苏莫若是不同意，那就是不给他面子了，也是不给他太史世家面子。
