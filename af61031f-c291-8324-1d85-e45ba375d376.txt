　　苍穹神殿中寂静无声，所有人都注视着正在瑟瑟发抖的海明江。
<br/>　　海明江，因太过愤怒，身体都颤.抖了起来。
<br/>　　唰！
<br/>　　下一刻，只见海明江身形一闪，抱着钱副阁主急速飞出了大殿。
<br/>　　“海阁主！”
<br/>　　苏莫见此立刻急乎一声，赶紧追了上去。
<br/>　　很快，苏莫便在上空水幕阵法之前追到了海明江，因为阵法已经关闭，任何人都出不去。
<br/>　　“海阁主，你现在要去哪里？”苏莫来到了海明江身前，沉声问道。
<br/>　　“苏宫主，立刻打开大阵，黄泉魔宗灭我天涯海阁总部，这个仇老朽不可能不报！”海明江满脸阴沉的大声吼道。
<br/>　　苏莫见此暗叹一声，随即立刻摇了摇头，道：“黄泉魔宗和阴阳神宫联手，若是我所料不错的话，还有北堂世家和殷家相助，您老现在去报仇，无异于以卵击石。”
<br/>　　闻听苏莫之言，海明江稍稍的冷静了下来，这点他何尝不知道，就算是没有北堂世家、殷家和阴阳神宫，他也完全不是黄泉魔宗的对手。
<br/>　　刚才他也是一时太过愤怒，有些冲动了，现在听到苏莫的提醒，他不禁眉头紧皱了起来。
<br/>　　“海阁主，现在先将钱副阁主安顿在此养伤，待今日大婚结束，就是我们向黄泉魔宗宣战的时候。“苏莫眸中精光闪烁，一字一句的说道。
<br/>　　“好吧！”
<br/>　　少倾，海明江点了点头，同意了苏莫的安排。
<br/>　　若想报仇，他的确只能借助苏莫和苍穹神宫的力量。
<br/>　　随即，苏莫和海明江又回到了苍穹神殿，重伤的钱副阁主被天涯海阁的强者带了下去，苏莫命人为他们安排了住处。
<br/>　　不过，经此一事之后，整个喜庆的大婚，气氛已经被彻底破坏了。
<br/>　　大殿中众人个个面色郑重，心思各异。
<br/>　　海明江没有心情再待下去了，再次离开了大殿，前往苏莫安排的住处。
<br/>　　这时，苏莫转头望向了帝释，天涯海阁遭到了黄泉魔宗的攻击，那帝玄宫呢？
<br/>　　此刻，帝释眉头紧皱，面现担忧之色。
<br/>　　见苏莫望向自己，帝释略一沉吟，道：“苏莫，本座要先回去了！”
<br/>　　帝释不放心帝玄宫，所以决定立刻返回。
<br/>　　不过，帝释也没有太多的担心，就算帝玄宫真的遭到了黄泉魔宗的攻击，也不会像天涯海阁这般损失惨重。
<br/>　　因为帝玄宫有内宫和外宫之分，内宫处于小世界中，入口有阵法封印，黄泉魔宗想要攻进去，短时间内很难办到。
<br/>　　“嗯！”苏莫点了点头，他没有阻止帝释，对方不可能不管帝玄宫的死活。
<br/>　　随即，苏莫打开了水幕大阵，让帝释离开了。
<br/>　　十三长老等其他帝玄宫的强者，帝释没有让他们回去，而是将他们都留在了苍穹神宫。
<br/>　　因为帝玄宫目前什么情况，还不知晓，众人回去的话，可能会遭遇危险。
<br/>　　而帝释实力强大，自然无惧。
<br/>　　苏莫本想和帝释一起前去帝玄宫，比较他也不想帝玄宫也遭难，但他最后还是放弃了这个想法。
<br/>　　其一是他走不开，其二是从苍穹神宫赶到帝玄宫，路上要花费不少的时间，若是帝玄宫真的遭到攻击，等他们赶过去，也已经晚了！
<br/>　　随着帝释离开之后，其他各大势力的强者，均是纷纷向苏莫提出了告辞。
<br/>　　现如今，苍穹世界面临着大变，众人哪里还有心思在苍穹神宫多待。
<br/>　　黄泉魔宗开始露出了獠牙，几乎人人心中都有紧迫感，谁都担心会被黄泉魔宗盯上。
<br/>　　对此，苏莫也没有挽留众人，将各大势力之人，全部送离了苍穹神宫。
<br/>　　宾客散去，所有宴席在几名傀儡人的安排下，也都全部撤销、清理，整个苍穹神宫也恢复了平静。
<br/>　　一场不算隆重的婚礼，就这样草草的结束了！
<br/>　　苏莫一个人坐在大殿中，独自沉思。
<br/>　　这一次黄泉魔宗出手，真的是打了他们一个措手不及。
<br/>　　天涯海阁不能再留在外界了，不然会被黄泉魔宗彻底消灭。
<br/>　　“苏莫！”
<br/>　　“苏莫！”
<br/>　　“宫主！”
<br/>　　这时，九名傀儡人和司空炎、姜枫然等人全部都走了过来。
<br/>　　司空炎、闻人天都、姜枫然、彩芸四人，修为都暴涨了一大截。
<br/>　　司空炎达到了武王境八重，闻人天都武王境七重、姜枫然武皇境九重，彩芸武皇境武皇境六重。
<br/>　　“宫主，这个黄泉魔宗狼子野心，必须早些毁灭！”金一道。
<br/>　　“我知道！”
<br/>　　苏莫点了点头，道：“黄泉魔宗不仅与我有大仇，更和试炼者有关系，绝对不能留！”
<br/>　　“灭杀黄泉魔宗，以宫主现在的实力，加上帝玄宫和天涯海阁之助，应该不是太困难吧！”石一道。
<br/>　　苏莫闻言摇了摇头，叹道：“各位前辈有所不知，黄泉魔宗有三位准帝强者，再加上阴阳神宫、北堂世家和殷家之助，想剿灭谈何容易！”
<br/>　　苏莫有些无奈，想灭了黄泉魔宗，以他现在的实力暂时还做不到，最起码他也要突破到武皇境四重的地步。
<br/>　　他若突破到武皇境四重，应该足以轻易战胜准帝强者，到时候灭了黄泉魔宗，问题不大。
<br/>　　“三位准帝？”众人闻言心中一惊，大感意外。
<br/>　　“各位前辈放心吧！我自有定夺！”苏莫摇了摇头道。
<br/>　　“苏莫，有什么需要我们帮忙的？”司空炎开口问道，他虽然一直在闭关，但现在也知道了苍穹神宫所面临的情况。
<br/>　　“有！”苏莫点头。
<br/>　　“什么忙？”
<br/>　　司空炎眼眸一亮，继续道：“苏莫你快说，我已经很久没活动筋骨了，若是去杀人，我绝对不会让你失望的！”
<br/>　　司空炎自信满满，修为的快速暴涨，已经让他的自信心爆棚了。
<br/>　　他修炼了一身的帝级上品武学，自己都不知道自己的战力，目前达到了什么地步！
<br/>　　“你们的任务……就是修炼！”苏莫道。
<br/>　　呃~~
<br/>　　苏莫此言一出，司空炎面色一呆，众人也均是面色怔怔，这也叫任务？
<br/>　　“你们的修为太低了，修为提升的速度也太慢了，帮不上什么忙，你们要争取尽快达到武皇境！”苏莫沉声说道。
<br/>　　“是！”众人点了点头，司空炎心中非常的无奈，天下间也就苏莫嫌弃他们修炼的慢。
<br/>　　“好了，司空炎、闻人天都、姜枫然、彩芸姑娘，你们抓紧时间回去修炼吧！”
<br/>　　苏莫摆了摆手，随即又对金一等人道：“各位前辈，苍穹神宫正常运转，其它事不用管！”
<br/>　　言罢，苏莫大步离开了苍穹神殿，留给众人一个背影。
