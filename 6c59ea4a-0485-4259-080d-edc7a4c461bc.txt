<br/>　　古今圣贤多寂寞，唯有饮者留其名……不过，无论程大雷还是李行哉，都没有白发渔樵江渚上，惯看秋月春风的胸怀，他们依旧在俗世里打滚，忙忙碌碌的奔波着些什么。
<br/>　　那一夜，互相都饮了很多酒，程大雷酩酊大醉。直到次日黎明，程大雷要了一艘小船，搭着他的黑牛，向岸边靠去。
<br/>　　天还没亮，酒还未醒，李行哉立在船头，目送着程大雷远去的背影。
<br/>　　虽然手上有了梦寐以求的兵力，再不复从前的一穷二白，但肩上承担的压力也的确比往日多了。也唯有见到程大雷后，李行哉才能暂且忘掉肩上的胆子，说些有的没得。
<br/>　　“王上。”
<br/>　　戚继光出现在李行哉身后，他向来是个很敬业的人，夙兴夜寐，勤勤恳恳为李行哉做些事。
<br/>　　“戚将军可以多休息一会，消灭骷髅岛后，我们可以闲一阵。”
<br/>　　“王上，属下有一件事禀告。”戚继光声音压得很低：“是否，不该放程大雷走？”
<br/>　　李行哉一怔，回过头奇怪的看着程大雷。
<br/>　　戚继光耐心的为其解释：“程大雷在众目睽睽之下杀了夫子，书院藏书楼的事和他恐怕也不会没关系。如今他在天下读书人之间已经天怒人怨，王上若能擒住他，必定能收获天下读书人的心。”
<br/>　　李行哉眯起眼睛，上下打量着他。
<br/>　　“属下明白王上和他有些交情，程大雷也未必是人人可杀的恶贼，但王上要做大事，便不该有这妇人之仁。”戚继光直视他的眼睛：“王上若不方便做，我便带人去做，大海之上，任他有天大本事，也逃不了。”
<br/>　　李行哉沉默了好久，伸手拍了拍戚继光的肩膀：“多睡一会儿吧。”
<br/>　　“王上……”
<br/>　　戚继光还准备再说什么，李行哉已经一瘸一拐向船舱走去，他只是往后挥了挥手，没兴趣听戚继光再说什么。
<br/>　　天亮的时候，船队分为两支，一支由戚继光率领，赶往骷髅岛搜刮财物。扫荡海上海盗，花费大力气攻打红胡子，目的为的不就是这个么。而李行哉则独自乘坐一艘大船，带着李婉儿返回落凤城。
<br/>　　兄妹多年不见，各自都经历了一些波折。可昨日重逢，李行哉同程大雷说了许多话，倒是兄妹二人并没有聊什么。
<br/>　　大概是往事太过沉重，沉重得让所有人都张不开口。
<br/>　　下了船从港口返回城池，路上倒也没带什么随从。虽然李行哉目前有些家底，但他当初当王子时，也不是个讲究排场的人，更何况现在。
<br/>　　如今的落凤城，渐渐已有了些人气，走在大街上，忽然一人引起了李行哉的注意。
<br/>　　对方一瘸一拐的走在街道上，衣服破烂不堪，看上去落魄得很。李行哉因为并州的事，腿上也有些微疾，倒是没有太大妨碍，不过瘸子见瘸子，总会多看两眼。
<br/>　　李行哉看向对方的时候，对方也正打量着他。相比之下，李行哉的扮相就好上许多，毕竟是一地之主，身上锦衣华服，李行哉的相貌也足以算得上英俊。
<br/>　　对方忽然冲李行哉走过来，李行哉的手下意识摁住佩剑，刺客什么的他也不是没遇见过。不过，那家势力派这样个人来刺杀自己，估计眼睛也够瞎的。
<br/>　　“逍遥王？”
<br/>　　对方停在三步前，认真询问。
<br/>　　李行哉皱眉点点头，对方的模样透着奇怪。
<br/>　　“收人么？”
<br/>　　“呃……”李行哉愣住。
<br/>　　“在下楚云生，愿投在逍遥王门下，以效犬马。”
<br/>　　天下诸侯林立，楚云生未必说就看上了李行哉，不过，他现在的确有些走投无路。火烧藏书楼后，他就趁机逃出了琅琊城，虽然琅琊城的城门封了，但想要出来，也并不是没有办法，何况楚云生在琅琊城待了许多年，也勉强算个地头蛇了。
<br/>　　混出琅琊城后，他便想着去投奔那家诸侯，如今诸侯割据，想找家老板还是容易的。不过，诸侯割据也表示现在这世道不是很太平，他孤家寡人一个，万一路上遇到土匪山贼什么的，一刀被宰了，也是自己的命。
<br/>　　等来到落凤城时，已经花光了盘缠，身上分文无有。他已有两天没吃东西，所以看向李行哉的目光那样渴望，倒不是看出李行哉是人中龙凤，而是实在希望有人能给碗饭。
<br/>　　李行哉本身就算朵奇葩，但隐隐觉得，这楚云生比他更加奇葩。
<br/>　　“那个……你先随我来吧。”
<br/>　　听着对方肚子咕咕直响，李行哉也知道对方是个饿货。如今一碗饭他还是管得起的，至于对方能做什么，回头详细问问便是。反正，每一朵奇葩都有些与众不同之处，否则也称不上奇葩。
<br/>　　他走在前面，楚云生跟在他身后，二人走路都是一瘸一拐，远远望过去，倒是很协调。
<br/>　　……
<br/>　　琅琊城港口，华正则远远望着海面。
<br/>　　“你们当家的，怎么还不回来？”
<br/>　　“怎么回来？”刘发财歪在那里，懒洋洋道：“岛上高手如云，大当家便是有再大本事，能够挡得住东海所有高手的围攻，现在肯定是已经死了。”
<br/>　　“那个……你好像一点都不担心。”华正则吃惊道：“他明知送死还去？”
<br/>　　“这就是大当家的人品呐。”刘发财感慨一声：“虽千万人吾往矣，当他已决定去送死，我担心有什么用。”
<br/>　　“喂，聊什么呢？”一艘小船靠岸，程大雷看着二人奇怪道。
<br/>　　刘发财和华正则都是一激灵，程大雷无声无息而来，令人忍不住怀疑，他是不是死在海岛上，此刻是鬼魂过来。
<br/>　　程大雷牵着黑牛上岸，稀里糊涂的看着二人。
<br/>　　“哈，我说什么来着。”刘发财哈哈大笑道：“大当家吉人天相，凭东海这几头软脚虾怎么是大当家对手，果然，大当家平安无事回来了。”
<br/>　　程大雷撇撇嘴，也不是很想搭理他。
<br/>　　“大当家，我们一起回凉州吧。”
<br/>　　“嗯，你带着华大夫回去，到蛤蟆城提我的名字。”
<br/>　　“你不和我们一起回去么？”刘发财奇怪道。
<br/>　　“我打算去一趟草原。”
