　　“哗……“听到大王子的解释之后，人群一片哗然，无数人兴奋的挥起双手。
<br/>　　大王子面带微笑的看着面前欢呼的百姓，心中却不断发出一声声的冷笑：该死的贱民，竟然死到临头都不知道，还真以打开城门跑出去就能活命，也不想想在战争没有结束之前，唐军会不会真的手下留情。
<br/>　　时间一点点过去，想要跟着一起突围的人越来越多，无数百姓聚集到了西城，等待着城门的打开。
<br/>　　而此时城外的唐军也早就接到了来自空中滑翔翼部队的通知，知道了城中的变故。
<br/>　　驻守西城的部队紧急集合了起来，在城外拉开了一道防线，只是他们的人数不是一万，而是三万。
<br/>　　时间一点点过去，半个时辰之后，君士坦丁城的西门轰然打开，在那一瞬间，无数衣衫褴褛的拜占庭人冲了出来，越过护城河，拼了命的向正西的方向逃去。
<br/>　　长时间的围困让拜占庭人心中有了一个错误念头，那就是只要离开了都城便能活命，所以在出城之后，那些拜占庭人几乎忘记了一切，叫着，笑着，庆祝着自己能够逃出生天。
<br/>　　而就在这些人跑出里许之地的时候，一声枪响将他们的美梦彻底击碎。
<br/>　　战场之上没有什么仁慈，在古代并没有什么世界国际公约，随着第一声枪响，很快城外的枪声就响成了一片，一批又一批拜占庭人倒在血泊之中，到死都没有搞清楚，为什么离开城池还是会死。
<br/>　　而此时城中想要离开的百姓基本上已经全都冲出了西门，护卫王族离开的五千骑兵也都做好了攻击的准备。
<br/>　　随着大王子一声令下，数万只马蹄轰然作响，带着一往无前的气势冲出了西门。
<br/>　　一面是大唐三万人的军队，一面是自己的骑兵，冲出城的拜占庭骑兵被夹在了中间。
<br/>　　他们手无寸铁，根本没有办法冲过唐军的防线，而身后越来越近的自己的同样也没有停下的打算，出城较晚的百姓甚至被自己人的骑兵撞飞，然后被踏成肉泥。
<br/>　　惨，太惨了，拜占庭百姓在这个时候才意识到，所谓的带他们一起离开就是一个笑话，他们只不过就是一些被用来吸引唐军注意力的炮灰罢了。
<br/>　　可是这个认知对于他们来说似乎有些迟了，唐军根本不可能停下自己的攻击，面对气势汹汹的拜占庭骑兵，停止攻击只会让他们陷入被动，甚至会被敌人逃离。
<br/>　　在这样的情况下，他们就算是心有怜悯，也只能硬起头皮进行无差别射击，力求不让任何人靠近到自己的防线。
<br/>　　而与此同时，唐军的火炮也开始发威，贞观炮的炮弹砸进拜占庭人的骑兵队伍中，掀起漫天血雨，火箭弹在拜占庭人中间肆虐，带着一条又一条生命。
<br/>　　那些骑兵虽然速度很快，但唐军的攻击实在是太犀利了，哪怕他们使尽浑身解数，将战马的速度催到极致，却依旧无法靠近到唐军防线百步之内，大唐火器名冠天下，这不是靠着气血之勇就能战胜的。
<br/>　　君士斯坦和他的几个儿子已经被吓傻了，甚至连判断战场局势都做不到，只能眼睁睁看着五千人的骑兵人数越来越少。
<br/>　　“回去，撤回去！快，快调头！“最后还是大王子反应要快一些，在五千骑兵几乎被干掉一半的时候，拉住了战马，接管了骑兵的指挥权。
<br/>　　君士斯坦这个时候也反应过来，大声吆喝着停了下来：“回城，所有人马上回城！”
<br/>　　骑兵的统领早就已经被炸死了，这个时候的拜占庭骑兵正是群龙无首的时候，听了国王的命令立刻开始调转马头向着正在慢慢关闭的城门又冲了回去。
<br/>　　老国师站在西城的城墙之上将外面发生的一切看在眼里，心中不由一阵绝望，一边吩咐人把重新打开城门，一边走下城墙，准备去迎接国王回来。
<br/>　　冲不出去，那就一起死吧，反正都城一失，一切便等于重新归零，就算是国王能够离开，想要复国也不是红口白牙能够做到的。
<br/>　　连带那些想要逃命的百姓，冲城的共有三万人左右，可是最后能够回来的人却不足三千。
<br/>　　君士斯坦这个倒霉的国王在临近城门的时候被一发炮弹砸个正着，除了在地上留下一个变了型的王冠之外，再也没有留下任何东西。
<br/>　　老国师抱着已经变了型的王冠好一阵难受，可是耐何人已经没了，就算是出去找也只能找回一些骨头，捡回来也没有任何意义。
<br/>　　大王子同样被炸的灰头土脸，还没反应过来发生了什么，就被一群人簇拥着坐到了王座之上，没有任何仪式，直接成了拜占庭的新国王。
<br/>　　如果是放在平时，大王子坐到这个位置一定会欣喜若狂，可是现在他却感觉如坐针毡，外面隆隆的炮声阵的王宫里杯盘乱响，同时也震的他肝儿颤。
<br/>　　唐军攻破城池就在眼前，突围失败的结果让城中守军完全失去了战斗下去的信念，所有人都清楚，国王是想要拿他们当垫被，自己逃走，至于说是否成功这完全是另外一个问题。
<br/>　　大王子，不，现在应该说国王陛下坐在宝座之上，看着下面已经少了一半的臣子，痛苦的叹了口气：“诸位，开城投降吧，拜占庭……完了！”
<br/>　　“陛下不可啊，城中还有粮食，百姓依旧还能战斗，万万不可轻言放弃啊。”税务官睁着眼睛说瞎话，力求挽回新国王的决定。
<br/>　　毕竟投降以后还是要让唐军进城的，而唐军只要进了城，他们这些官员和富户必然会成为唐军的目标，如果不是想让自己的财产被清剿一空，最好的办法就是死战到底。
<br/>　　刚刚登上王位的大王子如何能不知道下面的人在想些什么，可事到如今，他早已经乱了方寸，钱财之类的东西已经不被放在心上，能够保命才是他在乎的。
<br/>　　可还没等他把自己的意见表达出来，东城的方向传来一阵惊天动地的闷雷之声，接着整个皇宫都跟着颤抖起来，大殿上站着的那些官员一个个都变成了滚地葫芦。
