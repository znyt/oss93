　　苏莫缓缓的飞临了神剑山，他没有贸然的从高空中飞过去，而是飞落在了山脚下的城池之前。
<br/>　　“风雪城！”
<br/>　　苏莫伫立在城门之前，望着城门之上的三个大字。
<br/>　　这让他有些疑惑，此城与风雪有和关系？
<br/>　　不过，当苏莫走进城中，立刻了然。
<br/>　　城池之中，散发着一种冰寒冷厉的寒意，仿似空气之中，有数之不尽的雪花在飞舞。
<br/>　　“难怪叫风雪城！”
<br/>　　苏莫心中恍然，他发现城中的武者，身上大多蕴含着这种冰寒的意境。
<br/>　　这是一种道，一种剑道所带来的变化。
<br/>　　风雪城中不算繁华，但人口也不少，苏莫大步而行，没过多久便穿过了风雪城，来到了神剑山的脚下。
<br/>　　神剑山的脚下，山门高达数十丈，修建的极为气派。
<br/>　　十数名守山弟子，个个背负着长剑，自有一股凌厉的剑威弥漫。
<br/>　　苏莫心中沉吟，不知绫裳是否在终极神王殿？自己直接求见，会不会有些不妥？
<br/>　　还有就是，绫裳回到终极神王殿之后，应该不会再向以前那样隐藏了吧！
<br/>　　沉吟了片刻，苏莫大步来到了山门之前。
<br/>　　“来着何人？”
<br/>　　山门前，为首的一位黄衣弟子，立刻拦在了苏莫身前，大声询问。
<br/>　　“在下求见绫裳神王！”苏莫微微抱拳，沉声说道。
<br/>　　“见绫裳宫主？”黄衣弟子闻言，面色惊疑不定的打量着苏莫。
<br/>　　终极神王殿，目前有四大剑宫，分别是风雪剑宫、毁灭剑宫、紫霄剑宫和绫裳剑宫。
<br/>　　绫裳神王，自然就是绫裳剑宫宫主。
<br/>　　只不过，绫裳神王于数百年前陨落，结果却并没有真正死亡，在二十多年前才回到神剑山。
<br/>　　而这个消息，在终极神王殿，算是比较隐秘之时，并没有外传。
<br/>　　但是，眼前之人居然知晓，看来真的认识绫裳宫主。
<br/>　　“报上名讳，我为你通传！”黄衣弟子沉声道。
<br/>　　“青璇！”苏莫微笑着说道，他并未报上真名，但是，青璇这个名字，只要绫裳听到，就会知晓是他。
<br/>　　“稍等！”
<br/>　　黄衣弟子点了点头，便进入了山门之中，向神剑山上而去。
<br/>　　苏莫便静静的等待了起来，一边等待，他一边打量神剑山。
<br/>　　神剑山上建筑很多，延绵而上，而在上方类似于剑格的位置上，建筑最多，宫殿成群。
<br/>　　尤其是神剑山的最顶端，坐落着一片天宫，虽然不大，但却有天上宫阙的气势。
<br/>　　苏莫约莫等了半刻钟，黄衣弟子回来了。
<br/>　　“跟我来吧！”
<br/>　　黄衣弟子招呼一声，便再次向神剑山上而去，苏莫不敢怠慢，急忙跟上。
<br/>　　神剑山很高，而且极为陡峭，黄衣青年不急不缓，苏莫也只能跟在对方的身后。
<br/>　　片刻之后，苏莫跟着黄衣弟子，来到了神剑山的上方。
<br/>　　这里是一处环形的平台，犹如是一柄宝剑的剑格护手，较为宽敞。
<br/>　　四大剑宫，均是坐落在此处，各有宫门。
<br/>　　黄衣弟子带着苏莫，进入了绫裳剑宫之中，最后停在了一座剑塔之前。
<br/>　　此塔不大，高约三十丈，古朴无华。
<br/>　　黄衣弟子将苏莫带到塔门之前，便转身离去，只留下了苏莫一人。
<br/>　　苏莫看了看紧闭的塔门，眉头微皱，绫裳在塔内吗？
<br/>　　“上来吧！”
<br/>　　就在此刻，一道悠悠的女音，在苏莫的耳畔响起，正是绫裳的声音。
<br/>　　随后，塔楼的大门，无声的打开了。
<br/>　　苏莫闻言，便点了点头，走进了塔楼之内。
<br/>　　塔楼只有五层，苏莫顺着塔内的阶梯，直接就来到了顶层。
<br/>　　塔楼的顶层，是一间宽敞的修炼室，苏莫如愿以偿的见到了绫裳。
<br/>　　此时的绫裳，穿着一声海蓝色的长裙，容颜和当初没有丝毫的变化。
<br/>　　修炼室之中，苏莫和绫裳相对而坐。
<br/>　　“苏莫，你能来到这里，可喜可贺！”绫裳目视着苏莫，朱唇轻启，声音无波无澜。
<br/>　　苏莫能来到这里，没有出乎她的预料，只是，比她预料的要早了一些年。
<br/>　　打量了苏莫一番，见苏莫修为已达二品低级，绫裳心中略有感慨。
<br/>　　苏莫的修炼速度，着实够快，比她都差不了太多。
<br/>　　“你说过，神域能救活芊寻月！”苏莫同样凝视着对方，声音低沉。
<br/>　　绫裳闻言，沉默了下来，少倾之后，微微点了点头，道：“以你的性格，来此之前，应该已经有所了解了吧！”
<br/>　　“嗯，听说紫霄神王拥有让人死而复生的本领！”苏莫点了点头。
<br/>　　“紫霄神王的确是有这个能力，不过，却也不是能轻易的让人死而复生，你不要抱太高的希望！”绫裳叹道。
<br/>　　她对于紫霄神王，实在是太熟悉了，对方的本领，她也很清楚。
<br/>　　但是，生死元神和生死法则，却也不是能拯救一切陨落的生灵，毕竟，紫霄神王的修为和法则，还没有修炼到巅峰。
<br/>　　她此言，是让苏莫有心理准备，免得若是失败了，会很失望。
<br/>　　当然，她同样希望，芊寻月能活过来，毕竟，她曾为人弟子。
<br/>　　“我明白！”
<br/>　　苏莫点了点头，她明白对方的意思，随即抱拳道：“多谢你了！”
<br/>　　“谢就不必了！”
<br/>　　绫裳微微摇了摇头，随即她绝美的脸庞之上，罕见的露出了一丝笑容，道：“芊寻月是我的师尊，救活她也是我的愿望。”
<br/>　　苏莫闻言，心中有些感触，对方能说出此言，让他心中欣慰。
<br/>　　最起码，对方还没忘记，其不仅是绫裳，也是宏青璇。
<br/>　　“这么多年了，你的修为居然还没有恢复？”苏莫打量了绫裳一眼，对方的修为，只比他略高，二品中级的修为。
<br/>　　这远远低于他的预料，在苏莫看来，对方如今的修为，应该有四品、乃至五品，才最合理。
<br/>　　“天赋使然！”绫裳微微的叹了口气，这一世的肉身，比之上一世，天赋差的太多太多了。
<br/>　　上一世，她曾经乃是万古天骄，天赋在整个神域都是屈指可数的存在。
<br/>　　但是，这一世，元神只是普普通通的青莲元神，而体质只是纯阴之体。
<br/>　　这种天赋，在苍穹世界都只能算是中上，而在神域，简直不值一提。
<br/>　　若不是她前世是神王，凭这一世的天赋，一辈子都不可能成神。
<br/>　　“天赋？”苏莫闻言恍然，青璇的天赋的确是一般，看来是天赋限制了对方修为的恢复。
<br/>　　不过，他目前而言，倒是没有什么元神，能给予对方。
<br/>　　“紫霄神王目前不在神剑山，你且等待一段时间！”绫裳道。
<br/>　　“好！”苏莫点了点头。
<br/>　　“你修炼剑道，也有剑类元神，而终极神王殿便是以剑道闻名神域，既然来了，那就在此地获取你的机缘！”绫裳沉吟了一番，轻声说道。
<br/>　　“什么机缘？”苏莫好奇的问道。
