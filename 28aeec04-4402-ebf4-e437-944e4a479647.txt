        “……”童筱颖继续沉默着。
<br/>
<br/>        连华生双眼注视着前方，手握方向盘，接着说道：“他以前总是会执着于他自己的事情，不会因为任何人而放弃。但现在，他却为了你选择了释怀。或许是因为失去后才懂得珍惜的缘故，但更多的是他愿意为了你而去改变自己。”
<br/>
<br/>        言下之意，是他也希望她童筱颖，愿意为了傅夜沉而去改变自己。
<br/>
<br/>        两个相爱的人，在一起磨合，不是包容，就是为对方改变自己。
<br/>
<br/>        连华生希望童筱颖能明白。
<br/>
<br/>        良久后，童筱颖突然开口道：“谢谢你。”
<br/>
<br/>        连华生下意识地瞥了童筱颖一眼，嘴角微微扬了扬。
<br/>
<br/>        布置婚礼的这件事情，傅夜沉并未马虎，连婚礼策划都出了好几个方案，派人送到了连华生的手中，连华生再转交给了童筱颖供她选择。
<br/>
<br/>        童筱颖最终挑选了“海边婚礼”的策划案。
<br/>
<br/>        就连设计师设计好做出的婚纱，也是上门来供她选择。
<br/>
<br/>        其实这些准备事宜，傅夜沉很早前就做好了，只是一直没有机会展现出来，现在这个时机，再好不过了。
<br/>
<br/>        一个星期后，婚礼如期举行。
<br/>
<br/>        童筱颖坐在连华生为她布置好的待嫁的闺房之中，等着化妆师过来为她化新娘妆。
<br/>
<br/>        化妆师来了两个女人，一进门就开始开化妆箱忙碌起来。
<br/>
<br/>        就在童筱颖放松警惕的时候，其中一个女人突然用一块手帕，捂住了童筱颖的口鼻，在童筱颖还未来得及挣扎的时候，就已经晕了过去。
<br/>
<br/>        随后，女人给外边的人打了通电话，说自己漏了眉笔，需要自己的助手送过来。
<br/>
<br/>        十分钟左右，屋内又进来一个戴着口罩的女人。
<br/>
<br/>        当女人摘下口罩后，露出了一张与童筱颖一模一样的脸。
<br/>
<br/>        苏漫雪打了个响指，另外那两个女人立即将童筱颖藏在了放置婚纱的布箱子里。
<br/>
<br/>        紧接着，这两个女人开始有条不紊地替苏漫雪化新娘妆。
<br/>
<br/>        从傅昼景那儿得知，童筱颖和傅夜沉将在今天对外补一次正式的婚礼时，苏漫雪心里便生了这一计策。
<br/>
<br/>        一个星期前，傅夜沉忙于筹备婚礼，她就在暗中忙于如何掉包新娘。
<br/>
<br/>        正巧，傅昼景的那个名义上的老婆季思妍不打算放过她，她便把童筱颖交出，让季思妍整整童筱颖。
<br/>
<br/>        所以，当童筱颖醒过来的时候，看到的人不是别人，正是对苏漫雪恨之入骨的季思妍。
<br/>
<br/>        季思妍就坐在她对面的一把木椅上，手中还把玩着一把水果刀。
<br/>
<br/>        童筱颖对这个女人的印象很模糊，感觉自己认识她，好像又不认识她。
<br/>
<br/>        她记起了部分以前的事情，同时又忘了部分以前的事情。
<br/>
<br/>        但是一种女人的直觉告诉她，面前的这个女人，跟傅昼景有关系。
<br/>
<br/>        “苏漫雪，你当你的女明星，拍你的电影，演你的戏，不好吗？你非得要去勾引我的阿景做什么？”季思妍红了眼，说话的语气里透着一股阴森。
<br/>
<br/>        童筱颖四下看了看，从屋子的结构来看，感觉这里应该是一幢洋房的地下室。
<br/>
<br/>        她的手脚被绑在了椅子上，嘴巴上被一块胶布状的东西给封住。
<br/>
<br/>        不能言语也不能动弹，只能眼睁睁地看着季思妍拿着刀子在面前晃来晃去。
<br/>
<br/>        “你勾引阿景也就算了，为什么你还要怀上他的孩子呢？你想做什么啊？生下孩子来跟我争吗？”季思妍龇牙咧嘴地质问道。
<br/>
<br/>        童筱颖压根就没心思听她在这儿废话，双手背在椅背后不停地来回摩擦着试图挣脱，一心只想如何从这里逃出去。
<br/>
<br/>        “不如，我毁了你这张跟童筱颖长得一模一样的脸，让阿景看看，你对他来说还有什么意义？”季思妍突然变得凶神恶煞起来。
<br/>
<br/>        就在她举起刀子，准备向童筱颖的脸蛋挥去的时候，突然“乓”地一声，那边的木门突然被人踢开，重重地撞到了一旁的墙壁上。
<br/>
<br/>        “阿妍，住手！”傅昼景疾步冲过来，从后抱住季思妍，并夺走了她手中的刀子。
<br/>
<br/>        季思妍随之揪住傅昼景的衣领，并对他拳打脚踢起来：“你这个负心汉，你为什么要这么对我？我到底哪里对不起你？我为你付出了这么多！你为什么还要包养苏漫雪？还让她怀上你的孩子？你这样做，对得起我和尚轩吗？”
<br/>
<br/>        “就算我对不起你，你也不必为了我做这种要坐牢的傻事！”傅昼景对季思妍呵斥道。
<br/>
<br/>        如果不是他发现她这几天的异常安静，他真没料到，她会花钱请人绑架苏漫雪。
