　　大战在持续，第五城、第六城和第八城之人，在人数和实力都没有优势之下，已经拉开了落败的序幕。
<br/>　　片刻功夫，三城之人便只剩下了二三十人，而且各个遭到围攻，险险环生。
<br/>　　这二三十人，哪里还敢恋战，纷纷逃回了自己所在的城池。
<br/>　　而第一城、第二城和第三城之人，经过这么久的战斗，也只剩下了八、九十人左右。
<br/>　　“诸位，现在第五城和第六城、第八城，都没有多少人了，我们应该个个击破！”
<br/>　　一众第一城、第二城和第三城之中，聚拢在一起，公孙长绝沉声说道。
<br/>　　“你别忘了第七城和第九城！”叶凌天提醒道，现在，他们三城的实力下降了一大截，而第七城和第九城还处在巅峰，一直没有动手。
<br/>　　所以，他们必须提防起来，不然只会成为螳螂，被黄雀一口吃下。
<br/>　　“这样吧，挑一部分实力强的人，去将第五城、第六城和第八城个个击破，留下一部分人手，以防第七城和第九城之人！”阎魔鬼圣沉声说道。
<br/>　　“嗯，这样比较稳妥！”
<br/>　　“就这么办！”
<br/>　　众人纷纷点头，虽然他们现在人手不多了，但是，每个城中还有一部分坐镇之人。
<br/>　　而且，之前很多受了重伤的人，都返回了城中，倒是也不惧第七城和第九城。
<br/>　　随即，他们立刻分出了五十余人，由叶凌天和阎魔鬼圣带领，开始去攻击第五城。
<br/>　　至于余下的三十余人，则负责压阵，以防其它城池之人。
<br/>　　当然，三十余人太上，又从第一城、第二城和第三城之中，抽调了三十人，共计达到了六十余人。
<br/>　　在叶凌天等人的攻击之下，第五城没过多久，护城大阵便被攻破了。
<br/>　　第五城人数不多，总共只剩下了二十余人，完全不是叶凌天等人的对手，在抵抗了片刻之后，便被彻底击溃，城中的阵基被摧毁，所有人遭到了淘汰。
<br/>　　其它城池之中，并未前来插手，第六城和第八城之人并未出手，虽然他们知道会被个个击破，但是，已经无力反抗了。
<br/>　　至于第七城和第九城，依旧毫无动静。
<br/>　　随后，叶凌天等人，又开始攻击第六城的护城大阵。
<br/>　　第七城之中，涂赢和众多的人马凝视着第六城，个个都在等待出手的时机。
<br/>　　“涂赢，我们什么时候出手？”一名高大的天才，低声向涂赢问道。
<br/>　　“再等等！现在他们的实力，依旧强于我们！“涂赢沉声说道，虽然第一城和第二城，第三城折损了大量的人手，但是，三城之人联合之下，依旧要远远强于他们。
<br/>　　主要是，他们第七城之中，没有什么厉害的人物，都是普通的万古天骄。
<br/>　　虽然他们现在不出手，第六城和第八城会被击溃，但是那个时候，第一城和第二城、第三城之人，还会折损不少，到时候两方的实力就相差不大了。
<br/>　　“那等会之后，我们是要击败这三城吗？”有人问道。
<br/>　　“不，我们和他们联合！”涂赢摇了摇头，面色凝重的道：“与他们联手，击溃第九城！”
<br/>　　“他们会合我们联合吗？”
<br/>　　“到时候，他们的实力不会比我们强多少，他们会知道怎么选择！”涂赢沉声说道，第九城才是大患，想必谁都很清楚。
<br/>　　战斗在持续，第五城被击败之后，没过多久，第六城也被击败，而紧接着是第八城。
<br/>　　叶凌天带领的人，折损了一小部分，但终归是大获全胜，将第五城和第六城、第八城全部击溃。
<br/>　　至此，神昆小世界中的九座城池，只剩下了第一城、第二城、第三城、第七城、第九城。
<br/>　　叶凌天败了三城之后，便和公孙长绝等人汇聚在了一起，浩浩荡荡约莫九十余人。
<br/>　　“诸位，接下来怎么行动？”叶凌天向众人问道。
<br/>　　“我们现在的人手已经不多，而还有第七城和第九城，不能大意！”阎魔鬼圣道。
<br/>　　轰！
<br/>　　就在此刻，远处的虚空一阵轰响，陡然炸开，一道人影从虚空深处飞了出来。
<br/>　　“苏莫！”众人一见这道人影，顿时惊讶，苏莫出来了，那就是逃出了古魔太子之手！
<br/>　　果然，苏莫已经强大到，连古魔太子都无法奈何了！
<br/>　　众人面色各异，对于公孙长绝等一些其它势力之人，他们自然是希望苏莫不要回来，或者是被古魔太子镇压。
<br/>　　毕竟，苏莫的实力太强，最后肯定要和他们争夺最终的胜利。
<br/>　　但是龙族之人，或者是人王宫之人，见到苏莫返回，均是大松了一口气。
<br/>　　飞出虚空深处，苏莫目光环视四周，看到第五城、第六城、第八城的护城大阵均已不在，他心中明了，他们已经暂时胜了。
<br/>　　只不过，龙腾并没有回来，因为龙腾所在的第五城被攻破，自然也被强行传送了出去，所以只有他一人回来了。
<br/>　　“诸位，情况怎么样？”苏莫飞身来到众人身前，沉声问道。
<br/>　　“目前，只剩下了第七城和第九城，均是毫无动静。”公孙长绝沉声说道。
<br/>　　对于龙腾和古魔太子没有回来，他们并没有意外，两人肯定都被淘汰了。
<br/>　　“我们还有多少人手？”苏莫目光凝视第七城和第九城，人手是关键，这两城之中可是有两百余人。
<br/>　　“总共不足一百五！”叶凌天道，这已经是包括他们三座城池之内留手的人了。
<br/>　　所有人加起来不到一百五，人数处于弱势，好在他们强者众多，战力应该有优势。
<br/>　　“这么少！”苏莫闻言皱眉，人手居然损耗了一半以上了。
<br/>　　嗡~~~
<br/>　　就在此刻，一直毫无动静的第七城之中，一道挺拔的身影，大步的走了出来。
<br/>　　“苏莫，好久不见！”涂赢遥望着苏莫，微笑着说道，他乐意看到苏莫回来。
<br/>　　他和苏莫两人，从神路到神域，算是不打不相识，接下来也将互为对手，不过在此之前，要先解决第九城。
<br/>　　不过，苏莫不愧是苏莫，能从古魔太子的手中逃回来，当真没有辱没了轮回分身的名头。
<br/>　　“涂赢，你是想和我们联合？”苏莫打量了涂赢一番，沉声问道。
<br/>　　“不错，只有我们联手，才能击败第九城！”涂赢重重的点了点头。
<br/>　　“好，我同意！”苏莫没有犹豫，立刻就同意了。
<br/>　　第九城是大敌，他们联合起来才是最好的办法，不然无法破局。
<br/>　　其他人亦是纷纷点了点头，同意这个提议。
