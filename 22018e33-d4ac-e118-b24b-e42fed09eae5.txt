　　小岛上。
<br/>　　苏莫盘膝坐在一块大石上，眼帘低垂，心神沉入体内，查看那魔魂的情况。
<br/>　　那魔魂潜伏在他的吞噬武魂之中，没有任何动静，但他知道此魔魂并没有死。
<br/>　　呼！
<br/>　　轻呼一口气，苏莫心中暗叹，希望此魔魂不要再搞出什么乱子！
<br/>　　不过，此魔魂极强，本来都要夺舍他了，为何魔念突然衰弱，后继无力了呢？
<br/>　　思索了片刻，他也没有想到所以然，便不再多想。
<br/>　　“苏兄，这些魔甲人真是穷啊！”
<br/>　　聂乾和韩盈坐在苏莫不远处，聂乾清点完自己获得的战利品，无奈的说道。
<br/>　　“魔甲人被困在魔云海，哪有什么资源财富！”苏莫摇头道。
<br/>　　按照神武录的记载，魔甲人一族以前也是非常强大的族群，后来被人族杀的几乎灭族，残部被困在了魔云海。
<br/>　　“已经不错了！这种魔晶虽然远不如灵晶，但也价值极高！”韩盈笑着说道。
<br/>　　随即，苏莫手掌一番，也开始查看他获得的战利品。
<br/>　　这些魔甲人的确很穷，他总共获得了二十余枚储物戒，其内除了三十余件帝级兵器之外，只有不到两百块魔晶。
<br/>　　不过，魔甲人虽穷，但廖云剑却是并不穷，而且是相当的富有。
<br/>　　廖云剑的储物戒中，各种帝级兵器十余件，小灵晶足有三百块，其中杂七杂八的宝物也有不少。
<br/>　　不过，其中有四枚玉简，却是吸引了苏莫的兴趣。
<br/>　　这四枚玉简是四种榜单，分别是：太古万族榜、太古武魂榜、太古异体榜、太古大道榜。
<br/>　　太古万族榜的玉简，苏莫本来就有，而另外三种榜单他只听说过太古武魂榜，太古异体榜和太古大道榜以往根本没有听说过。
<br/>　　他首先了查看了一番太古武魂榜的玉简，发现其内记载了一万种武魂，而且基本上都是他从未见过的武魂。
<br/>　　这些武魂排名，不分武魂和战魂的区别，因为战魂本来就是武魂进化而成。
<br/>　　廖云剑的融灵兽战魂也在其中，不过却是排在第九千九百多名。
<br/>　　第一名，名为‘鸿蒙紫气武魂’！
<br/>　　第二名，太初之魂！
<br/>　　第三名，万界圣魂！
<br/>　　第十九名，吞天武魂！
<br/>　　第二十名，通天神树武魂！
<br/>　　第二十一名，九龙弑天武魂！
<br/>　　……
<br/>　　苏莫一一查看这太古武魂榜上的武魂，很快便将一万种武魂都查看了一遍。
<br/>　　不过，他却是并未发现自己的这种武魂，太古武魂榜上虽然有好几种拥有吞噬能力的武魂，但与他的武魂并不相同。
<br/>　　比如，第十九名，吞天圣魂！
<br/>　　第九十三名、吞灵武魂！
<br/>　　第三百九十八名，七彩吞天蟒武魂！
<br/>　　这些武魂虽然都有吞噬的能力，但按照太古武魂榜上的介绍，与他的吞噬武魂完全不同。
<br/>　　“吞天圣魂！”苏莫呢喃自语，这排名第十九名的武魂，居然能吞噬天地万物，号称无所不吞，无所不噬！
<br/>　　这就恐怖了，他的吞噬武魂，也就只能吞噬一些魂类和一些灵气、血液、血脉等能量体而已。
<br/>　　相比较而言，单论吞噬的能力，他的吞噬武魂远远不及这吞天圣魂。
<br/>　　不过，他的武魂却是有融合的能力，无论是融合武魂还是血脉体质，这是吞天圣魂所不具备的能力。
<br/>　　就算如此，这号称能吞天噬地的吞天圣魂，也是无比的恐怖。
<br/>　　而如此恐怖的吞天圣魂，居然才排在太古武魂榜第十九名，当真让人费解。
<br/>　　至于排名在其之上的那些武魂，比如第一名的鸿蒙紫气武魂、第二名的太初武魂等等，苏莫完全没有概念。
<br/>　　而且，玉简上对于这些排名前列的武魂，介绍的非常少，只说威势气吞万界，威力霸绝环宇，至于为何如此强大？有什么能力？根本没有介绍。
<br/>　　“苏兄，你在看太古武魂榜？”聂乾闻听苏莫的自语，好奇的问道。
<br/>　　“嗯！这些武魂的确强大，真是闻所未闻！”苏莫点了点头，心有感慨的说道。
<br/>　　“是啊！”
<br/>　　聂乾赞同的点头，随即眸露惊奇的说道：“其实，你的武魂肯定也能名列太古武魂榜！”
<br/>　　聂乾心中非常的好奇，好奇苏莫的武魂到底是什么武魂？
<br/>　　这几****也仔细考虑了一下，并不是太古武魂榜上的任何一种武魂，但绝对比太古武魂榜上大多数武魂更强大。
<br/>　　“并没有我这种武魂！”苏莫摇头道。
<br/>　　“苏兄有所不知，这太古武魂榜，只收录出现过的超级武魂，或许你这种武魂在这无尽宇宙中是首次出现，自然便不会有排名！“聂乾笑着说道。
<br/>　　“也许吧！”苏莫叹道，对方说的也有道理，他这种武魂若是首次出现，自然不可能有排名。
<br/>　　不过，排名倒是无所谓，他主要是想了解自己的武魂。
<br/>　　毕竟，这么多年了，他吞噬武魂的能力还在不断的开发，他很想了解自己武魂所有的一切。
<br/>　　“苏兄，你的武魂和我们学府神子的武魂倒是很像！”韩盈开口说道。
<br/>　　“神子？”苏莫闻言一怔，沉声问道：“他是什么武魂？”
<br/>　　对于神武学府唯一的神子，苏莫了解的不多，只知道是学府地位最高的弟子。
<br/>　　“神子的武魂是吞灵武魂，能吞噬一起能量，太古武魂榜排名第九十三名，当然，现在早已进化为战魂，吞灵战魂！”韩盈道。
<br/>　　苏莫闻言讶然，神武学府神子的武魂，居然是排名第九十三的吞灵武魂！
<br/>　　这倒是和他的武魂颇为类似，只不过吞灵武魂并不能吞噬血脉和他人的武魂，比之他的吞噬武魂，应该是有所不如。
<br/>　　“聂兄、韩姑娘，有一事需要两位帮忙。”沉吟了一番，苏莫开口说道。
<br/>　　“何事？”两人疑惑的问道。
<br/>　　“我武魂的能力，还希望两位守口如瓶！”苏莫目光灼灼的望着聂乾和韩盈。
<br/>　　他吞噬武魂的能力，还是不宜外传，免得惹出不必要的麻烦。
<br/>　　事实上，在从阎魔地宫中出来之前，苏莫都有将两人灭口的打算。
<br/>　　但他这个念头一起，便立刻打消了。
<br/>　　因为这两人的人品还不错，颇为重义，是值得相交的朋友，他不想杀这两人。
<br/>　　“苏兄请放心，你对我有救命之恩，我绝对不会泄露分毫！”聂乾面色郑重的说道。
<br/>　　“我也是，绝对不会泄露苏兄武魂的信息！”韩盈亦是点了点头。
<br/>　　“好，我相信两位！”苏莫面露微笑。
