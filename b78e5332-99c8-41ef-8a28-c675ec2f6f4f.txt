<br/>				“让我解了他的奴隶契约？原来，城主也知道他已经是我的奴隶了？”凤九似笑非笑的睨了城主一眼。<br/>　　<br/>　　听到这话的城主，看了那低着头的儿子一眼，微顿了一下，这才对凤九道：“今天的事，是小儿荒唐了，还请凤公子不要跟他计较，毕竟他是我的儿子，若是沦为公子的奴隶，这传出去，我岂不是成了外面的笑话？所以还请凤公子高抬贵手，只要凤公子肯解了他的契约，我定会补偿凤公子。”<br/>　　<br/>　　凤九轻笑着：“城主让我如何相信解了契约之后我不会被杀？”<br/>　　<br/>　　闻言，城主神色微动，他深深地看了凤九一眼，道：“凤公子请放心，只要解了小儿的奴隶契约，由我做主，这件事情就此结束，定不会为难凤公子半分。”<br/>　　<br/>　　凤九笑了笑，摇了摇头：“不，不够，这样我可不敢解开他的契约。”她笑了笑，站了起来，道：“这地方有天地规则，要我解开他的契约也可以，只要城主以及少城主承诺解开契约之后，不会找我麻烦，那么，我可以解开契约。”<br/>　　<br/>　　她的声音微顿，瞥了那一旁的凌天羽一眼，又对城主道：“不过，在此之前，我有一事还请城主可以告知。”<br/>　　<br/>　　听到这话的城主眉头微拧，他看了凤九半响，最后问：“凤公子想知道什么？请说。”<br/>　　<br/>　　凤九站了起来，缓步走到他的面前，道：“我想知道，蓬莱仙岛的登仙梯在哪里？”<br/>　　<br/>　　城主一怔，脸上尽是错愕之色：“什么？登仙梯？”<br/>　　<br/>　　他的目光微动，盯着凤九看了良久，问：“凤公子怎么想找登仙梯？”<br/>　　<br/>　　“城主只需告诉我，在哪里可以找到登仙梯就行。”她缓声说着。<br/>　　<br/>　　闻言，城主笑了笑，道：“其实，这登仙梯据说是在蓬莱仙岛不错，但是，在这里的人几乎没人看见过。”<br/>　　<br/>　　他的声音一顿，笑道：“据闻，登仙梯是可以通向另一片天地的桥梁，也是验证一个人实力的地方，只不过，我在这里这么久，却也是从没见过这登仙梯的，更不知道登仙梯是在蓬莱仙岛的何处。”<br/>　　<br/>　　“连城主也不知登仙梯在哪？”她有些不信。<br/>　　<br/>　　“呵呵，不是我不告诉凤公子，而是我确实不知道，我只听说过那登仙梯可以通往另一片天地，也知道登仙梯上设有重重考验，稍有不慎一身修为就会作废。”<br/>　　<br/>　　他看着凤九，目光带着莫名意味：“据闻，凤公子是刚到城里来的，怎么会知道逢莱仙岛有登仙梯一事？莫不是有什么高人指点？若真如此，那么，这位高人难道没有告诉凤公子，这登仙梯所在何处吗？”<br/>　　<br/>　　见他神色不是作假，凤九心下不由沉思：难道，连他也不知道这登仙梯在哪里？连这里的城主也不知那登仙梯在哪里，那么那天机老人怎么又会知道？而且还让她来这里找什么登仙梯，如今四处打听也没人知道那地方位于何处？她又要往何处去寻呢？<br/>
