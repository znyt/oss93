　　巫人王的强大，超出了苏莫的预料。
<br/>　　不使用战魂的话，他就算动用混天魔域，也完全落入下风。
<br/>　　虽然他修炼了圣级上品的混天魔功，但毕竟境界较低，才第一层的境界，威力也就和尊级上品功法差不多。
<br/>　　眼见巨龙之爪再次抓来，苏莫急忙调动体内魔力，并且催动了郭剑青的那柄长剑战魂，再次一剑斩击而出。
<br/>　　动用了战魂之力，苏莫的气势大增，剑威暴涨足有五成。
<br/>　　咻！
<br/>　　凌厉的剑气冲光而起，锋芒耀眼，撕裂苍穹，狠狠的斩在了巨龙的龙爪之上。
<br/>　　轰隆隆！！
<br/>　　一震爆响，龙爪被斩碎，剑光斩过，巨龙庞大的身躯也被撕裂开来，而后彻底崩溃。
<br/>　　“九龙葬天！”
<br/>　　巫人王的怒喝声再次响起，攻击如同排山倒海，不给苏莫丝毫喘息之机。
<br/>　　只见其再次一拳击出，恐怖的拳茫如同九天神光，化为了九条庞大的巨龙，咆哮着向苏莫飞扑而来。
<br/>　　这九条巨龙的威势，每一条都比之前的更为强大，磅礴的龙威铺天盖地，强大的力量奥义席卷苍穹，威势一时无两。
<br/>　　而且，九条巨龙的气机彼此相连，更增加了此击的威力。
<br/>　　苏莫见此，脸色凝重不已，这巫人王不愧是巫族绝世天才，根本不是巫族那些普通天才可比。
<br/>　　对方的这武技，也是绝对强大，估计最低都是尊级下品武技。
<br/>　　“灭！”
<br/>　　苏莫不敢硬挡，一边身形后退，一边连忙出剑，剑光闪烁，不断的斩击。
<br/>　　这一次，他除了吞噬战魂之外，几乎爆发了这具肉身的最强战力。
<br/>　　体内魔力运转到了极致，不仅催动了长剑战魂，还催动了十几种剑类武魂。
<br/>　　反正都是剑之魂，在他催动之下，气息完全一样，就算大祭司等人也不会发现异样。
<br/>　　就算发现了什么不同寻常之处，最多发现他剑之战魂的威力强大的过分而已！不会认为他有多种剑魂！
<br/>　　动用了十几种剑武魂之后，苏莫身上剑威再次暴增，形成了强大的剑势，仿若他整个人就是一柄屹立在天地之间的神剑。
<br/>　　咻咻咻！！
<br/>　　无数漆黑如墨的剑气，裹挟滚滚魔浪，汹涌澎湃，铺天盖地的向九条巨龙袭杀而去。
<br/>　　轰！轰！轰！！
<br/>　　一声声惊天爆响，震动了整片星空，巨龙纷纷被斩中，而后剑气与巨龙发生了猛烈的爆炸。
<br/>　　汹涌的气浪席卷九天，森然的魔气逆冲苍穹，方圆万里之内尽皆毁灭，化为了一片虚无之地。
<br/>　　大祭司等观战之人，个个双眸微挑，这巫尘的实力真是遇强则强啊！这是什么剑之战魂，威力居然如此之大！
<br/>　　嗖！
<br/>　　苏莫斩灭了九条巨龙之后，身形不退反进，直接向着巫人王冲了过去。
<br/>　　他已经有了对付巫人王的办法，在不动用吞噬战魂的情况下，打败巫人王。
<br/>　　当然，具体能否击败对方，还需要一试才能知道。
<br/>　　若是不动用吞噬战魂就能击败巫人王，那真是太好不过了，暴露的风险会降到最低。
<br/>　　“找死！”
<br/>　　见苏莫不再后退，反而向自己冲了过来，巫人王眸中顿时爆射出森然的笑容。
<br/>　　吼！
<br/>　　而后，他的背后再次有虚影升腾而起，是一只体长达十丈的黑色猛虎。
<br/>　　这猛虎全身漆黑如墨，身上毛发如钢针，巨大的虎瞳呈现血红之色，极为恐怖。
<br/>　　这种黑虎名为天魔虎，亦是一种天地异兽，比之蛟龙还要强大的存在。
<br/>　　而天魔虎战魂，也是名列太古武魂榜之上的超强战魂，排名二千七百九十五名。
<br/>　　至此，巫人王的两种战魂全部释放了出来，分别是紫蛟战魂和天魔虎战魂。
<br/>　　两种战魂全部是兽类战魂，对于攻击力均有强大的加持效用。
<br/>　　“死吧！”
<br/>　　巫人王厉喝一声，背后的紫蛟战魂和天魔虎战魂齐齐咆哮，他双拳齐出，两道浩大的拳茫瞬间化为了一龙一虎，向苏莫飞扑而且。
<br/>　　这一龙一虎，庞大无比，凶威盖世，气势如山崩海啸，引动狂风席卷天地。
<br/>　　昂！吼！
<br/>　　一龙一虎呈夹击之势，瞬息便临近了苏莫。
<br/>　　苏莫顿感头皮发麻，这一击威力实在太强了，恐怖的威势，几乎让的他的浑天魔域都开始溃散了。
<br/>　　不敢硬挡，苏莫身形连连闪烁，急速向一边躲避。
<br/>　　巨龙从苏莫身边飞过，并没有击中他，但那巨大的猛虎却是利爪一探，直接向着他的脑袋上罩来。
<br/>　　轰隆隆！
<br/>　　利爪还未击中苏莫的脑袋，强大的威势便引动虚空一震爆炸，恐怖的气机直接压在了苏莫的心头。
<br/>　　“不好！”苏莫心中一惊，电光火石之间他不再攻击而是全力防御。
<br/>　　唰！
<br/>　　一块硕大的黑色盾牌出现在了他的手中，他立刻举盾封挡。
<br/>　　这块盾牌，便是斩杀赵城得来的战利品。
<br/>　　此盾虽然没有达到尊级的级别，但品质极佳，防御力超强。
<br/>　　下一息！
<br/>　　轰！
<br/>　　一声爆响，猛虎的利爪狠狠的击在了黑色盾牌之上，顿时如同陨石砸落大地，天地为之一震。
<br/>　　苏莫的身躯更是受到了巨大的震荡，如遭重击，整个人直接被虎爪拍飞了下去。
<br/>　　嗖！
<br/>　　苏莫身形急速下坠，仿若划破星空的流星，瞬息千里，眨眼间便飞到了万余里之外。
<br/>　　“哈哈！巫尘，你必败无疑了！”
<br/>　　巫人王张狂的笑声响起，整个人身形一动，急速向苏莫冲了过来。
<br/>　　他完全占据上风，轻易就将巫尘完全压制，一切都在他的掌控之中。
<br/>　　苏莫努力稳住身形，全身气血沸腾，逆冲而上，忍不住喉头一甜。
<br/>　　他的手臂血肉都被震裂了，露出森森白骨，鲜血潺潺而流。
<br/>　　望着急速冲来的巫人王，苏莫立刻就想动用吞噬战魂，但他还是忍住了。
<br/>　　不到生死存亡之际，绝对不能动用吞噬战魂，一旦露出破绽，一切将前功尽弃。
<br/>　　不仅吞噬战魂不能用，其它吞噬得来的战魂都不能用。
<br/>　　大祭司是何等人物？何等眼力？他若是动用了太多的战魂，就算不释放出战魂，也逃不过对方的法眼。
<br/>　　所以，为了安全起见，他不能太夸张，此战他最多还能动用一种战魂！
<br/>　　这一种战魂不是吞噬战魂，也不是巫烈的烈焰石战魂，更不是赵城的金刚猿战魂，而是他才得到不久的——梦魔战魂！
