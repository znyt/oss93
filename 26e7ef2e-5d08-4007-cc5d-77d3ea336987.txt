　　第542章
<br/>　　徐州，琅琊城。
<br/>　　程大雷和刘发财牵着牛，随着行人一起入城。难得的是，二人入城的时候，竟然没遭到城门守军的刁难。程大雷事先准备好的钱袋，也没有派上用场。
<br/>　　“琅琊城是诗书礼乐之乡，在这里家家路不拾遗，夜不闭户。”刘发财脸上荡漾着异样光彩，似乎来到琅琊城，他的道德底线也蹭地一下提高了。
<br/>　　程大雷把钱袋收回怀里，牵牛走在大街上。只见街道两旁商铺鳞次栉比，男人宽袍大袖，女人锦衣华服，人人一脸和善。
<br/>　　刚进城的时候，程大雷还担心自己这幅相貌暴露了行迹，后来却发现，这担忧完全是多余的。琅琊城比他怪的人大有人在。凶神恶煞的江湖莽汉，一身戎装的军人，金发碧眼的海外商人，还有充满异域风情的西域胡姬。
<br/>　　“今天这里人好像很多？”程大雷呐呐叹了一声。
<br/>　　“不单单是今天，琅琊城一向都是如此繁华。”刘发财道：“天下分成二帝十八王，各路诸侯都在招兵买马，招纳贤才。孟夫子的万卷书院是人才窝子，万卷书院的学子向来是各路诸侯争抢的对象。而天下读书人，也愿意在万卷书院历练一番，然后择主而事。”
<br/>　　各路诸侯需要人才，天下学子也愿意习得文武艺，货卖帝王家。所以这琅琊城一向热闹，程大雷下意识叹了一句：“原来这是个大型人才市场。”
<br/>　　“什么？”刘发财没太听懂程大雷的话。
<br/>　　“没什么，咱们继续走，你继续聊。”程大雷道。
<br/>　　“琅琊城可是个好地方。帝国不知有多少双眼睛盯着这里，只要稍稍表现好一些，就会有诸侯拉拢你。在学院的衣食住行都有人包下来，送钱送物送女人，先定下来再说。”
<br/>　　“呀，还可以送女人啊，看来万卷书院的校规不怎么严格么？”程大雷咂舌。
<br/>　　“当然啦，食色性也么。才子不风流能叫风流才子。”
<br/>　　刘发财说这话时，不无羡慕的意思。这也自然，帝国传统还是尊重读书人的，向来是士农工商，如刘发财这种坑蒙拐骗行走江湖贼人，根本就是下九流。
<br/>　　前方道路忽然拥挤起来，程大雷看到人都挤在一座楼前，楼内怎样景象他看不清楚，单单大门口都挤满了。
<br/>　　“哈，我知道那是什么地方了。”刘发财感慨一声：“琅琊最大的青楼，红院。”
<br/>　　“咦，这诗书礼乐之乡，还有青楼？”程大雷像个乡巴佬头一次进城般不断发问。
<br/>　　“食色性也，食色性也么。”刘发财摆摆手：“看今日的情况，是有人在红院中论道，可惜我们挤不进去，不然也能长长见识。”
<br/>　　“论道？”程大雷又一次不解地问。
<br/>　　“如今帝国形势复杂，未来帝国将会何去何从，二帝十八王中，谁可执掌天下，这些事情都有得聊。读书人么，各抒己见，坐而论道。每逢这样的事，也是显露才华的好机会。只要你有见解，刚辩论完，就会有人找上门，很多时候，几家势力抢一个人，打得不可开交。”
<br/>　　“总之，琅琊城是个好地方，只要你有才华，就不愁吃不上饭。曾经有个来自南地的穷书生，分文无有，却被红院最出名的花魁养了起来，在万卷书院的一应开销都由花魁负责，也是一件风流韵事。”
<br/>　　“后来呢？”程大雷问。
<br/>　　“后来的事情我怎么知道，我也是听人说的。”
<br/>　　二人一路走，一路聊，穿过拥挤的人群。一路上，碰到两个在路中央下棋的，醉酒狂歌的，还有几个在桥头比武的……
<br/>　　读书人么，也不都是文弱书生。儒家六艺：礼、乐、射、御、书、数。各路诸侯在这里选拔人才，也不仅仅挑选谋士，还包括武将。甚至木匠、铁匠……只要有一技之长，都能在琅琊城找到饭碗。
<br/>　　费了颇大事，程大雷和刘发财来到万卷山前。
<br/>　　万卷书院就在万卷山上，此地紧邻大海，站在山脚就能闻到海洋的咸味。耳边听得咚咚的声音，只是海浪拍击万卷山传来的声音。据说万卷山下有一片空穴，海浪拍在上面，宛若敲鼓，方圆百里可闻，名之曰海鼓，乃是琅琊城一景。
<br/>　　程大雷来到山前，首先映入眼帘的是一座白玉牌坊，上面只写两个字：忠礼。
<br/>　　人站在牌坊脚下，下意识生出一种景仰之心，程大雷微微点头，倒是气魄的紧呐。
<br/>　　“程当家，我还忘了问，您打算怎么上去？”刘发财问。
<br/>　　“走上去啊？”程大雷奇怪道：“我又不会飞。”
<br/>　　“唉，我就忘了说，您果然就不知道。”刘发财道：“您想打探消息，自然需要见到孟夫子，但那孟夫子，可不容易见到。”
<br/>　　“怎么讲？”
<br/>　　“各路诸侯都盯着书院的学子，如我们刚才看到，红楼论道，桥头比武，当街对弈，只要表现好，总会被各家势力看守。但这些人，也只是一般人才。孟夫子的弟子中，有七贤十二秀，是各家势力想要拉拢的对象。但想拉拢他们的人太多了，普通势力想见他们一面都不容易。这还是孟夫子的弟子，您想想那孟夫子是随便可以见的么？”
<br/>　　“难道就没有法子见到这孟夫子？”
<br/>　　“法子当然是有了。”刘发财道：“有三种法子，第一种就是身份。如果帝国天子到了，孟夫子无论如何也要出来迎接的。而各路诸侯王，就得递上名帖，这就要看孟夫子的时间了，等上几天，大概也有机会见上一面。”
<br/>　　“我也算一路诸侯吧。”程大雷试探道：“假若报上我的名字……”
<br/>　　“万卷书院最不缺的就是人才，那七贤十二秀各个都是文武双全，此外更有七十二剑阵，号称从无败绩。”
<br/>　　“哈哈哈。”程大雷仰头大笑：“说第二种法子？”
<br/>　　“第二种就是钱。”刘发财道：“君子爱财么，万卷书院是接受资助的，只要你拿出的钱财足够多，特别多，孟夫子可能在喝茶之余与你见上一面。”
<br/>(本章完)
