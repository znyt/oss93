　　苏莫心中急思，考虑自己该询问什么问题。
<br/>　　这些问题还要与修炼一关，不然对方不一定回答他。
<br/>　　思索了片刻，苏莫抬起头颅，朗声问道：“请问神长老，这个世界上有没有什么方法，能让人起死回生。”
<br/>　　苏莫的声音响彻而起，可问出的问题，却是貌似与修炼没有什么关系。
<br/>　　听闻苏莫之言，黑衣人普阳神长老怔了怔，显然没有想到苏莫会问出这种问题。
<br/>　　不过，既然苏莫白白浪费机会，他也不会多说什么。
<br/>　　沉默了片刻，普阳长老淡淡的道：“有！”
<br/>　　“是什么方法？”苏莫闻言急忙问道，他之所以问出这个问题，自然是想要救活芊寻月。
<br/>　　当初，阎魔鬼圣告诉他，想要救活芊寻月，唯有找到轮回之门。
<br/>　　但是，苏莫并不会只听从一人之言，毕竟，阎魔鬼圣也不是无所不知的人，说不定就有什么办法，是对方也不知道的。
<br/>　　“我是否要算这是第二个问题？”普阳神长老的声音淡淡的响起。
<br/>　　“呃……可以算！”苏莫迟疑着说道。
<br/>　　“在你得到满意的答案之前，便算你是一个问题吧！”
<br/>　　普阳神长老摇了摇头，随即徐徐说道：“想要起死回生，则要重塑神魂，不过，若是神魂消散太久，则回天乏术！”
<br/>　　苏莫闻言心中微沉，消散太久则回天乏术，芊寻月神魂消散已经十几年了！
<br/>　　而现在，后者只剩下一丝执念，而且连这丝执念也已经快要彻底消散了。
<br/>　　沉默了良久，苏莫深深的吸了口气，他没有让这位普阳长老帮忙，看能否救活芊寻月，因为对方不可能帮他。
<br/>　　“第二个问题，请问神长老，五行之力，如何能彻底的融合在一起？”苏莫有问出了第二个问题，则是对于他来说，同样非常重要的问题，因为他一直无法将五行之力，彻底的融合在一起。
<br/>　　“五行之力？”普阳长老闻言沉默了下去，仿佛陷入了思索之中，良久之后，才开口道：“木，五行之始也；水，五行之终也；土，五行之中也。此其天次之序也。木生火，火生土，土生金，金生水，水生木，此其父子也；五行之力想要融合，需秉承造化之机，不可无生，亦不可无制。无生则发育无由，无制则亢而为害，必须生中有制，制中有生，才能运行不息，相反相成！”
<br/>　　普阳长老徐徐说道，言语颇为复杂，但其中貌似点出了五行融合的要点，虽然只是理论，但却是直指源头。
<br/>　　苏莫闻言，眸中闪过一丝迷茫之色，他好像把握住了什么，但又好像什么也没有明白。
<br/>　　“你还有最后一个问题！”普阳长老的话音再次的响起。
<br/>　　“何为生中有制，制中有生？”苏莫沉声问道，问出的依旧是与上一个问题有关。
<br/>　　“所谓生中有制、制中有生，便是一种互相制约、相互依存的状态，维持在一种微妙的平衡之中！”
<br/>　　普阳神长老说道，这些东西他也是凭自己的见识讲解，但具体能否做到，就不得而至了，毕竟他不修五行之力。
<br/>　　“明白了！”苏莫点了点头，心中稍有明悟，但其中还是有很多不懂的地方，因为具体如何做，完全是没有头绪。
<br/>　　“好了，你回去吧！”普阳长老的声音再次的响起。
<br/>　　苏莫闻言心中暗叹，这一次指点，居然只能提出三个问题，收获虽然也有，但还是远远没有达到心中的期望。
<br/>　　但是，对方已经下了逐客令，他要不好再继续询问了。
<br/>　　“告辞！”苏莫向对方抱拳，随即躬身后退，而后离开了后院。
<br/>　　待苏莫离开之后，院落中陷入了寂静之中，普阳长老的身影，如同石化在水潭上的石雕，亘古长存。
<br/>　　片刻之后，一阵脚步声响起，一道蓝色的倩影走进了后院，倩影正是云悠悠。
<br/>　　“见过师尊！”云悠悠向普阳神长老微微躬身。
<br/>　　“悠悠，你来了！”普阳长老的声音响起，而后其身体缓缓的转了过来，露出一副年轻的面容。
<br/>　　普阳神长老非常的年轻，看起来不足三十岁的模样，面目俊朗，给人十足的阳刚之气。
<br/>　　当然，这只是外表而言，真实的年轻并不会表现在脸庞之上。
<br/>　　就比如苏莫，如今已经三十余岁，看起来却依旧是十八、九岁的模样，而且以他的修为，就算再过百年，依旧是这番年轻。
<br/>　　“师尊，那个苏莫为何会这般厉害？”云悠悠沉声问道，这也是她来的目的，这是她比较好奇的地方。
<br/>　　“此子的确不简单，修五行之力，还想要将五行之力彻底的融合在一起！”普阳长老沉声说道，对于苏莫修炼之路，他还是颇为惊讶的。
<br/>　　这种修炼之路，或许非常的强大，但是阻力同样也是极大，很难走的顺畅。
<br/>　　“这五行之力居然这么厉害？”云悠悠闻言，俏脸上露出惊讶之色。
<br/>　　“五行之力的确不凡，而且他应该是已经融合到了一定的地步，才会有越级碾压其他妖孽天才的战力！”普阳长老说道。
<br/>　　“原来如此！”云悠悠微点螓首，难怪苏莫这么强大，原来修炼了特殊的武道之路。
<br/>　　“悠悠，你无需在意其他人，努力提升自己的修为才是正途！”普阳神长老的面上，露出温和的笑容。
<br/>　　“师尊，我很努力的！”云悠悠乖巧的点了点头。
<br/>　　“呵呵，努力就好，我会让你在百年之内成圣！”普阳神长老道，其言语轻松至极，仿佛成圣非常容易一般。
<br/>　　“多谢师尊！”云悠悠立刻向对方行礼，她心中对这位师尊非常的崇敬，对方对于她的指导，一直都是毫不吝啬，各种珍贵的资源宝物，更是源源不绝。
<br/>　　“嗯，希望你不要让为师失望！”普阳神长老道。
<br/>　　“师尊放心，悠悠定不会让师尊失望！”云悠悠面色坚定的说道。
<br/>　　普阳圣长老微微颌首，旋即便挥了挥手，道：“回去吧！”
<br/>　　“弟子告退！”
<br/>　　云悠悠再次恭敬的施了一礼，便转身款款而去。
<br/>　　望着云悠悠纤细的背影，普阳神长老眼眸之中，留露出莫名的神光。
