　　“天卓道长后来被法术界追杀，居然还找到他，把他杀了，逼问出三口鼎的下落，但是只找到一口，当时半个法术界一起出动，才将这口鼎中的亡魂超度，将鼎毁掉。至于另外两口，则下落不明……
<br/>　　后来元末的时候，据说有一口鼎现世，被一个妖王占据，本来是想吸收鼎中的妖气，结果鼎中的怨气实在太强，最后妖王自己也被反噬，被鼎给吸收了……之后那鼎中有一些游魂泄露出去，到处杀人，引起大明朝廷的重视，还是刘伯温亲自召集天下法师，一起设法将鼎封印起来，造了一个八卦台，将这口鼎在阳光下暴晒一年，等到其中的怨气散去大半，这次召集众人一起作法，才将里面的亡魂都超度了……”
<br/>　　道渊真人听到这里，说道：“我也听过这件事，据说当时作法的时候还出现意外，死了不少法师。”
<br/>　　叶少阳还真没听过他们说的这些事，关于炼尸缸，他最多只是在典籍里看过一些只言片语的讲述，不光是这件事，叶少阳自从跟毛小方打交道之后，从他嘴里听到了很多自己不知道的法术界的秘闻，一开始还很好奇，后来打听他看过的那些典籍，才知道大部分自己都没看过，但是两人的时代相隔不过百年，按说是不会出现这么明显的知识断代。
<br/>　　叶少阳仔细思考过这个问题，觉得只有一个可能，就是在之后那个破四旧的时代，有很多典籍之类的被毁掉了，而且在那段法术界最沉寂的岁月，很多东西都产生了断代，法术界的组织也完全被破坏，直到80年代才开始慢慢修复……
<br/>　　叶少阳撇开这些杂乱的想法，说道：“不是鼎吗，怎么成炼尸缸了？”
<br/>　　毛小方道：“听说是那鼎口大肚子小，模样很像水缸，就给取了这么个诨名。”
<br/>　　“这么说，世上还存在两口炼尸缸？”
<br/>　　“不知道，不过……如果那古墓里真是炼尸缸，那麻烦可就大了。”
<br/>　　三个道士互相看去，都是一脸的不知所措。
<br/>　　叶少阳还是头一回听说这么邪性的东西，如果真的有，那真是邪到不能用言语来形容。
<br/>　　“不过，炼尸缸就算再凶，也只是个死物，怕得就是被人所利用……”叶少阳道，“炼尸缸怎么会出现在古墓里？”
<br/>　　这问题当然没人能回答，甚至他们都不知道这古墓的主人是谁，也无从调查，现在唯一的期盼就是那位妙心真人真的能来，可以告诉他们一些事情。
<br/>　　道渊真人和毛小方都是行游天下，这种灵异事件对他们来说算是分内之事，自然不会放任不管。叶少阳心思完全不在什么古墓上，但既然遇到了，也不可能袖手旁观的，美华自然也是跟着他。
<br/>　　“对了少阳子，我这次来之前，从我们龙虎山上打听到一个消息。”道渊真人突然说道，“桃花山最近发出法帖，召开水陆龙华会，要共同选出一位年轻才俊，号称法术界首席弟子，重点培养，将来能号令法术界行事。”
<br/>　　叶少阳一听就皱起眉头，没等开口，毛小方就嚷起来，“桃花山疯了？”
<br/>　　道渊真人耸了耸肩，道：“确实很是诡异，说起来，当今法术界一直被法术公会统治，大家都很不满，也曾想过怎么样抵抗，把人间法术界重新夺回到自己手中，桃花山想到培养青年才俊，这个法子是不错，但是并不现实。”
<br/>　　毛小方道：“是不现实，法术界二代弟子，张晓寒等人都成了法术公会的走狗，等这帮人上位，法术界更是被人家掌控，哪还有什么未来？”
<br/>　　道渊真人点头说道：“不过最关键的不是在这，他们有忧患意识是正常的，只是……这根本就不是桃花山该关心的事。”
<br/>　　毛小方冷笑说道：“谁说不是来着。”
<br/>　　“说起来，我们几大门派中，除了昆仑山等几个门派跟着法术公会走，其余我们几家，其实都在暗中培养自己的年轻弟子，为将来做准备，天下之乱，大派担当，一向是我们几大门派的责任，不是我自夸，法术公会虽然掌控全局，但也奈何不了我们几大门派，但是桃花山……这么说的话，会非常危险。”
<br/>　　叶少阳听到这里，也算是明白了怎么回事，说道：“法术公会还会灭门不成？”
<br/>　　“那倒不会，不过会通过各种手段打压小的门派，让他们无法立足，以桃花山的实力，想要跟法术公会抗衡，简直就是螳臂当车。大家起初都觉得莫名其妙，我们龙虎山派人前去调查，发现一个情况……”
<br/>　　道渊真人缓了一口气，说道：“桃花山上，似乎来了一个陌生人。”
<br/>　　叶少阳和毛小方情绪被调动起来，都一本正经地听他说。
<br/>　　“一个陌生人，住进了桃花山上，据说占据了掌教山河真人的修所，整天修炼，从来不下山，这就是前段时间的事，桃花山突然发出法帖，极有可能跟这位陌生人有关。”
<br/>　　毛小方道：“难道是桃花山有什么太上掌教之类的出关了？”
<br/>　　很多门派的掌教不会等到死了再让位，都会在生前让位给后人，自己躲起来清修，这些家伙在清修的时候，都很老了，修炼也都达到了瓶颈和上限，基本上就算再苦修多少年，也不会有什么突破，不过凡是总有例外，也有一些老家伙在山里潜心修行十年八年之后，一遭顿悟，突破新的境界。
<br/>　　一个门派的实力，比的是综合实力，不过上限却是取决于门派中最强之人的境界，很多中小门派最多只有一两位天师，如果突然多了一个地仙，一下子就会脱颖而出，有资格参加几大门派的活动和会议，假如出了一位灵仙，更不用说，至少在这灵仙挂掉之前，在法术界也会拥有相当的地位。
<br/>　　因此，桃花山这反常的行为和突然多出来的陌生人，才会让毛小方觉得是不是山中突然多了什么不世出的强者。
