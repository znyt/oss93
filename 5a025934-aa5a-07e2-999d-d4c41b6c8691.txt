　　第621章
<br/>　　小白狼自从与程大雷失散后，便在附近寻找他的下落。路途中，遇到陆长迢调动兵马，便一路尾随到此地。
<br/>　　如今两家重逢，也将程大雷的座骑黑牛带了过来。程大雷跨上黑牛，顿时信心大增，提着剑向外冲杀。黑天半夜，也分辨不得方向，程大雷、小白狼、白元飞就像是没头苍蝇一般往外闯。
<br/>　　不知逃了多久，东方已经露出鱼肚白。见此刻追兵还没有追过来，程大雷才算是松了口气。
<br/>　　“大当家，这位是……”白元飞看向小白狼。
<br/>　　“喔，大家都是自己人，以前青牛山便在一起。”程大雷抓紧时间平复着呼吸。
<br/>　　白元飞和小白狼互相抱了抱拳，算是打过招呼。
<br/>　　“大当家，咱们下一步往哪儿走？”白元飞问道。
<br/>　　程大雷四周看看，刚才慌慌张张往外逃，也没有分辨方向，现在也是不知道自己身在何处。
<br/>　　“咱们还是该回去。”程大雷道：“蛤蟆城在西北方向，往西走总是错不了的。”
<br/>　　三人已经迷路，认准了方向，背着太阳往西走。还没行多久，就听见前方传来打斗声。
<br/>　　程大雷小心翼翼凑过去，见陆长迢指挥兵丁形成一个包围圈，中间围着的正是莫将难、柳追风、周桃几人。
<br/>　　昨夜程大雷往外逃，莫将难和柳追风也向外逃，误打误撞两家撞在一起。眼下莫将难已经落单，身边又没了韩醒这样的高手，柳追风立刻挥着剑去找莫将难报仇。
<br/>　　双方的战斗还未决出胜负，便被追兵赶上了，陆长迢将他们围在一起，等他们力气耗尽，便一网打尽。
<br/>　　“大当家，咱们别趟这浑水，早早撤吧。”小白狼小声道。
<br/>　　“对对对。”程大雷连忙点头：“咱们赶紧溜。”
<br/>　　三人转过身就要向远处逃去，这时候他们可不希望与官兵遇见。就在此时，东方童无意间向四周望了一眼，正好和程大雷的目光撞上。
<br/>　　“呔，那是程大雷，可别让他走了。”
<br/>　　一石激起千层浪，陆长迢急忙指挥士兵围杀过来。知道程大雷本事了得，他们不敢靠近厮杀，只在远处放箭。箭网如雨，一层又一层迫向程大雷。想要逃命的程大雷无奈何被他们围在当中，正好和莫将难两波人会在一起。
<br/>　　三方见面，都是面面相觑，莫名其妙觉得有些尴尬。三人昨夜在一起，时而为敌，时而为友，面对官兵的追杀没头苍蝇一般向外蹿，却没料到竟又在此地被围在一起。
<br/>　　陆长迢看着这一幕，忍不住哈哈大笑：“好好好，今天你们已经走投无路，我看你们往哪里逃，今日杀了你们三贼，也是我大功一件。”
<br/>　　程大雷道：“陆将，咱们昨夜就见过，不是说井水不犯河水。”
<br/>　　“呔！你这恶贼，还有脸与我说这话，今天本将军不杀你，我就不姓陆。”
<br/>　　陆长迢又记起昨夜被程大雷一脚踹下马背的事情，现在他的座骑还在白元飞手中，想起这件事他就一阵无名火起。在他心里，莫将难和柳追风倒是可以活捉，但程大雷却必须死。
<br/>　　程大雷也知道今天这件事不可能善了，他看了看莫将难和柳追风，二人现在都是伤痕累累，估计派不上什么用场。
<br/>　　他亮出匹夫剑，面向陆长迢道：“来，听说陆将手下也是高手如云，来来来，谁敢与本当家过手。”
<br/>　　“呸。”陆长迢狠狠啐了一口：“傻子才和你单打独斗，来人，与我放箭。”
<br/>　　程大雷一阵无语，名声太大了也是不好，现在想要找个单挑的人也不容易。
<br/>　　陆长迢一声令下，手下弓兵布成一排，同时向程大雷几人放箭。现在，莫将难和柳追风都对程大雷恨得牙痒痒，他不出现倒还好些。可他一出现，陆长迢就一点不留情了。
<br/>　　几人挥舞兵器，击落射来的羽箭。如今弓箭虽然威力并不大，但毕竟是最方便的远程武器，而且双方距离又近。阵阵破空声在耳边响起，程大雷偶尔还帮白元飞或小白狼扫落几枚羽箭。
<br/>　　可莫将难与柳追风就没有这么好的武艺了，尤其是柳追风一心一意护着周桃，昨夜一场激战，他早已是身受重伤。冷不防一枚射来的羽箭没有拦住，射中他的肩膀。
<br/>　　在一轮箭雨的空当，程大雷轻轻吁了口气。如今自己还勉强撑得住，可一直这样下去，等自己力气耗尽，终究难逃万箭穿心的下场。
<br/>　　必须想个办法破局，程大雷目光遥遥落在陆长迢身上，所谓射人先射马，擒贼先擒王。
<br/>　　陆长迢感受到了程大雷的目光，心中一激灵：这恶贼想要做什么。
<br/>　　“陆长迢！”程大雷突然喝了一声。
<br/>　　“干嘛。”陆长迢下意识道。
<br/>　　“你活够了没有？”
<br/>　　“什么？”
<br/>　　“本当家问你活够了没有。”
<br/>　　程大雷忽然喝了一声，一拍座下黑牛向陆长迢冲去。黑牛撒开蹄子，就像一头下山猛虎般，以锐不可当之势向陆长迢冲去。
<br/>　　速度越来越近，越来越近，陆长迢骇得手足颤抖，口中忙喝：“放箭，放箭。”
<br/>　　弓兵准备好后，又一轮箭雨扑面而来，这次主要是招待程大雷。程大雷挥舞匹夫剑，将射来的羽箭一枚枚击落。很快，他便已迫近陆长迢，口中大喝一声，身子忽然跃起，身无彩凤双飞翼施展开来。
<br/>　　陆长迢慌张慌张往后退，身边护驾的兵将乱糟糟撞在一起，顿时人仰马翻。程大雷从空中落下时，黑牛已闯入敌军阵中，程大雷正好落在黑牛背上。
<br/>　　他轻轻舒臂，抓住陆长迢的盔甲，直接将他从马背上摘了下来。双腿轻轻一夹，便带着陆长迢返回白元飞身边，咚地一声，将他丢在地上。
<br/>　　等陆长迢站起来时，两柄刀已经搁在他的肩膀上，一柄是小白狼的，一柄是白元飞的。
<br/>　　在场的所有人，无论是官是匪，看到这一幕都大惊失色。程大雷窜入乱军丛中，如入无人之境。所有人脑海里都想着一句话：以他的本事，不说万人敌，千人敌是绝对没问题的。
<br/>(本章完)
