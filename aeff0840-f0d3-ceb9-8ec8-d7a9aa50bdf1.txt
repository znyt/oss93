　　“闭关千年了！”
<br/>　　苏莫心中自喃，有些紧迫之感，虽说到了神王级别，提升肯定极为困难，就算再恐怖的妖孽，都要消耗大量的时间。
<br/>　　但是，帝一魂和拓跋青云，绝对不能用常理来衡量，万一其中一人率先成为高级神王，那一切都完了。
<br/>　　不过，帝一魂和拓跋青云，均是在闭关，对于他来说，也算是一件好事。
<br/>　　毕竟，他的危险性，就降低了不少。
<br/>　　“拓跋流明，我饶你不死，你就在这里呆着吧！”苏莫朗声说道，言罢，他的意念便退出了吞噬旋涡之中。
<br/>　　此人暂时不能杀，以后或许是他的一个筹码。
<br/>　　拓跋流明沉默不语，他的心中，对于苏莫的身份，有了一个猜测。
<br/>　　这个猜测，让他心惊不已，但是，他不会说出来。
<br/>　　只有活着逃走，他才能将这个消息，告知地王宫以及他的二叔。
<br/>　　随即，苏莫不再理会拓跋流明，开始融炼古剑山的万化神剑元神。
<br/>　　他本来就有一个万化神剑元神，加上又掌控了终极剑道，自然要将剑道的威力最大化。
<br/>　　所以，他自然要融炼古剑山的万化神剑元神，将其融炼成自己的本命元神。
<br/>　　这一点，倒是不困难，几个时辰之后，苏莫便成功融炼，拥有了两个混沌万化神剑元神。
<br/>　　至于千幻元神，以及云沐的天罚神瞳元神，他倒是没有融炼，因为，他的神魂不够强大，无法再行融炼一个本命元神。
<br/>　　当然，千幻元神和天罚神瞳元神，就算不成为本命元神，他也能使用，只不过效果会降低很多而已。
<br/>　　随后，苏莫意念一动，进入了空间戒指之中。
<br/>　　空间戒指之内，辽阔的大地一望无际，巨大的终极神剑，矗立在大地之上，震撼人的心灵。
<br/>　　“融炼此剑，方能认主！”苏莫看着巨大的终极神剑，面上带着一丝笑意。
<br/>　　他虽然能控制此剑了，但还不是此剑真正的主人，不能彻底如意的掌控此剑，所以还需要融炼此剑。
<br/>　　融炼的方法，倒也简单，就和融炼本命灵剑一样。
<br/>　　当然，只有掌控了剑源之人，才能融炼终极神剑。
<br/>　　少倾之后，苏莫盘膝坐在了终极神剑之上，滴出自己的本命精血，开始了融炼。
<br/>　　融炼终极神剑，比本命灵剑要困难很多，但是，苏莫也只是用了一个时辰，便融炼了终极神剑。
<br/>　　意念一动，终极神剑急速缩小，化为了一柄四尺大剑，落到了苏莫的手中。
<br/>　　“真是一柄不凡的神器！”
<br/>　　握着手中的终极神剑，苏莫心中欢喜不已，这终极神剑极重，难以想象的重，几乎比宇宙中的一颗星辰还要重。
<br/>　　若是修为不够高，估计都拿不动此剑。
<br/>　　最主要的是，此剑不仅是一种正统的宝剑，也是一柄灵剑。
<br/>　　随即，苏莫意念一动，终极神剑便消失无踪，进入了他的识海之中。
<br/>　　一切完成之后，苏莫立刻了空间宝器之中，开始赶路。
<br/>　　他所去的方向，是天龙神朝的方向，这一次他要去龙族。
<br/>　　飞行之中，苏莫意念一动，终极神剑出现在脚下，他御剑而行，速度顿时爆增数倍。
<br/>　　此时此刻，他御终极神剑的速度，快到极致，比普通的本命灵剑要快的多。
<br/>　　当然，御剑飞行的速度，还是没有他第一次召唤终极神剑之时，此剑飞到他身前的速度。
<br/>　　毕竟，召唤终极神剑，靠的是剑源的联系。
<br/>　　而御剑飞行，靠的是精神力，两者不可同日而语，但是御剑飞行，只能靠精神力。
<br/>　　苏莫心中大喜，如此速度，无论是逃命还是赶路，都快了数倍。
<br/>　　随即，苏莫不再多想，闷头赶路。
<br/>　　而在苏莫一心赶路之时，万道剑冢一役发生的事情，也传遍了终极剑域，向其他界域传播。
<br/>　　“一个叫做秦陨的妖孽，掌控了终极剑道，取走了终极神剑，毁掉了终极神王殿的山门！”
<br/>　　此事，如同一轮风暴，以终极剑域为中心，向各大界域席卷。
<br/>　　一时之间，此事掀起巨大的波澜，无数的人为之震惊。
<br/>　　传闻，这个秦陨极为逆天，天赋足以吊打任何万古天骄的天赋。
<br/>　　此人，修炼混沌道统，练就了混沌不灭体，拥有万化神剑元神、不败战气元神、千幻元神等超级强大的元神，天赋震古烁今。
<br/>　　虽然这个消息有些让人难以置信，但是，并不是空穴来风，很多人都见证了秦陨的天赋。
<br/>　　无数的为之震惊，无数的人在猜测，这个秦陨是何需人也，为何如此的逆天。
<br/>　　不过，对于普通的武者而言，根本不可能知道秦陨的底细，也只有少部分人，才知晓真正的内幕。
<br/>　　……
<br/>　　原神剑山所在之地，此时此刻，新建了一座城池。
<br/>　　城池的轮廓已经建成，但其内还不完善，很多的建筑正在修建。
<br/>　　足有数十万武者在忙碌，不过，因为均是修为强大的武者，修建的速度非常之快，每时每刻都有宫殿落成。
<br/>　　一座古朴的宫殿之中，此刻有五人在坐，坐于主座之上的人，自然便是终极神王。
<br/>　　大殿左边端坐着紫霄和绫裳，右边端坐的是风雪神王和毁灭剑王。
<br/>　　“毫无疑问，这个秦陨就是苏莫，也是天尘宇一直在找的人！”终极神王环视众人，声音低沉。
<br/>　　“殿主，现在当务之急，是找到这个苏莫的下落！”风雪神王说道，不管这个秦陨是谁，和天尘宇是什么关系，找到此人才是当务之急。
<br/>　　终极神王颌首，目光看向绫裳，问道：“绫裳，你认为苏莫会去何处？”
<br/>　　因为之前，他已经得知，苏莫是通过绫裳进入的终极神王殿，所以他才会有此一问。
<br/>　　“殿主，苏莫以前对我有恩，我才会帮助他加入终极神王殿，所以我并不知晓他会去何地？”绫裳摇了摇头道，她心中却是暗暗感慨。
<br/>　　因为，她也没有想到，苏莫能得到终极剑道，并取走终极神剑，这大大的出乎了她的预料。
<br/>　　“是吗？”终极神王闻言，眼眸微眯，此话他只信一半，因为，绫裳不可能帮助一个不知底细之人，加入终极神王殿。
<br/>　　略一沉吟，终极神王又道：“绫裳，此事你有责任，追查苏莫之事，你必须亲力亲为，不能懈怠！”
<br/>　　他并未逼迫绫裳，因为，他也知晓，苏莫既然得到了终极剑道，并取走了终极神剑，绝对不会泄露自己的行踪，这点绫裳应该没有说谎。
<br/>　　“明白！”绫裳重重的点头。
<br/>　　“你现在即刻启程，前往轮回神朝天王宫，寻找天尘宇，了解苏莫的具体底细！”终极神王吩咐道。
<br/>　　“是！”
<br/>　　绫裳起身，向终极神王抱拳，便转身离开了大殿。
<br/>　　在绫裳走后，终极神王目光环视紫霄神王、风雪神王和毁灭剑王三人。
<br/>　　“紫霄，现如今的绫裳，可能已经不再是以前的绫裳，你可明白？”终极神王的目光，落在了紫霄的身上。
<br/>　　“明白！”紫霄点了点头，他理解殿主的意思，如今的绫裳，心可能早已不是完全在终极神王殿了。
<br/>　　“你暗中跟着她，她极有可能与苏莫有特殊的关系，若是一旦有异，立刻将她擒拿，若是她与苏莫见面，那就将两人一起带回来！”终极神王吩咐道，现如今，他对于绫裳已经不完全信任了。
<br/>　　“殿主放心！”紫霄点了点头，长身而起。
<br/>　　“还有，不能让她发现你！”终极神王补充道，若绫裳没有背叛终极神王殿，此法会让其心寒，所以还是要小心为上。
<br/>　　绫裳毕竟是他一手教导出来的弟子，在他心中，还是有很大的分量的。
<br/>　　“殿主放心，以绫裳现如今的修为，不可能察觉的到我的存在！”紫霄沉声说道，言罢，便转身而去。
<br/>　　“你们两人，安排门下弟子，给我调查苏莫的下落，就算翻遍整个神域，也要给我查到苏莫在何处！”
<br/>　　随后，终极神王又对风雪神王和毁灭剑王吩咐道。
<br/>　　“是！”两人抱拳领命。
<br/>　　“还有，据韩生所言，时空界域永恒神宫弟子公孙嘉和苏莫相识，立刻派人前去永恒神宫！”终极神王又吩咐道。
<br/>　　“明白！”风雪神王和毁灭剑王领命，便纷纷告退离开，各自去安排人手调查苏莫的行踪。
<br/>　　风雪剑宫和毁灭剑宫，足有弟子三十万，其中不乏大量擅长追踪与查探的弟子，全部被安排了出去，开始大规模的追查苏莫的行踪。
<br/>　　而终极神王，也没有置身事外，离开了终极神王殿，动身前去混沌神朝。
<br/>　　那苏莫修炼的正宗混沌道统，练就了混沌不灭体，必然和混沌神朝有关系。
<br/>　　所以，他要前去混沌神朝，面见混沌神朝如今的掌权之人，混沌元王。
