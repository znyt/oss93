　　“第一层尽头的屏障被破开了。”
<br/>　　周遭剑修，脸上尽是不可置信之色。
<br/>　　萧逸的身影，缓缓穿过了屏障，渐渐消失在所有人视线中。
<br/>　　一众剑修，虽面露闲艳羡之色，却也只能望而却步。
<br/>　　那一步，他们踏不出去。
<br/>　　一秒之后。
<br/>　　萧逸已然消失在所有人眼中。
<br/>　　而第一层尽头的屏障，亦恢复了正常。
<br/>　　“该死，这家伙怎么做到的。”冯济的拳头，握得劈啪作响。
<br/>　　“怎么可能。”即便是一众老牌剑修，亦面露不可置信之色。
<br/>　　倒是那些学宫的剑道弟子，仍旧面露倨傲之色。
<br/>　　“哼，就算让他进入了第二层空间又如何。”
<br/>　　“先不说他能否在第二层空间得到机缘，即便得到了，废物仍旧是废物。”
<br/>　　“不错。”几个五痴学宫剑痴弟子冷笑一声。
<br/>　　“令狐忘师兄等各学宫首席，此刻还在天藏学宫受各大强者培养呢。”
<br/>　　“若是他们在，何时轮到这小贼耀武扬威。”
<br/>　　“令狐忘师兄，何须来此。”学宫弟子得意道。
<br/>　　“剑帝碑内的好处虽也不凡。”
<br/>　　“但说到底，哪里比得上五大学宫的底蕴。”
<br/>　　“各种修炼资源与重宝，各位首席唾手可得。”
<br/>　　“便暂且让这小贼嚣张几分，他日，定要他知晓何为尊卑。”
<br/>　　一众学宫弟子、长老，似是天然有着倨傲。
<br/>　　周遭其他剑修闻言，虽是皱眉，但也未反驳什么。
<br/>　　毕竟，五大学宫的底蕴，确实恐怖。
<br/>　　哪怕真在这剑帝碑内得了莫大好处，怕也比不上五大学宫的倾力培养。
<br/>　　……
<br/>　　剑帝碑，第二层空间内。
<br/>　　萧逸身影一闪，就此进入。
<br/>　　第一层尽头处的屏障，破开的要求，其实并不难。
<br/>　　待得武者走过剑碑主人的一生经历后，就代表着剑气之路已走完。
<br/>　　但，最后还有一剑。
<br/>　　便是属于剑碑主人一生的最强一剑，亦是‘死亡一剑’。
<br/>　　能否劈出这一剑，参悟这一剑，便看参悟者自己的本事了。
<br/>　　这一剑，便是劈开屏障的一剑。
<br/>　　在萧逸的画面中，那斩破星辰的一剑，就是斩星前辈的死亡一剑。
<br/>　　那一剑之后，斩星前辈便陨落了。
<br/>　　具体原因，萧逸也不知晓。
<br/>　　那些画面，终究是断断续续的画面，而不是真正的一生。
<br/>　　这般算起来的话，斩星前辈活过的岁月，也只在万年左右。
<br/>　　算上之前的战斗、成长，还有最后的一悟三千年，加起来也就万年左右。
<br/>　　这般算来，斩星前辈，并非是寿命走到尽头而陨落的。
<br/>　　“闭关三千年，便只为那最强一剑，湮灭星辰。”
<br/>　　“之后，便黯然陨落。”
<br/>　　萧逸眉头紧皱，“斩星前辈最后的生命中，到底经历了什么？”
<br/>　　这是萧逸不得而知的。
<br/>　　说回之前除却提升剑道外的好处，那便是‘剑心’变得无比坚固。
<br/>　　万年岁月的沉淀、磨练，萧逸的修为，没有半分提升，那是因为那都是虚幻的画面。
<br/>　　可他的剑道层次，却在其中得到极大提升。
<br/>　　他的‘剑心’，也在那悠长沉淀下，变得坚固无比。
<br/>　　那最后的一剑，看似斩碎星辰；实则，也将萧逸在这悠长画面的参悟中所有的负面影响，一一斩碎。
<br/>　　随着那最后一剑的劈出。
<br/>　　走过斩星前辈的一生经历，不仅思想记忆上没有半分影响，连同‘剑心’也在那一瞬间坚固百倍。
<br/>　　这才是真正的莫大好处。
<br/>　　这般坚固剑心，萧逸放在以前绝不敢想象。
<br/>　　他甚至确定，哪怕他日后超越了绝世强者层次，乃至更强层次之后，都不会产生半分心魔。
<br/>　　即便他现今修炼‘气血一道’，任那血珠吸收再多气血，也休想影响他心神分毫。
<br/>　　萧逸笑笑。
<br/>　　此次剑域盛事，果真没白来。
<br/>　　锵…锵…锵…
<br/>　　这时，第二层空间内，无数剑鸣声响起。
<br/>　　萧逸放眼望去。
<br/>　　整个第二层空间，同样一望无际，大小应当与第一层空间差不多。
<br/>　　但，第二层空间内，没有半座剑碑。
<br/>　　有的，仅仅是一把把分散在各处的利剑。
<br/>　　这些利剑，萧逸认得其中一把，正是斩星。
<br/>　　如果没有猜错的话，这些剑，正是一位位剑碑主人生前的剑。
<br/>　　数万把神兵利器，看似杂乱无章地伫立着。
<br/>　　实则，萧逸一眼便看出，剑刃伫立，自成剑阵。
<br/>　　此时，数十把剑刃凌空而起。
<br/>　　远处，两道身影，试图通过剑刃的拦路。
<br/>　　两道身影，正是林子风与桌万宗。
<br/>　　远处，林子风，显然也注意到忽然现身的萧逸。
<br/>　　“萧逸师兄。”林子风瞬间放弃了与剑刃的交战，身影一闪，瞬间来到萧逸身旁。
<br/>　　“子风师弟。”萧逸笑笑。
<br/>　　远处，原本与数十剑刃鏖战的桌万宗，少了林子风的帮忙，独自抵挡，显然抵挡不住。
<br/>　　只能气急败坏地闪身回到原地。
<br/>　　“子风，你做什么？“桌万宗愤愤问道。
<br/>　　“嗯？萧逸小贼？”桌万宗皱眉看着萧逸，“你竟也能来第二层？”
<br/>　　林子风为理会桌万宗，而是笑着看向萧逸，“我早便知道，以萧逸师兄的本事，第一层绝对拦不住你。”
<br/>　　“呵。”萧逸笑笑。
<br/>　　林子风，向来对萧逸有着无比的自信。
<br/>　　“第二层的考验是什么？”萧逸问道。
<br/>　　林子风指了指周遭利剑，道，“便是通过这万剑之阵。”
<br/>　　“事实上，在第一层空间中，伫立于剑碑之旁的利剑，不过是一道虚影。”
<br/>　　“剑的真身，一直在第二层。”
<br/>　　“第二层之内，自成禁制与剑阵；这一把把神兵利器，就是阻挡我们前往第二层尽头的阻碍。”
<br/>　　“这数万把神兵利器，可不是等闲之物。”萧逸皱起了眉头。
<br/>　　林家，作为剑帝碑的守护者；而林子风，一是拥有剑帝血脉的天骄，二是林家少家主。
<br/>　　自然的，他对剑帝碑内的空间无比清楚。
<br/>　　而这数万把利剑，每一把，可都是经历过无数鲜血洗礼与惊天大战的。
<br/>　　这一把把神兵利器，都曾陪伴着剑碑主人走过了峥嵘一生，每一把都非凡物。
<br/>　　以此而成的剑阵，其威力…
<br/>　　林子风苦笑一声，道，“便是这剑阵的威力太强，所以我和桌万宗二人来此八天了，也毫无办法。”
<br/>　　“顶多就是顶着剑阵，前行不过数十里。”
<br/>　　对比偌大个第二层空间，这数十里，不过是个开端。
