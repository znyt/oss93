　　第531章
<br/>　　酒过三巡，菜过五味。黄老汉亲自起身，为程大雷和刘发财斟满酒杯。
<br/>　　“二位再用一些酒，荒村僻野，用这些薄酒招待二位，老汉给二位赔罪了。”
<br/>　　刘发财已饮了不少酒，说话满口酒气，行事也放肆些，两眼直勾勾盯着一旁伺候的丫鬟。
<br/>　　“老丈这是哪里话，能有这一桌酒菜，也足见你们的用心了。”刘发财闷了一口酒，在丫鬟的手上摸了一把。
<br/>　　黄老汉呵呵笑着，目光扫到这一幕，假装没瞧着。
<br/>　　“二位果然有江湖豪侠的风范，日后定能有一份大前程，到时候世人知道我招待了二位，也是我黄家祖宗的荣耀。”
<br/>　　“雀儿……”黄老汉笑笑：“今夜你到程大侠房中，好好服侍程大侠。”
<br/>　　刘发财一听这话，顿时乐不可支，满口酒差点喷了出来。
<br/>　　程大雷倒是没多大反应，依旧闷头吃菜。
<br/>　　“刘壮士似乎不爱说话，来来，请满饮此杯。”
<br/>　　“路上太累了，还没缓过来。”程大雷随口道了一声，忽然心中一动，道：“我还要向黄老汉打听一件事？”
<br/>　　“壮士请讲”黄老汉发现一件奇怪的事，当程大雷开口说话时，滔滔不绝的刘发财绝对不开口。
<br/>　　“请问这附近有没有知名的大夫，最好是姓华的？”程大雷问。
<br/>　　黄老汉皱眉想了一会，摇摇头道：“老汉粗陋寡闻，却是没有听说过。如果壮士需要，老汉可以帮忙打听一下。”
<br/>　　程大雷摇摇头，没有说什么。他在黄家无非歇歇脚，就连对刘发财，也谈不上过分在意，他的主要目的还是寻找华氏一族。
<br/>　　在兖州打听不到，自己就要去徐州，究竟能不能访到华氏一族，程大雷其实没有多少把握。
<br/>　　黄老汉盯着程大雷，一时之间有些走神。
<br/>　　所谓是名士自风流，程大雷在蛤蟆城习惯了一言九鼎，手下对他的话言听计从。等程大雷不愿意和刘发财逗闷子的时候，身上那股跋扈劲儿就露了出来。
<br/>　　刘发财和程大雷朝夕相处，程大雷安静下来的样子，看着让人怕。所以不知不觉间，二人的相处，便是以程大雷为主。
<br/>　　黄老汉锦衣玉食了大半辈子，闲着没事就想着怎么琢磨人。这种感觉对他来说太明显了，于是心里就有些想不明白：怎么这个无足轻重的程大雷，却是说话算数那个。
<br/>　　他盯着程大雷，程大雷模样自然谈不上英俊，说一句不丑已经算拍马屁了。而他说话吃饭，毫无风度可言，但却自有一股气派。
<br/>　　相比之下，刘发财就显得有些不值一提了。
<br/>　　喝了不少酒，说了不少话，黄老汉大概已经能摸清刘发财的城府，而对这个程大雷，他还是摸不透。对方就像一块石头，又丑又硬，可究竟里面是石料还是玉石，黄老汉就完全摸不透了。
<br/>　　再看那个刘发财……
<br/>　　呀，该不会认错了吧。这样一个人，怎么会是那个传说中的杀胚。
<br/>　　黄老汉激灵一下，心中有了不好的预感。这要是自己儿子把大军引过来，发现抓到一个冒牌货……那该如何是好。
<br/>　　他心乱如麻，一时间说话做事都大失分寸，一张脸白了红，红了白。猛抬头，见程大雷坐在那里笑盈盈的盯着他。
<br/>　　一瞬间，黄老汉的冷汗唰地冒了出来。
<br/>　　程大雷的确在观察黄老汉，他当然猜不透黄老汉的心思，可黄老汉做事透着一些怪。
<br/>　　昨夜来到黄家，黄老汉虽然没有冷落二人，但也并没有表现得太过热情。相比昨天，今天他热情得有些过分了。
<br/>　　每一句话每个动作，像是都刻意讨好二人。问题是程大雷和刘发财有什么好讨好的？
<br/>　　无事献殷勤，非奸即盗，他究竟藏着什么心思。
<br/>　　“二位壮士，来来，喝酒喝酒。”黄老汉又端起酒杯，反正事情到这一步，已经不能回头，那就先灌醉二人再说。
<br/>　　“来，喝酒。”
<br/>　　程大雷也抖索起精神，无论对方什么有龌龊心思，不过就是兵来将挡，水来土掩，自己一剑在手，又何惧之有。
<br/>　　此时此刻，黄为善已经领着大军，快马加鞭返回黄土镇。
<br/>　　“大家快些，快些！”
<br/>　　黄为善骑马走在队伍最前方，他今日白天入城，托老父的关系，又用了不少银子，才见到兖州城有兵权的人。
<br/>　　好说歹说，才出动兖州的方石隆，引得一千兵马。
<br/>　　“好混蛋，若到时候不是那人，你敢耍老子，你黄家几十口的人头就都别要了。”方石隆骑在马上道。
<br/>　　“方将军，千真万确，只要抓到那人，方将军也是大功一件。”
<br/>　　“抓到……能杀了就不错了。”方石隆道了一声：“唉，借你吉言吧。”
<br/>　　一千人马奔向黄土镇，黄沙古道上烟尘滚滚。
<br/>　　这个时候，天已经黑下来，黄土镇外十里处，埋伏着一队人马。
<br/>　　长须鬼孙山起，饿死鬼朱刚，白面鬼张萌等鹰嘴岭八位当家，都出现在此地。
<br/>　　一匹快马疾奔到此地，马上人道：“诸位爷，已经打探清楚，姓程的现在就在黄土镇的黄家，黄家张灯结彩，摆下了宴席，一帮人正在吃酒。”
<br/>　　“好，咱们早就想动黄家，今天正好一并抢了黄家。”饿死鬼朱刚道。
<br/>　　“吃酒，吃得好酒。”张猛呵呵冷笑：“程大雷，今日我就将你人头当酒壶，祭我大哥在天之灵。”
<br/>　　一共三百余人，今夜鹰嘴岭是倾巢而动，势要血洗黄土镇。而十里地对于这些山贼来说，也不过是眨眼就能到的距离。
<br/>　　黄土镇外，一棵枯死的老树下，一个老者拄着木杖，剧烈咳嗽一阵，引得头顶老鸦呱呱大叫。
<br/>　　“滚开啦，我还没死呢，等着啄我的尸，你们还得等好久呢，都毒死你们。”
<br/>　　老者恶狠狠的骂了一阵，吐出一口浓痰。
<br/>　　这当然就是从大雪峰一路过来的孟玄青，他一大把年纪，能走到此地可不太容易。
<br/>　　却是没想到，能在此地碰见程大雷。他已经思考太久，看有没有杀死程大雷的机会。
<br/>　　出门太急，没有带什么毒药，用普通毒药，似乎也显不出西蜀巫医的本事。
