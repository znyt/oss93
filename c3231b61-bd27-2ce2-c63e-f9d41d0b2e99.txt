　　千选战结束了，人也陆续的出来了，苏莫和公孙长绝等人，均是通过神昆小世界的出口出来了。
<br/>　　对于这个结果，有人欢喜有人忧。
<br/>　　“苏莫，我朝太子呢？”
<br/>　　“苏莫，贺兰殇呢？”
<br/>　　苏莫乍一出现，魔长烽和不死族的一位长老，便立刻大声质问了起来。
<br/>　　苏莫闻言，立刻心中恍然，看来龙腾并没有将古魔太子放出来。
<br/>　　“贺兰殇没事！”苏莫沉声说道，随即，他意念一动，吞噬元神释放而出，一道人影瞬间便被抛了出来，正是贺兰殇。
<br/>　　“苏莫，你找死！”
<br/>　　贺兰殇乍一出来，便怒发冲冠，不过，看到此刻自己已经在神昆小世界之外，他还是压制住了心中的怒火。
<br/>　　“苏莫，我朝太子呢？”一声暴喝，震动天地，魔长烽一步踏出，瞬间来到了苏莫的身前，滚滚魔威狠狠的压迫到苏莫的身上。
<br/>　　“魔长烽，你发什么疯？”龙昌身形一闪，便挡在了苏莫的身前，直面魔长烽。
<br/>　　“龙昌，我朝太子和苏莫一战，现在生死不明，你说此事难道与苏莫无关？”魔长烽冷声道。
<br/>　　“古魔太子或许迷失在了虚空深处！”龙昌道。
<br/>　　“可笑！”魔长烽闻言，立刻冷笑一声，道：“你当我朝太子是虚神境的武者吗？”
<br/>　　此言他根本不信，以古魔太子的修为，区区那片刻的功夫，怎么可能迷失在虚空深处？
<br/>　　“苏莫，你知否古魔太子在哪里？”龙昌没有回到魔长烽，而是向苏莫问道。
<br/>　　“不知！”苏莫摇了摇头，既然龙腾没有将古魔太子放了，自然是有自己的打算，他肯定不能说出来。
<br/>　　当然，他也不清楚，龙昌大长老是否知晓，还是故意装模作样的询问。
<br/>　　“你看，苏莫并不知晓古魔太子在哪里，至于怎么找到他，那就是你们魔族的事情了！”龙昌向魔长烽说道。
<br/>　　魔长烽面色难看，因为，他之前已经暗中用族中秘术查探过了，都没有感知到古魔太子在何地。
<br/>　　“好了！”
<br/>　　这时，未天神王朗喝一声，道：“千选战已经结束了，神域神阵更换阵基，将由龙族进行，大家将控制神阵的九块阵牌，交于龙昌大长老吧！”
<br/>　　中众人闻言，面色各异，但是，对于这种规矩，他们也无法提出异议。
<br/>　　随即，未天神王、魔长烽等各大神朝之人，纷纷将一枚黑色的玉牌，交给了龙昌。
<br/>　　这黑色的玉牌，乃是控制神域神阵的神物，非同小可，只有集齐九枚，才能操控神阵。
<br/>　　九大神朝，分别掌管一枚玉牌，轮回神朝是由兽王宫掌管，天池神朝是有一个叫御天神王殿的实力掌管。
<br/>　　“十年之内，阵基便会更换完毕！”龙昌收起九枚玉佩，郑重的说道，所谓的更换阵基，就是更换支撑神域的星辰。
<br/>　　神域的大阵，阵基便是宇宙中九十九万九千九百九十九颗星辰，而以前所用的那些星辰，灵气都要耗尽了，变成废弃的星辰。
<br/>　　所以，要重新选择九十九万九千九百九十九颗灵气浓郁的星辰，如此，神域才会更加稳固，源气才会更加的浓郁。
<br/>　　“嗯，有劳龙族了！”
<br/>　　不少强者纷纷点头，越早将阵基更换，对于神域来说就越好。
<br/>　　“诸位，既然今日事已了，那老夫便返回龙都城了！”龙昌向众人抱拳，朗声说道。
<br/>　　“后悔有期！”不少强者拱手。
<br/>　　“我们走！”随即，龙昌招呼一声龙族的众人，便飞天而起，一群人浩浩荡荡的向天龙神朝的方向而去。
<br/>　　见龙族之人离开，不少人眸中闪过异茫，但是，并没有人阻拦。
<br/>　　而后，其它各大神朝各大势力之人，也纷纷告辞，返回各自所在的势力。
<br/>　　……
<br/>　　长空之上，龙昌带着苏莫等人，向天龙神朝疾驰而去。
<br/>　　“苏莫、龙腾，那古魔太子到底在哪里？”
<br/>　　开口之人是龙族二长老龙柏，他转头凝视着苏莫和龙腾，之前两人不方便说，但现在已经没有了外人。
<br/>　　众人闻言，纷纷转头望着两人，他们都很想知晓。
<br/>　　“古魔太子在祖龙塔之中！”龙腾沉声说道。
<br/>　　“祖龙塔！”众人闻言，纷纷心中一震，果然啊，古魔太子败了，还被生擒住了。
<br/>　　“还是把他放了吧，不然的话，亘古魔国不会善罢甘休的！”龙柏微微皱眉，沉声说道。
<br/>　　“就算放了，我估计他们也不会善罢甘休！”龙昌眼眸微眯着说道，言罢，他大袖一挥，劲力卷住所有人，道：“为防事情有变，我们必须尽快返回龙都城。”
<br/>　　言罢，龙昌大长老立刻加速，速度暴增十倍，向天龙神朝火速而去。
<br/>　　这一次的千选战，诡异无比，很多人都想对付苏莫，所以他不得不提防。
<br/>　　至于古魔太子，先将此人带回龙都城再说。
<br/>　　“大长老，亘古魔国，霸天神朝，不死神朝应该都已经站在了帝一魂一方！”飞行之中，苏莫向大长老说道，这是他最为担心的。
<br/>　　亘古魔国站在了帝一魂一方，这是可以确定的，霸天神朝和不死神朝等，暂时还不能确定。
<br/>　　“此事，回去之后再做商议！“龙昌沉声说道，他自然看出了此事的不同寻常。
<br/>　　但是，现在，他必须尽快赶回龙都城，多留一刻，就多一分变数。
<br/>　　连续飞行了几个时辰，一直都相安无事，很快，众人进入了幽冥神朝境内。
<br/>　　幽冥神朝很大，跨越幽冥神朝之后，距离天龙神朝就不远了。
<br/>　　不过，就在众人进入幽冥神朝境内没有仅仅半个时辰，异变陡然升起。
<br/>　　天穹之人，大长老龙昌和二长老龙柏，裹挟着所有人，正在疾驰，突然，一根根粗大的光柱，在周围万里之内冲天而起。
<br/>　　这些光柱呈现青紫色，足有数千根之多，形成了一个巨大的圆形，将众人包围在了中央。
<br/>　　而此刻，这每一根光柱之人，都伫立着一道身影。
<br/>　　“不好！”龙昌面色一变，立刻停下了身影，眸中闪过犀利的精芒。
<br/>　　“是幽冥太厄古阵！”二长老龙柏的面色，瞬间铁青了下来。
<br/>　　而苏莫和龙腾等百余人，此事个个心中惊疑不定，这幽冥神朝居然在此设伏。
