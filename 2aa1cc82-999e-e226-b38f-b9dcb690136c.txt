　　人有人道，邪魔有邪魔的道。
<br/>　　邪魔千变万化，各有长处，人力所不能及。于是人类才发明了各种法术来对付邪魔，道门的四柱五行、九宫八卦，佛门的三定五门、六字真言等等，这些都是人之道，邪魔是没法修炼和掌握的，只有一种例外，就是鬼术，也就是法师死后，将一身罡气转化为魂魄可以掌握的气息，尽最大可能保留修为，并且按照生前掌握的法术规律来修行。
<br/>　　碧清既然能够将封印自己的阵法吸收、收为己用，那只有一种可能：她曾经是人间法师！
<br/>　　短短的一瞬间，叶少阳想到了这许多，并且通过推理得出这个判断。
<br/>　　“你们修为都不错，正好拿来祭阵，也够我吸收一段时间的！”碧清的声音，从四面八方传来。
<br/>　　声音刚落，五行阵突然旋转起来，一股股恐怖的力量，从五个方向铺天盖地地涌来，叶少阳一行人立刻感受到了巨大的压力，好在这几位也都是强者，联手一起，一时间也能挡住，不过都知道支撑不了多久。
<br/>　　“我们结阵抵挡，你们按我说的方位站好！”
<br/>　　妙心指挥大伙站好，自己手持法杖，念诵咒语，在大伙的配合之下，临时布了一个阵，将在场几个人都罩住抵挡着五行阵中层层涌来的恐怖力量和那些厉鬼的攻击。
<br/>　　五行阵运转了一会之后，静止下来，大伙松了口气，但是没过多久，五行阵再度运转，一时间火光冲天，直逼众人，就算有结界抵挡住了大部分威力，众人仍然感到炙热难耐。
<br/>　　“这是地狱烈火！”叶少阳悚然说道，心中更加骇然，妈个蛋，这妹子居然能操控地狱烈火！
<br/>　　“是阵法，阵法召唤来的。”妙心说道，“之前一波是罡风，现在是火海，之后还有落岩、冰芒和诛心剑……这是五行杀阵……被发动到这种强度，正是不可思议，我们撑不到第五波的。”
<br/>　　叶少阳一边作法，一边问道：“那怎么办？”
<br/>　　“阵法运转，五行相生相克，必然有生门和死门，在不断变幻……每一次转变方位，生死二门也在转变，所以我们现在暂时安全，但如果我们站着不同，这里早晚还是会转变成四门，所以攻击是一波一波的，五波之后，循环往复，没有终结。我们只有不断移动，变幻方位，才能暂时保证安全……”
<br/>　　梧桐听她说完，立刻问道：“这里一共被阵法分割出六个空间，生死各半的话，有三处为生，三处为死，我们怎么知道往哪边移动，万一撞进死门呢？”
<br/>　　“只能试试了，我一时之间也没法断出阵法机理……”
<br/>　　生死有命，至少有搏一把的机会，最多也是九死一生，一直站在这，那是必死无疑。因此，大伙在听了妙心的话之后，也都没有意见。
<br/>　　妙心左右查看一番，被光线分隔开的其余几个空间，从外面看都是风平浪静，实在不知道该去哪一个，只好随便选中一个，让大家等着，自己去试试。
<br/>　　叶少阳一把拉住她，说道：“你这一去，万一是死门，就出不来了。”
<br/>　　“那怎么办，在这等死吗？”
<br/>　　“跟着我走，保证安全！”
<br/>　　叶少阳说完，朝着右侧一个空间跨了过去，众人没拦住，叶少阳一步迈了进去，立刻被空间隐藏，看不见了。
<br/>　　大伙面面相觑，谁也没想到叶少阳这么果决，说走就走。
<br/>　　“要过去吗？”妙心怔怔地问。
<br/>　　“试试，我相信少阳子！”毛小方说完自己也迈了过去。
<br/>　　梧桐和妙心互相看一眼，点了点头，先顶住那些鬼魂的攻击，让卢晓清把陈晓宇唤醒，带着他的猴王一起过去。
<br/>　　“你们去吧。我不走！我不相信他！”陈晓宇大声嚷着。
<br/>　　在这生死攸关的时刻，没人理会他负气的话，反正他也清醒了，于是不管他，陆续跳过了空间之间的连接线。
<br/>　　就剩下陈晓宇自己一人了，呆呆着看着众人消失的方向，叹了口气，也跟着跳了过去。
<br/>　　要死大家一起死。
<br/>　　可万一他们都没死，就自己一个死了，那多倒霉？
<br/>　　陈晓宇抱着这个态度，怎么也要跟上大部队。
<br/>　　隔壁的空间，果然风平浪静，连那些鬼魂的侵扰都没有。
<br/>　　大伙都长出了一口气。妙心想到什么，问叶少阳：“你是蒙的？”
<br/>　　叶少阳摇摇头，“既然是五行阵，那就要遵循五行的规律，二十五种可能的变化，万变不离其宗，只要掌握规律，就知道生门在哪了。”
<br/>　　妙心皱眉道：“二十五种变化，每一种都有可能，你怎么确定哪一种是对的？”
<br/>　　“我自有推演的办法。”叶少阳笑了笑，这个不好解释，也没时间解释：自己会先天八卦，先天八卦，不光是卦象，五行八卦本为一体，在先天八卦的基础上，叶少阳能推演出各种基于五行八卦为基础的阵法机理，这在后天八卦上是不可能的。
<br/>　　就好像是掌握了乘法九九表之后，能轻松算是各种数字的乘法，对不懂的人来说就很不可思议。
<br/>　　等了一会，叶少阳又指了一个方向，钻进了相邻的另一个空间，果然又是安全的。
<br/>　　“大师兄，你怎么也跟来了？”叶少阳冲陈晓宇挑了挑眉毛。
<br/>　　陈晓宇脸色通红，心中憋屈到了极点，但是不敢对叶少阳发作。
<br/>　　在阵法变幻的间隙，叶少阳带着大家不断移动方位，钻进不同的空间，一路安全，没再遇到过危险，连那些厉鬼的侵扰也不再有。
<br/>　　“我们就这样一直躲下去？”梧桐问道。
<br/>　　“虽然这五行杀阵是借助场地形成的，但是驱使起来也是费力气的，我就不相信她能一直陪我们玩藏猫猫。”叶少阳皱着眉头说道，“我只怕，她还有别的手段。”
<br/>　　碧清的声音，再度从不知道哪个方向传来，声音里也带着一些吃惊：“你为什么可以摸到五行杀阵的规律？”
<br/>　　“雕虫小技，这算什么。”叶少阳故意夸张地说道。
<br/>(本章完)
