　　那是一个叫墨瑟的新人类，不知道等阶为几，不清楚他的底细，但曹春辉似乎需要墨瑟帮什么忙，前后态度客气异常，而墨瑟来此底下研究院与曹春辉面谈的某次，也不知是故意还是巧合，弄坏了某个重要设备，导致这座底下基地的系统有几分钟的失灵。
<br/>　　曹春辉没有责怪墨瑟，因为底下研究院并没有发生什么不对劲的状况，但墨瑟在此之后也再也没能进入这里。
<br/>　　不过曹春辉并不知道，就在那系统失灵的短短几分钟内，有几名在这座巨大研究院内打杂的年轻人，出其不意的路过胡梦浩的所在地。
<br/>　　与他进行了短暂无比，不过十几秒的碰头。
<br/>　　正是这十几秒的时间，让胡梦浩的暴躁情绪一扫而空，并在之后很长的一段时间里都隐忍不发，等待时机。
<br/>　　那几名打杂人员对他发出了暗号，正是当初他与楚涵达成共识之际，被告知的特殊暗号！
<br/>　　这几人正是混入了这座研究院的狼牙暗部成员，只不过由于种种原因无法表明身份，也无法向外界传递情况，但这长久时间的埋伏，也让他们对这座研究院的里里外外都越来越熟悉。
<br/>　　狼牙暗部混入研究院的目的，除却要帮助胡梦浩牵制曹春辉，并且挖出这座研究院的背景之外，还有更重要的一点就是想办法救出楚云天。
<br/>　　任务下达之时，有关楚云天的身份并没有说明，但根据其姓氏及年龄，有点脑子的人都能从中看出此人与他们最高长官之间的联系。
<br/>　　所以当这批成功混入曹氏研究院，在此处长期埋伏的狼牙暗部成员，第一次见到了楚云天本人时，那种激动的情绪根本难以言喻。
<br/>　　楚云天这个名字在曹氏研究院内并不陌生，因为相比较胡梦浩这位副院长的刻意无作为，楚云天在很多研究上都有独到见解，是曹氏研究院研究项目上推进的重要功臣。
<br/>　　所以曹春辉对楚云天很看重，很多重要项目都交给他来做。
<br/>　　楚云天自末世爆发后不久，就一直属于被曹氏研究院控制的状态，他不知道外面的情况如何，更不清楚当下自己的妻儿是否活着。
<br/>　　胡梦浩那种与狼牙暗部相认的情况，也自然无法在楚云天这里实施。
<br/>　　于是一干狼牙暗部的成员就只能干着急，胡梦浩也毫无办法，研究院内没有可以绝对的私聊机会。
<br/>　　尤其是像胡梦浩身为副院长，还是前门萨内排名相当高的成员，与楚云天这般同样出自门萨排名第十的高智商者的任何接触，别说闲聊，哪怕一个走廊偶遇的相互点头，都会被细节放大监控！
<br/>　　所以在很长的时间里，楚云天毫不知情的进行着研究，胡梦浩和狼牙暗部的成员却连表明身份都难，更无法向其传达有关楚涵的消息。
<br/>　　这一状况一直维持着，直到忽然有了银市倾巢大战，紧接着很快分界线南北联盟军的碰撞，而后南方大尸潮爆发，一桩又一桩的战事接二连三，震荡了整个华夏之后，研究院内的曹春辉也隐隐不安了起来。
<br/>　　于是在南方大尸潮战役以狼牙战团大获全胜告终后，这座底下研究院内忽然来了一批贵客，曹春辉院长不仅亲自迎接，甚至还叫上了副院长胡梦浩和楚云天全程陪同。
<br/>　　贵客一来就进入了研究院内最豪华的会客间，此处常年不会开放，甚至胡梦浩和楚云天都是今天才知道，原来研究院内还有这样一间屋子。
<br/>　　其内的豪华程度无法用语言表达，总之极尽奢侈！
<br/>　　这一批贵宾很奇怪，大部分人都分散在这间屋子的各个角度，就像是在守卫。
<br/>　　唯独一人背对着所有人，随意的站姿带着一丝优雅，正在观赏墙上的壁画。
<br/>　　曹春辉带着楚云天站在此人身后，胡梦浩由于双·腿已断常年坐着轮椅，才有幸成为这间屋子里唯一坐着的人。
<br/>　　只不过这种坐在当下，更加显得像是怜悯。
<br/>　　这一喧宾夺主的状况不仅没有让性情阴狠的曹春辉不满，反而连头都不敢抬，卑微无比的安静站在原地，似乎前方背对着他们的那个人一万年不转身，他就会如此卑微的低头站在这里一万年都一动不动。
<br/>　　胡梦浩好奇，楚云天也好奇，两人都忍不住悄悄看前方，想要知道眼前这位贵客是何方神圣。
<br/>　　豪华的会客间内一片安静，带着压抑的气氛持续了良久，曹春辉的额间都已经溢出密密麻麻的汗水。
<br/>　　此时那位贵客似乎看够了壁画，他没有转身，就这般背对着所有人乍然开口：“曹春辉，南署基地的大尸潮一战，你怎么看？”
<br/>　　曹春辉一个哆嗦，挤出笑容回答道：“恕我这里位置远，对南方的战事，消息还不灵通，但听说是大胜。”
<br/>　　“既然消息不灵通，那我也不妨让你知道一些。”贵客的声音平淡，继续道：“南署基地的大尸潮战役，尸潮数量为一千五百万，狼牙战团人数为九万，战役时长为七天整，战绩为一千五百万丧尸全灭一个不活，狼牙战团伤亡一万多人。”
<br/>　　毫无情绪的声音，配合着这一系列惊心动魄的数字，让曹春辉的心脏剧烈跳动。
<br/>　　胡梦浩愣了愣，他当然知道狼牙战团是楚涵的麾下战团，当听闻这一战绩的时候，也差点没绷住情绪。
<br/>　　真特么牛啊！
<br/>　　楚云天是唯一不清楚状况的人，但也单纯的对这一惊人数据，表现出了震惊。
<br/>　　这个狼牙战团，厉害的逆天啊！
<br/>　　“你怎么看啊？”贵客再次发问。
<br/>　　曹春辉一瞬间冷汗浸·湿了后背，面上满是惊恐的情绪：“狼牙，呵呵……”
<br/>　　这时候的贵客缓慢转身，露出了他明显区别于亚洲的容貌，异色的瞳孔里尽是冷漠。
<br/>　　他盯着曹春辉开口：“顺便一提，这次大尸潮战役的战前准备及过程中，狼牙战团大肆用到的药剂有三种，分别是丧尸引诱剂，新人类觉醒剂，等阶提升剂，这三种药剂都是根据丧尸研发出来的。”
<br/>　　“我说曹春辉曹院长，你研究丧尸这么长时间了，怎么一样东西都没研究出来呢？”
