<br/>				过了几天，陈默除了去父母哪里不时的看看，陪陪父母外，其他时候都窝在老宅这里，每天除了修炼就是躺在那里养神。<br/>　　<br/>　　是真的养神，丝毫没有其他动作和学习，只是闭目养神。<br/>　　<br/>　　这是因为，这几天他每天都要去乾坤珠的灵兽养殖区域看看，但是每次进去就会被压制住。<br/>　　<br/>　　到现在为止，还真的没有看清过那个巨大的阴影是什么样子呢，每次也坚持不了多久就会被送出来，不过倒是能锻炼他的神识。<br/>　　<br/>　　就这么几天，神识已经增加到了12米以上，还真是厉害。所以他真的非常欢喜，天天都要这么去折磨一下自己，然后利用一阵天来恢复神识。<br/>　　<br/>　　不过他的这种情形，比一些老头还老头子，也让何老那帮子人看了后，都有些嫉妒。<br/>　　<br/>　　这么年轻的家伙，又有这么好的技术和医术，还这么懒散，这不是浪费是什么？<br/>　　<br/>　　不过前几天才和陈默聊过，如果现在再去说他，可能陈默就不是和颜悦色了，而是一口口水吐到脸上了。<br/>　　<br/>　　再说了，无论怎么样，陈默也是有可取之处的，不仅能够自己研发植物营养液，还能凭借这个东西给自己带来丰厚的报酬。<br/>　　<br/>　　所以，教育的话就少说，只能渐渐引导就是了。不过这样下去也不是事情，还是要给他找些事情做的。<br/>　　<br/>　　其实何老他们这么关心陈默的工作，也是有一些原因的。何老他们一帮老人，经过上次吴老的身体检查后，还有这次纪老的身体恢复以后，所有知道情况的人们都有些眼红了。<br/>　　<br/>　　谁不想身体健健康康的，谁不想能吃能睡？尤其是这些人，有权有势的，当然是最好的服务最好的医疗了。<br/>　　<br/>　　所以知道陈默的医术这么好，当然都是希望能让他给自己看看，有病治病，无病保健。<br/>　　<br/>　　但是陈默却是这么懒散，又不出诊，所以何老很是傻眼。而且，由于陈默将剩余能接受病人数量交给何老负责，其实也是一个麻烦。<br/>　　<br/>　　不过好在何老还是很有威信的，安排了几个急需救治的老人，可能在过几天就能送过来了。<br/>　　<br/>　　病人转到陈默这里，不是一个容易的事情，首先就是要经过一系列的检查，还有手续，另外还需要很多人的首肯。像他们这种人，要是在转移过程中~出了事情，那么谁来负责？<br/>　　<br/>　　所以商定好的几个人，也是安排的伤病比较多的人，成天因为疾病的困扰而疼痛等等。<br/>　　<br/>　　陈默没有想到何老等人的心思，是那么的弯弯绕绕的。他想的就很简单，让自己出诊的话，那是绝对不能去的。不过来到自己这里的，那么能治疗的就治疗。<br/>　　<br/>　　前几天，陈默开车去了一趟镇上，参加镇子里的集市，买了十来只小狗，然后让人做了一溜的狗窝，放在了菜地区域。<br/>　　<br/>　　这是上次答应陈金贵的事情，一直拖到现在才做到的。主要是考虑到晚上，就陈金贵一个人看守菜地，有些人力不足，那么多养些狗，也能起到警戒作用。<br/>　　<br/>　　另外陈默还找上席止涵，让他将安防监控设备增加一部分，在蔬菜区域和养殖区等等都增加一些摄像探头，用来保障这里的安全。<br/>　　<br/>　　其实这些手段都是陈默明面上的，实际上还是要靠他自己布设的感应阵法。<br/>　　<br/>　　就这么一个星期左右，外边通向葫芦谷的双向车道已经全部都敷设完毕，清一色的硬化路面，而且是按照省道施工，标准的路灯杆数，以及排水沟渠、强弱电管道等等，全部都是最高标准施工。<br/>　　<br/>　　葫芦谷内的工程也在顺利进行中，不过由于工程两比较大，还不到一半的工程施工完成，看来还有的等。<br/>　　<br/>　　不过这一阵陈默的别墅盖好了，并且经过装修后，已经差不多能够住人了。陈默去验收了一下，还是不错的。<br/>　　<br/>　　陈明没有糊弄，都是严格按照要求施工，并且所有的装修材料都是用的最好的材料。整个别墅4层加一层地下室，溜达了一圈，感觉还不错，也就和明叔做了最后的结账。<br/>　　<br/>　　然后就是找人打扫别墅，购买家具等等。所有的东西都安装完毕后，他自己又走了一圈，就是给每个房间施放洁净术，将房间中的潮气、家具的气味，装修的气味、灰层等等全部清理干净。<br/>　　<br/>　　转着圈看了一边后，没有什么问题了，这才开车去父母哪里。他准备让父母搬过来住着，然后将父母的那套院子拆除，重新翻盖。<br/>　　<br/>　　这件事和陈明说过，也让他回去准备了。但是还没有和父母商量过，一直拖到现在，都不知道父母同意不同意，不过自己要是好好劝一下，另外再将弟弟陈辉的名头用一下，可能父母也会同意。<br/>　　<br/>　　这一段时间，随着陈默别墅建成后，村里人也是非常的眼热。没有想到二娃回来这么几个月，就弄下了这么大的阵势，真的是让人意外有羡慕嫉妒。<br/>　　<br/>　　所以很多村民都是跑到家里，和陈建国、付慧丽两个人拉家常，另外就是借钱等等。<br/>　　<br/>　　有的时候，农村人借钱非常的让人无语。都是十块二十块，有的时候五十的借。但是却不会超过一百元，这样借钱，很多时候是不准备还的。<br/>　　<br/>　　农村人有的时候也是非常讲究的，如果超过一百，那么砸锅卖铁的都会还了。就像是陈默一家，以前陈建国借的钱，有几百也有几千，这些钱每一笔都记录在一个小本本上，等家里有钱的时候，就会一笔笔的归还。<br/>　　<br/>　　有的比如一百二十块钱，都会完整的还回去，并且还会在还钱的时候买些礼品什么的，作为还钱时候的敬意。<br/>　　<br/>　　当然，如果借钱的时候，讲了利息的，那么一笔笔也会计算清楚后归还。对于这种钱，农村人是非常讲究，一定会有借有还的。<br/>　　<br/>　　不过，要是借的钱不超过一百元，零零碎碎的，而且还是那种家里富裕的人，那么在村里人来讲，这就是打秋风，也就是说你家富裕了，那么就过来蹭些甜头等等。<br/>　　<br/>　　这钱，一般是打着买东西钱不凑手，或者差几块钱等等，就跑过来找你借钱。有的时候甚至就是借个几毛钱，再或者要些菜油、盐、酱油之类的东西。<br/>　　<br/>　　这个钱，脸厚心黑的人，倒是会记录在书本上，然后有借有还就是了。但是放在普通好面子的人身上，那么就是肉包子打狗，一去不回！<br/>　　<br/>　　有的时候，你不想借钱出去都不行，别人会在后面指着你的脊梁骨，然后到处传扬你有多么抠搜，多么的吝啬，将你的口碑弄臭。<br/>　　<br/>　　所以，有的时候在农村生活，真的有些无奈，家长里短的传的是全村人都能知道。<br/>　　<br/>　　现在，陈建国和付慧丽也就是在经历这种状态。以前是没有钱，也没有什么人来找他们。但是自从陈默的别墅建立起来后，看的人多了，有心的人也就多了。<br/>　　<br/>　　大家虽然不知道陈默的蔬菜卖多少钱，但是知道一个4层别墅的小院子，能值多少钱。也就是从这里，就可以估算出陈默大概的金钱。<br/>　　<br/>　　因此，七大姑八大姨的，全部有事没事情的到了陈默家里，陈建国和付慧丽有时候抹不开面子，就会借出去些钱，也能想到有很多都还不回来了。<br/>　　<br/>　　而且，还有各种亲戚上门，给陈默介绍对象的，那栋别墅屹立在哪里后，村里很多说媒的人就自觉的朝家里跑。<br/>　　<br/>　　：。：<br/><br/><br/>Ps:书友们，我是小学嗣业，推荐一款免费小说App，支持小说下载、听书、零广告、多种阅读模式。书友们快关注起来吧！<br/><br/><br/>
