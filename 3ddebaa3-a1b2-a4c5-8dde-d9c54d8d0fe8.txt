　　这一刀来的毫无征兆，威力极大，看攻击而来的方向，貌似是第四城之人。
<br/>　　咻！
<br/>　　玄力的刀光，撕裂空间，发出刺耳的厉啸之身，瞬间就临近了龙双双。
<br/>　　苏莫见此，眼眸一凝，不过，他并未出手救援，龙双双可是能登上祖龙塔第八层的人物，不可能会被轻易击败。
<br/>　　果不其然，龙双双的反应极快，陡然转身，一抓凌空抓出。
<br/>　　霎时之间，一道巨大的爪影，如同上苍之手，狠狠的抓在了刀光之上。
<br/>　　轰！
<br/>　　一声爆响，震动整个神昆小世界，刀光被抓碎，爪影也同样崩溃，化为浩大的余波，横扫八方。
<br/>　　“谁？”龙双双冷喝，目光环视前方，想要找到攻击之人。
<br/>　　不过，攻击之人并未现身，她只能感觉到前方的空间有细微的波动，此人隐匿了身形。
<br/>　　“苏莫，我们要不要派人前去相助？”第三城之中，公孙长绝遥望着远处的情况，低声向苏莫问道。
<br/>　　“不用！”苏莫摇了摇头，龙双双不会轻易被击败，再者，他要看看第二城的态度。
<br/>　　任谁都能看出来，龙双双是要去第二城，那么第二城是什么态度呢？
<br/>　　而且，龙羽在第二城，此人若是在第二城有极大的话语权，应该不会坐视不理。
<br/>　　“滚出来！”
<br/>　　高空之上，龙双双一声娇喝，双手齐出，不断的向前方抓去，道道爪影铺天盖地的击出。
<br/>　　此人隐藏的很好，但是，此地地域不大，她轻易就能逼对方现身。
<br/>　　只要擒拿了对方，就能逼问出对方是第几城之人，以及统领是何人？
<br/>　　不过，就在此刻，异变再起，只见第四城之中，陡然飞出了四人，齐齐向龙双双出手了。
<br/>　　剑气、枪芒、拳劲、掌印，四道攻击裹挟滔天的威势，排山倒海一般袭向龙双双。
<br/>　　而那位隐藏在暗中之人，不知什么时候，居然跑到了龙双双的后方，也在同一时间出手了，一只巨大的源力手掌，向龙双双猛然抓去。
<br/>　　“不好！”苏莫见此，心中一惊，但是此刻，他就是想要救援，也已经来不及了。
<br/>　　这出手的四人，倒不是什么有名的人物，都是天池神朝不同界域之人。
<br/>　　眼见龙双双仿佛面临巨大的危险，第二城之中，终于有反应了。
<br/>　　吼！吼！
<br/>　　只听第二城之中，陡然传出了两声震动天地的龙啸之声，恐怖的音波浩荡而出。
<br/>　　这是有两位龙族之人，施展了洪荒龙息，强烈的龙息如同凶猛的巨兽，一股攻向了龙双双后面的源力大手，一股攻向了另外四人的攻击。
<br/>　　轰！轰！轰隆隆！！
<br/>　　几人的攻击，纷纷相击在了一起，爆响不断，爆开的冲击波如同蘑菇云一般升腾而起。
<br/>　　“双双，快进来！”
<br/>　　就在此时，一道声音响彻而起，只见第二城的护城阵法，裂开了一道口子。
<br/>　　这声音并不是龙羽的声音，但显然也是龙族之人。
<br/>　　龙双双眼疾身快，身形一闪，便通过大阵的裂缝，进入了第二城之中。
<br/>　　而那第四城的几人，也没有多留，同样山巅般的退回到了第四城。
<br/>　　“诸位，这几人是什么人？”苏莫望着第四城的方向，向众人问道。
<br/>　　“这四人分别是大夏神王殿、御天神殿、烈焰神殿和雪月神殿的人，都是天池的势力！”有人沉声说道。
<br/>　　苏莫微微的点了点头，至于那位隐藏之人，应该是暗影族，他对暗影族颇有了解，应该不会错。
<br/>　　这第四城，倒是隐藏的极深，虽然出来了五人，但是底细完全没有暴露。
<br/>　　苏莫静静的等待了起来，等待龙双双归来。
<br/>　　第二城之中。
<br/>　　龙双双身形一闪，便落在了城中，目光环视左右。
<br/>　　此时，她的眼前有众多的人，比之第三城还要多几人。
<br/>　　龙羽、贺兰殇、阎公子均在，还有其他近二十位本身修为在九品之人。
<br/>　　总体而言，第二城的实力，比第三城要强大不少。
<br/>　　嗖嗖嗖！！
<br/>　　龙双双刚一落下，便被一众第二城的人包围了起来，几乎是围的水泄不通，插翅难逃。
<br/>　　“你来此意欲何为？”
<br/>　　“第三城之中有什么人？”
<br/>　　“告诉我们第三城的情况，不然你就出局吧？”
<br/>　　不少人纷纷开口，面色冷然，若不是因为龙羽和其他两位龙族之人，他们立刻就会将龙双双擒拿。
<br/>　　“好了，诸位肃静！”
<br/>　　龙羽上前一步，抬手示意众人安静，随即对龙双双问道：“双双，第三城都有什么人？你来此的目的是什么？”
<br/>　　“龙羽哥，我是奉了公孙长绝的吩咐，来和你们结盟的！”龙双双沉声说道。
<br/>　　“哦，公孙长绝在第三城？”贺兰殇闻言，立刻上前一步，目光凝视着龙双双的眼眸，他想要看出龙双双是不是在骗他们。
<br/>　　毕竟，现如今的九城争霸，除了自己人之外，任何人都不能信任。
<br/>　　“嗯，他之前来探查过，知道你们第二城的情况，所以让我前来！”龙双双郑重的道。
<br/>　　“第三城还有什么人？”
<br/>　　不远处，一身暗红色长袍的阎魔鬼圣，淡淡的问道。
<br/>　　“我们第三城没有什么强者，除了公孙长绝之外，就只有魔野、力闫等少数几人！”龙双双道。
<br/>　　“是吗？”阎魔鬼圣面色淡淡，对于龙双双的话，他只信一半。
<br/>　　“公孙兄的意思，是我们两座城池联合起来，如此才能击败其它城池！”龙双双又道。
<br/>　　言罢，他瞥了龙羽一眼，眼眸深处闪过微不可查的异色。
<br/>　　龙羽见此，立刻心领神会，龙双双是在告诉他，苏莫在第三城。
<br/>　　虽然他心中不喜苏莫，但他心中明白，苏莫也是代表他们天龙神朝，所以，他会以大局为重。
<br/>　　“我同意和第三城结盟！”龙羽朗声说道，既然苏莫在第三城，那么他自然要结盟。
<br/>　　以苏莫的强势，想必在第三城之中，必然是统领一城之人，公孙长绝也不太可能左右苏莫。
<br/>　　众人闻言，面面相觑，龙羽和龙双双都是龙族之人，自然愿意联合，但是，他们却要考虑各方面的情况。
<br/>　　贺兰殇和阎魔鬼圣对视了一眼，均是沉默不语，心中暗自思量。
<br/>　　“贺兰兄，阎兄，你们觉得如何？”龙羽看向贺兰殇和阎魔鬼圣，沉声问道。
<br/>　　现如今，这第二城之中，他并不能言出法随，而是要和贺兰殇、阎公子商议。
<br/>　　第二城的行动，由他们三人商量着决定。
