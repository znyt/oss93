　　一群没有理论基础，但确有一定操作经验的方士被李承乾忽悠的头晕眼花，最后在自己都不知道发生了什么的情况下，上了李承乾贼船，稀里糊涂的就开始了制备感光材料的工作。
<br/>　　当然，这个过程李承乾也不是闲着没事儿作，他必须在现场看着，时时指导这些新手如何操作，操作时要注意什么。
<br/>　　就这样，时间不知不觉的一点点的流逝，直到半个月之后，所有感光材料和显影液全部配制完成。
<br/>　　一块块巴掌大小的玻璃被切割好放在一起，一辆马车被封死了所有的窗子，只留下一个可以出入的小门。
<br/>　　马车的车厢上被开了一个洞，在洞的位置上镶嵌着一块凸透镜，并且在里面用一块黑色的帘子挡住。
<br/>　　液式显影法，用特制的感光材料均匀的涂在玻璃上面，然后在玻璃未干的时候就开始照相，曝光时间五到十五秒。
<br/>　　然后还要在玻璃未干的情况下就开始显影、定影，在玻璃上形成负片，否则的话前面的一切工作就全白费了。
<br/>　　当然，定影之后就不再需要注意了，只要找一个合适的时机把玻璃上的负片影印到纸上便可以了。
<br/>　　这些东西说起来简单，但实际操作起来却需要一定的经验，比如感光材的配比，曝光的时间、显影、定影液的配制以及显影、定影需要的时间。
<br/>　　所以一切东西弄好之后，李承乾在‘高手’的配合下，钻进了马车的车箱里面，对着自己的‘兰若寺’开始了不间断的测试。
<br/>　　‘高手’此时已经彻底懵逼了，完全不知道李承乾到底在搞什么东西，只知道在他的指挥下不断的泡玻璃。
<br/>　　这样的日子又持续了大概半个月时间，终于在一个风和日丽的下午，一张黑白的‘兰若寺’照片呈现在眼前。
<br/>　　“成功了，我们成功了。”兴奋的李承乾拿着巴掌大的纸片四下乱窜，吓的杨雨馨以为他疯魔了，追在后面一个劲叫他停下。
<br/>　　“小丫头，看看，看看，这就是我们住的地方！”李承乾跑了一圈之后，把手里的纸片递给追上来的杨雨馨。
<br/>　　结果小丫头根本看都没看，扯过李承乾就把手搭在他的额头上。
<br/>　　“干什么，本宫没病！”李承乾一把将小丫头的手打开，继续将手中的照片递过去：“看看这个，看看！”
<br/>　　“这是什么？”看李承乾恢复正常，杨雨馨松了口气，接过照片低头看了起来。
<br/>　　谁知，只看了一眼，小丫头就一声惊叫，结结巴巴有些失语的问道：“殿，殿下，这，这是……这是……”。
<br/>　　“怎么样，是不是很清楚？”李承乾没有注意到小丫头苍白的脸色，炫耀的说着。
<br/>　　“殿下，你吓到她了！”夜魅悄然出现杨雨馨的身边，伸手在她的肩背上捏了几下。
<br/>　　吓到？李承乾愣了一下，这才注意到，小丫头拿着照片一个劲的哆嗦着，像是看到了什么可怕的东西。
<br/>　　“怎么了这是？”
<br/>　　“殿下，您这是仙法么？为什么……为什么这个会如此清晰，这么真实？”夜魅从杨雨馨的手中将照片拿过来，眼神在照片和大殿之间来回扫视了好几遍，有些疑惑的问道。
<br/>　　“这叫照片，可以将所有的东西在极短的时间里印到纸上，而且和原来的东西一模一样。”李承乾并不在乎是否换了一个听众，持续的兴奋让他现在只想尽快与人分享。
<br/>　　给自己照相，然后把照片找个地方存起来，千百年之后的那些考古的家伙们就有机会看到大唐时英俊的太子殿下，这是多么有成就感的一件事情！
<br/>　　不过让李承乾尴尬的事，当他自己站在马车前，想给自己来一张自拍的时候，杨雨馨和夜魅却将他死死的拉住，说什么也不让他过去。
<br/>　　“不是，你们两个到底要干啥？难道要一起？放心下一张我们一定来张合照。”李承乾挣扎不过，有些颓然的说道。
<br/>　　“不行的殿下，这东西会吸收人的魂魄，您决不可以去照！”相对于有些慌乱的杨雨馨，夜魅的头脑还是很清楚的，意思表达的也很明确。
<br/>　　“啊？！吸人魂魄？谁说的？”李承乾有些愕然。
<br/>　　“如果不吸收魂魄，那大殿为什么会出现在纸上？”夜魅抖抖手中‘兰若寺’的照片说道。
<br/>　　“胡说八道，这就是简单的物理成像，和印刷的道理一样，你听过印刷会吸人魂魄的么？”李承乾有些哭笑不得的说着。
<br/>　　后世的时候曾经听人说过，照相机刚刚出现的时候，很多人都害怕灵魂被那个小盒子吸进去。
<br/>　　当时看的时候以为是扯犊子的说法，一些人瞎编的，现在看夜魅和杨雨馨的表现，似乎这说法并不是空穴来风。
<br/>　　李承乾用了大概盏茶时间来尝试说服夜魅，但最后发现一切都是徒劳，无奈之下将目光转向呆立一边的‘高手’：“你来，站在这！”
<br/>　　“不，不，殿下，殿下饶命！”‘高手’脸上的表情瞬间变的惨白，当初被李二打入大牢都没有出现过的恐惧笼罩全身。
<br/>　　他可是要飞升成仙的人物，所以并不怕死，但是如果把灵魂印到那个玻璃片上，永远被困在其中，这是他绝不接受的，他它可去死，也不想被困住。
<br/>　　秀才遇着兵有理说不清的感觉让李承乾觉得有些崩溃，照个相而已，怎么搞的像杀人全家一样。
<br/>　　把夜魅和杨雨馨拖到一边，三人一番交涉，最后取得了一个共识，去找一个突厥奴隶，用奴隶作试验，看看照完相之后那个奴隶会不会死。
<br/>　　于是一行人赶上马车，直奔医学院，那里还有一些收尾的工程没有弄完，还有些突厥人在那里工作，到时候随便抓来一个照一下，所有的事就会全部真相大白了。
<br/>　　不过，很多事情都是在巧合之下发生的，李承乾等人的马车刚刚离开‘兰若寺’还没到百步，就被老头子带着一群人给堵在了半路之上，而且老李脸上写满了不痛快：“逆子，又要干什么去？”
