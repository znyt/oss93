　　鸿钧道祖先是将各种大道道则，之后才开始讲准圣境界的修炼之法。
<br/>　　“大罗金仙之后者，准圣也。何为准圣？准圣者，圣人之准备也，乃成圣之过渡。”
<br/>　　“欲突破准圣境界，则需修行斩三尸之法。”
<br/>　　斩三尸之法？众人心中疑惑不解。
<br/>　　鸿钧老祖继续道：“所谓三尸，三尸之为物，实魂魄鬼神之属也。欲使人早死，此尸当得作鬼，自放纵游行，飨食人祭拜。每到六甲穷日，辄上天白司命，道人罪过，过大者夺人纪，过小者夺人算。”
<br/>　　“三尸者，一曰青姑，伐人眼，令人目暗面皱，口臭齿落；二曰白姑，令人腹轮烦满，骨枯肉焦，意志不升，所思不得。故求仙之人，先去三尸，恬淡无欲，神净性明，积众善，乃服药有效，乃成仙。”
<br/>　　“三尸者，毒也，人身上有三尸神，名三毒，上尸明彭琚，管人上焦善恶；中尸名彭瓒，管人中焦善恶；下尸名彭娇，管人下焦善恶。”
<br/>　　“上尸住玉枕关，中尸住夹脊关，下尸住尾闾关，每逢庚申甲子，诣奏善恶。又有九蛊作害不浅，阻塞三关九窍，使其真阳不得上升。”
<br/>　　“而九蛊俱有其名，一曰名伏蛊，住玉枕窍；二曰龙蛊，住天柱窍；三曰白蛊，住陶道窍；四曰肉蛊，住神道窍；五曰赤蛊，住夹脊窍；
<br/>　　六曰隔蛊，住玄枢窍；七曰肺蛊，住命门窍；八曰胃蛊，住龙虎窍；九曰蜣蛊，住尾闾窍。三尸住三关，九蛊住九窍，变化多端，隐显莫测，化美色，梦遗阳精，化幻境，睡生烦恼，使修士生灵大道难成矣。”
<br/>　　“所谓斩三尸，便是将自身的善念、恶念、执念通过先天灵宝寄托斩去，每斩一尸，道行便会增长一倍，三尸皆斩尽后，就是大寂灭境界。之后三尸合一，化身与本体彻底融合不分彼此，此时可以以身化亿万，离那圣境只差一步之遥，到时只需机缘一到，便可立地成圣。”
<br/>　　每斩一尸，道行法力就会增加一倍？三尸尽斩，只需机缘一到，便可立地成圣？众人心中震惊不已。
<br/>　　“斩去三尸乃是用善、恶、执三念寄托于先天灵宝之中，借助于那先天灵宝一举斩去这善、恶、执三念便可使自身的心境处于圆满状态，每斩去一尸，自身心境便会圆满一分，对着天道的理解便会容易一分，待到三尸斩尽，自身心境便会圆满，将无时无刻处于心境通明的状态，参悟这天道自然容易许多。”
<br/>　　“敢问道祖，是否必须用先天灵宝为寄托，才可斩去三尸。”有人提问道。
<br/>　　“凡是先天之物，皆可。但先天之物亦关系到三尸修为的高低，自身品级越是上乘，斩去三尸后，修为道行便会越高。”
<br/>　　啊？在场众修士大多都目露悲苦之色，他们之中，大多数人都没有先天灵宝，那岂不是说，自己斩不成三尸了？那不就意味着自己不能突破准圣了吗？
<br/>　　众修士心中一片苦涩，下定决心，此次回去之后，定要到洪荒之中到处走走，看看能不能搞到几件先天灵宝，就算搞不到，抢也要抢几件。
<br/>　　特别是妖族的太一，他看向元始的目光，更加的不善了。要知道，他的灵宝可是先天至宝啊，就这么被元始索走了，他斩三尸之物都需要重新去找。他不恨元始才怪。
<br/>　　众修士同时向着鸿钧拜服道：
<br/>　　“恳请道祖传授吾等斩三尸之法。”
<br/>　　“好，今日贫道便传授尔等斩三尸之法。”
<br/>　　……
<br/>　　圣人讲道果真非凡，只见地涌金莲，天光大震，仙音阵阵，又有天女散花，火海冰川显化，无比非凡。
<br/>　　无上大道从鸿钧口中道出，只听得众人时而欢声大笑，时而皱眉不语，时而嚎啕大哭，神态各异，虽不能完全听懂，但或多或少能够记下一些，只待回山后静静参悟。
<br/>　　云台之上，鸿钧不管台下众人反应，自顾自的讲解起这无上大道，一个个大道铭文浮荡于虚空之中，在紫霄宫中来回闪烁，玄奥莫名的气息从紫霄宫内浮荡而出。
<br/>　　众人一个个沉浸在这玄妙的斩三尸之道中，不可自拔。
<br/>　　当——
<br/>　　时间匆匆而过，三千年讲道时间一眨眼便过去了。
<br/>　　宫殿之中，钟声一响，众修士都从参悟大道的状态之中醒悟过来，望向鸿钧。
<br/>　　此次讲道后，鸿钧并没有像第一次那般，讲完道后便消失不见。他看着众修士，面无表情，淡淡开口道：
<br/>　　“三千年已过，第二次讲道结束。汝等之中，却是有几个与贫道有缘，当为吾之弟子。”
<br/>　　“啊？圣人鸿钧道祖要收徒了，若是被道祖看中，以后谁还敢得罪？”
<br/>　　众修士醒来一听鸿钧这话，顿时眼睛都亮了起来，纷纷盼望着能够被鸿钧道祖作为弟子收下。
<br/>　　“三清为盘古元神所化，身具开天大功德，根基深厚，却是与贫道有缘，当入我玄门。元始、老子，通天，你们可愿意？”鸿钧看着元始三人，悠悠开口道。
<br/>　　“吾等愿意，弟子拜见老师。”
<br/>　　三清似乎早知如此，立即跪拜道。
<br/>　　不是吗，鸿钧只收几个弟子，而紫霄宫中只有六个蒲团座位，用意已经十分明显了，只有那些脑子转不过来的蠢人，才不够清楚。
<br/>　　“赐你们鸿蒙紫气，此乃圣道之基。”
<br/>　　鸿钧说着，大袖一挥，众修士只见，三道闪烁着紫色光芒的光源飞出，并迅速融入元始、老子、通天元神之中。
<br/>　　元始三人感受着元神之中，天道韵律飘然，似乎这鸿蒙紫气与天道有关，时时刻刻散发着道韵，让自己的道行稳步提升，眼中不由闪过一丝激动之色，朝着鸿钧拜谢。
<br/>　　元始心中激动，这鸿蒙紫气终于到手了，他的三大最初本源大道中的灵魂本源鸿蒙道就可以开始修行了，成不成圣他不在意，他在意的是鸿蒙紫气之中蕴含的鸿蒙大道。
<br/>　　没有鸿蒙紫气又如何，他一样可以证道。而且，证道是最为关键的事，能不借助外物，就不借助万物。借助外物证道，那就意味着你的道有了限制，不是最好的道。
<br/>　　所以，像元始这样骄傲的人，又怎会用鸿蒙紫气证道成圣？
<br/>　　但元始不会，并不意味着其他人不会啊。在场的众修士也不是傻子，，眼见鸿蒙紫气与元始三人的反应，便可猜出不少。
<br/>　　“嘶！鸿蒙紫气？圣道之基？难道是有了这鸿蒙紫气就可成圣？”
<br/>　　所有人都望向鸿钧老祖，希望能够成为道祖的弟子，得到成圣之基。鸿钧却是脸色淡漠无比，看向第四个蒲团上的女娲，沉声道：
<br/>　　“女娲日后有大功德之事要做，可为吾之弟子。”
<br/>　　“女娲叩谢老师。”女娲娇躯一颤，脸色从之前的阴沉之中转为大喜，也向着鸿钧拜谢道。
<br/>　　鸿钧大袖一甩，顿时鸿蒙紫气迅速融入女娲身体之中。
<br/>　　感谢大家的支持，感谢各位道友的打赏。
