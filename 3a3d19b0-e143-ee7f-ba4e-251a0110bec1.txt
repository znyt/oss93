　　只见通体火焰缭绕的吸血蛇察觉异样，凶光的眸抬首一望，顷刻间笼罩在一片雷海之中。
<br/>　　雷声震天，电闪雷鸣，整个蛇谷都被映照在一片刺耳的光芒之中，掀不开眼。
<br/>　　三十米庞大的吸血蛇充斥在雷海中，它被淹没，被吞噬，耀眼的雷海中，倒影出一抹殷红。
<br/>　　雷海滔天久久不息，巨大的雷声如同山崩地裂，好像大地都被震得颤动起来与雷电交织在一起。
<br/>　　不远处的江行泽看到如此规模的雷海，咽了咽口水，心里直发怵，有着莫名的恐惧感。
<br/>　　吸血蛇不会被雷给劈死了吧。
<br/>　　青天白日，怎么会有雷，而且还不是一道，像是有无数的雷电，自银河从天而降。
<br/>　　霎时间，江行泽响起雷海出现之前的那道诡异而冰冷的声音。
<br/>　　这冰冷的声音既然让他有一丝熟悉，脑海中不禁的浮现一条掌控雷电的银白蛇。
<br/>　　顿时，他猛然的倒吸了一口凉气，狰狞的面容变的比厉鬼还要可怕。
<br/>　　他一只独眼龙四下寻找，忽然，他在一片火海的上空看到了一条金银蛇悬浮与空。
<br/>　　不是银白蛇，而是金银条纹灿然生辉的蛇。
<br/>　　但江行泽一眼就笃定，这就是那条将他害的人不人鬼不鬼的该死蛇。
<br/>　　好呀。
<br/>　　它终于出现了，自己报仇雪恨的机会来了，这回一定要吃它的肉，喝它的血，最后将它挫骨扬灰。
<br/>　　不，就算如此，也难解江行泽的心头之恨。
<br/>　　他真的恨透了这条蛇，这恨绵绵无绝期。
<br/>　　涂小安悬浮与空，蛇目泛着冷淡无波的眸光看着吸血蛇笼罩在自己制造的雷海下。
<br/>　　步入二阶蛇精层次，他的雷电之力威力大增，他甚至可以一口气同时释放二十道雷电。
<br/>　　每道雷电可以同时将一百个人类给活生生劈死。
<br/>　　此时，他占得先机，先发制人，将吸血蛇笼罩雷海，雷电疯狂的侵袭它的蛇躯。
<br/>　　要换做一般的觉醒蛇，十条命都没有了。
<br/>　　但它是吸血蛇，吸血蛇的强大恐怕远远比涂小安想象的要强上数倍。
<br/>　　就刚才，他飞到蛇谷内，看到了吸血蛇口吐火焰，将数万毒蛇活生生的给烧死，他诧异万分。
<br/>　　如今蛇谷的空气充分腐臭味，乌烟瘴气。
<br/>　　火焰之力？
<br/>　　吸血蛇既然跟他一样掌控了属性力量吗。
<br/>　　要知道能掌控属性力量，那可是非同小可的，涂小安是吞噬了变异鸡冠蛇开启了第一阶段性的进化，才得到了雷电能力。
<br/>　　除他之外，他还遇见了一位同样拥有属性力量的蛇种，那便是上古蛇精修蛇，它拥有御水之力，甚至可以顷刻间引发洪水灾难。
<br/>　　可涂小安万万没想到，此时的吸血蛇居然也掌控了这种属性能力，而且还是无比霸道的火焰之力，这是非常可怕的一种力量呀。
<br/>　　甚至不比他的雷电之力来的逊色。
<br/>　　自古以来，火就是一种无比霸道的元素力量，谁能掌握了火，就等于拥有了毁灭一切的力量，它是凶猛的，是可怕的，它静时，能让你感觉到温暖，它动时，可以摧毁一切事物。
<br/>　　不管是神话中的各路神仙，还是一些神兽，但凡掌控了火，那么都是无比可怕的存在。
<br/>　　一时之间，涂小安陷入了深思，不知道想着什么，但是一双蛇眸烙印在雷海下，变的贪婪起来。
<br/>　　如果他要是吞噬了吸血蛇，开启第二阶段性的进化，是不是也可以得到它的火焰之力？
<br/>　　要是这样，那么他就同时掌控两种属性力量了。
<br/>　　隐隐一想，涂小安整个人都忍不住的激动起来。
<br/>　　“嗷...”
<br/>　　此时，一声惊天的咆哮声自雷海激荡出去，一股骇人的恐怖气息迸发。
<br/>　　电闪雷鸣的雷海顿时直接被一股强大的力量震的溃散开来。
<br/>　　只见三十米的吸血蛇红火鳞片白烟缭缭，一双凶煞无比的蛇眸已经锁定在悬浮的金银蛇身上。
<br/>　　“吸血蛇，我们又见面了”
<br/>　　涂小安知道一招雷霆之怒是解决不了这条蛇，于是悬浮在上空，冷漠的跟这条蛇打招呼。
<br/>　　它们也算是“老友”见面。
<br/>　　自打这条传闻中的变异吸血蛇出世，涂小安就跟它纠缠上了，还是捕蛇人的时候，他就想靠着捕抓吸血蛇名声大噪，可惜，那次他失手了。
<br/>　　失手的结果，就是丢掉小命，之后他系统加身，化为家蛇，好不容易等自己强大起来，找吸血蛇报仇，但却半路杀出来一个蛇博士。
<br/>　　将它们两条蛇都给抓了去做实验。
<br/>　　实验之后，他跟吸血蛇同时都得了巨蟒基因，实力暴涨，如今再度相遇，涂小安成为二阶蛇精，掌控雷电之力。
<br/>　　可吸血蛇也恐怖如斯，掌控了火焰之力。
<br/>　　它们两条蛇，都是可以成为真正蛇王的存在，一条掌控雷电，一条掌控火焰。
<br/>　　但蛇王只能是一条，一山岂能容二虎。
<br/>　　如果让涂小安成为三阶生灵，成为真正的蛇王，统领蛇族一脉，那么自是白镇人民的福气。
<br/>　　可要是让吸血蛇成为三阶生灵，凌驾普通生物之上，那么就是一场灾难。
<br/>　　毫无疑问，它是一条邪蛇，涂小安代表正蛇。
<br/>　　正邪向来不两立，更何况涂小安跟这条蛇有不共戴天之仇。
<br/>　　吸血蛇凶光大放，看着悬浮的金银蛇，像是一点都不意外，发出阴冷沙哑恐怖的声音：“你终于出现了，本王等你很久了”
<br/>　　当初密林之战，不分胜负，反到被蛇博士螳螂捕蝉黄雀在后。
<br/>　　涂小安更是战意盎然，这一次，他不会在失手了，吞了这条变异吸血蛇，开启第二阶段性的进化。
<br/>　　“哈哈...该死的蛇，你总算来了，今天我们新帐旧账一起算”
<br/>　　一位黑袍男子冲到吸血蛇的前面，趾高气扬的看着金银蛇，说话间，还恨得牙痒痒。
<br/>　　嗯？
<br/>　　涂小安早就留意到有一个人类的存在，只是他的注意力一直放在吸血蛇的身上，此时这个人自己冲出来，怒刷存在感。
<br/>　　这人是谁？
<br/>　　正眼看之，涂小安诧异，差点从空中掉下来，天哪，好丑，好丑的人，既然有人长的这般丑陋，比鬼还吓人。
