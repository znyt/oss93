　　齐凡和慕容飞知道罗昊肯定是有着他自己的打算，于是就按照罗昊的吩咐，朝着冲向楼房的反政府武装人员射击着，掩护着李毅峰和徐旭东撤离。
<br/>　　罗昊握着龙牙军刀，凝神静待。
<br/>　　一名反政府武装人员端着AK自动步枪冲进了楼房，而他根本就没有注意到藏在门口墙壁背后的罗昊。
<br/>　　在反政府武装人员冲入楼房的瞬间，罗昊一把揪住对方的衣服，把他按在墙壁上，握在手中的龙牙军刀顺势刺进对方的脖子中，向一侧横拉，切断了对方的声带、气管和颈动脉，鲜红色的动脉血从反政府武装人员的脖子喷射而出，喷溅在墙壁上，猩红一片。
<br/>　　罗昊挥动龙牙军刀，将AK自动步枪的枪带割断，把对方的AK自动步枪抢到了自己手中，顺势从他身上的战术背心中抢了两个用胶布捆绑在一起的备用弹匣，塞到自己的战术背心中。
<br/>　　瞥见对方挂在腰间的小包中还有着手雷，罗昊也不客气，拿出一颗手雷拉掉保险销，然后塞在对方身上，一脚把那个反政府武装人员的尸体踢向了街道。
<br/>　　脖子上还在喷着鲜血的反政府武装人员尸体刚落在街道上，罗昊塞在他身上的手雷便发生了爆炸。手雷引发了连续爆炸，那名反政府武装人员身上剩余的手雷也发生了爆炸，对街道上其他的反政府武装人员造成了一定的杀伤，好几个来不及逃跑的反政府武装人员被炸死。
<br/>　　罗昊打量了一下自己手中这把抢来的AK自动步枪，不禁摇了摇头。这把AK自动步枪一看就属于缺乏保养，枪身有多处磨损和划痕，就连枪管上都是有着好几道很深的刻痕，这种武器的准星自然也没有什么可以指望的了，不过现在也没有选择，罗昊只能期望自己手中这把AK自动步枪不会炸膛。
<br/>　　“哥们，给点面子，前往别炸膛了。”
<br/>　　罗昊把AK自动步枪的快慢机拨到半自动模式，这次倒不是因为对弹药的考虑，而是担心全自动模式会造成枪管过热，导致武器炸膛。
<br/>　　哒！哒！哒！哒！哒！
<br/>　　罗昊以精准的半自动射击，对着街道上的反政府武装人员进行着点射，进行有效的杀伤。
<br/>　　咔！
<br/>　　而就在这个时候，罗昊手中的AK自动步枪发生了卡壳，一枚弹壳卡在抛壳窗处，使得后续的子弹没有能顺利进入枪膛中。
<br/>　　这种缺乏保养，并且损伤严重的武器发生故障是罗昊意料中的事情，可是在这个关键时刻发生故障，却是让罗昊彻底陷入了被动之中。
<br/>　　罗昊快速拉动枪栓，把卡在抛壳窗位置的弹壳退出，然后将子弹重新上膛；可就是这短暂的几秒钟时间，却是给了外面那些反政府武装人员有了反应的时间，等到罗昊把子弹重新上膛的时候，楼房外已经有两个反政府武装人员举起了RPG-7火箭筒，并且瞄准了楼房的方向。
<br/>　　罗昊几乎没有任何的犹豫，转身就朝着楼房的后门跑去。
<br/>　　咻！轰！
<br/>　　一枚火箭弹从天空中飞来，落在楼房外的街道上，发生剧烈的爆炸，将街道上那些反政府武装人员全都炸飞了出去。
<br/>　　火箭弹刚落下，一串机炮炮弹紧随而来，天空中有着直升机螺旋桨的呼啸声传来。
<br/>　　“来的还真是时候。”
<br/>　　罗昊把自己手中这把几乎能送进炼钢炉重铸的AK自动步枪仍在地上，用力挥舞了一下自己的拳头。
<br/>　　楼房外的反政府武装人员看到天空中那两架满载弹药的AH-64D“长弓阿帕奇”武装直升机，顿时被吓得魂飞魄散，这种可以携带多种空空武器和空地武器的武装直升机，留给他们的阴影太深太深了。
<br/>　　原本还气势汹汹的反政府武装人员此时却是落荒而逃，跳上在车顶上安装了.50口径重机枪的皮卡车，沿着街道逃窜，在逃跑的过程中，皮卡车车斗中的反政府武装人员把架在车斗中的重机枪转过头来，对准了天空中的武装直升机；只是还没等到皮卡车上的反政府武装人员射击，一枚“地狱火”导弹便是离开挂架，击中了街道上的皮卡车。
<br/>　　轰——！
<br/>　　皮卡车和皮卡车上的反政府武装人员化作一团巨大的火球，冲向天空，硝烟散开，那辆安装了重机枪的皮卡车就只剩下了一个铁壳子，汽车上的重机枪也被炸成了废铁。
<br/>　　俯冲而下的“长弓阿帕奇”在距离地面还有不到二十米的时候，猛地重新拉起机头，飞向天空，担任着低空警戒任务。
<br/>　　一架V-22“鱼鹰”运输机，在楼房不远处的空地降落，李然和几名美国海军陆战队员从运输机中跑下来，向楼房靠近。
<br/>　　李然看到罗昊、齐凡、慕容飞、李毅峰，还有被李毅峰背在身上，已经陷入了昏迷的徐旭东时，悬着的心总算是放下了一点。
<br/>　　李然甩手把一支加装了ACOG瞄准镜的M4A1自动步枪扔给罗昊，问道：“你们都没事吧？”
<br/>　　“噬蜂伤得很严重，我们几个都只是受了一些轻伤。”罗昊接过李然扔给自己的M4A1，熟练把子弹上膛，朝着停在空地上随时准备再次起飞的V-22“鱼鹰”运输机走去，对李然问道：“你是怎么找到这里来的？”
<br/>　　李然指了指罗昊手腕上的手表，说道：“你忘了里面有定位芯片吗？只要你们没有离开地球，不处于卫星信号被屏蔽的地区，我都能通过定位芯片找到你们。”
<br/>　　“高科技果然是个好东西。”罗昊看了眼自己手腕上的手表，说道：“其他人怎么样了？”
<br/>　　李然的脸色变得有些难看，对罗昊说道：“到目前为止，幸存的，我暂时只知道了你们。”
<br/>　　罗昊脚步猛地停了下来，盯着李然问道：“幸存的？他们都死了？”
<br/>　　李然沉默着没有说话。
<br/>　　罗昊抓住李然的肩膀，大声质问道：“回答我！他们到底怎么样了？！”
<br/>　　李然拉开罗昊抓在自己肩膀上的手，说道：“我们找遍了你们出事的地方，发现了电光、冰枪，还有虎鲨的尸体……”
