　　“第一件拍品。”
<br/>　　灵尊真人长啸一声，便是一个玉盒，呈了上来。
<br/>　　这玉盒虽然封闭，但是楚风眠也察觉出，在其中蕴含的精粹之极的灵力。
<br/>　　除了灵力，在其中蕴含的，还有一股浓郁到极致的生机。
<br/>　　一些将死老者，单单是闻到这其中味道，只怕都会可以在多活十年。
<br/>　　这生机，绝对是比起任何东西，都要浓郁。
<br/>　　“各位请看。”
<br/>　　灵尊真人，缓缓打开玉盒。
<br/>　　在其中，静静的躺着一枚三寸树枝，这树枝，翠绿无比，虽然只有三寸，却仿佛是活得一样。
<br/>　　这其中的灵力，跟浓郁极致的生机，正是从这三寸树枝的，散发出来的。
<br/>　　“建木？”
<br/>　　楚风眠目光看到这树枝的一刻，便是突然脱口而出了一个词。
<br/>　　“哦？楚公子倒是好见识。”
<br/>　　李长老在一旁，听到楚风眠开口，也是微微有些惊讶道。
<br/>　　“这正是天地第一树，建木的树枝。”
<br/>　　建木树枝，这种东西，居然出现了。
<br/>　　楚风眠心中也是极为惊讶。
<br/>　　建木，乃是号称荒古时代，存在的天地第一树，据说是贯穿天地，乃是天地的根本。
<br/>　　世间的种种生灵，许许多多都是因建木而生。
<br/>　　在荒古时代，建木据说都因为一场极为庞大战争破碎了，如今数十万年过去，没想到居然还会有着建木树枝的出现。
<br/>　　“这建木树枝，其中蕴含无数生机，生命之力无穷无尽，一旦是炼化，融入到身躯之中，足以炼化成了类似于战龙血脉这般的身躯，无论受到什么伤势，都可以恢复，几乎不死。”
<br/>　　楚风眠心中暗自想到。
<br/>　　这建木树枝，他要是得到，一齐圣祭的话，足以让楚风眠的恢复力变的更加强大。
<br/>　　倒时候真的是无法一招灭杀楚风眠的话，以战龙血脉跟建木的力量，足以直接恢复过来。
<br/>　　“建木树枝，谁要是可以得到这种东西，炼化到身躯之中，就几乎得到了不死不身！”
<br/>　　“通天阁居然是将这种东西拿出来拍卖了。”
<br/>　　“这建木树枝，我势在必得！得到他，在虚空海中闯荡，我也会安全无比。”
<br/>　　在下方，无数武者看到这建木树枝，都是沸腾起来，一个个都想要争夺。
<br/>　　“这建木树枝，李长老不知道有没有争夺的想法？”
<br/>　　楚风眠看着旁边的李长老，似乎并没有多少表情变化。
<br/>　　“建木树枝虽好，不过想要得到，要付出的财富也不少。”
<br/>　　李长老摇了摇头道。
<br/>　　“这样一枚建木树枝，虽然融入到身躯中去，足以让生机澎湃许多，但是遭遇致命伤，想要恢复也需要一段时间。”
<br/>　　“真正强者交手，胜负都是一瞬之间的事情，这建木树枝的作用其实并没有多少，付出极大财富去购买他，有些得不偿失。”
<br/>　　李长老所说的，正是这建木树枝的弱点。
<br/>　　像是实力达到生死境高阶的程度，一旦交手，几乎一息之间便是足以分出胜负，根本没有时间可以去慢悠悠的恢复伤势。
<br/>　　不止是李长老，周围天字包厢中，都没有多少人想要起了拍下的心思。
<br/>　　不过这只是寻常而言的，而对于楚风眠来说，他拥有战龙血脉，本身恢复能力便是强到极致，在加上这建木树枝的话，便是如虎添翼。
<br/>　　倒时候一瞬之间足以恢复全部伤势，比起不死魔尊那种，还要可怕的多。
<br/>　　“第一件拍品，建木树枝，起拍价，一万灵石！”
<br/>　　灵尊真人看到众人情绪点燃，也是在适当的时机，开口了。
<br/>　　一万灵石，所指的便是下品灵石，一块王品灵石的价格，便是在一万灵石左右，至于一块皇品灵石，则就是足足百万灵石的价值。
<br/>　　“两万灵石！”
<br/>　　“三万灵石！”
<br/>　　“五万灵石！”
<br/>　　下方人群中，一声声叫喊声，直接响起，
<br/>　　一尊尊武者，纷纷报价，建木树枝，毕竟是天地第一树的树枝，价值绝不会再十万灵石之下。
<br/>　　一万灵石的价格，正是为了让众人参与，点燃气氛。
<br/>　　价格一路飙升，很快，便是到达了十万灵石左右。
<br/>　　这个价格，已经不是一些武者，可以承受的了的了。
<br/>　　“二十万！”
<br/>　　楚风眠看到这报价逐渐变慢，才突然开口，喊了一个价格。
<br/>　　这建木树枝被楚风眠得到，的确足以带来不小的好处。
<br/>　　楚风眠现在最不缺少的，便是财富，只要是对他有利，这一届拍卖会中的东西，他便是要全部得到。
<br/>　　“楚公子，居然对着建木树枝有兴趣？”
<br/>　　李长老看到楚风眠出价，有些意外。
<br/>　　楚风眠的实力，他是见识过的，这种接近生死境高阶的实力，这建木树枝，对楚风眠的作用，应该是极为有限才对。
<br/>　　“毕竟是天地第一树的树枝，买来玩玩也行，反正也花不了多少财富。”
<br/>　　楚风眠随意笑了笑开口道。
<br/>　　这句话，令李长老直接无语。
<br/>　　二十万灵石，可就是二十块王品灵石的价格了。
<br/>　　这种财富，挥霍出去，居然只是玩玩。
<br/>　　就算是李长老，也不会敢做出如此奢侈的事。
<br/>　　感受着李长老一副看向暴发户的眼神，楚风眠倒是毫不在意。
<br/>　　建木树枝自然是有着他的用处，楚风眠也懒得解释，反正他现在的形象，就是一个暴发户了，不如彻底一点。
<br/>　　“二十万灵石，这个价格，还有人加价吗？”
<br/>　　楚风眠喊出二十万的价格之后，在场便是寂静了许多。
<br/>　　这个价格，已经不是下方的许多人，可以拿得出了。
<br/>　　而且就算是拿得出，也很少会有人，付出如此之大的代价，来购买建木树枝这样一个如同鸡肋一般的宝物。
<br/>　　“天字包厢中的人，居然花费如此大代价，来买这建木树枝。”
<br/>　　“这天字包厢中的人物，可都是大人物，这建木树枝对他们应该没有多少作用才对。”
<br/>　　“没准是哪个大势力的阔少呢。”
<br/>　　“算了我们还是不要议论了，天字包厢中的人物，我们得罪不起。”
<br/>　　下方一尊尊武者交谈。
<br/>　　就在这寂静之中，一道声音却是突然响起。
<br/>　　“三十万灵石！”
<br/>　　这声音，赫然是来自楚风眠旁边的天字十号包厢之中。
