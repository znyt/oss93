        &lt;h3&gt;第三卷  第三十三章  年少之诗&lt;/h3&gt;
<br/>
<br/>        诗一首
<br/>
<br/>        青春的色彩
<br/>
<br/>        你从云里姗姗而来
<br/>
<br/>        光的影，散发着七彩
<br/>
<br/>        生活不再是黑和白
<br/>
<br/>        翠的绿，是生命顽强的爱
<br/>
<br/>        樱花树下
<br/>
<br/>        飞舞的梦想
<br/>
<br/>        点燃了青春火热的情怀
<br/>
<br/>        诗二首
<br/>
<br/>        校园之歌
<br/>
<br/>        雪花，覆盖，梦想深埋
<br/>
<br/>        寒的冬，冻成了苍白
<br/>
<br/>        精灵的梦，爬满青苔
<br/>
<br/>        眼里的希冀
<br/>
<br/>        融化成春水，成泥
<br/>
<br/>        梦想，种下了种子
<br/>
<br/>        生根、发芽，冲破层层障碍
<br/>
<br/>        走向未来！
<br/>
<br/>        陈子昂看着这些熟悉的诗句，泪眼模糊，她以为历史老师曹禺弄丢了她的笔记，她以为他再也记不起她这个学生。
<br/>
<br/>        没想到，在这个地方，还有她存在的地方，还有人记得她。
<br/>
<br/>        青春已散场，无悔也无殇。
<br/>
<br/>        虽然，她没有上得了大学，可是她曾经努力过，那一千多个日夜，那年少的希望，那年少的张狂，都是那么美好，美好得就像那条清澈透明的河水，纯净而甘甜。
<br/>
<br/>        走过了，无悔。失去了，无望！
<br/>
<br/>        那么就让：眼里的希冀
<br/>
<br/>        融化成春水，成泥
<br/>
<br/>        梦想，种下了种子
<br/>
<br/>        生根、发芽，冲破层层障碍
<br/>
<br/>        走向未来，走向未来吧！
<br/>
<br/>        其实，想一想，陈子昂阴差阳错的也算是成就了自己的梦想，在和张函呆在一起的三年时间里，她无忧无虑，完成了成考，完成了考研，还学习了相关的电子信息类的大量知识，如果没有那三年，陈子昂可能还只会是一个农村姑娘，也有可能会面朝黄土背朝天。
<br/>
<br/>        是那三年富足安宁的日子，让陈子昂沉下心来努力学习，是张函给了她鼓励，给了她榜样，也是张函造就了她。
<br/>
<br/>        她有了足够的知识存储量，她才能给落雪当家庭教师，她有了足够的专业知识，才能立足中关村，在那个人才济济的地方拥有一席之地。
<br/>
<br/>        没有考上大学虽然遗憾，但是不足成为陈子昂前进路上的阻碍。
<br/>
<br/>        陈子昂站在广告墙前沉默着，泪水早已经很河！
<br/>
<br/>        “子昂。”江俞轩拿出纸巾，轻轻的替陈子昂擦去脸上的泪水。
<br/>
<br/>        “我都没有想到你还是个诗人啊。”
<br/>
<br/>        “那是当然。”陈子霖骄傲的说，他也才知道他妹妹已经是洛城小有名气的诗人。
<br/>
<br/>        “什么诗人？这个是我高中的时候写的。当初稿子都交给老师了，我都不知道老师还保存着呢。”
<br/>
<br/>        “走，咱去把版权要回来。”江俞轩。
<br/>
<br/>        “问谁要？我都没有老师的联系方式了，再说了，这么幼稚的诗，看着我自己都觉得好笑。”
<br/>
<br/>        “幼稚吗？我倒是从诗里看到了你小小年轻就有很大的抱负呢，难怪现在如此成功，陈子昂，你带着我飞吧。”
<br/>
<br/>        陈子昂笑了，江俞轩总是会在不经意间鼓励她，支持她：
<br/>
<br/>        “江家大少就是心胸宽广，看问题的高度都和别人不一样。”
<br/>
<br/>        “江家大少？房地产界一哥？”陈子霖惊问。
<br/>
<br/>        陈子昂微笑着看着她哥哥，陈子霖嘴巴张得能塞进去一个鸡蛋。
<br/>
<br/>        陈子霖走到了江俞轩身边转来转去，转来转去，江俞轩都被他转晕了。
<br/>
<br/>        “陈子霖，你晕不晕？你不晕我都晕了。”
<br/>
<br/>        “原来你就是神秘的江大少啊？”
<br/>
<br/>        “如假包换！”
<br/>
<br/>        “你那么有钱的，干嘛还赖在我妹妹的公司，你说你有何企图？”陈子霖现在倒是将妹妹看得很紧。
<br/>
<br/>        “.......”江俞轩要笑哭了。
<br/>
<br/>        “陈子霖，我结婚的时候邀请你吃喜糖？”
<br/>
<br/>        “你都没有经过我们的同意，你就准备结婚？门都没有？”
<br/>
<br/>        “我和张家姑娘结婚，要你同意什么？”
<br/>
<br/>        陈子霖心里一松，然后又失落了。
<br/>
<br/>        原来他只是想为难一下江俞轩的，其实，江俞轩这几天对妹妹真不错，妹妹嫁给他也很幸福，倆个人一般高，站在一起也算般配，虽然江俞轩个头比张函差那么一丢丢，但是妹妹年龄也大了不是？而且和张函也没有可能了，想起张函他心里就火大，如果有机会再见到，他一定要将他揍得满地找牙。
<br/>
<br/>        妹妹也不知道什么时候能将自己给嫁了，他哀怨的看了一眼江俞轩，又看了一眼陈子昂，将俩个人看得发毛。
<br/>
<br/>        “你这是几个意思？”
<br/>
<br/>        “什么几个意思？”
<br/>
<br/>        “你那眼神就活脱脱一个怨妇。”
<br/>
<br/>        “你就那么便宜的将自己给嫁了？”
<br/>
<br/>        “我们俩好像年龄相仿吧？你娃娃都一大堆了。”
<br/>
<br/>        “也是，你还是早点将自己嫁出去，省得天天跟在我们家子昂后面，我们子昂都嫁不出去。”
<br/>
<br/>        “他说你嫁不出去。”江俞轩笑得捂着肚子。
<br/>
<br/>        “有那么好笑吗？”陈子昂脸上挂不住了，她好像真的嫁不出去了，她身边优秀的对她好一点的一个个都结婚了。
<br/>
<br/>        “我不是嫁不出去，我是不想和你们一样将自己给贱卖了。”陈子昂傲娇的转过身不去理这俩个大孩子。
<br/>
<br/>        她不知道为什么曹老师一直会留着她的笔记本，还会在这里给她一席之地，而且她也不算是诗人，十七八岁的年纪，无论是思想还是行为都是幼稚的，只是有一点，她从来都没有放弃过努力。
<br/>
<br/>        无论之前是缺吃少穿还是被被人嘲笑，她都不曾退缩过，对于梦想，她有着深深的浓浓的渴望，在初中，她的目标是考上重点高中，走出那座大山，上高中，她的梦想是考上电子科技大学，虽然夭折了，她却不曾放弃。
<br/>
<br/>        陈子昂深深的知道，一切皆缥缈，唯有读书高，梦想是需要强大的信念和足够的能力来支撑的，而作为一个农村的穷孩子，她，只能用知识来改变自己，努力的让自己拥有足够多的知识，用知识来武装自己，用知识来改变自己的命运。
<br/>
<br/>        上天眷顾，让她认识了她生命中一个又一个的益友，让她在困苦无助的时候得到救助，政策利好，让她能在时代洪流中展示自己，如今的她，早已经跳脱了只为吃喝拉撒奋斗的境界，她不再是为了个人，在她的事业里，深深的埋着她的梦想。
<br/>
<br/>        看着在自己手中的项目一个个的完成，自豪感油然而生，她带领的科研团队研发出来的软件也是为时代的前进作出了贡献。
<br/>
<br/>        因为有了他们先进的信息化软件，他们完善的售后服务工作，合作单位才能更好的做好他们的本职工作。
<br/>
<br/>        陈子昂微微一笑，过去的已经都过去了，她还有更大的梦想，在她经营公司的这几年里，她知道如果有自己自主的品牌，不仅仅是为了生意。
