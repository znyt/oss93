　　所有的人都停手了，惊疑不定的望着大步走向贺兰殇的苏莫。
<br/>　　对于第六城和第八城而言，他们乐意看到苏莫和贺兰殇窝里斗，他们可以坐收渔翁之利。
<br/>　　“虽然不是真的不死，但在你面前就是不死！”
<br/>　　贺兰殇有恃无恐的说道，他的眼眸微眯了起来，面色稍显凝重。
<br/>　　虽然他无惧苏莫，但也并无把握战胜苏莫，毕竟，刚才那么强大的一掌都伤不了苏莫。
<br/>　　不过，因为他是不死族之人，天生立于不败之地，所以不会惧怕苏莫。
<br/>　　“那我就领教一番！”苏莫来到了贺兰殇前方千丈之处，手中剑光一闪，一剑直劈而出。
<br/>　　面对贺兰殇，苏莫也不会太过藏拙，立刻动用了真正的实力。
<br/>　　两种万化神剑元神、不败战气元神，血脉之力，全部动用了起来。
<br/>　　这一剑威势绝伦，不仅剑威浩荡九霄，锋芒绝世，更有一股无上的战意，汹涌澎湃。
<br/>　　这战意之强，无与伦比，仿佛能战天战地，毁灭一切，更如同亿万大山一般狠狠压了下来。
<br/>　　千丈的距离，可以说是剑气一挥，便瞬息而至。
<br/>　　“什么？”感受到苏莫这一剑的威势，贺兰殇面色一变，因为这一剑太强了，强到了他根本无法挡住。
<br/>　　不过，躲避根本来不及了，贺兰殇只能硬挡。
<br/>　　他的手掌猛然拍出，掌中源力浩荡，形成了一座源力小山，企图挡住这一剑。
<br/>　　轰！
<br/>　　剑气斩中了源力小山，小山轰然爆炸，化为漫天气浪滚滚。
<br/>　　而混沌剑气势如破竹，狠狠的斩在了贺兰殇的手掌之上，又是一声爆响，混沌剑气炸开，并伴随着血雾升腾而起。
<br/>　　嗖！
<br/>　　贺兰殇的身躯，被震退百丈，他的一只手掌已经消失，化成了血雾。
<br/>　　但是，仅仅是一瞬间，其消失的手掌，竟然飞速的长了出来，瞬间恢复如初。
<br/>　　“苏莫，你的确厉害，我们走着瞧！”贺兰殇脸色难看的说了一句，便转身向第二城飞去。
<br/>　　他知晓，自己不是苏莫的对手，想要擒拿苏莫或者是斩杀苏莫，根本不可能了，而苏莫想打败他，也不太可能。
<br/>　　所以，他不想再做无谓的战斗，准备返回第二城。
<br/>　　“你走不了了！”
<br/>　　苏莫声音冷漠，立刻再次出手了，并且准备一击绝杀。
<br/>　　对于这种偷袭他的人，他怎么可能放过，即便是得罪不死神朝，他也再所不惜。
<br/>　　唰！
<br/>　　苏莫手中长剑猛然一刺，霎时之间，一道巨大的剑影，如同千丈巨剑，向贺兰殇袭杀而去。
<br/>　　这一剑，是更加强大的吞天噬地，庞大的剑影，蕴含恐怖的吞噬之力。
<br/>　　这一剑威力更加的强大，并且，剑影击出之后，飞速的吸纳四周涌动的余波，壮大剑影的威力。
<br/>　　“可恶！”见苏莫再次出手，贺兰殇顿时大怒，面对这巨大的剑影，和强大的吞噬之力，他同样是无法躲避，只能硬挡。
<br/>　　只见他双掌齐出，掌印合而为一，化为了一只巨龟，挡在了身前。
<br/>　　轰！
<br/>　　在恐怖的剑影攻击之下，一切都是徒劳，巨龟随即被绞碎，庞大的剑影顿时击中了贺兰殇。
<br/>　　轰隆隆！
<br/>　　贺兰殇的身躯，在吞天噬地的轰击之下，一起爆炸开来，被击成了齑粉。
<br/>　　不过，贺兰殇有恃无恐，仗着他是不死族，根本不担心死亡。
<br/>　　只见在滚滚冲击波的之中，无尽的血气雾气飞速的凝聚，只需半个呼吸的时间，贺兰殇就能重铸肉身。
<br/>　　但是，苏莫怎么可能让对方重铸肉身，吞噬元神升腾而起，强大的吞噬之力，将大量的血雾和崩溃的冲击波，一股脑的吞噬了下去。
<br/>　　转瞬之间，现场恢复了平静，贺兰殇再也不可能重铸肉身了。
<br/>　　“这……！”
<br/>　　“苏莫那元神将贺兰殇吸收了？”
<br/>　　“这是什么情况？”
<br/>　　周围两三百人，个个目瞪口呆，不可思议的看着这一幕。
<br/>　　之前，他们已经看出来了，贺兰殇不是苏莫的对手，这并不让众人意外。
<br/>　　毕竟，苏莫的逆天，众所周知，而贺兰殇本身的修为，也就七品低级，无论是修为的根基还是法则的力量，都还差些火候。
<br/>　　但是，他们之前，倒是并不担心苏莫能将贺兰殇怎样，最多赶出大阵笼罩的范围而已。
<br/>　　然而此刻，他们都懵了，这贺兰殇下场到底如何，是被苏莫擒拿了？还是被苏莫杀死了？
<br/>　　“所有第三城之人，随我回城！”苏莫面色淡漠，环视一圈，朗声高喝道。
<br/>　　言罢，他二话不说，便向第三城返回而去。
<br/>　　现在，不宜在外恋战，大体的情况已经更清楚了，接下来就好办了。
<br/>　　剩下的第三城之人，面面相觑，没敢多留，纷纷紧随着苏莫而去。
<br/>　　“回城！”叶凌天见此，招呼一声，便也同样带着第一城的人，返回第一城。
<br/>　　跟随贺兰殇出来的第二城之人，哪里还敢多留，顿时作鸟兽散，纷纷返回。
<br/>　　古魔太子等第八城之人和第六城之人，并未阻拦，纷纷返回。
<br/>　　众人离去之后，古魔太子本想趁机将第六城之人全部击败，但是想到现在苏莫势大，他就没有出手，放过了那些第六城之人。
<br/>　　第六城貌似没有什么强者，威胁不大，但是，却可以为他所用，和苏莫等几城抗衡。
<br/>　　片刻之间，所有人全部回城，神昆小世界之内，陷入了安静之中。
<br/>　　但是，此时此刻，神昆小世界之外，却是并不太平。
<br/>　　“岂有此理，这苏莫未免未免太过狠毒了！”一位不死族的老者，面色铁青，怒声大喝。
<br/>　　此人头发花白，身穿暗青色长袍，脸庞狭长，暗含凌厉之色。
<br/>　　此人不是别人，乃是不死族的大长老，贺兰程。
<br/>　　“龙昌，若是贺兰殇有个三长两短，我们必然不会饶过苏莫！”另一位不死族的神王，面色难看的望着龙族大长老龙昌。
<br/>　　贺兰殇可是他们不死族万年难得一遇的天才，若是有个三长两短，他们决不罢休。
<br/>　　龙昌闻言，只是微微瞥了对方一眼，并未答话。
<br/>　　因为，连他也不清楚，贺兰殇到底如何了？
<br/>　　贺兰殇被苏莫的元神吞噬了，而苏莫的这元神，是轮回演变元神，是轮回之门演变出的元神，具体有什么能力，他们并不清楚。
