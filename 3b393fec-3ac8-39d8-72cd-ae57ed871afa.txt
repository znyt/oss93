　　　　随着时间的推移，死在三战队手中的高阶丧尸已经不计其数，可跳上城墙的高阶丧尸依然不断涌现，好似怎么都杀不光，这恐怕是狼牙战团成立以来遇到过最为密集的高阶丧尸群。
<br/>　　光是超级丧尸就出现过不下三头，这也说明当下这极短的时间里，丧尸的数量已经到达一百五十万之多。
<br/>　　而在南署基地内部坚守的狼牙战团总人数，却只有九万！
<br/>　　基地外围一大圈到处都是丧尸的尸体碎屑，高墙内的新武器每当一大·波丧尸涌现的时候，就发起了最强攻击，地面早已变得坑坑洼洼，丧尸被炸烂的躯体到处都是，有些地方甚至堆积成山。
<br/>　　空气中弥漫着令人作呕的腐臭气息，大量的黑血陷入地里，将南署基地的周围都变成了无比泥泞的环境。
<br/>　　两墙之间的一万五千人早已坚持不住，将大量的丧尸群放入基地之内，那些丧尸摇晃着脑袋，延着早已被规划好的道路前进，而后·进入南署基地内圈的那处迷宫。
<br/>　　无数的小队埋伏在各个拐角，他们手握武器等着那些丧尸走进，一旦进入最佳攻击地点，就会毫不犹豫的上前一通猛砸！
<br/>　　待在高台上的四百名射击队成员已经打出无数发子弹，迷宫·内还没出现需要他们援助的情况，但外墙的那些高阶丧尸，则是已经有不少逃脱了三战队的防线，在基地内到处作乱。
<br/>　　这些高阶丧尸大部分都被射击队第一时间解决，他们的命中率堪称恐怖，手握狼牙科技部制造而出的强力武器，六阶以下的高阶丧尸都难不倒射击队。
<br/>　　除此之外一直在时刻观察整个战区情况的侦查队，遇到迎面而来的高阶丧尸也会立即合伙解决，而其余大批的低阶丧尸则是任凭它们进入迷宫。
<br/>　　不得不说这些侦查队的成员都相当老练，哪怕没有指挥官的命令，也知道当下最应该做什么。
<br/>　　朝外攻击的新武器还在继续攻击，死在城外的丧尸已经多达七十多万，还有三十万则是被堵在沼泽之中亦或是失去行动能力，五十万进入了基地内部。
<br/>　　平均下来每一人就要杀死五至六头丧尸，加以配合的话，每一批小队都必须用最快速度杀掉几十头丧尸。
<br/>　　大尸潮来自那个没有生灵的最南方，这与他们在城市里遇到的大多数一阶丧尸完全不一样，能够从极远的最南边来到华夏的边境南署基地，这群丧尸本身的体能就相当恐怖。
<br/>　　所以实际上每一个小队面对的丧尸，大多都在二阶以上，这对于很多刚刚步入新人类，都只是一阶二阶的人来说，已经是他们最大的负荷。
<br/>　　基地外四面八方涌来的尸潮越来越多，从上空看这整片地域都成了丧尸的海洋，真正无穷无尽的一片尸海，而那一波又一波涌来的尸潮，则像是黑色的海浪，想要将最中心的南署基地吞噬。
<br/>　　南署基地也真正的成为了一处孤岛，被千万丧尸死死包围，已然走投无路。
<br/>　　除此之外高空还有呼啸不断的大量飞行丧尸，空中的作战更是激烈，驾驶着直升机的军法处成员很多都汗流浃背，那些飞行丧尸的敏捷度实在太高，经常需要极限躲避不然就会机毁人亡。
<br/>　　尤其是此时在看看下方地面，哪还有一点陆地？
<br/>　　密密麻麻全是丧尸！
<br/>　　而射击队的成员更是全力以赴，但凡飞行丧尸进入射程就使出了全部的注意力瞄准，只求能杀的越多越好！
<br/>　　不得不说陈少爷选出的这一百人射击队成员，一个个都命中率高的可怕，哪怕是在空中颠簸的状态里，哪怕周围不断有飞行丧尸呼啸而过，时刻有着生命危险，他们依然沉着无比的专注于瞄准。
<br/>　　飞行丧尸与普通的丧尸一样，唯有打爆脑袋才能算击毙，战斗直升机上的远程武器已经经过改良，攻击力自然非同凡响，但依然无法彻底解决飞行丧尸的威胁。
<br/>　　原因就是飞行丧尸太多了，多的就像是蝗虫！
<br/>　　一批又一批的飞行丧尸被击毙坠落，却不断有新的飞行丧尸从天边飞来。
<br/>　　空中作战的战斗直升机总计不过一百架，而高空的范围却宽广无比，空军要做的不仅是击毙，还要确保这些飞行丧尸不能突破他们的防线进入后方。
<br/>　　身后的基地可没有能够对付飞行丧尸的底牌！
<br/>　　只是随着这些飞行丧尸的数量不断增多，空军显然已经开始应接不暇，很多时候专注于瞄准的射击队成员刚解决完眼前这头，机身之后就立即冲过来另一只。
<br/>　　于是很快在空军内部形成了紧急的应对方案，相互调整了战斗位置，改为两两合作模式。
<br/>　　当一架直升机瞄准攻击时，另一架就需要密切注意周围是否有飞行丧尸偷袭，必要的时候给予提醒或是伸出援手。
<br/>　　狼牙在楚涵的带领下一向战斗模式激进，哪怕是射击队也是如此，于是在展开了合作方案之后，提醒一次没有，伸出援手的情况却在高空不断发生。
<br/>　　常常是一架直升机刚轰爆了一头飞行丧尸的脑袋，身后一只丧尸鸟就扑向机身尾翼，另一架直升机的射击队成员，就二话不说的瞄准，进攻！
<br/>　　而后‘轰’的一声将那头偷袭的丧尸鸟炸了个稀巴烂！
<br/>　　火光和冲击就这么擦着第一架直升机而过，在高空中划出一个抛物线状，紧接着就是良家直升机的错身而过，开始进行下一轮的一攻一守。
<br/>　　这种配合被狼牙空军打的很完美，但全程看在眼里的高少辉，则是彻底傻了眼。
<br/>　　“就不怕误伤？！”
<br/>　　他震惊无比的看着不断发生在眼前的一幕幕，开始高度怀疑这些空军是不是在玩极限，那些攻击飞行丧尸的炮火只要稍有不慎，就很有可能会攻击到同伴的直升机啊！
<br/>　　但更让他倍觉不可思议的是，这种让他提心吊胆的情况一次都没有发生，这些飞行员的技术是很好，但那些射击队成员的极限瞄准，就好像是开了挂！
