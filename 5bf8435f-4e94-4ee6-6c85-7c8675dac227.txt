　　第326章
<br/>　　程大雷骑在马上，挥舞着大斧，冲在队伍最前方。
<br/>　　这样的乱战是程大雷擅长的，一百单八斤的大斧抡开，当真是神挡杀神，佛挡诛佛。对方的军阵生生被程大雷杀出一条血路。
<br/>　　“弟兄们，跟我冲！”张肥马上大叫。
<br/>　　“杀杀杀，咱们豹字队什么时候都不能落了后。”高飞豹。
<br/>　　五支队伍一起冲锋，其实互相之间也有竞争。关鱼、赵子龙、秦蛮这算是稳重的，只是一味冲杀，并不会大吼大叫。最抢眼的就是张肥和高飞豹，二莽汉早就缠着程大雷要主动攻击，只是被程大雷压了下来。
<br/>　　一连压了半个月，二人心里早就窝着一团火。此刻有机会发泄出来，二人互相较着劲，都想争先，不愿落后。
<br/>　　一柄丈八蛇矛，一柄大刀，在战旗的助益下，完全成了战场绞肉机，所过之处，血流成河。
<br/>　　“哈哈，高二爷我看你莫要争了，今日定是我斩了姓莫的。”
<br/>　　“呸！张三，你也不怕风大闪了舌头！”
<br/>　　“嘿，那我们就走走看！”
<br/>　　“杀！”
<br/>　　五支队伍凝在一起，宛若一条战场黑蟒，摧枯拉朽，风卷残云。
<br/>　　十八家头领的队伍被蛤蟆寨冲散，大家直面莫鸣米本阵。
<br/>　　莫鸣米要重新估计蛤蟆寨的实力了，他开始撤退。
<br/>　　莫看张肥和高飞豹喊得最响，但冲在最前方的不是他们，而是关鱼。
<br/>　　黑色战神旗飘舞，关鱼骑在马上，当真如战神临时。战场之上，所见者皆惊，烈马驰过，大刀挥下时，人头滚滚而落。
<br/>　　程大雷骑在马上，已能看到莫鸣米的位置，二人相距大概是五百步，莫鸣米虽有重兵保护，但这样的距离，以程大雷的本事并不是做不到。
<br/>　　只要能将莫鸣米杀死，对方军阵不破自破，而蛤蟆寨也将取得这场战斗的胜利。
<br/>　　程大雷催马向前，关鱼等人紧随其后，程大雷挥舞着斧头，距离莫鸣米越来越近。
<br/>　　只见莫鸣米在战阵中央，一直在后退，手中令旗挥舞。
<br/>　　对方大军如浪潮一般退去，随即露出其下暗藏的『礁石』。
<br/>　　程大雷因为迫得最近，所以看得也仔细，只见从对方军阵中，出现层层身披黑甲的精兵。
<br/>　　手中持着长矛，列成整齐的方阵，一杆杆长矛宛若朝天的芦苇。
<br/>　　铛！
<br/>　　程大雷的斧头敲在一人的胸口，相撞冒出火花，因为程大雷的大力，对方虽然被击倒，可程大雷并没有攻破敌人的护甲。
<br/>　　重甲兵。
<br/>　　程大雷心里一激灵，随即冷汗浸湿后背，今日的一切，多像一场筹划已久的阴谋。
<br/>　　示敌以弱，诱敌深入，然后布下重兵，要将自己一网打尽。
<br/>　　“退！”
<br/>　　程大雷反应半点不慢，他口中发出一声怪叫，立刻下达命令。前队改后队，程大雷、关鱼几人断后，护送着队伍返回山寨。
<br/>　　莫鸣米眼中绽放出杀机，既然出来了，何必又要回去，我把你们引出来容易么。但他没有想到，程大雷反应可以这么快，而自己手下的重甲兵因为身披铁甲，速度自然快不了。
<br/>　　而轻骑兵的追击速度虽然能追上对方，可却又不是关鱼等人对手。
<br/>　　“狗贼，给我留下！”
<br/>　　从敌人阵中突然冲出一队人马，为首之人正是程大雷熟悉的，或者说徐神机更加熟悉的牧长庚。
<br/>　　牧长庚手持一杆长枪，直扑程大雷而来，牧长庚在蛤蟆寨受过的委屈，此刻都想了起来。三尸神暴跳，五灵豪气腾空，誓要杀了程大雷，洗刷程大雷的耻辱。
<br/>　　“关鱼，带大家先走。”
<br/>　　程大雷不退反进，用手中的大斧迎上了牧长庚。
<br/>　　“张猛！”程大雷做出吃惊的表情：“怎么是你？”
<br/>　　牧长庚桀桀冷笑：“没想到吧，狗贼，现在给我去死。”
<br/>　　“张猛，你怎么可以，怎么会是你！”程大雷狼狈躲避。
<br/>　　枪在牧长庚手中打了个旋，忽地一枪斜插，奔着程大雷的肋骨而来。以牧长庚的力气，这一枪若是扎中，直接可以将程大雷的五脏六腑捅个稀巴烂。
<br/>　　“我劈脑袋！”
<br/>　　本来动作缓慢的程大雷，速度突然变快，只是一招，便砍下牧长庚的脑袋。
<br/>　　马上的牧长庚变成了无头尸体，马又跑了一段，他的尸体才砰地一声倒地。
<br/>　　啧啧……
<br/>　　程大雷看着地上的尸体撇了撇嘴。
<br/>　　“亏你还是顶级武将，我才用一招你就倒下了。”
<br/>　　“狗贼，去死吧。”
<br/>　　呼天喝地的声音，程大雷回过头，看到一两千人向自己冲来。
<br/>　　“大当家，风紧扯呼！”关鱼回头大叫。
<br/>　　“扯呼扯呼。”
<br/>　　程大雷调转马头，一拍马屁股，一人一马向蛤蟆寨奔去。
<br/>　　最后总算是赶在大兵追过来之前逃回了蛤蟆寨。大门紧闭，逃回山寨的弟兄各就各位，都退到防守位置。
<br/>　　在一轮轮重箭抵过对方的攻击后，今天的战斗总算是结束了。
<br/>　　“大当家，他们退兵了。”徐神机道。
<br/>　　“是呐，退兵了。”程大雷心情不是很愉快，他心里明白，明日或许就是莫鸣米总攻的日子。
<br/>　　程大雷不敢闲着，先去见了云中龙三人。
<br/>　　“程当家，您没事实在太好啦，我们三个都在为程当家祈祷。”云中龙。
<br/>　　“好啦，好啦，别说废话了。”程大雷问：“你们三个带来多少人？”
<br/>　　云中龙三个对视一眼，报出一个数字。
<br/>　　“七百。”
<br/>　　程大雷心里盘算着，有这七百人加入，山寨的兵员已接近两千人。只不过这些人，当然不能和蛤蟆寨本来的弟兄相提并论，如今还无法作为战斗力量。
<br/>　　今天最令程大雷想不到的是，莫鸣米竟然藏着一支重甲兵。这样的重甲队伍不应该只在西方有么，莫鸣米是怎么折腾出来的。
<br/>　　重甲兵虽然人数不对，大概只在两千人左右，可却是莫鸣米手下最精锐的力量。
<br/>　　程大雷想到这点就直咧嘴，如何对付这支两千人的重甲兵，成了程大雷首先要考虑的问题，同时也成了今晚会议的主要内容。
<br/>　　重甲兵身披铁甲，弓箭很难造成杀伤力，兵器砍上去也只不过一个白印。当然，他们也不是没有弱点，比如行动缓慢，体力消耗大……
<br/>　　
<br/>(本章完)
