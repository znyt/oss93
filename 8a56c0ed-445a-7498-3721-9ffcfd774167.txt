　　当李然跟李毅峰解释龙隐部队的战略行动组的时候，Bravo小队搭乘的“黑鹰”直升机已经在楼房楼顶降落，麦克米兰带着Bravo小队队员跳下直升机，朝着楼梯内的罗昊、李毅峰、李然、慕容飞四个人跑来。
<br/>　　麦克米兰压低身体，对罗昊问道：“伙计，你还好吗？”
<br/>　　“你们如果再晚几分钟，可能就不那么好了。”罗昊向麦克米兰回答道。
<br/>　　麦克米兰看了眼被李然抓在手里的Parker，说道：“Parker，我们又见面了，没想到会以这样的方式的见面，命运真是令人难以捉摸。”
<br/>　　“你们认识吗？”罗昊对麦克米兰问道。
<br/>　　“认识。”麦克米兰点头回答道：“在阿富国的时候，我们曾经配合着执行过几次任务。后来我的部队离开了阿富国，而他跟西伦科特留在了那里。”
<br/>　　麦克米兰耸了耸肩，联系上詹姆斯，说道：“‘波塞冬’00，我们已经抵达军事监狱并且见到了‘八爪鱼’，现在准备撤离。”
<br/>　　联合行动司令部，詹姆斯看着通过战场视频系统反馈回来的画面，对着耳麦说道：“立刻带走‘八爪鱼’，并且从军事监狱中撤离，战斗机会在四分钟后抵达，并且进行空袭。”
<br/>　　“明白。”
<br/>　　麦克米兰松开耳麦，对罗昊说道：“‘波塞冬’00的话你们都听到了，两架F/A-18E‘超级大黄蜂’会在四分钟后对这座军事监狱进行空袭，把这座军事监狱还有这里的影子部队全都清楚，所以我们必须在空袭开始前从这里撤离出去，要不然，我们就要成为这次空袭的陪葬品。”
<br/>　　麦克米兰拍了拍身边一名Bravo队员的肩膀，说道：“Move！”
<br/>　　那名Bravo队员握着M4A1战术步枪，朝着停在楼顶上的“黑鹰”直升机走去。
<br/>　　“黑鹰”直升机上，机组成员控制着M134机枪，还在进行着防御型射击；“黑鹰”直升机旁，三名Bravo队员手持HK416A5战术步枪，也同样进行着防御射击，不断射击着还在通过绳索爬上楼顶的影子部队士兵。
<br/>　　军事监狱另一栋楼房楼顶，一名影子部队士兵将一具“毒刺”防空导弹扛在肩膀上，锁定了楼顶上那架随时准备起飞的“黑鹰”直升机，并且扣动了发射键。
<br/>　　“咻！”
<br/>　　导弹笔直的飞向“黑鹰”直升机。
<br/>　　轰！
<br/>　　被“毒刺”导弹击中的“黑鹰”直升机瞬间发生了爆炸，变成了一团火球。火光冲天而起。
<br/>　　“Damn！”
<br/>　　麦克米兰用手挡在自己眼前，遮挡住刺眼的火光，嘴里忍不住咒骂了一声。
<br/>　　直升机被导弹摧毁，显然是打乱了麦克米兰原来的撤退计划，不得不重新退回到楼房内。麦克米兰按着耳麦，对詹姆斯说道：“‘波塞冬’00，我们需要更改撤退计划。”
<br/>　　“是的，小伙子们。”詹姆斯站在屏幕前，小幅度的踱着步，对耳麦那头的麦克米兰说道：“但是你们必须抓紧时间了，战斗机将会在三分钟后抵达军事监狱，你们必须在炸弹落入军事监狱之前，带着‘八爪鱼’从那里离开！”
<br/>　　“Roger！”
<br/>　　麦克米兰松开耳麦对罗昊说道：“我们必须在三分钟内，从这个该死的破监狱里离开，这听上去似乎很难完成，但是我们必须做到，要不然就死定了。”
<br/>　　“倒计时三分钟！”
<br/>　　罗昊在自己手表上设置了三分钟的倒计时，然后对着李然示意了一下，押着Parker走在前面。他们上来方向的一段楼梯被炸断，现在只能从另外一头的楼梯下楼。
<br/>　　“Go！Go！Go！”
<br/>　　麦克米兰握着HK416A5战术步枪，跟在李然和罗昊身边朝着楼下快步走去。麦克米兰扫了眼周围的楼道，对罗昊问道：“你们到底是对这栋楼房做了什么？你们是想要把这里炸了吗？”
<br/>　　“本来是有这个打算，不过最后还是放弃了。”罗昊咧嘴笑道。
<br/>　　“莱昂对你的评价很高。”麦克米兰沉默了一会儿，突然开口对罗昊说道。
<br/>　　罗昊诧异地看向麦克米兰，问道：“你认识莱昂？”
<br/>　　“是的，我们是同一批进入海豹的，在新兵连的时候我们就认识，而且关系很好。”麦克米兰点头说道：“谢谢，谢谢你帮助了他，不然他很有可能会死在阿富国，虽然是死在恐怖分子的子弹下，但是真正害死他的，却是自己人。”
<br/>　　“只是碰巧而已。”罗昊在麦克米兰肩膀上拍了一下，看到突然从窗户外跳入楼房的影子部队士兵，立刻转过枪口，开枪射击，“遇敌，接火！”
<br/>　　“Fier！Fier！Fire！”
<br/>　　麦克米兰大声叫喊着，对着通过窗户跳入到楼房内影子部队士兵进行点射。
<br/>　　麦克米兰拿出一颗手雷，拉掉保险销，对着身边的人喊道：“Grenade！”接着按压保险夹，令得保险夹弹出，甩手把手雷扔向走廊中的影子部队士兵。
<br/>　　轰！
<br/>　　手雷落点附近的影子部队士兵全都被炸倒在地，好几个影子部队士兵脸上被手雷破片击中，瞬间变得血肉模糊。
<br/>　　借着手雷的压制效果，麦克米兰、罗昊等人飞快地朝着楼下跑去，可是刚跑到底楼，就被从楼房外照进来汽车大灯逼退回去。
<br/>　　两辆车顶上架着.50勃朗宁重机枪的悍马堵在了楼房外，堵住了罗昊、麦克米兰他们的退路。
<br/>　　“Bravo4！”
<br/>　　麦克米兰对着小队中的火力手使了个眼色，那名Bravo队员摘下自己背包上的M72LAW火箭筒，锁定了其中一辆悍马，然后按下发射键。
<br/>　　咻——！
<br/>　　火箭弹从火箭筒发射管中飞出，冲出楼房，正面击中了左侧那辆悍马车。
<br/>　　轰——！
<br/>　　被火箭弹击中的悍马车化作一团火球，火光冲天。右侧那辆悍马也因为爆炸，被掀翻出去，在空中“优雅”的转了圈，把汽车中的影子部队队员转的晕头转向后，重重的砸落在地面上，饶是悍马车坚固，此时也被摔得扭曲变形，汽车内的影子部队士兵被倒扣在地面上的悍马车压着，无法从车厢中爬出来，而破裂的油箱还在往外漏着油。
