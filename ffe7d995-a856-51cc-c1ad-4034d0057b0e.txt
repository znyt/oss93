　　接下来，阎魔鬼圣详细的为苏莫讲解了一番混天魔功的玄妙，以及各种修炼要点和注意事项。
<br/>　　整整讲解了几个时辰，让得苏莫对于混天魔功，有了非常透彻的了解。
<br/>　　，这种等级的功法，若是没有阎魔鬼圣的讲解，他就算有这门功法的全部法诀，也无法参悟其中的玄妙。
<br/>　　因为实在是太深奥了，以他现在的修为层次，根本理解不了。
<br/>　　“修炼此功法，最主要的是守住本心，意志坚定，方可练到高深境界。”阎魔鬼圣提醒道。
<br/>　　“前辈放心吧！不会让你失望的！”苏莫点头道。
<br/>　　“嗯！以汝的悟性，一年之内练成第一层境界，难道不大！”
<br/>　　阎魔鬼圣倒是对苏莫颇为有信心，言罢，继续道：“那汝修炼吧！又何不解之处，再来问吾！”
<br/>　　随即，他便不再出声。
<br/>　　少倾之后，苏莫抛却心中杂念，将混天魔功的各种要诀，在脑海中仔细梭理一遍之后，便开始了修炼。
<br/>　　此功法，共有七层境界，每提高一层境界，威力便能大上十倍，堪称逆天无比。
<br/>　　他只要能完全练成第一层境界，战斗力立刻便能暴涨七、八倍。
<br/>　　意念一动，苏莫体内神丹一震，磅礴的玄力从神丹中涌了出来，在他经脉中流转。
<br/>　　他开始正式修炼第一层境界的混天魔功。
<br/>　　这运转路线，与巫尘本身修炼的功法完全迥异，遍及全身十二正经、奇经八脉。
<br/>　　随着玄力在经脉中运转一个大周天，这些流动的玄力逐渐改变了属性，颜色向墨色转化。
<br/>　　本来平静的玄力中渐渐充满了狂暴的气息，这是玄力在向魔力转化，威能渐渐提升。
<br/>　　随着而来的，便是魔力刺激着苏莫的身体，让他全身都微微颤抖了起来。
<br/>　　苏莫意沉丹田，守住本心，一刻不停的修炼。
<br/>　　这是功法转换的过程，要持续很长的一段时间。
<br/>　　就在此刻，苏莫手掌一番，手掌中出现了十几块下品灵晶，他一把将其按进了身下的阵法凹槽之内。
<br/>　　因他本来修炼的功法等级只要帝级上品，神丹内的玄力极为有限，不足以让他将功法转化成功，还需要大量的灵气。
<br/>　　嗡嗡~~~
<br/>　　随着灵晶进入阵法凹槽，修炼室内的阵法，顿时启动了起来，沟通了天冥星之下的大地龙脉。
<br/>　　轰！
<br/>　　霎时之间，磅礴的灵气混杂着大地精气，从下方涌了出来，如同雾气升腾，充斥在整个修炼室中。
<br/>　　所谓大的大地精气，其实也是灵气，只不过蕴含浓郁的土属性而已。
<br/>　　“这么厉害！”苏莫心中惊讶不已，这修炼室内的灵气浓郁程度，足有外界的百倍不止。
<br/>　　难怪这么多无助年轻人，来此修炼，真是绝佳的修炼宝地啊！
<br/>　　“给我吞！”毫不犹豫，苏莫立刻稍稍催动了吞噬战魂，开始吸纳灵气。
<br/>　　浓郁的灵气受到吞噬战魂的吸纳，顿时疯狂的向他体内钻去，整个修炼室中的灵气，瞬间便被吸干。
<br/>　　不过，修炼室内的阵法，沟通了大地之下的龙脉，灵气源源不断。
<br/>　　苏莫静气凝神，一边吞噬灵气，一边修炼混天魔功。
<br/>　　功法的转换，不是一朝一夕的事情，必须要将整个神丹内蕴含的玄力，全部转化成魔力。
<br/>　　嗡嗡~~
<br/>　　时间缓缓流逝，阵法内涌出的灵气，根本跟不上苏莫吞噬的速度，所有灵气几乎是被吞噬战魂直接从阵法能吸扯出来的一般。
<br/>　　整个修炼室内，灵气浓郁到了极致，化为了粘稠的灵雾。
<br/>　　苏莫整个人完全沉浸在了修炼之中，但却不知道因为他的吞噬，影响了所有人。
<br/>　　外界。
<br/>　　依旧有不少人在等待，因为修炼室太少，根本不够用。
<br/>　　嘭！嘭！嘭！
<br/>　　就在此刻，很多修炼室的大门，纷纷打开了，其内的巫族年轻人均是走了出来。
<br/>　　“怎么回事？修炼室内的灵气怎么变得如此淡薄了？”
<br/>　　“不清楚啊！我正在冲击修为瓶颈，灵气突然就断了！”
<br/>　　“玛德，到底怎么回事？”
<br/>　　“难道地灵塔出问题了？”
<br/>　　众人纷纷开口，个个面带疑惑，甚至有的人怒骂出声。
<br/>　　有的人正在突破境界的紧要关头，灵气突然断了，别提多郁闷了。
<br/>　　所有人都是一脸闷逼，那些在修炼室之外等待的人，更是搞不清情况。
<br/>　　不多时，这第一层空间之内，几乎所有的炼器室都打开了，所有在炼器室内修炼的人，全部都走了出来。
<br/>　　唯剩下两间修炼室，还是大门紧闭，其中一座修炼室，正是苏莫所在的修炼室，另一间不知是何许人也。
<br/>　　“不会是龙脉内的灵气耗光了吧？”有人猜测道。
<br/>　　“这不可能，这第一层空间修炼室沟通的龙脉，足够使用几百年，现在也才几十年而已，怎么可能耗光？”有人立刻反驳。
<br/>　　“可能是龙脉出问题了，或者是地灵塔出问题了！”
<br/>　　众人不知道原因，只能如是的猜测道。
<br/>　　巫长远也是一脸郁闷的从修炼室中走了出来，他修炼还不足半个时辰，地灵塔就出问题了，心中也是非常无奈。
<br/>　　这时，最后的两间修炼室中，其中一间缓缓打开，一名俊逸的青年大步的走了出来。
<br/>　　这名青年身穿蓝衣，身形挺拔，脸庞如斧砍刀削，棱角分别。
<br/>　　此人，正是巫天御。
<br/>　　此刻，巫天御的脸色很不好看，他闭关了近一个月，就是要一举达到武帝境三重境界。
<br/>　　本来，他就要成功了，但最后时刻，因为灵气不继，让他功亏一篑。
<br/>　　巫天御虽然有很多的灵晶，但他可没有吞噬的能力，靠吸纳灵晶中的灵气修炼，实在太慢了。
<br/>　　“怎么回事？”巫天御大步而出，冷冷的向众人喝问了一声。
<br/>　　“天御兄，我们都是一样，修炼室中都断了灵气！”
<br/>　　“是啊！可能是龙脉或者地灵塔出了问题！”
<br/>　　“或许片刻之后就能恢复了！”
<br/>　　众人纷纷开口，面色客气至极。
<br/>　　巫天御在巫族之中，名气较为响亮，很多人都熟知他。
<br/>　　而且，上次苍穹世界一役，巫天御挡住了苏莫，让众巫族天才免于全军覆没，得到了几位祭司的重赏。
<br/>　　因为此大功劳，巫天御得以进入巫族祖地修炼一个月，修为突飞猛进，实力暴涨，井然成了族中的顶尖天才。
<br/>　　祖地，可是族中禁地，只有族长、四位祭司、少司以及那为数不多的核心长老，才有资格进入祖地。
<br/>　　巫天御本来的天赋，在巫族之中虽然算是不可多得的天才，不过却并没有达到顶尖天才的行列。
<br/>　　但因为得以进入祖地修炼一个月，其如今的天赋、战力，比之族中那些顶尖天才，都不差多少了。
