　　“对汝来说，太过遥远，汝目前的任务，只是强大自身！”模糊的虚影淡淡的说道。
<br/>　　“强大自身？”
<br/>　　苏莫闻言，面色凝重，继续问道：“前辈，我只是一具分魂，若是我足够强大，主魂怎么办？”
<br/>　　苏莫问出了心中的疑惑，他若是足够强大，帝一魂和拓跋青云都将被他镇压，难道对方不想复生了吗？
<br/>　　“不，汝若是能融合主魂，汝就是主魂！”模糊的人影，微微摇了摇头说道。
<br/>　　“哦？”苏莫闻言，眸中精光微闪，他若是能融合主魂，他就是主魂。
<br/>　　不过，他可不愿意做主魂，更不想成为轮回天尊。
<br/>　　“那您就不担心，我超越了主魂之后，而不融合，将其永远镇压，而前辈你，将永远无法复生！”苏莫沉声问道，这是他疑惑的地方。
<br/>　　按理来说，主魂才是对方复生的关键，他若是超越了主魂，对方还怎么复生？
<br/>　　“吾不求复生，但求你三人之中，有人能问鼎荒界，打破桎梏。”模糊人影悠悠的说道。
<br/>　　“荒界？”
<br/>　　苏莫闻言疑惑，对方不求复生，那么轮回的意义何在？
<br/>　　他就算问鼎，也与对方无关吧！
<br/>　　略一沉吟，苏莫问道：“荒界是哪里？神域吗？”
<br/>　　“汝所处的这个世界，四大星域、亿万小世界、包括如今的神域，都是在荒界之内！”模糊的人影没有隐瞒，徐徐说道。
<br/>　　“荒界之外，还有其它世界？”苏莫眼眸闪亮，既然这里是荒界，那么便有其它的世界。
<br/>　　这还是他第一次听闻这种隐秘，不禁大感好奇。
<br/>　　“鸿蒙诞与元始，元始演化三千世界，荒为其中之一。”模糊的人影说道。
<br/>　　“三千世界！”苏莫闻言震惊，难道在这个宇宙之外，还有两千九百九十九个世界？
<br/>　　“荒界存在了数千万年的岁月，从遥远的太古蛮荒，经历了武道的初始、繁盛、巅峰，再到如今的衰落凋零，一个纪元即将终结！”
<br/>　　模糊的身影，背负着双手，声音中充满了万古沧桑，悠悠仿若自语。
<br/>　　“纪元终结会怎么样？”苏莫好奇的问道，此时此刻，面对这位太古时代的至强者，他感觉自己无比的渺小。
<br/>　　“纪元终结，整个荒界之内，将荡然无存，无尽生灵、千万种族、亿万星辰，全部毁灭，无一幸免！”模糊的身影淡淡的说道。
<br/>　　“什么？这怎么可能？”
<br/>　　苏莫闻听此言，顿时心中一震，一切都尽皆毁灭？这不是世界末日了吗？
<br/>　　他的眉头，瞬间紧皱了起来，他感觉对方像是在扯淡，但是，他知道，对方不可能欺骗他。
<br/>　　模糊的身影，并未向苏莫解释什么，自顾自的继续道：“吾诞于蛮荒时代，带领人族在夹缝之中生存，茫茫武道，慢慢求索，终掌无上大道，定生死轮回；数千万年以来，不曾有一丝懈怠；然，想要超脱纪元，谈何容易？超脱纪元，这是荒界所有至强者的夙愿，为此不惜任何代价。”
<br/>　　言及此处，模糊的身影沉默了下来，气氛略显凝固。
<br/>　　苏莫同样沉默了下来，在此人的言语之中，他仿佛感受到了对方的人生。
<br/>　　对方生于太古蛮荒，摸索武道，不断的前进，从弱小到巅峰。
<br/>　　但是，到了巅峰之后，才发现，并不能真正的超脱。
<br/>　　“前辈，您是为了超脱纪元，所以才进入轮回？”苏莫面色郑重的问道，他心中明白，看来只有超脱纪元，才能在纪元终极之后生存下来。
<br/>　　“嗯！”
<br/>　　模糊的身影微微颌首，道：“纪元的终极，决定了荒界所有生灵的命运，吾唯有不断的轮回，才能有一丝希望！”
<br/>　　“明白了！”苏莫点了点头，他明白了对方轮回的意义，是想要超脱纪元。
<br/>　　“前辈，我们三位轮回之身，唯有融合，才能有希望超脱纪元吗？”苏莫面色凝重的问道，这是一个关键。
<br/>　　因为，一旦融合，他将不再是他。
<br/>　　“轮回之中，变数万千，一切皆有可能！”模糊的身影幽幽的说道。
<br/>　　“一切皆有可能！”苏莫闻言一怔，对方此言是什么意思？
<br/>　　难道说，轮回之身，可以不融合？也有超脱纪元的可能？
<br/>　　此时此刻，苏莫心中还有太多的疑惑。
<br/>　　比如，他若是超越了帝一魂而不融合，对方不是不能复生了吗？
<br/>　　再比如，怎样才能超脱纪元？
<br/>　　不是有三千大世界吗？能否去其它的大世界？
<br/>　　他的吞噬元神，到底是什么元神？
<br/>　　主魂不是拥有对方的意志吗？对方为什么还让他超越主魂？
<br/>　　等等等等，一切的一切，还是让苏莫有些疑惑。
<br/>　　“前辈……！”
<br/>　　苏莫还欲再言，对方却是抬手打断了他的话。
<br/>　　“汝走吧！去完成汝的任务！”模糊的身影挥了挥手道。
<br/>　　“这就走了，不是还有福泽吗？”苏莫疑惑的问道。
<br/>　　“吾之轮回之身，不需要福泽！”模糊的身影淡淡的道。
<br/>　　呃！
<br/>　　苏莫闻言无语，他还想得到一些好处呢，居然什么都没有！
<br/>　　“前辈，我再问您最后一个问题，若是我们三具轮回之身，最终没有融合，那您还能复生吗？”苏莫大声问道，如此逆天人物，他不相信没有底牌。
<br/>　　按照对方之言，纪元终极已经不远了，而对方不可能完全将希望寄予轮回之身。
<br/>　　“能！”模糊的身影口中，淡淡的传出了一个缥缈的字音。
<br/>　　随即，其一挥手，天地旋转，斗转星移。
<br/>　　一瞬间，苏莫发现，自己又回到了外界，位于平台的上空。
<br/>　　而下方平台之上，战斗还在继续，交战之人，乃是涂赢和姬月松。
<br/>　　此时此刻，战况极为激烈，姬月松整个人犹如一个骄阳，威猛无比。
<br/>　　但是，姬月松却是落入了下风，完全被涂赢压制。
<br/>　　周围的议论之声，不绝于耳，均是惊讶于涂赢的强大。
<br/>　　“段惊天出来了！”
<br/>　　“不知道他得到了什么福泽，羡慕啊！”
<br/>　　“以后肯定更逆天了！”
<br/>　　苏莫乍一出现，众人便发现了他，纷纷投来羡慕的目光。
<br/>　　苏莫闻言无语，他什么也没得到，白忙了一场。
<br/>　　当然，也不是毫无所获，首先他向玄天神王展现了自己的潜力。
<br/>　　其次，他得知了一些隐秘，或许这些不是什么大秘密，但是，若是不来此地，他很难知晓。
<br/>　　唰！
<br/>　　苏莫身形一闪，向下方飞去，稳稳的落在了沁儿的身边。
