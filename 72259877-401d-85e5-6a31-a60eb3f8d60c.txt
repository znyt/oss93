<br/>				    这一大清早，正在郭家的人忙着安排上路的事情时，郭家外面一群人气势汹汹而来，将郭家大门围堵住。<br/>　　<br/>　　    “去！拍门！”<br/>　　<br/>　　    为首的中年男子声音一落，便见一人快步上前，拍打着那郭家大门：“开门开门！快开门！”<br/>　　<br/>　　    门房的人听到动静打开一看，就见外面围堵了一大群人，而且一个个脸色都并不好看，不由的吓了一跳：“你们都是谁啊？要做什么？”<br/>　　<br/>　　    前面的人朝两边退开，露出了后面的中年男子以及那手上包扎着，用一条白纱挂在脖子上的男子。<br/>　　<br/>　　    看到这两人，门房终于认出了来人：“您、您是李家家主？”李家家主怎么跑他们这里来了？<br/>　　<br/>　　    “哼！”李家主冷哼一声，负着手而立着，他睨了门房的老者一眼，道：“我们今天是来拿人的！去跟你们家主说，让他把人给我交出来！”<br/>　　<br/>　　    听着这话，门房老者一惊，连忙招手唤来一名护卫，让他快往里面去禀报着。<br/>　　<br/>　　    正跟他父亲在厅中坐着，说着话的郭家主见管家神色匆匆进来，便问：“何事慌张？”<br/>　　<br/>　　    “家主，老太爷，李家的人来了，那李家主带着人围堵在我们大门前，说什么让我们把人交出去。”管事将外面的事情禀报了一下。<br/>　　<br/>　　    听到这话，坐在厅中主位的郭老太爷眉头一拧，看向他儿子，问：“这李家是来做什么？交什么人？”<br/>　　<br/>　　    “父亲有所不知，昨夜……”<br/>　　<br/>　　    郭家主将昨夜郭信宁跟他说的事情对他父亲说了一遍，最后，道：“我后来又让人去玲珑阁询问了，是那李家公子喝多了酒起了色心想调戏谢家那小姑娘，谁知调戏不成挥拳就要打谢玉堂，所以当时就被凤九给一脚踹下了楼梯。”<br/>　　<br/>　　    他声音一顿，道：“我本想着这事李家的人理亏，他们不会将事情闹大，不过，估计是打听到他们不是我们郭家人，所有才敢上门来要人吧！”<br/>　　<br/>　　    听着这话，郭老太爷摇了摇头，叹息道：“这李家小子心胸有限，目光更是短浅，可真是比不上他父亲李老头的处理作风。”<br/>　　<br/>　　    “父亲，我去看看吧！”郭家主边说着边站了起来。<br/>　　<br/>　　    郭老太爷端起茶杯抿了一口茶水，声音缓慢的道：“他既然能找上门来，估计不会轻易罢休，你看着处理吧！这种事情就不要惊忧到我们的客人了。”<br/>　　<br/>　　    “是。”郭家主应着，这才往外走去。<br/>　　<br/>　　    郭信宁听到府外的动静后，询问得知是李家的人找上门来了，顿了一下，他便往凤九他们所在的院子走去。<br/>　　<br/>　　    这事他父亲会处理，他还是去看看凤九他们准备好了没有。<br/>　　<br/>　　    到了院中，见他们正要出来，他有些诧异：“你们起这么早？吃过了吗？”<br/>　　<br/>　　    “吃过了，正要过去找你呢！”谢玉堂说着，便问：“我听小九说郭大哥和你祖父要跟我们一路走？”<br/>　　<br/>　　    “是，我们要去易宝楼。”郭信宁温声说着，而后道：“对了，糕点玲珑阁的人送来了。”<br/>
