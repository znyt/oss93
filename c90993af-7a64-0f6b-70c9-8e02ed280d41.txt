　　　　这白茫茫的天劫也不知道维持了多久，天地肉眼不可见，时空仿佛被静止定格。
<br/>　　直到众生物的眼睛恢复了清明，抬头望天，一片无根的空中，天空澄碧，纤云不染。
<br/>　　远山含雾，清风微凉。
<br/>　　这样子分明是天劫结束了，传说中七十二道必死天劫终于在惊天动地下结束了。
<br/>　　倏然，一个个撑着惊骇的目光朝着山谷地面放眼过去，试图寻找那道巍峨的蛇影。
<br/>　　天劫结束，那么蛇精是否还活着？
<br/>　　这次一个个心中也不在多下结论，因为这条蛇精打了太多次众人的脸，一次次不屈不挠的活下去，蛇影屹立不倒。
<br/>　　他在六十六道天雷天火的轰炸下都活了下来，那么最后这六道天雷天火虽然毁天灭地，但现在众生物希望可以看到奇迹。
<br/>　　必死天劫要是能渡过去，这也是对一些即将要证道精怪的巨大鼓舞。
<br/>　　可惜，这次它们又失望了，不想看到偏偏出现，现在想看到，但那条巍峨睥睨的蛇影却不可见了。
<br/>　　山谷地面秃噜一片，废墟一片，仿佛被人掘地三尺了一样，非常的狼藉。
<br/>　　那些坑坑洼洼，那些裂缝不知道蔓延多少里，甚至很多小山头都崩裂了。
<br/>　　哎...
<br/>　　无数叹息声在各个区域响起，那条变态的蛇精终究没能熬过去，当最后究极的天劫六合一降下，万物皆灭，他也没有例外。
<br/>　　没有任何生物可以在这样的天劫下活着。
<br/>　　没有，妖王也不行。
<br/>　　不意外，只是有点可惜了，因为那条蛇精太惊艳了，他渡劫的所作所为让人荡气回肠，将天雷天火视若无物，抓起扔掉，一拳粉碎，吞天雷吃天火。
<br/>　　这些壮举，试问当今天下，说能做到的。
<br/>　　而且要知道，这是传说中的七十二道必死天劫，威力远远比那些四九天劫强大不知道多少。
<br/>　　完全不是一个层次。
<br/>　　可惜了，真的可惜了。
<br/>　　如此惊艳的蛇精要是证道渡劫成功，那么他势必是妖灵山脉的第六大妖王，甚至将来的成就会比妖王更高。
<br/>　　但这一切都在天劫无情的摧毁下，灰飞烟灭，天还是那片天，地还是这片土地，那道蛇影却永远消失。
<br/>　　天道无情，生物修炼，越是血脉高贵，越是强大，就越艰难。
<br/>　　毕竟这个世界，神兽早就只存在于传说之中。
<br/>　　这种结局虽然悲烈了一点，但这绝不是偶然，而是必然。
<br/>　　必死天劫之下，没有侥幸。
<br/>　　猴儿山的两位猴大王叹息之余更是恨的咬牙切齿，该死的蛇精，死了就死了，但是瞧瞧因为他的渡劫，猴儿山成了什么鬼样子。
<br/>　　仿佛是一百年没有动物生活的荒山野岭，到处坑坑洼洼，完全没有了生机，简直是惨不忍睹。
<br/>　　要知道猴儿山之前可是好比花果山，比外面那些一级的旅游景点风景不知道要好多少。
<br/>　　现在经历一场天劫，翻天覆地，犹如从一个极品大美女，非要整容把自己给整惨了一样。
<br/>　　“大哥，那条蛇精死了是一了百了，但我们的损失找谁算账，我们的气找谁出”赤阳巨猿看到这样的猴儿山，简直要暴跳如雷。
<br/>　　猿猴王也是一脸日了狗，可谓天降祸事，好端端风景秀丽的猴儿山就成了这般鬼样子，被人糟蹋了一样。
<br/>　　气啊，气的要原地爆炸，偏偏正主已经在天劫的轰炸下，灰飞烟灭了。
<br/>　　人死如灯灭，你能怎么办，对方死的彻底，连尸体都没给你留下。
<br/>　　你想鞭尸撒气，都没地方下手。
<br/>　　忽然，猿猴王跟赤阳巨猿一个眼神对碰，非常默契的朝着一个方向望去，脸上充满了戾气跟杀气。
<br/>　　牛妖王感觉到两股冲天的杀气，顿时不寒而栗，然后二话不说悲悯的冲了出去。
<br/>　　“主人，主人，我的主人”
<br/>　　一道胖乎乎的身影踏空而下，落到了废墟的山谷地面上，似乎在寻找一些什么。
<br/>　　主人，你要是死了，老牛我怎么办，我也会死的很惨的。
<br/>　　牛妖王欲哭无泪。
<br/>　　他的身后，跟着一颗高大挺拔的大树，这大树行走起来非常的滑稽，一路行走，一路树叶簌簌而落，自带特效。
<br/>　　“主人，你真的GG了吗，老牛的心好痛，你死了，老牛同样也活不了啊”
<br/>　　牛妖王说的夸张，但是脸上却不见什么眼泪，而是在这坑坑洼洼的地面疯狂寻找。
<br/>　　可惜最后，一牛一树连一片蛇鳞都没找到。
<br/>　　很显然，涂小安死了，猴儿山两位猴大王一定会把气全部出在牛妖王的身上。
<br/>　　这头牛的处境现在可是非常的危险。
<br/>　　“主人，你真的死的好惨，连一丝渣渣都没留下”树精身上树叶疯狂的飘零，但奇怪的是，它的树叶依然茂盛，一点不见少。
<br/>　　一般的大树，按它这样动不动就簌簌而落的掉树叶，早就成了光秃秃的树了。
<br/>　　牛妖王想起要面对两个猴大王的怒火，就一副摇摇欲坠的样子，一手撑着树身，看着查理道：“主人现在死了，你伤心不”
<br/>　　树精查理道：“本查理虽然跟主人的时间很短，但本查理重情重义，主人死了，自然伤心欲绝”
<br/>　　“哼，你在放屁，你分明在幸灾乐祸”牛妖王铜铃大眼一瞪，很是吓人，心中憋屈，有气没地方出，只能找这个树。
<br/>　　树精被这一喝，有点小心虚道：“你怎么看出来的”
<br/>　　“你要是真的伤心欲绝，你肯定会掉果实下来，但现在老牛张望了半天，也不见你掉下一颗果实来”牛妖王一副福尔摩斯的样子，洞若观火。
<br/>　　树精：“......”
<br/>　　它既然无言以对，被分析的好透彻。
<br/>　　主人刚死，你就惦记我的果实，这样真的好么？
<br/>　　“死牛，你也比我好不到哪里去，主人刚死，你就惦记本查理的果实吃，混账”
<br/>　　牛妖王冷哼一声：“老牛我痛不欲生，想化悲愤为力量不行吗”
<br/>　　说不定过一会儿，他这辈子也没机会吃果实了。
<br/>　　“滚...”
<br/>　　一牛一树忽然开始没羞没躁的斗嘴起来，完全不分场合，不分环境，到是将两位猴大王弄的有点措手不及，它们还没出手，就自己内讧了？
<br/>　　就在这时，忽然...
<br/>　　一牛一树一个激灵，浑身笼罩了一股暖洋洋的光芒，旋即，它们的眼睛慢慢的睁大，嘴巴张的跟一口小箱子一样，呆若木鸡，那表情简直难以置信，仿佛天上掉馅饼。
<br/>　　这种画面同样在另外几个区域发生，只见通体九彩色的仙灵孔雀青枭，黄金瞳，还有三条蛇都是如此，身上莫名笼罩一股暖洋洋的光芒，然后气息就开始暴涨。
<br/>　　超等生物的气息愈演愈烈，席卷四方，惹的附近无数生物诚惶诚恐。
