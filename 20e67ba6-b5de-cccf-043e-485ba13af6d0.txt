　　召开拍卖会的地点在小镇歌剧院。
<br/>　　没人知道这个歌剧院在这个小镇上存在了多少时间，仿佛在这个小镇出现的那一天起，这个歌剧院就已经存在了。
<br/>　　在平时，这个歌剧院是一处旅游景点，有时也会有一些舞台剧在这个歌剧院上演，而等到每年源代码拍卖会开始的时候，这里就是拍卖会会场。
<br/>　　广场上的灯光全部打开，一道道光柱打在那栋至少有上百年时间的歌剧院上，就连歌剧院外墙上因为老化缘故而产生的裂纹，也是在灯光下清晰可见。
<br/>　　或许是为了保持歌剧院的原貌，那些老化开裂的外墙并没有被刻意修复，而这也使得这家歌剧院的历史和岁月痕迹更加浓厚了一些。
<br/>　　歌剧院门口铺着一条红色的地毯，地毯顺着尝尝的阶梯一直铺到歌剧院门口，就跟某些颁奖典礼一样。
<br/>　　地毯的宽度并不算宽，刚好够两个人并肩同行，红毯两侧只是象征性的拉起了带子进行隔离，并没有颁奖典礼上那么严密的安保措施，可是来这里参加拍卖会的人却都是很守规矩的顺着红毯往歌剧院里走去，并且在歌剧院大门口停了下来，接受门口两个穿着西装的年轻人使用金属探测器进行检查。
<br/>　　来这里参加拍卖会的有些都是世界知名武器公司的代表，身份不凡，可是到了这里，他们却是安分的按照这里的规矩来，这让罗昊感到非常意外，他很想知道，这个小镇背后到底有着什么样的背景，能够令这些大人物都乖乖听话。罗昊自然也是询问过子午这个问题，但是子午也并不知道，子午告诉罗昊，这个小镇背后不仅仅只是有一个背景，它的背后很有可能有一些国家的影子，在得到默认的情况下进行交易。
<br/>　　因为并不是什么正式的典礼，所以来参加拍卖会的人穿着都非常随意，他们中间甚至有人带着鸭舌帽和墨镜，把自己的脸遮掩起来，而在歌剧院门口负责检查的两个人也没有去追究，只要是出示了表明自己能够参加拍卖会资格的卡片，就能够入内。
<br/>　　“我感觉到在暗处有人在盯着这里。”
<br/>　　罗昊眼睛警惕的向四周扫视着，但是他并没有在黑暗中发现什么端倪，这些藏身在黑暗中的人非常具有经验，将自己很好的隐藏了起来。
<br/>　　“应该就是负责执行小镇规矩的人。”子午看了眼眼前被地面探照灯照亮的歌剧院，说道：“我以为这辈子不会再来这里了，没想到隔了十五年还会再次回到这里，去看看吧，号称这个世界上规模最大的源代码拍卖会。”
<br/>　　歌剧院门口，几个人分别出示了自己身上的红色卡片，随后在接受了检查后，便是走进了歌剧院。
<br/>　　歌剧院内部也是欧式风格，墙壁上挂着诸如《卡门》、《浮士德》、《蝴蝶夫人》这些世界著名歌剧的油画，但很显然这些油画只是高仿作品，虽然只是高仿作品，但是存在也有着很长的时间，这些油画存在的时间甚至比罗昊自己的年纪还要大，如果放到市场上，也能卖出一个不错的价格，当然，前提是有人它们精心的维护，而不是像这样随意的暴露在空气中，导致画布上有部分色块发生开裂并且脱落。
<br/>　　按照所持卡片颜色的不同，来参加拍卖会的人员被划分在不同的区域。最前面，第一、第二排是给黑色卡片持有者的；中间第三到第十排是给紫色卡片持有者的，剩下的全是给红色卡片持有者的，没有固定座位，可以随意落座。
<br/>　　罗昊、子午等人选择了一个视野还算不错的位置坐下。按照子午的说法，这些手持黑色、紫色卡片的人并不需要提防，因为他们手里的卡片就说明了他们本身的身份，所以就算是他们有想法也不敢轻举妄动，因为很容易就追查到他们；但是持有红色卡片的人份就不同了，他们全都是没有背景的自由人，单独或者小团队活动，什么样的人都有可能隐藏在其中，手持红色卡片的“自由者”真的就是没有身份的自由者吗？未必！
<br/>　　甚至有些公司在派出代表以公司的名义拿着黑色卡片坐在拍卖会最前排的时候，还有真正的高层伪装成自由者，拿着红色卡片混迹在拍卖会后排座位中，但是并没有人知道他们的身份。
<br/>　　职业的敏感，让罗昊在坐下后就开始向四周打量着周围人；李然也同样是在做着同样的事情，擅长观察学的他在这方面要更加得心应手，那些从进入拍卖会现场以后，就拿着手机看个不停的，李然认为很有可能就是混迹在自由者区域中的某些大公司真正的高层，他们现在对着手机正在看的应该是那些坐在黑色卡片区内，通过手机拍下来的这次拍卖会将会出现的拍卖品的信息，然后通过手机相互之间进行通过，商量拍卖对策。几家公司联手，低价买入再高价卖出，争取最具价值拍卖品的事情，在源代码拍卖会上不是第一次出现，每年都会有这样的事情发生。
<br/>　　子午这次以“G先生”的身份，提供了一套他自己编写的源代码，主要功能是用来防御正在世界各地肆虐的勒索病毒，这种属于防毒性质的电脑源代码一般都是品级最低的拍卖品，并不能拍出很高的价格，但是因为子午这套防病毒源代码是针对眼下很火爆的勒索病毒，这也使得它的价格比起普通的防病毒源代码要高出不少，拍卖的顺序，也被排在第十位，一个不算靠后，也不算靠前的位置。
<br/>　　源代码拍卖会的拍品顺序也是根据拍品的价格进行排序的，越是靠前的拍品，价格越低，相对的，它们的价值也越低，一般有实力的人对于这些排在前面的拍品是没有什么兴趣的，第十五件拍品之后，才是他们所要争夺的目标。
