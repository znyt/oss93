　　翌日，冰缘大会，如期而至。
<br/>　　不过一月时间多些，诺大个不夜城中，早已挤满了各大势力的人。
<br/>　　包括极寒之地本身的数百势力，来自其它地方的数百势力。
<br/>　　加起来，近七八百个大势力。
<br/>　　极寒之地这边，每个势力，无论大小，都有参加冰缘大会的名额。
<br/>　　而极寒之地外，则并非如此。
<br/>　　一些上不了档次的势力，并没有名额。
<br/>　　而哪怕是大势力，若不够强，就算拥有名额，名额也不会多。
<br/>　　大多势力，只有一到两个名额。
<br/>　　较强的，有三个名额。
<br/>　　而整个炎武王国，也只有裂天剑宗，拥有5个名额。
<br/>　　这大大小小各势力的天骄，加上陪同而来的强者，还有来自各地的独行武者等等。
<br/>　　让得现今的不夜城，强者云集。
<br/>　　地元境、天元境，随处可见。
<br/>　　萧逸也认不全这些势力。
<br/>　　只是与白冰雪等另外四人，在不夜冰宫之内等候。
<br/>　　周遭各势力武者，同样等待着。
<br/>　　不夜冰宫的弟子、执事等等，则在给众人做着最后的登记、核实等等。
<br/>　　冰缘大会，事实上也是一次比赛。
<br/>　　参与者，同样有排名之分，奖励等等。
<br/>　　而冰缘大会如此盛事，参与势力如此之多，自然的，核实和登记相当繁杂。
<br/>　　萧逸闲着无事，饶有兴致地打量着不夜冰宫。
<br/>　　不夜冰宫，如其名，就是一个冰宫。
<br/>　　其内建筑，大多冰砌而成。
<br/>　　那一个个浑然天成的冰雕、冰面，相当华丽而绚丽。
<br/>　　同时，萧逸昨日才发现，天际上的那些极光，武者沐浴其中。
<br/>　　有着增加修炼速度、稳定心神、轻幅增加肉体强度的特殊功效。
<br/>　　而不夜冰宫的冰面建筑，便让得这些照耀下来的极光，能通过不断折射，充斥于整个冰宫。
<br/>　　让得整个不夜冰宫，成了整座不夜城内，最好的修炼之地。
<br/>　　当然，整个冰宫因此散发着氤氲光芒，显得相当绚烂夺目，让人惊叹。
<br/>　　大概一个时辰后。
<br/>　　最后的核实，已然完成。
<br/>　　九位老者，凭空出现于不夜冰宫大殿。
<br/>　　正是不夜冰宫九大长老，每一位，皆是极寒之地中声名显赫之辈。
<br/>　　“诸位。”九人对着各大势力武者拱了拱手。
<br/>　　“承蒙诸位看得起，这次冰缘大会，由我不夜冰宫主持。”
<br/>　　“若有懈怠，招待不周之处，还请海涵。”
<br/>　　“各位冰宫长老言重了。”各势力武者点点头，拱手说道。
<br/>　　九位冰宫长老沉声道，“此次冰缘大会，即将开始。”
<br/>　　“首先，由老夫宣布此次的规则…”
<br/>　　九位冰宫长老中，明显有一人为首，如果没猜错，应该便是冰宫大长老了。
<br/>　　冰宫大长老，一身修为之深厚，绝对踏入巅峰强者行列。
<br/>　　在深厚修为的增幅下，如洪钟般的声音，响彻全场。
<br/>　　一番详细的讲解后。
<br/>　　冰宫大长老高声道，“诸位，可还有疑惑？”
<br/>　　“若无，冰缘大会第一轮，冰海之缘，马上开始。”
<br/>　　一众武者，纷纷摇头，并没有人发出疑问。
<br/>　　九位冰宫长老见状，点了点头，随即转身带路。
<br/>　　各势力武者，跟随而上，往极寒冰海而去。
<br/>　　冰缘大会第一轮，举行地点并不在不夜城内，而是极寒冰海。
<br/>　　极寒冰海，辽阔无比，乃是极寒之地的一处险地。
<br/>　　就像没有人知道极寒之地有多大一样。
<br/>　　没有人知道极寒冰海有多辽阔。
<br/>　　只知，极寒之地数十大城中，有不少大城，都是临着它而建立。
<br/>　　而这些大城，最近最近的距离，都在万里以上。
<br/>　　有的，甚至隔着数万里之遥。
<br/>　　可想而知极寒冰海之辽阔无边。
<br/>　　众人出了不夜城，另一头便是极寒冰海了。
<br/>　　远远看去，那是一片看不到边、看不到头的大海。
<br/>　　其上，不时可见冰山漂浮。
<br/>　　或者密集的冰块。
<br/>　　海面之下，隐藏着未知的凶险。
<br/>　　极寒冰海，被誉为险地之一，绝非没有理由。
<br/>　　随着冰宫长老的宣布开始。
<br/>　　各方天骄，一众选手，纷纷飞跃而出。
<br/>　　最终在海面漂浮的冰块上站立。
<br/>　　且，彼此隔着极远的距离，不可互相干扰。
<br/>　　最近的，都在数百米以外。
<br/>　　同一时间，不夜冰宫负责担任裁判和记录的执事，纷纷御空而起。
<br/>　　数百执事，分布在高空各处。
<br/>　　以他们的修为，本是不可能御空飞行。
<br/>　　不过，他们身为冰宫弟子，本身有着出色的操控风雪手段。
<br/>　　加之身上有不夜冰宫宫主临时布下的禁制，可暂时在高空中飞行，并抵挡极寒罡风数日之久。
<br/>　　同一时间，九位冰宫长老中，有三人飞跃而出。
<br/>　　同样分布在高空各地，注视着各参赛天骄，以防意外。
<br/>　　而各大势力陪同而来的强者，家族长辈等等，则沿着极寒冰海的冰面边缘，四处分散。
<br/>　　他们作为观赛者，自然自主寻觅一个适合自己观赛的好地点。
<br/>　　这方圆数千里内，都是不夜冰宫的势力范围。
<br/>　　周遭妖兽，早已被清除一空。
<br/>　　故他们可以在周围随意寻觅自己喜欢的观赛地方。
<br/>　　言归正传。
<br/>　　此时，萧逸站于冰块之上。
<br/>　　冰块的面积，大概只有十米长宽。
<br/>　　冰块之外，便是冰水了。
<br/>　　而距离他最近的叶铭，所漂浮的冰块，都在五百米开外。
<br/>　　这第一轮，冰海之缘。
<br/>　　指的是，在这极寒冰海中，有着大量的吞灵冰鱼。
<br/>　　这种吞灵冰鱼，甚是奇特。
<br/>　　乃是冰海上空，比其它地方更为绚烂，更为夺目的极光之下，所应运而生的一种拥有吞灵之力的冰鱼。
<br/>　　在这极寒冰海中，又蕴含着大量极寒精华，大量精纯天地灵气。
<br/>　　这些吞灵冰鱼，日夜吸收，以之为食。
<br/>　　故，这些冰鱼，其本身，就是由无数精纯天地灵气而成。
<br/>　　同时，它们吸食的天地灵气越多，体内蕴含的天地灵气就会越精纯。
<br/>　　简而言之，这是一种对武者修为的大补之物。
<br/>　　不过，这种吞灵冰鱼，异常敏感，且生活在冰海深处。
<br/>　　武者只能通过释放自身元力或者真气，吸引它们上来，随后捕捉。
<br/>　　所谓冰缘大会，顾名思义，在这片极寒冰地中，获得机缘。
<br/>　　而能获得多少机缘，获得多少好处，全凭各方天骄自己的本事和运气。
<br/>　　嗖，忽然，数百之外，一武者大手一挥。
<br/>　　一条透明而纯净的吞灵冰鱼，被他凌空吸摄而起。
<br/>　　“吞灵冰鱼，我捉到了。”那武者，显然极其兴奋。
<br/>　　嗖…嗖…嗖…
<br/>　　陆陆续续开始有人捉到吞灵冰鱼。
<br/>　　萧逸这边，也轻松捉到了数条。
<br/>　　吞灵冰鱼，入手像极了一块冰坨，但又极其生猛。
<br/>　　“吸。”萧逸低喝一声。
<br/>　　体内武魂一动，瞬间吸收手中力量。
<br/>　　吞灵冰鱼，由纯净的天地灵气而成，武者可直接吸收。
<br/>　　不过数秒，手中那条吞灵冰鱼，已被吸收殆尽。
<br/>　　而萧逸体内，已多了一股精纯力量。
<br/>　　“好精纯的天地灵气。”萧逸脸上一喜。
