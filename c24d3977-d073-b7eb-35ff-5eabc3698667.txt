　　炼器师公会，四楼。
<br/>　　一间辽阔的炼器室中，苏莫长身而立，他的身旁立着一鼎硕大的炼器熔炉。
<br/>　　而在他的前方，此刻正端坐着五名炼器师公会的炼器师，柴管事亦在其中。
<br/>　　这五名炼器师，除了柴管事是尊级炼器师之外，还有两名老者同时是尊级炼器师。
<br/>　　至于最后两人，是为中年男子，观其胸口的标志，只是帝级炼器师。
<br/>　　“小友，今日考核希望你能通过！”
<br/>　　柴管事满脸笑意，仔细的打量着苏莫，由衷的说道：“如此年轻成为帝级炼器的人，北陵星还从未有过，整个古灵星河都是屈指可数，希望你能让我等大开眼界！”
<br/>　　柴管事是真心希望苏莫能通过考核，好让他们分会，也出一名绝世的炼器天才。
<br/>　　其他几位炼器师，均是面带好奇之色的打量着苏莫，他们三日前便听柴管事说了此事，知晓有一名不足三十岁的年轻人，要参加公会的炼器师考核。
<br/>　　“呵呵！不会让柴管事失望的！”苏莫笑了笑说道。
<br/>　　“那好，废话也不多了，你开始吧！”
<br/>　　柴管事点了点头，继续道：“你要求炼制的重渊斧的材料，都在你身边的炼器熔炉之中。”
<br/>　　“嗯！”苏莫点头，他瞥了眼身边巨大的炼器熔炉，随即略一沉吟，道：“柴管事，我能否用自己的炼器熔炉？”
<br/>　　苏莫身边的炼器熔炉，是炼器师公会专门为考核者准备的，品质极高，乃是尊级炼器熔炉。
<br/>　　按理来说，品质越好的炼器熔炉，炼器的成功率越高。
<br/>　　但是，对于苏莫来说，却不是如此。
<br/>　　他一直用帝级炼器熔炉炼制重渊斧，已经练出了自己的感觉，若是乍一更换炼器熔炉，不熟悉的情况下，很可能导致炼制失败。
<br/>　　所以，他要求使用自己的炼器熔炉。
<br/>　　“看来你有更好的炼器熔炉，这当然可以！”柴管事点了点头，他还以为苏莫嫌弃这炼器熔炉品质不够。
<br/>　　苏莫也没有多解释，立刻取出了炼制重渊斧的所有材料，而后取出自己的炼气熔炉，准备炼制。
<br/>　　柴管事和另外四名炼器师见此，顿时神情一怔，面现愕然之色。
<br/>　　因为他们一眼便能看出，苏莫的炼气熔炉只是帝级品质。
<br/>　　几人有些意外，明明有更好的炼气熔炉，苏莫居然要用差的！
<br/>　　难道此子如此自信？
<br/>　　柴管事几人暗道。
<br/>　　苏莫将所有材料分门别类的整理了一番，一切准备完毕，他便准备开始正式炼制。
<br/>　　咚！咚！咚！
<br/>　　就在此刻，炼器室门外响起了急促的敲门声。
<br/>　　柴管事顿时微微皱眉，炼器师的考核是严禁打扰的，居然有人这个时候前来！
<br/>　　“进来！”柴管事立刻轻喝了一声。
<br/>　　咯吱！
<br/>　　随即，炼器室的大门被打开了，几名炼器学徒走了进来。
<br/>　　这几名炼器学徒，正是平常在一楼售卖兵器的学徒，卢云亦在其中。
<br/>　　而在几名炼器学徒的身后，还跟着一名八字胡中年人。
<br/>　　苏莫见此，顿时心中一凛，立刻就想将巫娴放出来。
<br/>　　因为这名八字胡中年人，正是巫族尊者，巫梵。
<br/>　　巫梵走进炼器室，一眼便看到了苏莫，眸中杀机一闪而逝。
<br/>　　不过，他没有立刻动手，因为这里是炼器师公会，他的实力在此地，根本不算什么！
<br/>　　“柴管事，这名巫族的尊者求见！”卢云向柴管事抱拳说道，他的眼角余光瞥了一眼苏莫，眸中闪过莫名之色。
<br/>　　他已经知道了苏莫乃是巫族的仇人，难怪苏莫之前向他打听巫族的情况，原来是有大仇恨。
<br/>　　刚才巫梵来到了公会，给了他和其他几名炼器学徒一人一件宝物，所以他们毫不犹豫的带对方来到了这里。
<br/>　　对于卢云来说，苏莫和他没有任何关系，他能得到好处才是最重要的。
<br/>　　而且，他也不想苏莫成为公会的炼器师，因为他收过苏莫不少好处，若是苏莫成了公会的炼器师，他说不定都要将这些好处吐出来。
<br/>　　“苏兄，别怪我不仁义！”卢云心中暗自冷笑。
<br/>　　“巫族人？”
<br/>　　柴管事心中疑惑，打量了巫梵一番，问道：“阁下来此所谓何事？”
<br/>　　巫梵闻言，向柴管事一抱拳，道：“大师，在下来此，是为了一个人。”
<br/>　　巫梵面对柴管事非常的客气，这不仅是因为柴管事是尊级炼器师，还因为柴管事的修为比他高的多。
<br/>　　所以，他不仅不敢动手擒拿苏莫，表面上还要保持应有的客气。
<br/>　　“何人？”柴管事问道。
<br/>　　“他！”巫梵手指一指苏莫，眸中冷光爆射，道：“他是我巫族的仇人，杀我巫族近千名后辈天才！”
<br/>　　“什么？”巫梵此言一出，不论是柴管事等五名炼器师，还是卢云等几名炼器学徒，均是心中一震。
<br/>　　杀死了近千名巫族后辈天才？
<br/>　　这还得了？
<br/>　　卢云等炼器学徒，虽然已经知道了苏莫是巫族的仇人，但也没想到此仇居然如此之大！
<br/>　　苏莫脸色阴沉，心中急转，他的精神亦是高度紧绷了起来。
<br/>　　一旦情况有变，他会立刻逃走，或者使用巫娴作为挡箭牌。
<br/>　　现在，他还不能放巫娴出来，不然到时候他用一个女人做威胁，怕是炼器师公会都会敌视他。
<br/>　　“此事当真？”柴管事目光望向苏莫，面带惊疑的说道。
<br/>　　“巫族之人想要杀我，我自然不会坐以待毙！”苏莫沉声说道，他没有否认，因为否认也没有任何用处。
<br/>　　“大师，你也听到了，此子已经承认了！”
<br/>　　巫梵眸中杀机骤然凌厉了起来，沉声道：“所以，今日在下要擒拿此子，还请炼器师公会不要插手！”
<br/>　　其实，巫梵没有把握擒拿苏莫，毕竟苏莫手中有巫娴作为人质。
<br/>　　但今日，他就算无法擒拿苏莫，也要将苏莫困住，到时候再想办法。
<br/>　　他最担心的事，就是炼器师公会插手，到时候就麻烦了！
<br/>　　苏莫闻言，意念已经沟通了虚界神图，只要他意念一动，就能将巫娴放出来。
<br/>　　他没有立刻逃走，因为他知道自己不可能逃过巫梵的追击，他现在就在等柴管事的态度。
<br/>　　若是柴管事不管此事，他只能立刻以巫娴为挡箭牌，然后极力逃走了。
<br/>　　柴管事闻言眉头紧皱了起来，心中沉思，他不想得罪巫族，但是，他真的很看重苏莫。
<br/>　　不足三十岁的炼器师啊！
<br/>　　整个北陵星都没有出现过这等天赋的天才！
<br/>(本章完)
