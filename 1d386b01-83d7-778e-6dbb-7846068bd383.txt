　　“仲军，你去，将这小子从洞府里抓出来，废了他的修为，让他知道得罪了我们风云会的下场！”
<br/>　　方师兄，指着旁边的一尊猴脸男子说道。
<br/>　　这方师兄开口，便是听起来有着一种君临天下，高高在上的样子。
<br/>　　仿佛是楚风眠的一切，都要随意受到他的赏罚一样。
<br/>　　“是。”
<br/>　　仲军闻言，脸色更是狰狞许多，配上他的脸庞，更是显得阴毒无比，脚下一动，直接来到了楚风眠洞府之前。
<br/>　　“小子，滚出来！”
<br/>　　仲军冲着洞府里，大声喝道。
<br/>　　在洞府之中，楚风眠听到这仲军的声音，不由的眉头一皱，却是坐在原地，懒得动弹。
<br/>　　看起来这样子，却是完全将仲军无视了一样。
<br/>　　看着楚风眠完全的无视了他的话，仲军的脸上露出几分怒色道。
<br/>　　“可恶！一个新人，居然敢无视我的话！果然是狂傲到没边了！”
<br/>　　“仲军，跟着小子不用废话，直接将他从洞府里面抓出来！”
<br/>　　后方的方师兄，不由的冷哼了一声道。
<br/>　　“是！”
<br/>　　仲军狰狞一笑，闻言，猛然灵力便是在他的拳中凝聚。
<br/>　　突然之间，一拳便是向着楚风眠的洞府轰击过去。
<br/>　　直接便是就要将楚风眠洞府上的无数禁制，全部轰碎。
<br/>　　“轰！”
<br/>　　这一拳轰击过去，下一刻的结果，却是出乎了所有人预料。
<br/>　　“噗！”
<br/>　　那刚刚轰击楚风眠洞府的仲军，突然退后连连，一口鲜血直接喷了出来。
<br/>　　这仲军，轰击楚风眠的洞府，想要将楚风眠洞府上布置的无数禁制轰破。
<br/>　　但是他的力量，却是完全无法撼动楚风眠布置而出的无数禁制，被反噬的力量，直接给反震的吐血。
<br/>　　这突如其来的一幕，令其他几尊风云会的弟子，都是一愣。
<br/>　　“怎么回事？”
<br/>　　“这仲军，居然都无法打破这禁制？”
<br/>　　一些围观武者，看着这一幕，都是不由的议论纷纷起来。
<br/>　　仲军，也是北邙学院外门的老牌弟子了，境界达到神力境九重，实力更是达到了巅峰圣者的地步。
<br/>　　在外门，也算是一尊有名有姓的人物了，如今却居然是全力轰击，都无法打破楚风眠洞府上布置的禁制。
<br/>　　这让不少人都是笑出声来，风云会的名声可没那么好，看到风云会的人吃瘪，不少武者都是心中偷笑。
<br/>　　“这风云会行事，霸道无比，每年的新人，不少都被他们欺负，真以为仗着武风云他们就可以无法无天，武风云都还未入人榜前十。”
<br/>　　“这新人小子，果然有着几分门道，一个御风境小子，居然可以成为新人王，肯定是有着独特的手段，果然风云会的人，都吃瘪了。”
<br/>　　不止是如今楚风眠这一座山脉的弟子，周围几座山脉的弟子，都是纷纷过来，围观者。
<br/>　　风云会的动静这么大，他们怎么会不知道，一些曾经跟风云会有些恩怨的弟子，都是毫不客气的大笑起来。
<br/>　　这一声声笑容，简直都是嘲笑，令那方师兄的脸色变得阴冷许多。
<br/>　　“方师兄，这小子洞府中的禁制，我打不破。”
<br/>　　仲军回去，看着那方师兄，低声说道。
<br/>　　“废物！”
<br/>　　“我们风云会，白白浪费了那么多资源，培养你了！”
<br/>　　方师兄一个巴掌，直接打在了这仲军的脸上，将他直接打飞出去。
<br/>　　在这众目睽睽之下吃瘪，这让方师兄都是极为恼怒。
<br/>　　尤其是周围的笑声，更是刺耳无比。
<br/>　　方师兄看向洞府中的楚风眠，冷冷的开口道。
<br/>　　“小子，你现在自己乖乖从洞府里面出来，我们风云会只打断你的手脚，这是你最后的机会！”
<br/>　　“要是你在继续抗争，跟我们风云会为敌的话，你很有可能就会死无葬身之地！你可要清楚！”
<br/>　　这方师兄的话中，简直便是浓浓的威胁，不言而喻。
<br/>　　在北邙学院中，的确是在强大的弟子，也不敢冒着学院中的禁制出手杀人。
<br/>　　但是所有北邙学院的弟子，都不可能一直在学院之中的，肯定会冲去与执行任务的。
<br/>　　一旦离开学院，这规则也就不适用了，每年死在外面的弟子，不是少数。
<br/>　　只有一小部分，是因为被外敌斩杀，大部分的弟子，都是因为学院的恩怨，被学院中的其他弟子斩杀。
<br/>　　这件事，在北邙学院之中，一直都是一个潜规则一般的存在，众人虽然知晓，却是没有人会捅漏出来。
<br/>　　但是这一次，方师兄却是直接明晃晃的开口了，毫不掩饰其中的杀意。
<br/>　　“小子，你以一己之力抗衡我们风云会，简直是自己的愚蠢，你现在乖乖滚出来，跪下，被我们打断手脚，我们就可以给你一条生路如何？”
<br/>　　方师兄看着楚风眠，淡漠的说道，这语气中的意思，仿佛就是给了楚风眠什么好处，法外开恩了一样。
<br/>　　跪在地上，被人打断四肢。
<br/>　　对楚风眠这个境界的武者来说，被人打断四只，几日时间也就可以恢复，不会伤及到身体。
<br/>　　但是若是这样做了，所伤及的，可不是武者的身躯了，而是尊严。
<br/>　　一尊武者，若是没有尊严，没有一颗向武之心，是绝对无法成为强者了。
<br/>　　对天才来说，这更是极为重要。
<br/>　　方师兄的这番话，与其说是开恩，但是人人都能听出来其中的狠辣。
<br/>　　一旦是楚风眠这样做了，那楚风眠将来终生可能就无法突破了，跟一个废人也没有任何的区别了。
<br/>　　这道理，楚风眠自然是知道的一清二楚，更何况让他跪下的人，在这个世界，还没有谁有这个资格。
<br/>　　“好大的口气，也不怕闪了舌头。”
<br/>　　楚风眠的脸上，露出几分嘲弄之意道。
<br/>　　“风云会的人，都是白痴吗？不知道你们风云会的武风云，是不是也跟你们一样的白痴。”
<br/>　　“废物，你就连我的禁制都无法打破，还敢如此大的口气，你不是说他是一个废物吗？那你过来看看，你能不能打破？”
<br/>　　楚风眠的话，几乎都在指着方师兄的鼻子骂。
<br/>　　不止是骂他一人，更是骂了这个风云会。
