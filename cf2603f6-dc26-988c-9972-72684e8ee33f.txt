　　“居然这么多人！”
<br/>　　苏莫一眼望去，这辽阔的空地之上，聚集了不下二三十万人，比以往大大的增加了。
<br/>　　苏莫仔细观看了一番，这些人大多都是四品或者五品的修为，也有少量六、七品之人，囊括了各种各样的种族。
<br/>　　有苏莫较为熟悉的飞灵族、黑鳞族、巫族，还有霸天族、圣猿族等等等等。
<br/>　　“估计是因为千选战的原因吧！”苏莫心中暗道，很快，他便印证了这个猜测。
<br/>　　他在人群之中，等待了起来，因为时空之城，是九天开启一此，他只要在此等待便可。
<br/>　　随着等待，周围无数嘈杂的声音，不可避免的流进了他的耳中。
<br/>　　此地的人，不少人都是在外千选战做准备，有些修为不足五品之人，想要尽快突破五品修为，有些修为在五品之上的人，想要增强实力，从而在自己所在势力之中，获得参加千选战的机会，也有些修为极大强大的人，想利用最后五年的时间，尽力的提升势力。
<br/>　　苏莫并未修炼，而是眸光不断的查看四周，看有没有对他有威胁之人，毕竟，虽然他改变了容貌和气息，但也不是绝对的安全。
<br/>　　“公孙嘉！”
<br/>　　少倾，苏莫神情微皱，他看到了一个老朋友，在这片空地的中央位置，有一大圈人伫立在一起，其中一人正是公孙嘉。
<br/>　　公孙嘉目前的修为，已经达到了四品高级，比苏莫要高不少。
<br/>　　“厉害！”苏莫暗暗点头，当初在万道剑冢之时，对方的修为还不到四品，现在居然达到了四品高级，这个速度比他都还要快。
<br/>　　不过，转念一想，公孙嘉是永恒神宫宫主之孙，时空之城可谓是在家门口，估计是经常来这里。
<br/>　　看似只过去了几年，说不定人家已经修炼了几十年。
<br/>　　这群人足有二十多人，修为都相当高，公孙嘉的修为几乎是垫底的存在。
<br/>　　其中，修为最高的人，是一位模样奇特的青年，此人穿着一声大红色的长袍，面若红枣，丹凤眼，给人一种极为另类之感。
<br/>　　此人的修为，高达七品，气息无比强大。
<br/>　　除了此人之外，还有一人修为高达六品高级，此人身穿华服，身高八尺，相貌俊朗。
<br/>　　“贺兰公子、公孙公子！”苏莫双眸微挑，通过那群人的谈话，以及周围不时指指点点的人，他立刻知晓了两人的身份。
<br/>　　那位身穿大红长袍，丹凤眼的青年，居然是不死族的贺兰殇，人称贺兰公子。
<br/>　　而那位高大俊朗的华服青年，乃是永恒神宫的万古天骄，天池神朝唯一一位公子级人物，公孙长绝，人称公孙公子。
<br/>　　而和公孙长绝月贺兰殇在一起的二十余人，有永恒神宫的天才，也有不死族的天才，还有一些其它界域的天才，基本上都是万古天骄级别的人物。
<br/>　　一众二十余人，伫立在这片空地的中央地带，引的周围之人频频侧目，低声议论。
<br/>　　“居然都是要进入时门！”苏莫听到了这群人的交流，他们都是要进入时门之中修炼。
<br/>　　时门之中，三十三重时光神殿，可以改变时间的流速，能够给予他们大量的时间，参悟法则等等。
<br/>　　苏莫关注了片刻，便不再理会，这些人除了公孙嘉之外，都和他没有任何瓜葛。
<br/>　　而他，也不会去和公孙嘉叙旧，毕竟，他的身份不能泄露。
<br/>　　这一次，他来到时空之城，只为获得大量的时间，从而尽快达到五品的修为。
<br/>　　从周围之人的口中，苏莫得知时空之门开启，就在明日，他就安心的等待了起来。
<br/>　　时间匆匆，一日的时间眨眼而过，很快，便到了第二日的中午时分。
<br/>　　高空之上，两座巨大的门户，再次出现了，是时门和空门。
<br/>　　时门和空门开启，大量的人从其内飞出，如同长河一般，络绎不绝。
<br/>　　望着两扇门户，苏莫很想去空门之中一探，因为据说，空门之内，有玄妙无穷的空间法则。
<br/>　　不过，他目前去空门没有意义，只有时门对他的帮助最大。
<br/>　　很快，时门和空门之中，不再有人出来，下方无数的武者，纷纷冲天而起，迫不及待的冲进两座门户之中。
<br/>　　甚至，有些刚刚出来之人，又再次返回。
<br/>　　大量的人，如同两条浩浩荡荡的长龙，飞速的涌进时门和空门之中。
<br/>　　只是短短的几个呼吸之间，数十万人便消失一空，均是进入了时门和空门之中。
<br/>　　苏莫跟随在人群之中，在公孙嘉等人的后面，进入了时门之内。
<br/>　　公孙嘉等人，完全没有发现他的存在，没有神王级别的修为，除非是有特殊的能力，不然不可能认出他。
<br/>　　苏莫对于时门之内的三十三重时光神殿，也算是颇为熟悉了，毕竟这是第三次前来了。
<br/>　　第一重神殿！
<br/>　　第二重神殿！
<br/>　　第三重神殿！
<br/>　　苏莫快速的掌握一重重神殿的时光法则，不断的前进，这时光神殿的时间法则，每一次来到这里都要重新参悟。
<br/>　　并且，掌控时光神殿的时间法则，出去以后并不能使用，只有以时光神殿的时间法则为根本，参悟出自己的时间法则，才能算是真正的法则。
<br/>　　当初，苏莫耗费了近八天的时间，才来到第十重时光神殿，而这一次，他只用了六天的时间。
<br/>　　“第十重时光神殿了！”
<br/>　　苏莫深深的吸了口气，这里的时间流速，高达四千多倍，换言之，他在时光神殿中剩下的最后三天时间，以这里的时间流速，可以变成三十多年。
<br/>　　不过，这对于苏莫来说，还是有些不太够，毕竟，到了他这个地步，想要突破一个境界，动辄需要十几年。
<br/>　　而且，由于上次强行冲破了境界，下一次突破，将需要更多的时间。
<br/>　　所以，三十多年的时间，虽然有可能让他达到五品的修为，但是也许还不够。
<br/>　　好不容易来了一次时空之城，他要为自己争取更多的时间，时间越是充足，他反败为胜的机会才越大。
<br/>　　第十三重时光神殿，时间流速，按照之前的规则来看，起码有四万倍，哪怕到时候只剩下了一天的时间，也会变成百余年。
<br/>　　所以，苏莫决定，继续前进，前往第十三重时光神殿，他相信，以他的悟性，一定能在九日时间之内，到达第十三重时光神殿。
<br/>　　而且，就算时间不够了，他也可以提前的退回来，下次开启再进来。
