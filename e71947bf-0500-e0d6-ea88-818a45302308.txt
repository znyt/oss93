        宋云汐身心一怔，倏地瞪大了眼睛。
<br/>
<br/>        随即，她又恢复了平静之色，闭了闭眼睛，隐忍地转过身去。
<br/>
<br/>        “你起来吧！我对你的公司以及联姻这事，不感兴趣。”宋云汐淡淡地说道。
<br/>
<br/>        宋志远不依不饶道：“云汐，爸爸知道，你还爱着温韦。爸爸向你发誓，温韦和雨柔之间是清白的。他们俩，真的不是你想象中的那个样子。更何况……”
<br/>
<br/>        “更何况什么？郑温韦主动跟我提分手的第二天，他就跟他的朋友们宣布他跟宋雨柔在一起了。速度还真是快啊！你居然还跟我说，他俩是清白的？”宋云汐转过身来，看着跪在地上的宋志远，双拳紧握。
<br/>
<br/>        宋志远刚想解释：“云汐，爸爸……”
<br/>
<br/>        宋云汐直接打断了他的话：“你起来吧！你这么跪着，是想折我的寿吗？”
<br/>
<br/>        “云汐，爸爸真的很需要郑家的资助……”
<br/>
<br/>        “公司既然经营不下去了，那就把公司卖了吧！就算郑家资助了又能怎样？过不了一年，公司又得垮！说到底，你和你那个小三的儿子，压根就不会管理公司！”宋云汐一脸淡然。
<br/>
<br/>        “你怎么能对你父亲说这种话！”一听这话，宋志远便气呼呼地从地上站起来。
<br/>
<br/>        宋云汐不以为然地咧了咧嘴：“不爱听实话，那就请走吧！我是不会嫁给郑温韦，你就死了这条心吧！”
<br/>
<br/>        “你——”宋志远气得脸红脖子粗，怒气冲天地举起手来。
<br/>
<br/>        宋云汐毫不畏惧地看着宋志远。
<br/>
<br/>        宋志远的那一巴掌，也迟迟未落下来。
<br/>
<br/>        他有气没地方出，但是也不敢再打宋云汐。
<br/>
<br/>        宋志远放下手来，深沉地叹了口气，然后转身离去。
<br/>
<br/>        宋云汐的脸色慢慢地变得有些惨白，她的贝齿在不经意间地咬住了下唇，心里一阵又一阵地抽痛。
<br/>
<br/>        第二天，冷晏琛又给她来送排骨汤了。
<br/>
<br/>        同样，他放下保温饭盒便走。
<br/>
<br/>        第三天，亦是如此。
<br/>
<br/>        直到她出院这天，宋云汐才按捺不住地开口问道：“你是不是在追求我？”
<br/>
<br/>        冷晏琛身心一怔，坦然地笑了笑：“我只是受我祖爷爷所托，来给你送的排骨汤。”
<br/>
<br/>        “那就是你祖爷爷希望你来追我的意思喽？”宋云汐直截了当地问。
<br/>
<br/>        毕竟，他们俩不再是那种青涩的年龄段，都已经到了要相亲的地步了，两个成年人之前就无需再藏着掖着了。
<br/>
<br/>        冷晏琛点了点头。
<br/>
<br/>        宋云汐看着冷晏琛，踱步走到了他的跟前，接着问道：“那你呢？你是什么意思？”
<br/>
<br/>        “我暂时对宋小姐没有任何想法，只是，祖爷爷年纪大了，不想惹他老人家不高兴。所以，我就把这汤送来了。”冷晏琛温文儒雅地说道。
<br/>
<br/>        宋云汐看得出，这个男人骨子里透着一股子的傲气。
<br/>
<br/>        难道是因为上次相亲时，她拒绝了他，他现在才会如此抗拒她的吗？
<br/>
<br/>        “我今天出院，想去拜访一下你的祖爷爷，谢谢他这几天送来的排骨汤，不知道现在可否方便？”宋云汐微笑着转开了话题。
<br/>
<br/>        冷晏琛点了点头，待宋云汐收拾了自己的行李后，便带着她去了祖爷爷傅炎廷的病房。
<br/>
<br/>        当宋云汐看到傅炎廷住的是VIP特级病房时，心里倏然一怔，下意识地睐了冷晏琛一眼。
<br/>
<br/>        她有些怀疑，冷晏琛的家庭背景是不是官家子弟。
<br/>
<br/>        傅炎廷看到宋云汐时很是高兴地招呼她过来坐，还让冷晏琛主动给宋云汐削苹果吃。
<br/>
<br/>        “老先生，要不，我今天再陪你下一会儿象棋。”宋云汐微笑着问道。
<br/>
<br/>        傅炎廷点了点头，一旁的管家林贵立即把象棋盘摆了出来。
<br/>
<br/>        “这回你得认真点，不能再让着我了！”傅炎廷一脸严肃地说。
<br/>
<br/>        宋云汐忍俊不禁地笑道：“那老先生待会若是输了，可不许不开心噢！”
<br/>
<br/>        “我可不是那么小气的老头子！”傅炎廷呵呵地笑道。
<br/>
<br/>        冷晏琛坐在一旁，不得不给他俩削苹果。
<br/>
<br/>        “宋小姐觉得我这曾孙如何？”傅炎廷一边开始下象棋，一边开门见山地问。
<br/>
<br/>        宋云汐瞄了正在切苹果的冷晏琛一眼，莞尔一笑：“长得很帅。”
<br/>
<br/>        “我这曾孙为人也不错。”傅炎廷立即补充道。
<br/>
<br/>        宋云汐眼珠子一转，大致明白了傅炎廷这话的另外一层意思，只是赧然地应了声：“嗯，确实不错。”
<br/>
<br/>        “我这老头子吧！已经是一只脚踏进棺材的人了，如果宋小姐不嫌弃，可否跟我这曾孙试着交往交往看？”傅炎廷摸着胡须说道。
<br/>
<br/>        宋云汐顿时瞪大了眼睛，完全没料到傅炎廷会如此直截了当地谈这件事情。
