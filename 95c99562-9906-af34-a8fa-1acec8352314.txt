　　“嗯？是裂天剑宗那位天骄。”
<br/>　　“还有沐家那小子，以及冰炎教那个怪物。”
<br/>　　九位冰宫长老，眉头一皱，目光看向下方没有动弹的三人。
<br/>　　“你们三人愣着做什么？快逃。”
<br/>　　冰宫长老们呵斥一声。
<br/>　　三人，丝毫没有动作，反而不约而同地同时出手。
<br/>　　三道强悍无比的攻击，重重轰在吞灵冰鲸皇身上。
<br/>　　吞灵冰鲸皇的身躯，庞大无比，简直如同一座移动的岛屿。
<br/>　　三人的攻击，在它面前，似乎显得很是渺小。
<br/>　　果然，三道攻击，打在它身上，丝毫没有效果。
<br/>　　反而似乎是打得它有些许疼痛感，让它忽然抖动了一下身躯。
<br/>　　一抖之下，再次海面沸腾，四面八方皆激起千丈巨浪。
<br/>　　萧逸一惊，手中寒霜剑，猛地一挥。
<br/>　　“寒冰覆海斩。”
<br/>　　如潮剑音，猛地凝聚成一道巨大的寒冰屏障。
<br/>　　将激向岸边的浪涛悉数挡下。
<br/>　　毕竟，现今大量武者正往岸边逃离。
<br/>　　而其中，还包括即将到达岸边的叶铭等人。
<br/>　　嘭的一声巨响。
<br/>　　以萧逸如今实力，凝聚出来的寒冰屏障，瞬间被浪涛轰成碎片。
<br/>　　但浪涛，好歹是挡了下来。
<br/>　　而激向其它方向的浪涛，则几乎将沿途一切都拍成了齑粉。
<br/>　　包括数十座巍峨冰山。
<br/>　　三人同时皱起了眉头，也同时露出了凝重的表情。
<br/>　　“寒冰覆海斩。”萧逸低喝一声。
<br/>　　这一次，并非凝聚成寒冰屏障。
<br/>　　而是如潮剑音，悉数打在吞灵冰鲸皇身上。
<br/>　　“暗冰武破掌。”另外一人，大喝一声。
<br/>　　手中一掌轰出，一道充斥着暴虐气息，以及极冷冰寒气息的暗黑色巨大手掌，凭空而现。
<br/>　　“冰炎蚀心。”另外一人，也大喝一声。
<br/>　　一巨大的白色火焰，从天而降。
<br/>　　奇怪的是，火焰上，丝毫感觉不到半分温度和热量。
<br/>　　取而代之的，是一种仿佛渗入灵魂的冰冷之感。
<br/>　　三道强悍攻击，打在吞灵冰鲸皇身上，瞬间产生了效果。
<br/>　　白色火焰，将它的身躯，烧出了一个巨大‘伤口’。
<br/>　　那黑色手掌，将它的庞大身躯，轰出了一个结实的掌印。
<br/>　　寒冰覆海斩的如潮剑音，则将它的身体刮得遍体剑痕。
<br/>　　这种吞灵生物，无论是吞灵冰鱼、吞灵冰鲸、抑或吞灵冰鲸王。
<br/>　　被打死的话，会瞬间化作精纯灵气，顷刻消散。
<br/>　　而被打伤，则会体内‘精纯灵气’外泄，那等同于它们的鲜血。
<br/>　　吞灵冰鲸王，此刻便是被打伤了，只不过，三人的攻击，虽将它的身体打出了‘伤口’，却也产生了一股寒霜，将这些‘伤口’冰封住。
<br/>　　导致精纯灵气并未外泄。
<br/>　　倒是将它打得无比疼痛，让它发出了惊天怒吼。
<br/>　　下一秒，吞灵冰鲸皇，没有半分犹豫，巨大的身躯，立刻下潜。
<br/>　　打算逃回冰海深处。
<br/>　　它不是傻瓜。
<br/>　　普通没有战力的吞灵冰鱼，尚且有灵智，尚且会逃跑。
<br/>　　吞灵冰鲸皇，其灵智，比吞灵冰鱼高得多，不亚于正常的妖兽，自然也会逃跑。
<br/>　　“孽畜，休想逃。”三人冷喝一声，连忙追击。
<br/>　　吞灵冰鲸皇庞大身躯下，却拥有着过人的速度。
<br/>　　转瞬间已潜入海面。
<br/>　　嗖嗖嗖。
<br/>　　三道身影，连忙飞入海中，快速追击。
<br/>　　高空中，九位冰宫长老，脸色大变。
<br/>　　“笨蛋，被追。”
<br/>　　“吞灵冰鲸皇，论实力，足以媲美最强大的七级巅峰妖兽。”
<br/>　　“若在海底下，连极境强者都奈何不了它，甚至退避三舍…”
<br/>　　冰宫长老们的话，未能说完。
<br/>　　因为，萧逸等三人，已不见了踪影。
<br/>　　同时，吞灵冰鲸皇那巨大身躯下潜的瞬间，再次激起滔天巨浪。
<br/>　　将这九人狠狠拍飞。
<br/>　　……
<br/>　　冰海之下。
<br/>　　三道身影，破水而行。
<br/>　　“沐冰，你少给我碍事。”刚才操控白色火焰那年轻人冷喝一声。
<br/>　　“哼，韩苍，碍事的是你，吞灵冰鲸皇，归本公子了。”沐冰冷喝一声。
<br/>　　萧逸皱了皱眉头，他认得这两人。
<br/>　　之前，有本事捕获吞灵冰鲸王的另外两方势力，便是这两人。
<br/>　　萧逸本对他们知之不详。
<br/>　　但如今听名字，却大致知晓了他们的身份。
<br/>　　沐冰，便是刚才施展暗冰武破掌之人。
<br/>　　乃是冰武教第一天骄。
<br/>　　之前，冰武教弟子虽与风花月等人一道。
<br/>　　但沐冰，并不在他们之中，而是独自一人，在别的范围。
<br/>　　至于韩苍，乃是冰炎教圣子。
<br/>　　说到冰炎教，其并不在一宫一门二教之列。
<br/>　　但其教派实力，绝对不可小觑，在这极寒之地，也属于霸主级别的大势力。
<br/>　　言归正传。
<br/>　　两人，忽然将目光看向萧逸，冷声道，“你是何人？”
<br/>　　“裂天剑宗，萧逸。”萧逸淡淡地回答了一声。
<br/>　　“裂天剑宗？”两人明显地皱了一下眉头。
<br/>　　韩苍冷声道，“裂天剑宗虽厉害，但在极寒之地，还轮不到你们呈威风。”
<br/>　　“最好别给本公子碍事，否则，本公子让你走不出这极寒之地。”
<br/>　　说罢，韩苍加快了追击的速度，直朝吞灵冰鲸而去。
<br/>　　沐冰没有说话，身上一阵气势爆发，同样加快了追击速度。
<br/>　　萧逸脸色淡然，脚步玄妙一动，也加快了速度。
<br/>　　随着不断深入海底，水中阻力和压力，越发恐怖。
<br/>　　但，游云杀步，却能精妙地抵消掉这些阻力。
<br/>　　当然，本身吞灵冰鲸皇的速度也极快。
<br/>　　三人足足追了近七八千米之深，才算勉强追上。
<br/>　　“孽畜，给我留下来。”韩苍与沐冰同时大喝一声。
<br/>　　两人大手一挥。
<br/>　　一道白色火焰，一道黑色手掌。
<br/>　　两道巨大的攻击，同时澎湃而出，直直轰向吞灵冰鲸皇。
<br/>　　三人都很清楚，这极寒冰海，深不可测。
<br/>　　越是下潜，阻力和压力就越发恐怖，他们的速度会逐渐变慢。
<br/>　　若不及时拦下吞灵冰鲸皇，再继续下潜，它的速度，将彻底超过三人，逃得无影无踪。
<br/>　　轰…轰…两声巨响。
<br/>　　吞灵冰鲸皇，被拦了下来。
<br/>　　两人在水中划过一道流光，瞬间拦在了前方。
<br/>　　“吼。”吞灵冰鲸皇，怒吼一声。
<br/>　　在这冰海深处，可是它的地盘。
<br/>　　三道恐怖的水流，陡然打响萧逸三人。
<br/>　　韩苍与沐冰，冷笑一声，“孽畜，打算拼死一搏吗？”
<br/>　　萧逸不语，但眉头紧皱，这三道巨大的水流，显然不简单。
<br/>　　果然，当三人真正被这水流拍中时，才感受到其中的恐怖。
<br/>　　噗…噗…噗…
<br/>　　三人同时被轰飞，口吐鲜血。
<br/>　　“好强。”萧逸脸色一凛，刚才那道水流，绝对无限接近于极境强者的层次。
<br/>　　怕是段云那个层次的强者在此，在这冰海深处，对上这头吞灵冰鲸皇，也撑不了一时三刻。
