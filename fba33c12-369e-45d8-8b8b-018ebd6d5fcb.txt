　　　　涂家后院。
<br/>　　任务归来的涂小安对于山野集团发生的事情一无所知。
<br/>　　蛇胎女人这个任务让他一下子收获了十万的信仰之力，系统商城也直接的开启。
<br/>　　一条金银蛇的脑海中出现了一幅虚拟的屏幕，涂小安淡淡的话：“商城显化”
<br/>　　“叮...恭喜宿主成功进入系统商城，所得到的信仰之力充当点数货币购买里面的物品”
<br/>　　刷....
<br/>　　（1）功能物品界面....（2）生物血脉基因界面....（3）（功法属性技能界面...。
<br/>　　转眼，三条选项出现在了涂小安的面前，每一种代表了不同的物品领域，就好比你是去买家具，还是买服装。
<br/>　　不同的选项，不同的物品。
<br/>　　涂小安好奇的先进入了功能物品的界面，顿时间，各种图片物品陈列其中，有些图片是亮着的，但绝大部分图片都是灰蒙蒙一片，像是还没解锁。
<br/>　　“强化果，能显著的改变自身体质，直至身体达到常人巅峰，需要1000点数购买”
<br/>　　“觉醒果，当人的身体达到巅峰，此果能激发提升身体的潜能，成为觉醒者，需要10000点数购买”
<br/>　　“飞行术，能让任何的生物拥有飞行的能力，需要100000信仰点数购买”
<br/>　　涂小安略略的看了几眼，亮着的物品，大致明白，是他现在有能力购买的。
<br/>　　目前他的信仰点数为十五万左右，之前抽奖轮盘抽到的飞行术，原来可以直接购买，十万点数就能搞定。
<br/>　　其实还算便宜，并不算太贵。
<br/>　　另外这个强化果跟觉醒果到有意思了，普通人先吃强化果，让身体素质变强，达到一定的强度，再吃觉醒者，可以直接成为人类觉醒者。
<br/>　　无论是人，还是动物，有了这两颗果实，将鲤鱼跃龙门。
<br/>　　好东西，而且都不算太贵。
<br/>　　涂小安心想着，可以给自己的老妈老姐弄几颗，试试效果。
<br/>　　在商城中在，这些算低价的物品，至于绝大部分灰蒙蒙的物品没解锁，意思就是告诉涂小安，你买不起。
<br/>　　涂小安之所以想进商城，最大的一个目标，就是想在里面，找一找有没有一种能让自己化形的宝贝。
<br/>　　按照境界来划分，涂小安随着等级不断的提升，要是突破三阶，是有可能直接化形。
<br/>　　但他等不了，想走捷径。
<br/>　　找....
<br/>　　亮着的物品涂小安都看了一眼之后，直接就朝着灰蒙蒙的物品扫描，都是一些非常神奇的东西，现实世界根本不可能出现，但是价格也是让人乍舌，从十万，到三十万，五十万，越是后面的物品，价格越高。
<br/>　　突然，涂小安的蛇眸猛然的一亮，他想要的东西出现了。
<br/>　　“化形术，拥有化为人形的虚假形态，1000000信仰点数”
<br/>　　备注：虚假形态，外貌与人无异，但遇到强大的外力，就会立刻溃散，变回原型。
<br/>　　涂小安看完介绍之后，心微微的一凸，虚假形态，就是给了你一张皮，虚有其表而已。
<br/>　　但就算如此，也不错了，涂小安以后终归要境界上的真正化形。
<br/>　　只是这个价格让涂小安有点捉禁见肘，既然足足要一百万的信仰点数，他拥有的飞行术，才不过十万点而已。
<br/>　　看来，要想化形，终归是逆天而行。
<br/>　　就好比境界上的化形，就要遭受天雷天火浩劫的洗礼，失败了就是死。
<br/>　　这种伪化形，虽然简单，但是也要让你狠狠的出血。
<br/>　　涂小安现在全部的信仰点数也就十五万，差太多了，看来他想伪化形，都不是一件简单的事情。
<br/>　　必须要赚取足够的信仰之力才行。
<br/>　　那就在等些天吧，涂小安确定了商城有这种东西，这种价格，心里也有了底气。
<br/>　　到时候将白镇蛇王庙的雕像换成自己，他的信仰之力将会源源不断。
<br/>　　累积个一百万并不算难。
<br/>　　接下来，涂小安也没兴趣看后面的那么灰蒙蒙的物品，看了也白看，因为他完全买不起。
<br/>　　退出功能物品界面，涂小安将目光投放到血脉基因界面上去。
<br/>　　这有点意思。
<br/>　　意念一动，点了进去。
<br/>　　“普通血脉，上古血脉，神兽血脉”
<br/>　　任何什么生物，分三种血脉等级，涂小安饶有兴致的看着，这是能将动物融合里面神奇的血脉进化，取而代之。
<br/>　　由于涂小安是蛇类，优先出现了很多毒蛇的血脉属性，往普通血脉那一行一看，涂小安既然看到了自己熟悉的蛇种。
<br/>　　太玄蛇，10000信仰点数。
<br/>　　变异吸血蛇，50000信仰点数。
<br/>　　变异鸡冠蛇，50000信仰点数。
<br/>　　变异魔鬼蛇，70000信仰点数。
<br/>　　商城中的普通血脉，就是觉醒血脉了，能将一条普通的蛇，融合血脉，进化出来。
<br/>　　再看上古蛇种血脉看去...
<br/>　　上古巴蛇，100000信仰点数。
<br/>　　上古海德拉，3000000信仰点数。
<br/>　　异兽变异九头蛇，5000000信仰点数。
<br/>　　各种各样的传说蛇种都一览无余，涂小安看的目瞪口呆。
<br/>　　不过上古血脉要融合，就必须是觉醒血脉的前提下，你找一条一般的毒蛇融合，那么毒蛇本身根本承受不住那么强大的血脉基因。
<br/>　　这就好比蛇博士做的实验，想要承受巨蟒基因，本身的蛇种都要非常的强大，否则将会承受不住的自爆。
<br/>　　这个商城有点变态，血脉基因更是神乎其神，只要涂小安有足够的信仰点数，那么他连神兽都能进化出来。
<br/>　　只有你想不到，没有做不到。
<br/>　　看的兴奋的涂小安往神兽那一列描上一眼。
<br/>　　霍...
<br/>　　神兽腾蛇，10000000信仰点数。
<br/>　　整整一千万的信仰点数，而且还要在拥有上古血脉的基础上融合，才能进化的。
<br/>　　在往别的神兽血脉看去。
<br/>　　神兽凤凰，30000000信仰点数。
<br/>　　神兽白虎，3000000信仰点数。
<br/>　　神兽玄武，30000000信仰点数。
<br/>　　神兽青龙。40000000信仰点数。
<br/>　　涂小安没有继续看下去了，都是天价，望尘莫及。
<br/>　　他这十五万，只能买神兽的一根毛而已。
<br/>　　目前的阶段，涂小安也没别的想法，就是累计信仰点数，先购买化形术在说，其它的都是浮云，路走一步一步走，饭要一口一口吃。
<br/>　　这些基因血脉，看着让人振奋，其实并没有那么简单，就算你买的起，你也要找到一头合适的强大生物融合。
<br/>　　否则一旦失败，就会前功尽弃。
