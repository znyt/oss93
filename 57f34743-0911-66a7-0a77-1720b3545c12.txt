        翌日。
<br/>
<br/>        滨江城的南郊是出了名的只有非富即贵的人才能买得起的别墅区，一条笔直的泊油路两边，是错落有致的别墅群。
<br/>
<br/>        晨阳冉冉地升起的时候，如碎金般闪目的霞光向四周照射，将那些犹如白鹤群立山丘的别墅染上了一抹似红似金的微光。
<br/>
<br/>        苏漫雪曾在这的花园别墅区里买了一套别墅，现在已经装修完毕了，但她的人却远在韩国还未回来。
<br/>
<br/>        这柏油路尽头，是一处私人禁地。
<br/>
<br/>        外边的入口是一扇黑色的大铁门，门柱两边有高高的长满带刺的蔷薇爬藤的铁栅栏，这里看似美观，实则暗藏在蔷薇花枝叶下的防盗铁丝非常危险。
<br/>
<br/>        此时，一个女孩的到来，打破了这里的静谧。
<br/>
<br/>        “滋滋滋——”
<br/>
<br/>        女孩按响了门柱上的可视对讲电话按钮。
<br/>
<br/>        而接听她来访电话的人，却是一个中年妇女。
<br/>
<br/>        傅夜沉从院子里晨跑回来刚冲了个澡，外边的女管家刘婶便叩门，说道：“大少爷，外边有个名叫季雅姗的女孩，自称是被您救过，想亲自向您当面道谢。”
<br/>
<br/>        季雅姗？！
<br/>
<br/>        傅夜沉剑眉微蹙。
<br/>
<br/>        他救过的人无数，他哪记得这个名叫什么“季雅姗”的女孩又是谁。
<br/>
<br/>        “不认识，打发她走。”傅夜沉拧上花洒，拿了条浴巾围在了腰上。
<br/>
<br/>        他打开了浴室门，从浴室里走出来后。
<br/>
<br/>        刘婶见他光着上身，连忙拿来一件干净的浴袍，给他递了过去。
<br/>
<br/>        傅夜沉的身姿修长挺拔，小麦色的肌肤上伤痕累累，看着很是让人心疼。
<br/>
<br/>        明明是含着金汤匙出生的贵少爷，却用自己血肉去保护着别人的安危。
<br/>
<br/>        刘婶感慨又感动着。
<br/>
<br/>        “大少爷，您如今已经退役了，有空的话，就常回临海城去陪陪老爷子。”刘婶像个慈母般絮絮叨叨地念着，“还有太太，她日盼夜盼，就盼着您这当儿子的能平安归来。”
<br/>
<br/>        “我知道的，刘婶。”
<br/>
<br/>        他那腹部的肌肉边上是性感的人鱼线，浴巾松松垮垮地系在腰间，隐隐约约的线条，差点就让刘婶这已经是当老妈子的年龄的人还能产生一种脸红心跳的感觉。
<br/>
<br/>        刘婶看得有些不太好意思了，连忙转身离开了傅夜沉的房间。
<br/>
<br/>        傅夜沉吃过早餐后，换上了西装革履，出了门，就在他上车前，是刘婶追出来把公文包递给他的。
<br/>
<br/>        他这种常年摸枪摸惯了的军人，一时间还真不适应这每天手提公文包按时去公司“报道”的日常。
<br/>
<br/>        傅夜沉上了自己的私家豪车，坐在了司机身后的座位上，他刚想闭目养神，岂料司机在这个点踩了个急刹车，一阵惯性导致他整个身体前倾，差点儿撞到前方椅背上。
<br/>
<br/>        “发生什么事了？”傅夜沉剑眉微微蹙动，一脸严肃地问。
<br/>
<br/>        这才刚出大门口，就遇上个不好的开始。
<br/>
<br/>        司机小陈一脸抱歉地回答道：“对不起，大少爷。我们的车，被一个女生给拦住了！”
<br/>
<br/>        就在司机小陈说这话的时候，傅夜沉身旁的车窗被人从外扣响了。
<br/>
<br/>        傅夜沉微微偏过头去。
<br/>
<br/>        只见车窗外站着一个高挑却很纤瘦的女生，她穿着白色衬衫，深蓝色格子裙，后背上还背着一个对她而言似乎有点儿大了一些的书包。
<br/>
<br/>        女生的头发束成一个干净利落的马尾辫，柳眉秀丽，眼睛水灵如清泉，鼻梁小巧，又是樱桃嘴，使她那张小小的脸庞添上几分朝气蓬勃。
<br/>
<br/>        傅夜沉本不想理会她，但是，当她抬起手来，指间捏着一条黑色的手链在他眼前晃动时，让他瞬间身心一怔，连忙按下了车门上的按钮，将车窗落了下来。
<br/>
<br/>        “救命恩人，我知道你不想见我。但是，我还是想把这个你当时为了救我而落下的重要的东西交给你。我特意找了一个技术极好的手工师傅，让她帮我编的手链。而且，里面我让师傅给我加了金线，很坚固的！所以，我祝你和你的爱人，幸福美满，长命百岁！”季雅姗微微躬着身子，将手伸进了车内。
<br/>
<br/>        傅夜沉看着女生递过来的青丝手链，既欣喜又欣慰。
<br/>
<br/>        这是那日在那幢烂尾楼里，被他编成手链，却在紧要关头、生死一线间救了他和那女生而被挂断了的“漫雪”的头发。
<br/>
<br/>        所以说……
<br/>
<br/>        面前这个女孩，就是那日，他救了的那个女孩？！
<br/>
<br/>        傅夜沉接过青丝手链后，并打开了车门，挪出自己的座位，语气依旧冰冷地说道：“进来吧！小陈，先送这孩子去滨江一中。”
<br/>
<br/>        “是！”司机小陈毕恭毕敬地应了声。
<br/>
<br/>        “那我就不客气啦！”季雅姗笑颜如花，立马钻进了车内，侧身看着傅夜沉，好奇地问道，“你怎么知道我在滨江一中念书啊？我又没佩戴学生证。”
<br/>
<br/>        “我曾在滨江一中念过高中。”傅夜沉坐在了副驾驶座后，淡淡地回答道。
<br/>
<br/>        他把驾驶座后那最安全的位置让给了女孩。
<br/>
<br/>        季雅姗非常知足又很有礼貌地道了声：“谢谢。”
<br/>
<br/>        傅夜沉并未再看她，而是把注意力集中在了手里的这条青丝手链上，有点儿心不在焉地问：“你是怎么知道的？”
<br/>
<br/>        他营救人质的那天，脸上是涂抹了油彩的。
<br/>
<br/>        哪怕他受了重伤，人质的家属想要感激，也只能去他上级领导那儿送锦旗，不能去医院探望他，以免泄露他的真实容颜。
<br/>
<br/>        季雅姗咧嘴一笑：“我姑姑是你的主治医生，我缠着我姑姑，让她告诉我的。至于你的住址嘛！嘻嘻——我花钱雇人查的！”
<br/>
<br/>        “总而言之，谢谢你帮我拿回并重新编织了这条青丝手链。以后，关于我的身份，希望你对外保密。”傅夜沉剑眉微蹙，并将这条青丝手链，放在了西装胸口处的内兜里。
<br/>
<br/>        季雅姗非常乖巧地就像小鸡啄米般猛点头：“救命恩人，你放心好了！我是不会出卖你的！如果不是你的话，我恐怕连今年的高考都没法参加了！”
<br/>
<br/>        “……”傅夜沉没再跟她搭话。
<br/>
<br/>        而这个女孩子简直就是个小话痨。
<br/>
<br/>        她聊天的话题，都可以问候他祖宗十八代了！
