　　“少主，没想到您的实力已经是达到了这一步，未来重振云羽天宫，指日可待！”
<br/>　　云老看着楚风眠惊叹道。
<br/>　　他见到楚风眠的时候，楚风眠才只是一名神力境的圣者，纵然是在九域之中，都不算是什么强者，只能算的上是一个天才。
<br/>　　这么短的时间过去，楚风眠从域外归来，如今实力已经是发生了翻天覆地一般的变化。
<br/>　　八劫古帝，楚风眠说杀就杀，这种实力已经是接近到了九劫古帝了。
<br/>　　云老选择将云羽天宫认主楚风眠，也是想要指望楚风眠可以重振云羽天宫，现在看到楚风眠的实力。
<br/>　　云老已经是感觉，这件事已经是没有那么遥远了。
<br/>　　“我这次回来回到九域，也有这个打算。”
<br/>　　楚风眠听到云老的话，目光闪烁开口道。
<br/>　　“不过这次，我要建立的一方势力，是剑道门，我是出身自剑道门的，不过在剑道门之中，我可以开辟一个堂口，去传承云羽天宫的功法。”
<br/>　　剑道门，就是曾经剑道之主建立的宗门。
<br/>　　在万年之前，剑道门弟子虽然稀少无比，但是每一位出去，都是足以独当一面的人物。
<br/>　　之前在九域天才战上的时候雨，楚风眠是剑道门弟子的消息，已经是暴露了出去。
<br/>　　如今楚风眠也有着足够的实力，不需要在隐藏什么了。
<br/>　　“剑道门，万年之前的剑道门？”
<br/>　　洛汐跟寒月漓听到楚风眠的话，都有些吃惊。
<br/>　　就连她们都不了解，楚风眠还有这样一段事。
<br/>　　楚风眠出身自剑道门的事，可以暴露出去，但是他二世为人的秘密，无论如何楚风眠也不敢说出去，只能够说他是了剑道门的传承。
<br/>　　“成为一方堂口，倒是无妨，只要是云羽天宫的功法可以传承下去，就足有了。”
<br/>　　云老也没有反对。
<br/>　　云羽天宫，只是一个名字，他要传承下去的，是曾经云羽天宫的功法，只要是云羽天宫的功法武技没有失传，宗门叫什么，也无所谓。
<br/>　　“到时候寒雪阁，也可以并入到剑道门之中。”
<br/>　　楚风眠看着寒月漓开口道。
<br/>　　他要建立剑道门，也打算庇佑寒雪阁，寒雪阁加入到剑道门之中，也会受到楚风眠的庇佑。
<br/>　　“可以。”
<br/>　　寒月漓点了点头。
<br/>　　“寒雪阁也愿意加入到剑道门之中，成为剑道门的一个堂口。”
<br/>　　如今的寒雪阁，也算是由楚风眠一手建立起的，如果没有楚风眠，寒雪阁都不知道覆灭过多少次了。
<br/>　　现在要让寒雪阁加入到楚风眠建立的剑道门之中，她也没有任何的意见。
<br/>　　“那剑道门，什么时候建立？开宗立派，这可是流传万古的事！”
<br/>　　洛汐也一旁也有些兴奋开口询问道。
<br/>　　“不急，建立剑道门前，我还要去北邙学院一趟。”
<br/>　　楚风眠沉声开口道。
<br/>　　“北邙学院，其实也是剑道门的弟子，我的大师兄，剑凌所创立的，我要去北邙学院，将北邙学院夺回！”
<br/>　　他这次回到九域，也打算去北邙学院，真正的将北邙学院掌控，毕竟这是楚风眠大师兄剑凌所开创的一方势力，决不能落入到外人手中。
<br/>　　“北邙学院？是你师兄所创？”
<br/>　　寒月漓看了楚风眠一眼，有些惊讶，突然间眼神闪过几分清醒道。
<br/>　　“怪不得，你可以得到北邙圣君的传承，原来是因为这样！”
<br/>　　寒月漓突然间的想到了当初楚风眠在北邙学院众圣堂之中，得到了北邙学院的创始人，北邙圣君的传承。
<br/>　　楚风眠对北邙学院来说，只是一个外人，就连北邙学院的许多古老大人物的后代，都没有得到北邙圣君的传承。
<br/>　　楚风眠居然是得到了，当时寒月漓就有些奇怪，但是也只能觉得是楚风眠的机缘。
<br/>　　原来在那个时候，楚风眠就发现了他跟北邙圣君的关系。
<br/>　　“可现在北邙学院，可是被阎罗星域的人所占据了。”
<br/>　　洛汐眉头一皱的说道。
<br/>　　“阎罗星域的人，可不好对付，阎罗星域中的强者，不在大罗天星域之下。”
<br/>　　“无妨，不过是一个陈家罢了，也不是整个阎罗星域！更何况以我现在的实力，纵然是面对九劫古帝，也足以全身而退！”
<br/>　　楚风眠冷声说道。
<br/>　　“更何况我绝不会让，北邙学院落到阎罗星域的人手中。”
<br/>　　楚风眠直接开口道。
<br/>　　现在九域的局面，还没有那么混乱，因为还有许多的强者没有来到九域。
<br/>　　等到达九域的强者越来越多，九域也就会越发的混乱了，到时候楚风眠在动手也会变得更加的麻烦。
<br/>　　现在出手，是最好的选择。
<br/>　　“我们走，这就去北邙学院！”
<br/>　　楚风眠纵身一跃，运转起灵力，化为了一道遁光向着寒雪阁外飞遁而去。
<br/>　　北邙学院跟寒雪阁，本来还有着很远的距离，至少要几个月的时间，才可以飞到。
<br/>　　但是现在楚风眠等人的实力，足以是在域外星空之中，有肉身横渡一座座大陆。
<br/>　　更何况是在这九域之中，他的遁光速度几块，几个呼吸之间，武胜国度就被他摔在了身后。
<br/>　　接下来，穿越一个个的国度。
<br/>　　现在整个北域，几乎都落入到了北邙学院的手中，本来北邙学院还做不到这一点。
<br/>　　可如今随着陈家入住北邙学院，北邙学院的实力不知道是强了多少倍，第一件事便是征服了整个北域。
<br/>　　只是寒雪阁附近的几座国度，一直都在反抗，不愿意并入到北邙学院之中。
<br/>　　所以北邙学院，才没少来人找麻烦。
<br/>　　一个个国度之上，都建立起了庞大无比的大阵，这些大阵，看来都是强者布置的。
<br/>　　一旦是全部开启，就足以化为一道道杀阵，靠近的武者妖神，都会受到雷霆一击。
<br/>　　看来这北邙学院的人，也清楚了九域即将到来的危机，布置好了这一切。
<br/>　　“九域大变样了。”
<br/>　　寒月漓在一旁轻声说道。
<br/>　　以前的九域，只是一个小地方，不能算安静祥和，却也算是悠闲。
<br/>　　可现在整个九域，就像是疯了一样，一片繁忙，处处都是因为争夺宝物的战斗。
<br/>　　不知道多少九域的势力，在这一次动乱之中被毁灭。
<br/>　　寒雪阁还算是有着寒月漓等人庇佑，才可以免受波及。
