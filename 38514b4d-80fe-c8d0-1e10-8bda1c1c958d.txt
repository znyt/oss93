　　龙祖现世，向天宣誓，威震洪荒。
<br/>　　没过多久之后，洪荒另外一个方向，一股红光冲天，那南方不死山之处，梧桐之上，有凤凰高飞，真身显化，强悍的气势席卷整个洪荒天地！
<br/>　　他们双翅展开，对日而鸣，身放五色光彩，冲天的气势，让苍穹悸动，八方风云倒卷，更有祥云朵朵，无数飞鸟环绕拜服。
<br/>　　“吾为风祖（吾为凰祖），天道在上，从今往后，飞禽一类以凤凰为首。”
<br/>　　凤凰声音刚落，便有麒麟出世，那真身巨大，却是龙头、鹿角、狮眼、虎背、熊腰、蛇鳞、马蹄，高三十万丈，足踏大地而吼：
<br/>　　“天道在上，此后洪荒，走兽类以麒麟为王，吾为麒麟王！”
<br/>　　他们的修为也与龙祖一样，各有突破，更进一步。
<br/>　　一时间，洪荒大地，山川河流，齐齐震动，如被此三族的威势所震慑一般。而在这个时候，洪荒之中，一些强横的大能，亦纷纷苏醒。
<br/>　　阴阳老祖洞府之处，阴阳老祖与乾坤老祖对视一眼，一时之间，默默无语。
<br/>　　……
<br/>　　西昆仑玉京山上，鸿钧老祖睁开双眼，头顶造化玉牒，造化之光缓缓垂下，道韵流转，大道弥漫。
<br/>　　“大劫开始了吗？贫道已斩出两尸，只差这最后一尸就能圆满了。天道难测啊！也罢，既然大劫已至，那便争取在此次大劫之中，踏出这最后一步，证道成圣！”
<br/>　　说完，便不再多言，再次闭上双眼，继续参悟造化玉牒上记载的大道，道光灿烂，玄之又玄。
<br/>　　……
<br/>　　洪荒西方灵山之上，煞气笼罩，黑雾翻滚，魔气滔天。突然，其内传出一道嚣张至极的声音：
<br/>　　“哈哈哈！大劫已经来了，打吧！打吧！尽情的打吧，你们打得越惨，本座的修为突破的就越快。哈哈哈！我要不要再加把火呢？！哈哈哈！”
<br/>　　“鸿钧，你等着，我很快就能突破了，本座要毁了这天地，毁灭这苍生，魔道屠天，证道成圣！哈哈哈！”
<br/>　　……
<br/>　　一时间，洪荒天地，风起云涌！
<br/>　　洪荒东部，一处隐蔽的洞府之中，蓝色宝光四散，宫殿华丽高贵，上书“玉虚宫”三字。其内，一中年道人刚刚炼化宫殿禁制，刻上自己的元神烙印，使之完全变为自己的宫殿，从此之后，这宫殿便完全属于他了，而不再是别人。
<br/>　　这道人便是元始，元始刚炼化宫殿，清点完宫殿内宝物之后，便听到龙、风、麒麟三族向天宣誓，他看着远方，面色不变，不由冷笑道：
<br/>　　“万兽朝苍，白鸟朝凤，麒麟盖世，还真是威风八面，威震洪荒啊！只是，你们也不看看自己是什么实力，哼！大罗金仙圆满，也敢出来称霸洪荒。你算什么东西！
<br/>　　就算在天道的帮助之下，突破现有境界，战力达到准圣初期。但，那又如何？还是太弱了，洪荒的水可是深着呢？比你们强大的多的是，就算本座自己，虽然只是大罗金仙圆满境界，但，虐你们，还是够了。
<br/>　　不过，既然大劫已至，那么，我的《玄黄不灭》也该突破下一重了。毕竟，实力太弱，玩不起来啊。”
<br/>　　不再多言，元始收起宫殿中的宝物，开始闭关突破。其实，在昆吾老祖宫殿之中，也并没有什么好东西，元始在其中只找到了三件先天灵宝，以及一些先天灵根，一个悟道蒲团，除此之外，便再无他物。
<br/>　　不知道是昆吾老祖真的只有这些灵宝，还是其余的放在他身上，自己亲身带着。
<br/>　　三件先天灵宝，分别是一件下品防御用的八卦紫寿仙衣，两件攻击灵宝中品玄阳链与下品火箭枪。其中，八卦紫寿仙衣威力一般，后世谁都会穿上一件，玄阳链，阳属性攻击灵宝，可困可守可攻，威力不错，至于火箭枪，鼎鼎有名，为后世哪吒三太子的法宝。
<br/>　　三件灵宝对于元始来说，威力太弱，他都懒得炼化，自己随便一印，都有着堪比灵宝的威力，这三件灵宝对自己可有可无。但，他还是收了起来，自己不用，以后赏给别人还是可以的。天予弗取，反受其咎，他还没有浪费到遇见宝物而不取的地步。
<br/>　　昆吾宫，不，现在是玉虚宫了，玉虚宫之中，共分五大宫殿，一间是大堂，原名为“独尊殿”，元始不喜，改为“天尊殿”，既符合他元始天尊的名号，又能彰显出他元始天尊的气势。此殿位于最中间，最为宽广宏大，也是最为华丽的一殿。
<br/>　　第二间是“悟道台”，元始没有改名，此殿最为清净，适合悟道修炼。第三间为“灵宝阁”，收藏各种先天后天灵宝。
<br/>　　第四间为“灵草堂”，栽种各种灵草、仙草、灵果、灵根。最后一间为演道场，作为道友之间相互印证道法的一处场所，大殿之外，各种阵法、禁制密布，防护其中，不让战斗之时破坏大殿。
<br/>　　元始来到悟道台，那是一个高高在上的崖台，崖台之下，是一片湖泊，湖水清澈见底，灵气逼人，白色莲花丛生，鱼虾嬉戏。崖台之上，仙音袅袅，仙鹤飞舞，道音袅绕，让人一眼看见，便能静下心来，忘却心中杂念。
<br/>　　此中也有许多阵法禁制，能够承受住准圣初期一击，强度已经足够了。毕竟这里只是悟道用的，又不是打架用的。
<br/>　　元始盘坐于悟道崖台之上的悟道蒲团上，取出四团四象神兽精血，精血一出，顿时空间威压显现，一团化为身绕赤金色神火的小朱雀，“啾！”小朱雀一出现，便发出一声鸣叫，在四周欢快的飞舞。
<br/>　　一团化为身绕金芒，全身雪白的小白虎，“嗷！”小白虎一出现，便仰天长啸，虎啸生风，也在四周奔跑。
<br/>　　一团化为身绕黑色水气，全身黑色的小玄武，“昂！”玄武一出世，就仰天大吼，发泄心中不满，它并没有四处乱跑，只是静静站在那里，仰天大吼。
<br/>　　一团化为身缠青色木气，全身青色的小青龙，“昂！”小青龙一出现，就欢快高吟一声，之后腾飞于空，腾云驾雾。
<br/>　　四团神兽精血一出现，就分别化为各自本体形状，用自己的行动来表示自己的欢快心情。
<br/>　　顿时，元始四周，四大神兽飞舞，奔啼鸣叫，威压四周天地。
<br/>　　神威如狱，神恩如海！
