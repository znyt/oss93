　　呼啦啦！
<br/>　　直升机的螺旋桨声音在空中彻响，洛冥和段江伟挤在一架直升机内，两人都没有说话，而是不断望着下发越来越近的南署基地，眼中更是流露出急切。
<br/>　　当越靠越近，两人都忍不住皱眉，腐臭的味道哪怕在高空，哪怕在机舱之内，竟然都能闻得到！
<br/>　　这是多少丧尸形成的腐血澡泽？
<br/>　　望着下方那黑漆漆一大·片的地方，两人心中震撼更是无以复加。
<br/>　　他们是来询问有关袁熙叶，有关觉醒剂，有关循环论的事，狼牙研究部的一切研究想要进行，都必须得到楚涵的允许才行，觉醒剂更应如此，所有研究所需的材料都不是凭空出现，基地管理者对这一项异常清楚，研究院是每个基地最重要的地方，研究材料往往每一份都记录详细，更别说特殊晶体这种稀有材料。
<br/>　　觉醒剂的研究成功，前期一定耗费了大量特殊晶体，这些材料的损耗不可能瞒得过基地首领。
<br/>　　所以只需要找楚涵一问，所有事情的真实与否都可以知晓！
<br/>　　但当两人急切的乘坐直升机来到南署基地上空，心中对以上事件的急迫感却被冲淡，取而代之的是发生在南署基地的这场大战。
<br/>　　听闻战报的一次次进度，没有亲眼所见的感观震撼，越近越是如此！
<br/>　　“太颠覆，也太震撼了！”段江伟感叹出声，声音有些颤抖。
<br/>　　洛冥也满是感慨：“真不知道他们究竟是怎么做到的，这些丧尸都快接近南都城的数量！才短短七天……”
<br/>　　直升机在飞近南署基地上空的时候就开始降落，随着距离地面越来越近，诡异的感觉也让两人心脏一顿。
<br/>　　“不对！”段江伟豁然起身，双目惊悚道：“怎么这么安静！？”
<br/>　　随着他的话音落下，整个就直升机的机舱内都一片寂静无声，两名飞行员及洛冥都微微冷汗直冒。
<br/>　　此时他们已经快要降落，但除了直升机的螺旋桨声，周围竟然安静的可怕，什么声音都没有！
<br/>　　空城？
<br/>　　段江伟瞪大着眼睛，透过窗户望着下方场景：“都是丧尸尸体，到处都是，全部死了。”
<br/>　　洛冥同样在观察，与段江伟轻声交流：“随着丧尸的数量越来越少，嘶吼声变弱，基地安静也是正常的，但总不至于狼牙战团的那几万人都是哑巴吧？”
<br/>　　“更何况战斗声呢？”段江伟越想越不对，下令道：“先别急着降落，低空飞行绕一圈！”
<br/>　　“先绕着高墙看看。”洛冥补充道：“不是说那一圈高墙上是三战队在战斗么？”
<br/>　　两名飞行员立即照做，很快直升机便飞到了那一圈高墙的上空，因为是低空飞行，高墙上的场景顿时被尽收眼底。
<br/>　　“和基地内的情况一样，只有丧尸的尸体。”段江伟的神情越来越严肃，面色也变得难看至极：“没发现任何狼牙战团的人，或者应该说，没看到一个活人……”
<br/>　　整个南署基地，当下都好似一座死城，充满死尸的城池！
<br/>　　机舱内安静的针落可闻，所有人的面色都有些发白，他们延着高墙飞行了大半圈，见到的统统都是同样的场景，满地的丧尸尸体，全部被砸烂了脑袋没有原样，高墙旁边与内墙之间的那空白地带，更是丧尸尸体遍地铺满。
<br/>　　“是不是大战已经结束，狼牙战团撤离了？”一名飞行员紧张的开口：“要是出事，也不可能这些丧尸都全死了啊，怎么看都像是已经屠杀完了，狼牙，狼牙战团赢了吧？”
<br/>　　洛冥冷着脸，道：“继续飞，把整个基地都绕一圈，一处角落都不能放过！”
<br/>　　直升机继续飞行，在这寂静到恐怖的基地上空一圈又一圈的飞着，随着时间的推移南署基地的大部分地域上空都已经飞过，但死一般的安静依旧，好似某种诡异又邪恶的力量，让机舱内的四人忍不住涌·出一股恐惧感。
<br/>　　狼牙战团的那么多人，都去哪了？
<br/>　　凭空消失？！
<br/>　　时间又过去良久，洛冥已经闭上了眼睛，他没有想到此行竟然会是这种结果。
<br/>　　“长官，已经全部查看一边了……”一名飞行员迟疑着开口。
<br/>　　“继续，再找一遍。”洛冥开口，不肯死心。
<br/>　　这不是楚涵的风格，更不是狼牙的风格，大战结束后的安静太不对劲！
<br/>　　这场创世纪的大战，打完后直接撤离？
<br/>　　他们习惯性焚烧丧尸尸体的战后清理呢？
<br/>　　战役的详细数据统计呢？
<br/>　　甚至输赢，都没有任何汇报……
<br/>　　这根本不正常！
<br/>　　忽的就在所有人都心头冒出了绝望想法之时，段江伟猛然出声：“等一下！”
<br/>　　洛冥豁然睁眼望向他：“发现了什么？”
<br/>　　“这里我们之前飞过一遍。”段江伟道。
<br/>　　洛冥皱起眉：“这是第二遍，有什么问题？”
<br/>　　段江伟目光呆滞，指着下方道：“这些尸体怎么没有脑袋砸烂？”
<br/>　　整个基地内的场景都差不多，密密麻麻躺着成堆的丧尸尸体，到处都是黑色的丧尸血，根本分不清躺着的是人还是丧尸，但既然是躺在那里一动不动，基本上就是死去的丧尸无疑。
<br/>　　所以之前他们寻找的重心，也是放在了哪里有站着的人，而不是躺在那里的尸体。
<br/>　　可当段江伟的话出口，机舱内的其余人也发现了不对，这片地带同样躺着密密麻麻的尸体，衣衫褴褛身上满是丧尸黑血或腐肉，但是与其他地带的丧尸堆不同，这些尸体的脑袋没被打烂。
<br/>　　没打烂脑袋的丧尸，根本不会死吧？
<br/>　　“再低一点。”洛冥出声，带着紧张。
<br/>　　直升机又一次降低高度，在这些成片的尸体上飞过，随着观察，四人都越看越觉得不对。
<br/>　　“这是，这皮肤？”段江伟瞪大双目：“这些人的皮肤根本不是灰白色！”
<br/>　　此时在他口中的尸体，已经被称呼为人，可见下方这些躺着的躯体，身份已经呼之欲出。
<br/>　　“降落！”洛冥立即下令。
<br/>　　显然，狼牙战团的人已经找到，眼下正躺在地上生死不知，从远处看跟基地内那些百万数量丧尸没太多差别，就像尸海中的一部分！
