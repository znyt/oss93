　　此时，焚寂就在面前，慕清霄感觉，阴阳玉中的魂剑已经迫不及待起来。
<br/>　　压制住魂剑的躁动，慕清霄叹了口气，他非常清楚，现在他可以吞噬焚寂，但后果就是韩云溪的性命。
<br/>　　焚寂是太子长琴被贬人间，魂魄经过榣山时，被龙渊铸剑师角离以“血涂之阵”强行取其命魂四魄所造，太子长琴魂魄分离之后，进入焚寂的一半剑灵即陷入沉睡。
<br/>　　随着女娲封印了七把凶剑，焚寂被封印于乌蒙灵谷，遣其族人世代看守。
<br/>　　千年后，焚寂被欧阳少恭找到抢夺时，看守此剑的大巫祝韩休宁将焚寂剑灵移入了已死的儿子韩云溪体内，
<br/>　　韩云溪是一个极其复杂的存在，因焚寂而亡，又因焚寂而生。
<br/>　　如果现在夺走焚寂，剑灵与剑身失去联系，韩云溪估计也会彻底死去，韩云溪死去，焚寂就不完整，这不是他想要的结果。
<br/>　　“呼......总算是压制住了焚寂的煞气。”
<br/>　　紫胤真人撤回法力，脸色变得有些苍白，显然在幽都便已经被焚寂给侵鉵，来天墉城的路上也压制着焚寂。
<br/>　　他虽是剑仙，但也经不起如此长时间的消耗。
<br/>　　见他撤回法力，涵素真人走上前去，道：“紫胤真人，到底是怎么回事？”
<br/>　　“哎，说来话长......现在，我用空明幻虚剑印，压制住焚寂，同时也封印了这孩子体内的焚寂煞气。”
<br/>　　紫胤真人话语落下，涵素真人皱眉，道：“按照你所说，焚寂煞气已与这孩子融为一体根本就无法驱除，假以时日，恐怕这孩子免不了被焚寂煞气所吞噬啊。”
<br/>　　紫胤真人叹息一声，他何尝不知，被焚寂煞气吞噬，韩云溪便会坠入魔道，但只要还有一线希望，他便会全力拯救这孩子。
<br/>　　“乌蒙灵谷世代镇守焚寂剑，如今这孩子更因焚寂沦为孤儿，我与他母亲乃是故交，只要还有一线希望，我都要保住他的性命。”
<br/>　　紫胤真人执意要救韩云溪，涵素真人见劝说不果，也不再阻挠，道：“紫胤真人，你受伤了？”
<br/>　　“无妨，闭关修养一段时间就能恢复。”
<br/>　　紫胤真人摆了摆手，现在心思都在韩云溪与焚寂上，虽然自身伤势严重，但现在也无暇顾及。
<br/>　　“也罢，既然你将这孩子带来天墉城，便是与我天墉城有缘，焚寂就由我们来负责封印吧。”
<br/>　　“如此，那便劳烦掌门真人。”
<br/>　　紫胤真人微微点头，涵素真人能帮忙封印焚寂最好不过，他的伤势，确实需要闭关一段时间。
<br/>　　“见过紫胤真人。”
<br/>　　“无需多礼。”
<br/>　　见到慕清霄哦，紫胤真人含笑点头，看向慕清霄时微微一愣，欣慰道：“空明幻虚剑，你已经入门了？”
<br/>　　要知道，空明幻虚剑可是他一身剑技所成，哪怕当初涵素真人修炼，都足足修炼了数十年。
<br/>　　而面前的青年，居然只用了一年的时间，就将空明幻虚剑入门，天赋妖孽程度，简直匪夷所思。
<br/>　　“勉强入门罢了，想要施展，恐怕还需要很长一段时间。”慕清霄如实说道。
<br/>　　紫胤真人欣慰一笑，不再说话，至于涵素真人也是满脸笑容，慕清霄天赋越妖孽，对天墉城来说，就越有利。
<br/>　　为人处事谦虚，性格不骄不躁，这份心性难能可贵。
<br/>　　可以说，涵素真人坐上掌门多年，做的最正确的事情就是，将慕清霄拉拢到了天墉城的阵营。
<br/>　　“红玉，你先带这孩子下去休息吧。”
<br/>　　“是，主人。”
<br/>　　此时，一直站在紫胤真人身后，默不作声的红玉，抱起韩云溪向剑塔内部走去。
<br/>　　红玉，上古庆枫族族人，紫胤真人的剑灵，宿体为古剑·红玉。
<br/>　　成为剑灵以前的红玉，只是庆枫部一名普通女子，因外出远行而逃过部族覆灭之灾。
<br/>　　回到族中后，看到被夷为平地的庆枫，被曝于野外的尸骨，悬于房梁的红衣，红玉那时只为复仇而存在，只求一个痛快，宁可堕入非道，永出轮回。
<br/>　　为报灭族之仇，自愿成为剑灵，求于师从龙渊的姒父以她的生魂铸成古剑·红玉，借助拥有绝世剑术的炤夫人之手复仇。
<br/>　　作为报答，红玉答应守护炤夫人血脉生生世世，直到血脉断绝，者炤夫人的后人不再需要她。
<br/>　　千年光阴如梦境一瞬，在安陆的黄昏里，红玉被她守护了许久的炤夫人血脉还以自由之身。
<br/>　　后来，她成为昆仑山天墉城紫胤真人的剑灵，奉命跟随、保护私自离山的百里屠苏。
<br/>　　她用自己的见识与智慧适时的帮助着同伴，然而与同伴们一起经历的事情，也给了红玉很多启示。
<br/>　　在旅途中为屠苏的性情所折服，将他视作自己的亲人，追随屠苏去蓬莱决战。
<br/>　　最终，红玉回到昆仑山，从此不离紫胤真人半步。
<br/>　　即便彼此之间并非红尘****，却有超脱世俗的相处之道，只怕就此天荒地老。
<br/>　　慕清霄颇为好奇的打量了红玉一眼，这还是他第一次遇到剑灵，与活生生的人完全没有任何区别。
<br/>　　待红玉抱着韩云溪离去，慕清霄也收回目光，视线再度落在焚寂上。

<br/>　　五堂长老，算上慕清霄，总共六位全都聚集于此。
<br/>　　“警钟敲响，到底发生了何事？”凝虚真人等人纷纷面露困惑之色。
<br/>　　当看到剑塔中，煞气肆意的焚寂后，涵究真人目光一凝，惊呼道：“好浓郁的煞气，这是焚寂剑！”

<br/>　　见他们一个个惊呼的模样，涵素真人微微摇头，道：“都安静下来，吵吵闹闹成何体统。”
<br/>　　待各堂长老安静下来，涵素真人便开始诉说事情的来龙去脉。
<br/>　　“就在半个月前，乌蒙灵谷被人所灭，焚寂剑乃是执剑长老带回天墉城，你们来的正好，便与本座一起将着凶煞之剑封印，免得凶剑为祸人间。”
