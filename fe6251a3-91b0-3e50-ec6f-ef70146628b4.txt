        一个人的时候，她总是在胡思乱想，然后浑浑噩噩地度过一整天。
<br/>
<br/>        傅氏集团现在都由外聘的CEO接管了，是不是代表了他傅夜沉的归期遥遥无期？
<br/>
<br/>        办公桌上的台历上，童筱颖画上了三十五个叉，代表着她有三十五天没有见到他了。
<br/>
<br/>        设计图的审核工作，她每天都在做，事情做完后的空暇时间里，她握着签字笔，在便利贴上写了一遍又一遍：“大哥，你回来吧！我不生气了！”
<br/>
<br/>        后面，她写着写着，变成了：“大哥，你回来陪我，我想你了。”
<br/>
<br/>        童筱颖发觉自己已经有了拿起手机，按下傅夜沉手机号码的习惯，却总是在最后一个按键上选择了返回。
<br/>
<br/>        明明，其实，她可以选择拨通按键。
<br/>
<br/>        但是她没有。
<br/>
<br/>        到底是跟自己过不去，还是跟他过不去，她心里已经彻底分不清了。
<br/>
<br/>        原本，她一直坚持认为是他傅夜沉的错，但是熬了这么多天，她熬不下去了，开始反省，是不是自己做错了。
<br/>
<br/>        在部长孙莉莉那儿软磨硬泡，拜托孙莉莉间接地从新任CEO那儿要到了傅夜沉的出差地址后，她童筱颖将手头上的工作都交接好后，办了签证，不打招呼就订了机票飞去了法国。
<br/>
<br/>        高耸的铁塔下，喷泉淙淙绚丽涌出；周遭是古典华丽洋溢着浪漫诗意的哥特式建筑；蓝天白云，空气里，弥漫着各种浓郁的花香……
<br/>
<br/>        但是，她不是来旅游的，导致她下飞机后，因为语言不通，出租车司机将她带到了旅游景点，与她要去的地方差了好远。
<br/>
<br/>        在这种人生地不熟的异国，用英语都无法交流的情况之下，童筱颖意识到彻底迷路了之后，拿出手机，拨通了傅夜沉的手机号码。
<br/>
<br/>        电话响了许久，才被接听。
<br/>
<br/>        当她带着期许唤了一声“大哥”的时候，听筒里却传来了落以柔的声音。
<br/>
<br/>        “你有什么事吗？傅总现在在开会。”
<br/>
<br/>        这是傅夜沉的私人号码，居然被落以柔拿在手上，可想而知，他傅夜沉是有多信任落以柔了。
<br/>
<br/>        一丝伤感浮上心头，但她很快又将这种感觉给压制了下去。
<br/>
<br/>        “我来法国了，但是我迷路了……找不到你们的那个地方。”童筱颖吞吞吐吐地说道。
<br/>
<br/>        而听筒那边的声音突然间没了，过了一会儿，落以柔才回复道：“你发个定位给我，我派车过去接你。”
<br/>
<br/>        “好。”想必，这是傅夜沉的意思吧！
<br/>
<br/>        童筱颖应了声后，挂了电话，然后发了个定位给那边。
<br/>
<br/>        在原地等了近两个小时，才有一辆黑色的豪华轿车来接她，如果不是落以柔在车内，她或许还得犹豫或是再打电话问清楚自己是否该上车。
<br/>
<br/>        落以柔一如既往的一袭米白色职业装套裙，出落大方，一丝不苟，明明她看起来与她年纪相仿，却比她要成熟干练。
<br/>
<br/>        车子驶出了豪华的城市，路过麦田，又蜿蜒而上。
<br/>
<br/>        在一片紫色的薰衣草花田之外，有一座偌大得像城堡一样的白色别墅屹立在晚霞之中，景色美得像身处童话世界。
<br/>
<br/>        下车后，落以柔带着童筱颖进了别墅，途径主道的时候，还有菲佣向她俩礼貌性地欠身行礼。
<br/>
<br/>        别墅的围墙内，除了有花园、草地，还有一片像湖泊一样水蓝的浅滩，浅滩上有几只白色天鹅在戏水游玩，这里比傅夜沉在滨江城里的那座私宅还要豪华、奢侈、美丽。
<br/>
<br/>        “这段时间，你们就住在这里吗？”童筱颖一边打量着这里，一边好奇地问。
<br/>
<br/>        落以柔却对童筱颖翻了个白眼，不冷不热地回答道：“傅总都已经买下这里了，我们自然是都住这里。”
<br/>
<br/>        “买下了这里？”童筱颖回过神来，难以置信地看向落以柔。
<br/>
<br/>        这座别墅确实很大，按建筑学上的分析来说，这别墅完全可以用于生活和办公一体化。
<br/>
<br/>        也就是说，带着一个十多人的团队，坐在家里办公，完全没问题。
<br/>
<br/>        所以，如果她不主动来，他傅夜沉是打算长久在这里住下去吗？
<br/>
<br/>        又或者说，她现在来这里的意义是什么？
<br/>
<br/>        落以柔将童筱颖领去了傅夜沉的房间，并让她好好地待在房间里不要到处乱跑。
<br/>
<br/>        “他还在开会吗？”童筱颖按捺不住地问。
<br/>
<br/>        落以柔却极其不耐烦地回答：“傅总在招待贵宾，你来得很不是时候。所以，你现在要做的，就是待在房间里，哪里都不要去。待会，会有菲佣把晚餐送到房里来。”
<br/>
<br/>        “我什么时候可以见他？”童筱颖又问道。
<br/>
<br/>        落以柔忍不住对童筱颖翻起白眼来：“我说你怕是戴了一个假的助听器吧？”
