　　火牙宗主见状，双眼一眯。
<br/>　　“都说你紫炎易霄性格淡漠，视人命如蝼蚁。”
<br/>　　“今日一见，果然如此。”
<br/>　　在他看来，易霄这等声名在外之辈，必然会极其在乎名声。
<br/>　　“呵。”萧逸淡漠一笑，“收起你那些鬼话吧，对我无用。”
<br/>　　火牙宗主的定力，果真厉害。
<br/>　　这般情况下，竟然还能镇定出言，甚至打算如之前那般，通过言语扰萧逸心神。
<br/>　　只是，这些话语，这些攻心之言，在萧逸面前，不过是个笑话罢了。
<br/>　　“你的这些鬼话，除了激怒我之外，别无用处。”萧逸冷笑一声。
<br/>　　嘭…
<br/>　　手中紫炎，猛地打出。
<br/>　　几乎是紫晶灵炎落下的瞬间，地面之上，一个火焰大阵，顷刻爆发。
<br/>　　那堆数以万计的武道冰晶，顷刻置身火海之内。
<br/>　　“混蛋，你敢？”火牙宗主瞳孔一缩，再不复镇静。
<br/>　　“就是现在。”萧逸双眼一眯。
<br/>　　嗖…身影，忽然化作了一道火光，速度快到了极点。
<br/>　　火牙宗主还未反应过来，萧逸已来到他身前。
<br/>　　火焰包裹着的手掌，轻轻一扣，反扣住了火牙宗主扣在廖灵儿咽喉的手掌。
<br/>　　下一秒，一拳轰出。
<br/>　　恐怖的一拳，竟是瞬间将火牙宗主轰飞百步。
<br/>　　“额。”火牙宗主，竟是闷哼一声。
<br/>　　不过，他已来不及对萧逸出手，而是暴喝一声，“快灭掉那些火焰。”
<br/>　　周遭邪修之中，不乏修习水之一道和冰雪一道的武者，纷纷出手。
<br/>　　一些修为高深的邪修，亦试图通过元力强行灭去火焰。
<br/>　　这边，萧逸冷笑一声。
<br/>　　“前辈，你好厉害。”身旁，廖灵儿美眸闪烁不停。
<br/>　　“之前我心头那声冷哼，是前辈在提醒我恢复清明吧。”
<br/>　　“那火牙宗主虽擅攻心之言，却不仅没有影响到前辈；反倒被前辈瞬间捉住了机会，雷霆一击，将他轰飞。”
<br/>　　萧逸闻言，瞥了廖灵儿一眼，不语。
<br/>　　但他知道，廖灵儿没有猜错，这丫头，倒是冰雪聪明。
<br/>　　火牙宗主的攻心之言，不仅没有对他起效。
<br/>　　反倒被他反过来言语一激，露了破绽，他这才有了雷霆一击，轰退火牙宗主。
<br/>　　至于刚才的雷霆一拳，则正是‘崩界拳’。
<br/>　　这一月时间的剿杀邪修，萧逸可不是在做着无用功。
<br/>　　他杀邪修，自然有他的目的。
<br/>　　或者简单来说，他早已将邪修当作了他的历练对象。
<br/>　　这些邪修，最喜欢弄些操控妖兽之事。
<br/>　　故这些邪修身上，多的是妖兽精血。
<br/>　　一月时间，横扫近百分部，萧逸的收获，可是极为丰厚。
<br/>　　而他的肉体力量，也短短一月时间内，连跳两重，达到了圣皇境六重的地步。
<br/>　　刚才，他便是瞬间耗尽了体内所有修罗之力，加上火焰的爆发，这才有了那恐怖的速度，瞬间来到火牙宗主身躯。
<br/>　　而最后的一拳，更是以所有修罗之力，以施出最强的一记崩界拳。
<br/>　　这才有了一拳轰退火牙宗主的效果。
<br/>　　不过，也仅此而已了。
<br/>　　那一拳，虽震退了火牙宗主，却并未能伤到火牙宗主。
<br/>　　对比他的控火一道所赋予的实力，还有剑道所赋予的实力，现今肉体一道，还是不够看。
<br/>　　暂时来说，修罗战体只能在一瞬间消耗所有修罗之力的情况下，起到一些奇效。
<br/>　　如刚才那一拳。
<br/>　　不过，这已经够让萧逸满意的了。
<br/>　　毕竟，他现今肉体一道才圣皇境六重。
<br/>　　这门功法，一直修炼下去，日后必然能给他更恐怖的战力；甚至于真正达到极高层次时，必有一拳破天之力。
<br/>　　只是，这门功法，也确实逆天，所需的妖兽精血，乃是一个天文数字。
<br/>　　在萧逸的判断中，修炼这门功法，怕是一个霸主势力的底蕴都难以支撑得起。
<br/>　　不过，这些邪修，倒是给了他‘以战养战’的途径。
<br/>　　这些邪修，特别是邪修分部，都存着大量的妖兽精血。
<br/>　　另一边，火牙宗主等人，全力试图扑灭着燃烧着的熊熊紫炎。
<br/>　　只不过，这显得有些徒劳。
<br/>　　“没用的。”萧逸淡漠出言。
<br/>　　“那堆武道冰晶之下，早便被我布下紫炎大阵。”
<br/>　　“而武道冰晶之内，则早被我烙下了火焰印记。”
<br/>　　火牙宗主等人，试图扑灭紫晶灵炎，不过是徒劳之事。
<br/>　　先不说在阵法增幅下，紫晶灵炎燃烧惊人。
<br/>　　单说那些武道冰晶，里头早就被烙下了火焰印记。
<br/>　　也就是说，早在萧逸手中凝聚出紫晶灵炎，引动大阵之时，这些武道冰晶外表看似无损，实则里头早已被燃烧成虚无。
<br/>　　即便紫晶灵炎被扑灭，这些武道冰晶也早成无用之物了。
<br/>　　“这就是传闻中的紫炎易霄吗？”廖灵儿看着身前的萧逸，美眸中，惊喜连连。
<br/>　　刚才，她显然是太激动和兴奋了，还一口一句‘前辈’地叫着。
<br/>　　可现在，她显然已反应了过来。
<br/>　　站在她面前的，哪里是一个脾气古怪的老家伙，根本是一个冷酷的年轻人，而且还是传闻中的紫炎易霄。
<br/>　　另一边，火牙宗主咬牙切齿，拳头握得劈啪作响。
<br/>　　紫晶灵炎，焚毁万物，根本不是寻常的风水能浇灭，寻常冰雪能消融的。
<br/>　　他只能眼睁睁地看着那数以万计的武道冰晶被烧成灰烬，而无可奈何。
<br/>　　“完…完了…”
<br/>　　火牙宗主脸色一白。
<br/>　　“火牙分部，还有周遭数百分部数年心血，毁于一旦。”
<br/>　　火牙宗主只觉一阵头晕目眩。
<br/>　　“宗主。”周遭长老，亦是大惊。
<br/>　　“此番所有武道冰晶和武道力量毁于一旦，先不说总府怪罪下来，单单护法那里，就无法交代。”
<br/>　　“该死…该死。”火牙宗主的身躯，颤抖着，那是愤怒到极点的表现。
<br/>　　一双锱铢欲裂的眼眸，看向萧逸。
<br/>　　“紫炎易霄，你好大的胆子。”火牙宗主几乎是一字一顿，咬紧了牙。
<br/>　　“怪便怪你自己太过得意。”萧逸冷笑一声。
<br/>　　“若非你话太多，也不至于给我机会布下紫炎大阵。”
<br/>　　“紫炎易霄，还有你这个臭丫头。”火牙宗主怒火滔天，直视萧逸与廖灵儿。
<br/>　　虽然他也不得不惊讶于易霄的手段，那无声无息便瞒过他在暗地里布下阵法的本事，确实厉害。
<br/>　　连他这个成名多年的绝世强者，都无法感知出其中的气息，难怪紫炎易霄会如此神出鬼没。
<br/>　　但在他看来，若非廖灵儿，易霄还有一众武者，怎会出现在这里。
<br/>　　“今日，你们都得死。”
