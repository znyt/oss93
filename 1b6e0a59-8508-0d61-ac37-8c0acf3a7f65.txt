陆寒霆将挺括的后背倚靠在沙发里，动作优雅的晃动着手里的红酒杯，他就喜欢看她吃醋，只有吃醋的时候一向清丽聪慧的她才会失了分寸，在他面前露出女孩的撒泼和娇蛮。
<br/>
<br/>        她就是一个小缩头乌龟，如果不用华容来刺激她的话，她永远不会敞开自己的心扉让他住进去。
<br/>
<br/>        ……
<br/>
<br/>        苏希跟夏小蝶的订婚party就在今晚举行，夏小蝶春风得意，早就打电话通知夏夕绾不要忘了今晚的订婚party，嘱托她一定要来亲眼见证自己的幸福。
<br/>
<br/>        夏夕绾笑着应下了，今晚的订婚party她当然不会缺席，她真的准备了一份神秘大礼。
<br/>
<br/>        这时陆老夫人走了过来，“绾绾，奶奶跟你说一件事，奶奶待会儿要去寺庙里祈福烧香，为了保佑你们早点给我生下重孙，我还要在山上住几天，就不回来了，你打电话跟寒霆说一声，让他早点回来陪你。”
<br/>
<br/>        “知道了奶奶。”
<br/>
<br/>        陆老夫人走了，夏夕绾拿出手机，其实她不想给陆寒霆打电话的。
<br/>
<br/>        那晚在酒吧的总统套房里他走后就没有再回来，两个人也没有再联系。
<br/>
<br/>        不过既然奶奶交代了，她就打个电话告知他一声吧。
<br/>
<br/>        夏夕绾拨出了他的手机号码，悠扬的铃声响了一遍，然后电话被不急不缓的接通了。
<br/>
<br/>        陆寒霆低醇磁性的嗓音传递了过来，“喂。”
<br/>
<br/>        夏夕绾纤白的手指拽着手机，“陆先生，奶奶让我告诉你一下，奶奶去寺庙里烧香祈福了，要在那里住几天才回来。”
<br/>
<br/>        “奶奶还说了什么？”
<br/>
<br/>        奶奶最后还交代让他早点回来陪她…
<br/>
<br/>        不过这话夏夕绾是不会告诉他的，一旦告诉他就很像是她要他回来陪她的。
<br/>
<br/>        这时陆寒霆那端突然传来了一道妩媚好听的嗓音，“总裁，关于这份合同我有点不明白的地方，想请教你。”
<br/>
<br/>        夏夕绾纤长的羽捷一颤，女人对有些东西特别敏感，她一听就认出了这道声音。
<br/>
<br/>        这道声音就是那天接听了他手机的声音。
<br/>
<br/>        这时夏夕绾就听到了陆寒霆格外低醇温和的嗓音，“哪里不明白，我教你。”
<br/>
<br/>        夏夕绾贝齿咬住了红唇。
<br/>
<br/>        这时陆寒霆好像想起了她，问了一句，“你还有事？”
<br/>
<br/>        “没事了，我只是没想到陆先生还有潜规则女员工的癖好，我不打扰你了！”
<br/>
<br/>        夏夕绾直接将电话给挂断了。
<br/>
<br/>        将手机放在包里，夏夕绾烦闷的深呼吸一口气，她没有想到那个女人是他手下的员工。
<br/>
<br/>        听说很多老总会将手下年轻貌美的女员工包养成自己的情人，出差的时候一起带上，工作娱乐两不误，原来他陆寒霆也是这种人！
<br/>
<br/>        夏夕绾用力的闭上眼不会想他的事情，她要赶去苏希和夏小蝶的订婚party了。
<br/>
<br/>        ……
<br/>
<br/>        这一次的订婚party是在香槟山庄举行的，相当的奢华浪漫。
<br/>
<br/>        夏小蝶穿着米兰订制回来的白色露肩纱裙，长发编成了花辫，还戴着璀璨的皇冠，美丽的像一只花蝴蝶。
