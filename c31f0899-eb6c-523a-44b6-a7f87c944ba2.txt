
<br/>　　<br/>　　    王惊蛰躺在床上枕着胳膊，思绪一时无两，这龟丞相一样的黄九郎给他一种探不到底的感觉，脸上写着高人两字的绝对是个装逼份子，但这种你摸不到深浅的，妥妥是个深藏不露的武当张三丰。<br/>　　<br/>　　    “当初，韩观山要是有他一半的见地，也不至于了啊……”王惊蛰摇头感叹了一声。<br/>　　<br/>　　    货比货得扔，人比人得能比死了，韩观山出场自带背景音乐，黄九郎出场无声无息，但看结果，其实就高下立判了。<br/>　　<br/>　　    入夜，火车轰隆直响，王惊蛰闭上眼睛没过多久就睡着了，黄九郎的问题他已经暂时不去思考了，来年京城再见再说。<br/>　　<br/>　　    时间一晃过去一夜，隔天清晨之后吃过早饭，没过多久列车抵达长安车站，王惊蛰背着简单的行囊出来后，直接在车站就上了一辆通往骊山景区的大巴，临近中午左右他就进了秦村。<br/>　　<br/>　　    来到王二楼家的时候，王仙芝正坐在院子里裹着棉袄晒太阳，旁边放着一壶茶水，两个杯子，这副装扮就是典型的农村大叔的打扮，要说哪里和别人有点不一样的，就是王仙芝的脸上写满了沧桑和故事。<br/>　　<br/>　　    父子俩人时隔大半年再见，反应都挺平淡的，王仙芝抬头说了一句来了，王惊蛰点头说了声嗯，我进屋去洗把脸，这就算是他们打的招呼了。<br/>　　<br/>　　    王惊蛰简单收拾了一番后，也来到院子里，主动拿起茶壶给杯子里倒上茶水，然后拿出烟递给王仙芝一根。<br/>　　<br/>　　    王仙芝接到手里，抽了几口后，嘴里忽然没头没尾的冒出一句：“人活一世，长大成年了，不管活成什么样，是苦是甜，是平坦还是坎坷，都不能怨天尤人，怪不得天地也怪不得父母了，因为你生下来就是个道理，你不管活成什么样都不能怨，但总归自己得珍惜吧？”<br/>　　<br/>　　    王惊蛰“吧嗒，吧嗒”的裹着烟头，几口就把一根烟给抽没了，说道：“对不起……”<br/>　　<br/>　　    王仙芝说的，是王惊蛰为吴满弓借三年阳寿的事，王惊蛰本就到三十就有一道坎，却在这个节骨眼上又没了三年的命，王仙芝自然一眼就瞧出来了。<br/>　　<br/>　　    王惊蛰说对不起，不是冲着自己借出去的三年阳寿，而是他以一个儿子的角度和父亲来说的，他不管出于什么理由，哪怕就是天大的道理，这三年命没了，无疑相当于是在王仙芝的心头上割了三刀。<br/>　　<br/>　　    人生一世，草木一秋，你活的是自己，但也有人在看你活着。<br/>　　<br/>　　    王仙芝说了一句之后，就没在这个话题上接着唠下去了，他没问缘由也没问事后怎么办，因为事情已经都发生了，你这时是埋怨还是后悔，那都跟放屁一样，话说的再多一股小风吹来也没了。<br/>　　<br/>　　    父子两人喝着茶水，多数的时候都是王惊蛰在说，王仙芝在听，说的无非就是上次一别之后发生的种种，一直聊到最近这段时间。<br/>　　<br/>　　    王仙芝说道：“那份丹方要是成了，你得先给向征送去一份，你和他的区别在于，你最后要的是逆天改命，这个丹方却能让他脱胎换骨，他比你需要得多”<br/>　　<br/>　　    王惊蛰点头说道：“了解，先可着他来吧”<br/>　　<br/>　　    王仙芝又说道：“你给向征送去，向缺必然会记着你这个天大的人情，以后也必定会为你鞍前马后的还这个人情，到也不是我们父子太市侩，而是人么，人与人之间的交往，就是你来我往的，有先礼才有后报，关系才能接着维持下去，君子之交平淡如水，这种事是不存在的，毕竟君子也交不到什么朋友，人谁无欲无求啊……”<br/>　　<br/>　　    “我懂，但愿能把这方子搞了吧”<br/>　　<br/>　　    “敢想就会有，没什么难的，那条柳仙怎么会平白无故的就给你送了这份礼，不还是因为你身带一道天道气运奏效了么，气运傍身，你还怕什么？”<br/>　　<br/>　　    王惊蛰“嗯”了一声，忽然直视着王仙芝的眼睛，开口问道：“我在八拐里坡见到爷爷了，他死了”<br/>　　<br/>　　    王仙芝皱了皱眉，没有吭声。<br/>　　<br/>　　    “爷爷，是为我去的八拐里坡，也是为我而死的吧”王惊蛰眼圈通红的说道：“到底，为的什么？”<br/>　　<br/>　　    “他大限将至了，就算不去八拐里坡，他也剩下不了几年了”<br/>　　<br/>　　    “但我就想知道，他为什么去的？”王惊蛰捏着拳头问道。<br/>　　<br/>　　    “你既然碰见你爷爷了，那他告诉你了么”<br/>　　<br/>　　    “没”<br/>　　<br/>　　    “那你就等着他亲口告诉你吧”王仙芝说完，撑着椅子就站了起来。<br/>　　<br/>　　    王惊蛰随即也起身，喘着粗气问道：“如果有一天，有个机会摆在你面前……你，你是不是也会像爷爷那样？”<br/>　　<br/>　　    王仙芝静静的看着他，良久之后，才淡淡的说道：“我和你说过龙背山的故事，还记得？”<br/>　　<br/>　　    王惊蛰当即一愣，抿了抿嘴唇，点头说道：“就是因为记得，我才这么问你”<br/>　　<br/>　　    “啪”王仙芝拍了拍他的肩膀，说道：“畜生尚且如此，更何况人呢？我刚才和你说，人活一世不管活成什么样，都怨不了天地和父母，其实这后面应该还有一句话，就是不管你活成什么样，天地无眼看不见，这世上能看见的，也就只有你的父母了……”<br/>　　<br/>　　    王仙芝说完，走了。<br/>　　<br/>　　    王惊蛰闭着眼睛深深的长吐了口气，嘴里挤出两字：“谢谢！”<br/>　　<br/>　　    传说远古时有一座大山在云层当中，山下有一湖泊，是为方圆千里百姓日常饮水所用，后来一天有头小龙来此游玩，一时兴起把整个湖中的水全都给吸走了，由此惹下了大祸，附近的百姓没有水吃加上天气干旱，就死了不少的百姓。<br/>　　<br/>　　    上天知道这件事后要责罚那头小龙，于是想将他禁锢在干枯的湖底，准备让云层里的大山落下来镇压他。<br/>　　<br/>　　    于是，小龙王的父亲老龙王得知此事，千里迢迢赶来，以自己的龙背抗住了那座山，祈求上天说他可为小龙偿还孽债，只为给他一次改过自新的机会。<br/>　　<br/>　　    最后，山落了下来，被老龙扛住了，一直把它压到了干枯的湖底。<br/>　　<br/>　　    小龙得救了，老龙永世不得脱身，至此这座山就叫龙背山了。
