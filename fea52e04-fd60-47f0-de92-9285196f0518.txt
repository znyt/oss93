　　“有些宝物，我有办法得到，但是，有些宝物，需要靠你自己！”
<br/>　　龙腾说道，言罢，他指了指司空炎，道：“司空炎之所以能提升这么快，也是因为有宝物之助，他得到的那些宝物，都是专门给予奴仆的奖励！”
<br/>　　在龙族之中，有些宝物对于龙族之人无用，但是对于人族或者是其他的异族却是很有效果。
<br/>　　所以，这些宝物，龙族就用来奖励给下面的奴仆。
<br/>　　“怎么样能得到这些宝物？”苏莫沉声问道。
<br/>　　“只有在特定的时间才有机会，最近没有机会，下次有机会我再找你！”龙腾摇了摇头道，苏莫一次闭关就是十几年，他中间没敢去打扰，只去找了司空炎。
<br/>　　“嗯！”苏莫点了点头。
<br/>　　片刻之后，三人离开了空间宝器之中，回到了大殿。
<br/>　　苏莫再次闭关修炼，准备一举突破到三品中级。
<br/>　　密室之中，苏莫取出了大量的五行源石，开始快速的吞噬了起来。
<br/>　　经过十几年的沉淀，一切都水到渠成，半年之后，苏莫成功的突破到了三品中级，实力再进一步。
<br/>　　突破修为之后，苏莫也没有出关，继续修炼。
<br/>　　而在他闭关的日子之中，龙腾在龙族之中，也名声大震。
<br/>　　因为，龙腾在龙族天才的切磋当中，击败了几位龙族妖孽天才，其中不乏同为五爪金龙的龙族妖孽。
<br/>　　龙腾的天赋实力，让得龙族很多高层惊讶，因此龙腾在龙族的地位，也水涨船高。
<br/>　　甚至，一位龙族神王，更是收了龙腾为义子。
<br/>　　又过去了三个月，这一日，苏莫的密室之外，响起了沉重的敲门声。
<br/>　　苏莫闻声，便结束了修炼，起身打开了密室的大门。
<br/>　　密室之外，龙腾一身金色长袍，长身而立。
<br/>　　“苏莫，给你弄了点好东西！”龙腾面带微笑，随即手掌一番，掌中出现了两枚樱桃大小的血红色果实。
<br/>　　“这是什么宝物？”苏莫沉声问道，他能感觉到，这两枚果实之中，蕴含庞大无比的血气。
<br/>　　“这是我龙族血灵龙的血液滋养之下，而诞生的神果，能强大气血，坚固肉身！”龙腾道，言罢，他便将两枚神果递给苏莫。
<br/>　　“不错！”苏莫点了点头，便收起了两枚神果。
<br/>　　“还有，你和司空炎，过几日有一个得到大量宝物资源的机会。”龙腾面色郑重的继续说道。
<br/>　　“什么机会？”苏莫好奇的问道，大量资源宝物，这他可不能错过。
<br/>　　虽然他有很多五行源石，但都只是普通的资源，与一些天材地宝差别很大。
<br/>　　“我龙族经常会得到一些特殊的宝物，这些宝物对于龙族之人无用，便会赐予下面的奴仆，而一些顶级的宝物，十几二十年便会统一发放一次，五日之后，便会发放下来！”龙腾说道。
<br/>　　“都有什么宝物？我和司空炎都能得到？”苏莫眼眸一亮，龙族可是掌控一朝之地，所拥有的宝物肯定不凡。
<br/>　　虽然，这些都是龙族之人弃之不用的宝物，但是对于人族和其它异族来说，绝对不是凡物。
<br/>　　“什么宝物不清楚，但绝对都是顶级的宝物！”
<br/>　　龙腾面色凝重，继续道：“想要得到宝物，还需要靠天赋和实力，以你和司空炎的天赋实力，轻而易举就能得到很多。”
<br/>　　“嗯！”苏莫点了点头，既然都是顶级的宝物，那他一定不能错过。”
<br/>　　“不打扰你了，五日后，我再来找你！”龙腾道。
<br/>　　“好！”苏莫点了点头，随即便再次返回了密室之中。
<br/>　　取出两枚龙腾给予的果实，他张口就吞服了下来，果实入腹，立刻化为汹涌的药力。
<br/>　　这果实的药力，极为特殊，能激发他气血的生命力，强化他的全身经脉骨骼和血肉。
<br/>　　只不过，两枚果实，药力有些，对于苏莫来说，效果并不是太可观。
<br/>　　毕竟，他的天赋太逆天了，不是逆天的宝物，不可能让他有太大的改变。
<br/>　　不过，果实虽少，但聊胜于无。
<br/>　　吸收了两枚果实的药力之后，苏莫便继续修炼了起来，巩固三品中级的修为。
<br/>　　时间流逝，五日时间，眨眼而过。
<br/>　　这一日清楚，龙腾再次敲响了密室的大门，苏莫走出了密室。
<br/>　　密室外，龙腾和司空炎两人均在。
<br/>　　“苏莫，走，我们去奴务府！”龙腾道。
<br/>　　“苏莫，今日我们兄弟俩，一定多弄点宝物！”司空炎穿着一声朴素的青衫，但是眼眸却是极为闪亮，目中充满了澎湃的自信。
<br/>　　“呵呵，我们走吧！”苏莫笑了笑，并未多言，随即三人一起，向奴务府走去。
<br/>　　今日的奴务府，格外的热闹，因为龙族发放给奴仆的宝物，就是在奴务府中发放。
<br/>　　有络绎不绝的奴仆，纷纷进入了奴务府，有的是独自一人，有的是三五成群，有的有主人的带领，有的则是自行前来。
<br/>　　苏莫和司空炎，跟着龙腾走进了奴务府，便看到，所有来此的人，都进入了一座偏殿之中。
<br/>　　“那偏殿通往一处小世界，发放宝物就在小世界之中！”龙腾边走，便向苏莫和司空炎说道。
<br/>　　“具体怎么分配宝物？”苏莫好奇的问道。
<br/>　　“龙都城是龙族的核心之地，城中有奴仆数十万，而宝物有限，自然是要给予天赋好的奴仆。”龙腾道。
<br/>　　“难道又是有什么考验？”司空炎问道，具体他也不清楚是什么情况，前几年他也得到了一些宝物，但只是领取的一些普通宝物。
<br/>　　“你们进去就知道了！”龙腾微微一笑。
<br/>　　言语之中，三人走进了偏殿之中，霎时之间空间变换，斗转星移。
<br/>　　随后，三人便出现在了一处小世界之中。
<br/>　　这处小世界，显然是人为开辟而出，方圆只有千余里，是一片光秃秃的大地，没有任何的生机。
<br/>　　而此时此刻，大地之上，聚集了足有二十万人。
<br/>　　这些人大多都是奴仆，来自各种不同的种族，只有一小部分是龙族之人，均是一些奴仆的主人。
<br/>　　“看，是龙腾！”
<br/>　　“龙腾前些年收了两个奴仆！”
<br/>　　“那个修为低的人叫炎空，据说仗着主人地位高，平常眼高于顶，连龙族之人都敢顶撞。”
<br/>　　苏莫三人的到来，不免引起了一些龙族之人和奴仆的关注。
<br/>　　因为，现如今，龙腾在龙族的风头极盛，可谓是春风得意。
<br/>　　“你又在外面搞事了？”苏莫眉头一皱，转头看向司空崖，他知道别人说的炎空，就是司空炎。
<br/>　　他和司空炎两人，进入龙族之后，都不会使用真名。
<br/>　　“这些奴仆，跪久了起不来了！”司空炎瞥了撇嘴，虽然他也是奴仆的身份，但心中从来不把自己看成奴仆。
<br/>　　他对于这些龙族的奴仆，非常的鄙视，因为有些人在龙族之人面前，卑微的如同蝼蚁一般。
