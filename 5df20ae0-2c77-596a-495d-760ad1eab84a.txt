　　北方看台上，云悠悠柳眉微瞥，心中也是觉得这个苏莫是不是有些太自大了。
<br/>　　以一敌四，而且还是炎澜天等妖孽天才，武帝境修为的武者之内，整个明虚星河都没有人能战胜吧！
<br/>　　或许等她达到武帝境九重之后，能做到这一步，但武帝境八重的修为，可能性微乎其微。
<br/>　　只有阴泽长老眸光闪亮，面上带着微笑，仿佛是能看穿苏莫一般。
<br/>　　战台上。
<br/>　　苏莫与炎澜天四人相对而立，凌厉如刀的目光交锋在一切，让得虚空不断的被撕裂出一道道口子。
<br/>　　炎澜天四人的气势，全部爆发了出来，如同山崩海啸，呈大浪席卷之势，向苏莫暴冲而来。
<br/>　　雷云皓是妖族，没有战魂，但其全身雷光闪烁，充满了毁灭般的气息。
<br/>　　炎澜天属于黑炎族，战魂便是本命黒焰，他全身黒焰熊熊，化为了黑色的火人，恐怖而又炙热的火焰之力，将周围的空间全部焚化。
<br/>　　灵枫和卓剑豪人两人的战魂，则是纷纷释放而出。
<br/>　　灵枫的战魂比较古怪，是一片淡白色的光点，不知道是什么战魂。
<br/>　　而卓剑豪的战魂，毫无意外是剑之战魂，而且是两柄剑之战魂，一黑一白，锋芒无尽，剑威摄人，恐怖无比。
<br/>　　面对四人如同狂涛骇浪般的气势，苏莫如同屹立在大海中的礁石，自巍然不动。
<br/>　　他的身上，同样有冲天的剑威爆发，这股剑威旁边、浩大，凌厉，恐怖，直接将四人涌来的气势威压切割而开。
<br/>　　四人的气势不可谓不强，就是一般的武尊境一二重武者，都要被吓的胆寒，不敢有对抗之心。
<br/>　　所有观战之人无不心惊，此刻苏莫身上爆发的剑威，实在是恐怖，早已超越了武帝境武者的范畴。
<br/>　　那卓剑豪乃是剑修，本命剑客，但身上的剑威和苏莫一比，简直是花猫和猛虎的区别，根本不可同日而语。
<br/>　　五人就这么对峙着，苏莫没有率先出手，炎澜天四人也没有出手。
<br/>　　就这样，五人对峙了约莫十几个呼吸之后，炎澜天等人终于动手了。
<br/>　　吼！
<br/>　　只见炎澜天双手一推，掌中无尽的黑焰喷涌，瞬间形成了一只长达百丈的黒焰猛虎，猛虎咆哮一声，裹挟滔天凶威，无尽火浪，向苏莫飞扑而去。
<br/>　　卓剑豪手臂一挥，刹那之间，剑光爆闪，瞬间便有近两百柄本命灵剑出现。
<br/>　　近两百柄本命灵剑，组合在了一起，形成了一只长龙，向苏莫袭杀而去。
<br/>　　这长龙是由长剑组成，堪称剑龙，威势滔天，恐怖骇人，惊天动地。
<br/>　　这正是一种本命剑阵，名为天龙剑阵，由一百八十八柄灵剑组成，可以轻易秒杀一般的武尊境一重武者。
<br/>　　与此同时，灵枫和雷云皓也没闲着，只见灵枫身形一闪，瞬间出现在了苏莫的后方，一拳轰响苏莫的后背。
<br/>　　这一拳极为强大，拳力极度凝练，白色的拳茫如同一道犀利的光芒，洞穿一切，威势一时无两。
<br/>　　而雷云皓则是身形冲天而起，瞬间掠上了高空，手臂一挥，一道长达千丈的恐怖雷霆从天而降，如同天地雷罚，又如九天之上斩下的神刀，直击苏莫的头顶。
<br/>　　几乎就是一瞬间，苏莫仿佛陷入了绝境之中，前方有炎澜天和卓剑豪的联合攻击，后方有灵枫的恐怖一拳，上方还有雷云皓的雷霆一斩。
<br/>　　所有观战之人，全部都紧张了起来，甚至很多人都哗啦啦的站了起来，比如道宗的弟子。
<br/>　　因为炎澜天四人的合击，实在是太强了，搞不好苏莫会在这一波攻击之下陨落。
<br/>　　“最好被轰杀！”扬天成却是心中大喜，若是苏莫在炎澜天四人的合击之下，被彻底的轰杀，那真是太好不过了！
<br/>　　阴泽长老眼眸微眯了下来，虽然他觉得苏莫很强，应该有和炎澜天四人一战的实力，但也是不得不防意外的发生，若是苏莫陨落了，那可是大损失。
<br/>　　轰隆隆！！
<br/>　　震天的轰响传荡八方，炎澜天四人的攻击，说来极慢，但实则迅疾无比，瞬间便粉碎长空，临近了苏莫。
<br/>　　就在无数的人为苏莫担心之时，就在扬天成心中激动不已之时，一直淡然而立的速度，突兀的消失不见了。
<br/>　　轰轰轰！！
<br/>　　苏莫突然消失，让得四人的攻击无可避免的碰撞在了一起，顿时响起震天的爆响，恐怖的爆炸之力，瞬间席卷八方，逆冲九天。
<br/>　　轰隆隆！！
<br/>　　滔天的气浪滚滚涌动，席卷了整个战台，炎澜天四人均是微微后退，免得遭到强烈的冲击。
<br/>　　“人呢？”四人立刻查看四方，但完全没有了苏莫的踪迹。
<br/>　　所有观战之人，亦是心惊不已，苏莫又动用了隐藏身形之法，躲过了攻击，而且隐藏了起来。
<br/>　　炎澜天四人，全部警惕了起来，个个满脸凝重的打量四周，这种局面让他们略显被动。
<br/>　　就在四人仔细查看四周之时，蓦地，一道恐怖的三色剑气突然出现在天穹之上。
<br/>　　剑气恐怖无比，浩大无边，距离雷云皓极近，不足十里距离，直接斩向雷云皓的后背，让得他面色剧变。
<br/>　　与此同时，足有百柄灵剑出现，形成十条犀利的灵蛇，向炎澜天飞射而去。
<br/>　　十条灵蛇剑威滔天，气息恐怖，呈合围之势，瞬间洞穿层层长空，瞬间便来到了炎澜天的周身。
<br/>　　“什么？”雷云皓和炎澜天惊骇不已，面色大变，突然的攻击让两人措手不及，而且这三色剑气和灵蛇剑阵的攻击，那恐怖的威势让两人感到全身冰寒，如坠冰窖。
<br/>　　由于距离太近，攻击来得太过突然，躲避已是来不及了，两人根本来不及多想，仓促之间只能极力防御。
<br/>　　“起！”雷云皓暴喝一声，全身雷光爆闪，形成了一个巨大的雷球，将他整个人完全的包裹了起来，形成了完美的防御。
<br/>　　雷球之上，密密麻麻的电弧，如同上万电蛇游走，惊人无比。
<br/>　　而另一边的炎澜天，全身黒焰暴涨数倍，在他周身百丈之内，形成了一片特殊的火焰领域，同样形成了完美的防御，防御力比之雷云皓只强不弱。
