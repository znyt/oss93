　　第565章
<br/>　　今天的山顶静悄悄。
<br/>　　程大雷蹑手蹑脚摸到山顶，大部分人都聚集在半山腰阻拦燕不归，山顶上本无多少人。而此时天色已暗下来，视线也不是很好，程大雷也不担心被人发现。
<br/>　　但，程大雷还是遇到了一个不算很大，却很难解决的麻烦。
<br/>　　他不知道刘发财在哪里。
<br/>　　万卷山是很大的，可以藏人的地方也是多的。而在程大雷脑海中，根本没意识到书院还有大牢这种地方存在。
<br/>　　如果自己一间房子一间房子找，似乎也不太现实。而随便抓个人问问，岂不是暴露了行踪。也不知道，在书院心里，是燕不归比较重要，还是杀掉自己重要。
<br/>　　程大雷在山上走着，偶然闯进一栋小亭子中，亭子里坐着一个人，一声不发，如果不是程大雷无意停下脚步，还真容易把对方忽略掉。
<br/>　　以程大雷的心思，当时就要杀掉对方，避免走漏消息。可看清楚对方模样时，程大雷却愣住了。
<br/>　　楚云生明显也愣了愣，随之他直起身，冲程大雷抱拳行礼。
<br/>　　“程先生，好久不见。”
<br/>　　程大雷摸摸鼻子：“你知道我？”
<br/>　　楚云生无言沉默。
<br/>　　看来已经是暴露了，只是不知这楚云生有没有背叛自己，程大雷也在犹豫要不要杀掉对方。
<br/>　　“那个……你可知道刘发财在哪里？”
<br/>　　楚云生点点头，为程大雷指明了方向。
<br/>　　程大雷松了口气，虽然他对读书人没什么好印象，但任何一人都不会喜欢背叛。当然，至于楚云生有没有背叛自己，还要确认刘发财的位置后才知道。万一他与自己指的地方是个陷阱呢？
<br/>　　“你与我一起去？”
<br/>　　“我还要在这里想一想。”
<br/>　　“随你去吧。”
<br/>　　程大雷摆了摆手，看楚云生的呆呆傻傻的样子，也不像会搞陷阱的人。他实在没时间在此地耽搁，谁知道燕不归能拖多久，自己还是速战速决。
<br/>　　他并不知道，楚云生说的是实话。从早上开始，他一直坐在这里纹丝不动，只做一件事，那就是想。
<br/>　　虽然没有走出亭子，但陆陆续续也有消息传到他耳里。知道书院准备引程大雷到白沙岛上，知道燕不归来了，现在也知道程大雷来了……但自己究竟能做什么，他还没有想到。
<br/>　　知道的事情再多，也无法掩盖他本身的势单力薄。在这场棋局中，自己并不是与其他人势均力敌的黑白棋子，这是一盘象棋，自己只不过是一枚微不足道的『卒』。哪怕自己有刘发财的本事，也能提着刀做些事。
<br/>　　可他只是一枚『卒』，太容易被人忽视。不过，真正的对弈高手并不会忽略『卒』的作用。只要在合适的时机，将它摆在合适的地点，它也可能逆转整个棋局。
<br/>　　但现在，楚云生还不知道合适的时机是何时，合适的地点是何地。
<br/>　　所以，他还在想。
<br/>　　风吹草动，夜静无声，他也不知还要继续想多久。
<br/>　　大牢内，刘发财是一副生无可恋的表情。
<br/>　　在短短一天内，他遭受了世间最残忍的酷刑。世人只当书院有夫子这样的圣人，怎么知道书院也有李亚子这样的杀贼。
<br/>　　他并不是不想屈服，也绝无义气要和程大雷同生共死，他甚至是迫不及待的想要冲书院投怀送抱，然而，他自己实在没什么可招的。
<br/>　　“我想你们一定搞错了什么。我真的不是程大雷的同伙，虽然我很想是。如果我是的话，我一定将他有几条内裤都告诉你们。”刘发财苦着一张脸道：“你们真的以为，凭我就能引程大雷上钩，嗨，别做梦了。”
<br/>　　李亚子歪坐在椅子上，审了一天一夜，他已经有些乏了。外面燕不归的事情，他当然也听说了，所以便将其他人派了出去，由自己守着刘发财。
<br/>　　对于刘发财的话，他没往心里去。通过接触发现，刘发财是个嘴很碎的人，偏偏李亚子生性不怎么爱说话。于是，在冲刘发财用刑的过程中，下手也就狠了些。
<br/>　　“大兄弟，我实话告诉你，我一点不恨你。我最恨的是程大雷那恶人，假若他此时在我面前，我一定撕烂了他。”刘发财道：“来，你把我放了，我和你们一起定个计划，一定要杀了此贼。”
<br/>　　“你觉得……”李亚子打了个哈欠：“当真会将你活着送到海岛上，做钓鱼的诱饵？”
<br/>　　“这……”刘发财一楞。
<br/>　　“明白告诉你，你活着或者死了，对我们并不太重要。把消息放出去最重要。”李亚子懒懒道：“我的任务就是在此地看着你，等时辰到了，亲手送你上路。”
<br/>　　刘发财一下呆住：“你没有开玩笑吧。”
<br/>　　“我这个人，不是很喜欢开玩笑。”
<br/>　　刘发财宛若被一道闷雷劈中，半晌一句话也说不出来。他在世上晃荡了大半辈子，也干过不少缺德坏事，平日也总说脑袋掉了不过碗大个疤，老子一辈子快活够了，死了也就死了吧。但事情真的发生在身上，方才意识到，死，当真不是一件好玩的事情。
<br/>　　本来还奇怪，书院怎么脑袋抽疯了，要将自己当诱饵。现在才明白，自己连当诱饵的资格都没有。
<br/>　　李亚子挑眉欣赏他的表情，这是他为数不多的乐趣之一。普通人遇到这种事，都是心如死灰，万念俱休。
<br/>　　“那个……实不相瞒，我和程大雷情同手足，我们早发过誓，不求同年同月同日生，但求同年同月同日死。我们两个的关系……可好啦。”
<br/>　　李亚子愣了愣，这种埋在棺材里还要伸手的人，当真是第一次遇见。
<br/>　　“呦，我还真没想到，你对我感情这么深。”
<br/>　　这时候，一道懒懒的声音响起。刘发财和李亚子同时一激灵，就见一条大汉晃悠着走进大牢。
<br/>　　刘发财先呆住了，此时此刻，他早已有必死的觉悟，完全没想到，程大雷会当真出现。
<br/>　　李亚子腾地一下站起，手扶住身下的太师椅，双眼直勾勾盯着面前的大汉。
<br/>　　“程大雷？”
<br/>　　“不错，是我。”程大雷摸了摸鼻子：“惊不惊喜，意不意外。”
<br/>(本章完)
