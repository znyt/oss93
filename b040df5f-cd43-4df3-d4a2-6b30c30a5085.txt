　　苏莫一袭灰白色长衫，立身于土丘上，目光炯炯有神，与所有人对视。
<br/>　　此处聚集的人，大多都是武帝境的武者，并且修为达到高阶武帝的人比比皆是。
<br/>　　只有极少部分人是武尊境强者，这些人多是带领各自势力天才的长辈。
<br/>　　在无尽的人海之中，苏莫发现有很多异族，比如眼眸血红的血瞳族之人，不过他并未发现巫族之人，还有强大的巨灵族、飞灵族、黑鳞族，想来都还未到来。
<br/>　　打量了少倾，苏莫干脆直接盘膝坐了下来，不理会无数注视的目光，打坐调息了起来。
<br/>　　众人打量了苏莫片刻，便不再关注了。
<br/>　　也没有人上前挑战苏莫，虽然现在是打败苏莫，证明自己的好机会，但现在真圣界即将开启，挑战已经没有什么意义。
<br/>　　在真圣界中获得好的战绩，的真圣之气、留名真圣碑，才是证明自己的最好机会。
<br/>　　以往，哪一个能在真圣碑留名的人，后来不是名震古灵星河，受人敬仰。
<br/>　　可以说，只要能在真圣碑留名，将来晋升武圣的几率将能大大的提高。
<br/>　　苏莫静气凝神，不理会外界嘈杂的声音，静心打坐。
<br/>　　炼制了大量的本命灵剑，耗费了他很多的精血，导致精神疲惫，虽说这几日恢复了很多，但还是没能恢复到巅峰。
<br/>　　不过，这也并不会影响他多少的战力！
<br/>　　谢彬、任云都等人，见苏莫打坐调息，便也没有打扰。
<br/>　　不同派系的人，各自聚集在一起，低声交谈了起来，议论此地的一些有名的天才。
<br/>　　其它势力之中，也有一些与吴晋和唐婉相熟的强者，前来和两人打招呼，互相谈笑。
<br/>　　时间流逝，很快变过去了一个时辰。
<br/>　　“快看，飞灵族的人来了！”
<br/>　　就在此时，一阵惊呼声突然响起，惊动了所有人。
<br/>　　苏莫立刻睁开了眼眸，顺着众人的目光望了过去，便见到天际的尽头飞来一支巨大的羽毛。
<br/>　　不错，是一支巨大的羽毛。
<br/>　　这羽毛呈纯白之色，长约数千丈，巨大无比，遮天蔽日，急速向此地疾驰而来。
<br/>　　而在羽毛之上，伫立着大量的飞灵族之人，粗略一扫不下上千人。
<br/>　　这些飞灵族之人有男有女，男子英俊、女子貌美，全部都是背生双翅，仿佛是精灵一般。
<br/>　　这些飞灵族之人的翅膀，也是各不相同，男子的翅膀多时黑色、紫色和灰色，而女子的翅膀多时白色、绿色或者淡粉色。
<br/>　　对于飞灵族，苏莫了解的不多，也没有具体的接触过。
<br/>　　他只知道飞灵族是古灵星河最强大的异族之一，与巫族、巨灵族、黑鳞族，号称是古灵星河四大异族。
<br/>　　这四大异族的实力，远超其它的异族，诸如血瞳族之类的异族，根本不能与这四大异族相提并论。
<br/>　　飞灵族之人天生异禀，速度在同阶武者之中，无人能出其右，双翅一扇，穿梭虚无，让一般的同阶武者根本锁定不住。
<br/>　　四大最强的异族之中，除了巫族之外，其它三大异族的单体实力，都是远超人族。
<br/>　　但这三大异族虽然单体实力强大，但族人却是最为稀少，没有任何一族的人数超过百万。
<br/>　　尤其是单体实力最为强大的巨灵族，其整个族群，也只有二、三十万族人。
<br/>　　呼吸之间，巨大的羽毛疾驰而至，而后一名名飞灵族之人飞了下来。
<br/>　　“好强！”苏莫暗暗心惊，这飞灵族的天才之中，居然全部都是高阶武帝，连一名中阶武帝都没有。
<br/>　　这飞灵族的人数虽少，但实力比他们神武学府，不知道强大了多少倍。
<br/>　　上千名飞灵族之人，落在一处空旷无人之地，领头之人只有一名飞灵族的老妪，身上没有任何气息，实力深不可测。
<br/>　　飞灵族之人到来以后，也没有与任何人打招呼，此族行事极为低调，在古灵星河很少与其它族群相处。
<br/>　　“苏兄，你看，那人便是飞长云，真圣榜第三！”
<br/>　　任云都来到了苏莫的身边，目光望向飞灵族的人群中，低声对苏莫说道。
<br/>　　苏莫顺着对方的目光望了过去，便看到了一名非常高的飞灵族青年，此人足有近两米，站在一众飞灵族人之中，如同鹤立鸡群一般。
<br/>　　但此人很瘦，穿着一袭青色的短衫，仿佛是一根青色的竹竿一般。
<br/>　　最吸引人的地方，是此人的一对翅膀，居然是银色，与其他飞灵族之人完全不相同。
<br/>　　“苏兄，此人虽然低调，但据说实力超强，是这次真圣碑留名的必然人选！”任云都道。
<br/>　　“或许吧！”苏莫点了点头，此人武帝境九重的修为，能排在真圣榜第三，实力和天赋便可见一斑。
<br/>　　“苏兄，你有没有把握战胜此人？”任云都问道，他倒是对苏莫颇有信心。
<br/>　　当初在环山矿脉，任云都可是亲眼所见，苏莫接住了血瞳族尊者的一击。
<br/>　　而那时候的苏莫，不过才是武帝境五重的修为，现在突破到了武帝境六重，在任云都看来，苏莫铁定有了武尊境一重的实力。
<br/>　　“战胜？”
<br/>　　苏莫闻言略作沉吟，道：“真圣碑留名，难道就必须战胜所有人吗？”
<br/>　　“这个倒不是！”任云都摇了摇头，道：“真圣界类似于一种考核，对于天赋和实力都有侧重，实力越强，自然成功率越高，难道苏兄不知道？”
<br/>　　任云都有些疑惑，真圣界考核的事情，一般人都知晓，苏莫居然好像不知晓！
<br/>　　“我了解的不多，任兄，这真圣界的考核情况，你能否详细的讲解一番？”苏莫沉声说道，能提前知晓真圣界内的情况，对他来说自然更好应对。
<br/>　　“当然可以！”
<br/>　　任云都沉吟了一番，讲解道：“真圣界内是一处独立的小世界，其内有一座巨大的真圣碑，只有触碰到真圣碑，才能得到真圣之气，以及在其上留名；而想要触碰到真圣碑，就要经历通圣三关！”
<br/>　　“通圣三关，顾名思义只有通过这三关，才能在真圣碑留名，这第一关……！”
<br/>　　任云都徐徐的讲解，苏莫仔细的聆听，准备将通圣三关的情况，详细的了解一番，就在此刻，异变陡生。
<br/>　　天穹之上，虚空突然崩裂，突兀的出现了一道长达十丈的裂缝。
<br/>　　一名身形高大的老者，背负着双手，从裂缝中踏步而出。
<br/>　　苏莫见到这名老者，顿时脸色大变，因为这名老者，正是天冥星巫族的大祭司——巫仑！
