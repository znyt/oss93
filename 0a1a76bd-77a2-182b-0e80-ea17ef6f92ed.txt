        “我早就到了，只不过候在门外没进来。所以，你们刚刚的谈话，我听得一清二楚。你直接告诉那个季大小姐，二少爷的下落。无非是想让那位季大小姐自己去解决他们三个之间的三角关系问题！阿沉，但是，你有没有想过，你把这个问题，其实是间接地丢给了童筱颖去做决定。童筱颖若是知道自己的老公婚内出轨，而且这小三还找上门来了。不知道，童筱颖到时候会如何抉择？到底是维护自己的婚姻若无其事地继续和二少爷在一起呢？还是就此成全了季大小姐与二少爷离婚？”连华生不禁摇头叹气地感慨起来。
<br/>
<br/>        “你来我这，就会为了跟我说这事？”傅夜沉冷冷地反问。
<br/>
<br/>        连华生看不透傅夜沉的心思，尴尬地咧嘴一笑，识趣地言归正传：“当然不是，就是关于临海大学马智杰的那个案子，还有上次在军校里发生的学生打靶走火的事件，有了新的发现。”
<br/>
<br/>        “这两个案子有关联？”傅夜沉诧异道。
<br/>
<br/>        连华生笑容渐敛，转瞬间，非常严肃地点了点头。
<br/>
<br/>        滨江城，花园别墅区。
<br/>
<br/>        从大哥傅夜沉那儿回到家后，傅昼景独自一人在房里收拾摊满整张床的女性衣服，每一件他都折得工工整整，才摆进他买给筱颖的那只粉色行李箱。
<br/>
<br/>        衣服放妥了，他又准备起日常用品，一件件清点完后再塞进几瓶胃肠药和感冒药。行车箱内的物品归纳得井然有序、一目了然。再三检查后，他才阖上行李箱。
<br/>
<br/>        待会，他会去临海城接筱颖，然后明天和筱颖一起出国去澳洲定居。
<br/>
<br/>        这段日子，其实他每天都过得浑浑噩噩。
<br/>
<br/>        他妒忌大哥应有尽有，记恨爷爷的偏心。
<br/>
<br/>        但最终，还是爷爷把他从这水深火热之中给拉了出来。
<br/>
<br/>        爷爷帮他和筱颖在筱颖所就读的大学附近买了一套房子，还送了他一辆车，而且还帮他在一家上市的设计公司找了一份安稳的建筑设计师的工作。
<br/>
<br/>        就连筱颖在澳洲念大学的四年学费，爷爷也全包了。
<br/>
<br/>        傅昼景想好了，他不再计较过去，从此以后，和筱颖一起，在澳洲好好过日子。
<br/>
<br/>        钱可以不用赚太多，够花就行了。
<br/>
<br/>        等筱颖大学毕业后，他就和筱颖生两个孩子，然后就此在澳洲过一辈子。
<br/>
<br/>        傅昼景把护照放入行李箱内，完成最后一道程序，终于松口气，阖上行李盖。
<br/>
<br/>        而从此以后，他在滨江城里所有的东西，都被爷爷傅炎廷悉数没收。
<br/>
<br/>        童筱颖被傅老太爷傅炎廷先接去了傅家大庄园，傅昼景随后才赶到。
<br/>
<br/>        就连傅夜沉，也被傅炎廷给临时叫了回来，只为了一家人在家一起吃一顿最后的晚餐。
<br/>
<br/>        暮色降临。
<br/>
<br/>        傅家大庄园大门口的红灯笼亮了起来。
<br/>
<br/>        童筱颖却独自一个人，站在台阶上徘徊。
<br/>
<br/>        直到傅昼景提着行李箱，从出租车上下来，童筱颖看到他后，欣喜若狂地跑下台阶，扑入了傅昼景的怀中。
<br/>
<br/>        朦脓的夜色下，怀里的童筱颖，仰着娇俏的脸蛋，微染了霓虹灯的光芒，显得格外楚楚动人。
<br/>
<br/>        她深情款款地凝视着他，一双被泪花浸湿的明眸，美得窒息，小巧的鼻梁下，她那犹如樱花般淡粉的嫩唇在慢慢地引诱着他。
<br/>
<br/>        童筱颖因为见到他而激动得热泪盈眶，她差点就以为自己真的变成了苏漫雪，从此以后就再也见不到她的小卫了。
<br/>
<br/>        还好，这只是她的一场噩梦，梦醒了，回到现实，她还是她自己，她仍旧可以和小卫在一起。
<br/>
<br/>        傅昼景的喉咙紧了紧，慢慢地低头，朝她凑近：“筱颍……”
<br/>
<br/>        “小卫……”
<br/>
<br/>        童筱颖凝视着傅昼景那张帅气的脸，在她自己的瞳孔里越放越大。
<br/>
<br/>        他离她越来越近、越来越近……
<br/>
<br/>        童筱颖下意识地闭上了眼睛。
<br/>
<br/>        忽然，两道强烈的车灯灯光朝他们照了过来。
<br/>
<br/>        两人被照得刺眼，不约而同地往前望去，只是刚看到那两道强烈的远光灯时，那两道远光灯随即就灭了。
<br/>
<br/>        一个司机率先从车内下来，然后小跑到后座旁，拉开了车门。
<br/>
<br/>        道路旁的霓虹灯下，一个颀长的身影从车内走出来，带着一股霸者之气，犹如君临天下。
<br/>
<br/>        大哥？！
<br/>
<br/>        童筱颖身心一怔。
<br/>
<br/>        傅昼景也没料到，爷爷傅炎廷会把大哥傅夜沉也叫回来。
<br/>
<br/>        傅夜沉从车前走过，他身着一袭黑色的衬衫，被这朦脓的夜色浸染，飘逸的碎发下，是一张英俊的脸。
<br/>
<br/>        他们兄弟两，都有一半的血液是来自彼此唯一的父亲冷卫峰。
