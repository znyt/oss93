　　苏莫知道妖族的寿命，要远远高于人族，但听闻秦不死的话，他还是难免震惊。
<br/>　　毕竟他才活了二十多年，居然就能遇到活了几千年的存在！
<br/>　　几千年，这么长的岁月，对于人族和一般的妖族来说，都是非常恐怖的存在！
<br/>　　“前辈，你经历了三次大劫，应该有非常详细的了解，我们有机会出去吗？”苏莫眸光灼灼，对方既然经历过三次大劫而不死，那想必非常的了解。
<br/>　　“机会的确有，但需要有详细的计划！”秦不死点头。
<br/>　　苏莫面色郑重，等待对方诉说。
<br/>　　“巫族试炼者降临，会打通外界和苍穹世界的通道，待试炼者全部来到苍穹世界，通道就会关闭，等到两个月后，通道会再次打开，接引试炼者离开！”
<br/>　　秦不死悠悠开口，言罢，他沉默了少顷，继续道：“只要进入通道中，就有机会离开苍穹世界！”
<br/>　　听闻秦不死的话语，苏莫微微皱眉，这么容易？只要进入通道，就能离开？
<br/>　　就在苏莫疑惑之时，秦不死，面色凝重了下来，继续说道：“但是，在通道之外，有巫族的尊者坐镇，而且不止一位，一旦有苍穹世界的武者出去，他们会挥手镇杀。”
<br/>　　“尊者？”苏莫闻言，顿时心中一震，面露骇然之色，尊者应该就是远古传说中的武帝之上的存在，武尊！
<br/>　　苏莫心沉到了谷底，有武尊境这种神话般的强者坐镇的话，那谁能逃的出去？
<br/>　　别说他才武皇境八重，就算是突破到了武帝之境，也难挡武尊强者一击吧！
<br/>　　他本以为，以他如今的实力，就算不能彻底打败试炼者，也没有人能杀他了。
<br/>　　逃出去的希望很大，但现在听了秦不死之言，他的心中充满了阴霾。
<br/>　　可是，若是无法离开，他就会被困死在苍穹世界，无法离开，他就永远无法救活芊寻月。
<br/>　　还有就是宏青璇，苏莫曾经猜测，巫森既然是巫族的后人，很可能会趁机离开苍穹世界。
<br/>　　一旦对方离开的话，宏青璇也将会被带走。
<br/>　　而他若是无法离开，就再也无法拯救宏青璇了！
<br/>　　苏莫面色难看了下来，心念电转，少顷，他望着秦不死道：“那我们还有机会吗？”
<br/>　　“有！”秦不死点了点头，沉声道：“唯有让那些武尊强者不敢出手，我们才能活命！”
<br/>　　“不敢出手？”苏莫闻言疑惑，如何才能让武尊强者不敢出手？
<br/>　　“不错！我们要在试炼者中，找到身份不凡之人，生擒活捉，待出去之时，要挟巫族的尊者！”秦不死眸中精光闪烁。
<br/>　　“这真的可行？”苏莫心中有些惊疑，毕竟，武尊强者是何等人物，岂是轻易就能要挟的！
<br/>　　“别无它法了！只能一试！”
<br/>　　秦不死叹了口气，眸中露出回忆之色，道：“老夫经历第一次大劫之时，冲出了通道，被巫族尊者瞬间镇杀；第二次大劫之时，隐藏在一件空间宝器之中，故意让一名试炼者获得，最终依旧被巫族尊者发现；第三次大劫时，老夫夺舍了一名试炼者，但还是无法逃脱，被瞬间识破！”
<br/>　　秦不死脸庞上充满了落寞，以他的血脉，他的天赋，就算是在外界，也是不可多得的存在，本该遨游太虚，威凌环宇。
<br/>　　但是，却被困在苍穹世界数千年，这真是莫大的悲哀！
<br/>　　苏莫闻言久久无语，他也感受到了对方的落寞，三次努力均以失败告终，的确够悲惨。
<br/>　　不过，三次被镇杀，对方是如何活下来的呢？
<br/>　　秦不死虽然看出了苏莫眸中的疑惑之色，却并没有过多解释。
<br/>　　“小友，你的实力也不差，你我联手，计划得当，并不是没有成功的可能！”秦不死目光凝望着苏莫。
<br/>　　他知道，苏莫不会拒绝，虽然他刚刚苏醒，对于苏莫完全不了解。
<br/>　　但他相信，任何一个苍穹世界的巅峰强者，都渴望出去，渴望离开这个天地牢笼。
<br/>　　苏莫闻言轻轻的点了点头，能有人合作，他自然不会拒绝。
<br/>　　虽然他不了解此人，但他们的目的相同，有共同的目的，就能精诚合作。
<br/>　　随后，苏莫和秦不死聊了很久，详细商谈接下来的计划。
<br/>　　每一个步骤都有商谈，详细的讨论。
<br/>　　不过，具体的情况，还要待大劫到来之时，见机行事。
<br/>　　毕竟，到时候试炼者降临，局势瞬息万变，什么情况都有可能发生。
<br/>　　再者，还不清楚这次试炼者的实力，其中会不会有武帝强者，还尚未可知。
<br/>　　而且，苏莫从秦不死的口中得知，试炼者的实力极为强大，每一个都是巫族中的后辈天才。
<br/>　　每一个试炼者，都有强大的越级战斗能力，苍穹世界的武者，面对同阶的试炼者，基本上只有被秒杀的份。
<br/>　　两个时辰之后，苏莫才离开了蛮荒圣山，返回苍穹神宫。
<br/>　　飞驰在天穹之上，苏莫的手掌一翻，手中出现了一根羽毛，这是一根灰白色的羽毛，其上每一根毛发，都有暗光流转，显得极为不凡。
<br/>　　“不死鸟？”苏莫眸中精光一闪，低声自语。
<br/>　　这根羽毛是他在离开宫殿之时，从宫殿的角落里随手抓取。
<br/>　　据他仔细观察，此羽毛非常像是传说中的远古异兽不死鸟的羽毛。
<br/>　　这几年，苏莫时间充足，经常抽空去观看苍穹神宫中留下的一些远古典籍。
<br/>　　其中，就有一些远古异兽的介绍。
<br/>　　按照古籍的记载，在远古时代以前，苍穹世界乃至外界，将一些强大无比的异兽，称之为神兽。
<br/>　　更是列出了太古神兽榜，罗列了一万种强大无比的异兽，而不死鸟便是其中之一。
<br/>　　当然，远古战象和八翼飞龙蛇，同样榜上有名。
<br/>　　不过，不死鸟的排名，排在前五百名，比远古战象和八翼飞龙蛇的排名，高出太多了。
<br/>　　“应该不会错！”苏莫心中暗道，他凭借古籍的记载于描述，基本上能断定，这羽毛就是属于不死鸟。
<br/>　　而秦不死的本体，就是不死鸟，只不过，对方肯定不是纯正的不死鸟，其血脉有多精纯，那就不得而知了！
